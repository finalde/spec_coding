# 第476章 元婴今卖首

第476章 元婴今卖首

「嗯？」

看着面前突然出现的四位修士，李凡微微有些诧异。

自从此世打着【上古天机宗修士】的名号搅动风雨以来，李凡就变得格外小心谨慎。

只要外出，就一定会用【只手遮天】秘术隐藏自己行踪。

上一世能够屏蔽玄网感知的秘术当真不凡，偌大的万仙盟，想要见李凡一面的修士不知凡几。

但却始终没人能够如愿以偿。

不想今天，终于迎来了几位不速之客。

「你们是谁？」李凡开口问道，不指望对方会如实交代，只是为了单纯的拖延时间。

感应着四位黑衣修士流露出的气息，李凡心下稍安。

比天阳弱了不只一筹，应该是元婴境界。

「找我何事？」李凡明知顾问道。

「等会你就知道了！」其中一位冷声说道，「布阵！」

分立四角，各自有一道金光自黑衣修士手中的法器射出。

金线在空中辗转反复，霎时间在李凡周边建造成一个金光奕奕的法阵。

宛若一鼎巨钟，将李凡困在其中。

李凡双目微动，想要躲避，却感觉周围的空间都仿佛凝固似的，挣脱不得。

「收！」四人齐声喝道。

金钟快速收缩，越来越小。

金色的轮廓，眼看就要把李凡彻底束缚。

「分光困龙阵……」

被困阵中，李凡却是丝毫不慌。

苦修阵法之道数世，他自然是一眼就认出了这四位黑衣修士联手施展的是什幺法阵。

乃是当今流传在世一等一的困阵，可由至少三位以上修士联手施展。

布阵的人越多，阵法的威力也就越大。越阶擒拿，也是易事。

四位元婴合力，按照正常情况下，即便是化神修士，也能轻易捕捉了。

但……

李凡心中冷笑，虽然动弹不得，但是从身体各处，激射出数十道【大五行寂灭剑】。

毁灭性的黑光四散而出。

感受着黑光中弥漫的可怖气息，四名黑衣修士俱是微微色变。

下意识的就要躲闪。

可马上，这些黑衣修士就骇然的发现，黑色光剑的目标不是他们。

而是分光困龙阵的各个角落中！

「不好！」

其中一位修士失声惊呼，因为他终于察觉，从李凡身上发出的黑光，精准无比的击打在了分光困龙阵的各个关键节点上。

「轰！」

金黑色光芒剧烈碰撞，原本宛如实质的金钟，刹那间变得黯淡了少许。

牢不可破的困阵，也由之松动。

「雕虫小技，也敢班门弄斧！」

光影变幻中，困龙阵中李凡的身影陡然消失不见。

「除了推衍之术外，此人居然还精通阵法？」

「这怎幺可能？」

刹那间，四位黑衣修士脑海中都闪过这个念头。

「情报有误，先撤！」

为首的修士低喝道。

然而，正欲撤离的他们，却莫名发现，天空陡然暗淡了下来。

仿佛有一片乌云，遮蔽了天空。

随后，数十道金光从乌云中飞出，以迅雷不及掩耳之势，在空中画出了更大的金钵，将四人笼盖在内。

「分光困龙阵？！」

黑衣修士们身躯剧震，双眼满是难以置信之色。

不仅顷刻间以其人之道还治其人之身，而且这阵法看上去，竟然比起他们四位元婴修士联手布置的，还要精妙！

虽然阵法威力稍逊，但相较于修仙界广为流传的那个版本，却是多出了一丝未知的改进。

正是这点变化，让他们想要破阵，也要花上一番功夫。

到了这个时候，四位黑衣修士也不得不全力以赴了。

巨大的金钵中，独立雪峰、迷雾丛林、如剑直山、广阔平原，四种独特的洞天景象相继出现。

黑云之下、金钵之上，李凡驻足而立。

俯视着下方四位元婴修士，心中杀意弥漫。

这几人并非都精于阵法一道，布阵还需法器相助。所以仓促间布下的法阵自然有些许瑕疵。

若是对付寻常修士，这些瑕疵自然无所谓。

可李凡在眼中，他们布下的【分光困龙阵】，可谓是破绽百出。

破阵而出后，他立马反制。

先是发动只手遮天秘术，遮蔽天空，隔绝外界对这里的可能窥探。

用天玄镜中购买的【妄言】法器，干扰灵符通讯。

随后布下【爆法惊神阵】，暂时使自己的道法招式上升到元婴境界。

再以分光阵，暂时将几人困住。

「我筑基时，就可力斩元婴。」

「如今修为已经晋升至金丹后期，观你们四人，如插标卖首耳！」

「翻天！」

巨大的手掌从天而降，金钵中的四位修士，如瓮中之鳖，无处可逃。

雪峰寒意飘然、雾林中发出鬼魅的尖叫、笔直的山峰迎头撞上、平原中升起袅袅炊烟……

洞天景象演变，四人手段尽出。

四色光芒纠缠、聚合在一起，和翻天掌碰撞在一处。

但，没用！

如纸糊的般，一触即溃。

巨掌当头压下，在惨叫声中，四人身躯寸寸碎裂。

洞天景象再现，血肉在蠕动中，重新变化成完整身体。

翻天掌余势不绝，不再管对方的肉身，转而朝着四大洞天轰去。

「砰！」

洞天震颤，山峰、雾林、平原中，仿佛迎来末世。

巨大的轰鸣声中，道道巨大的裂纹在洞天法域中浮现。

山峰倾塌、雾林树倒、平原地裂……

仿佛下一刻，四大洞天就要彻底被毁坏。

但是这几位黑衣修士配合极为默契，在这生死关头，竟然没有人退缩。

全身法力运转到极致，四人合力，竟然硬生生将翻天掌抗下。

巨掌虚影消弭无踪，不断震动的洞天这才慢慢恢复平静。

四人一脸惊魂未定的样子，连忙抽取着周围的灵气，弥补着自身洞天伤势。

同时取出丹药服下，寻找着李凡的身影。

「这式神通威力巨大，那李凡定然也是消耗颇多。我们抓住这个机会，破开困龙阵！」

「此人绝不是表面看上去的金丹修为，说不定他早已经恢复了伤势。」

「不错，一定要把这个消息传递回去。此人很有可能，是万仙盟抛出的诱饵！」

四人一边抓紧时机疗伤，一边用神识沟通道。
