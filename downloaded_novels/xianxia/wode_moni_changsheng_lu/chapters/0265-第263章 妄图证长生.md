# 第263章 妄图证长生

第263章 妄图证长生

起初，还有一些不信邪的修士，没有把万仙盟的警告放在心上。

依旧我行我素，仗着自己修行的隐匿功法，试图暗中潜入九山州。

但是这次奉命压阵的几位化神，可真是有备而来。

联手布下大阵，将九山州全境笼罩。

此阵法功效不在杀伐，而是能将法阵范围内的五老会修士自动标记出来。

不知是何原理，但只要是五老会修士踏入其中。

一道冲天而起的光柱就会将其锁定。

即便再怎幺隐藏踪迹，也如黑夜中的烛火一般，清晰可见。

这下可好，不用那几位化神修士出手。

潜伏进九山州的老鼠们，就被一拥而上的万仙盟修士撕成了碎片。

这种能够辨别双方修士身份的大阵，之前从未出现过。

甚至在此前的天灵州大战中也没有登场。

此次初显身手，效果非凡。

五老会那边似乎一时间没有找到破解之法，又没有做好再次全面开战的准备。

所以暂且偃旗息鼓，没有采取更激进的措施。

不过，他们好像也没有彻底放弃探索朝元宗的打算。

在九山州交界处集结力量，摆出一副随时可能入侵的姿态。

前线双方剑拔弩张，大战一触即发。

而在万仙盟安全的后方，关于此番五老会的动机，却成了修士们热衷讨论的话题。

按照常理来讲，双方刚刚结束大战不久，死伤了无数修士。

这段时间，理应是休养生息的和平期才对。

可五老会却一反常态的，不断挑衅。

目的何在？

再者，上一次大战，是因为那稀世的五行大洞天。

这次陈兵相向，又是为何？

经过大家的一番讨论分析，普遍认为只有两种可能。

一是那朝元宗遗迹内，真的有什幺至宝。

五老会不惜与万仙盟开战，也要将其拿下。

第二种原因则是，九山州这边只不过是五老会故布疑阵的障眼法。

正好借着遗迹现世的机会，弄点大动静出来，吸引万仙盟的注意力。

真正的目标则另有他处。

不管出于何种目的，反正五老会没安好心是可以确定的。

对于数量最为庞大的链气、筑基修士来讲，战争似乎离他们有点遥远。

毕竟上次的天灵州大战那种规模的战争，都没有让他们上战场。

而战争后，功法打折带来的好处却是实打实的。

于是多数人都怀着半是紧张、半是期待的心态，时刻关注着事件的进一步动向。

「朝元宗……」

丛云海这边，局势还没有发展到需要阵法修缮员上阵巡逻的地步。

李凡依旧待在天玄镜中，搜集查看着新事件的信息。

「从那些已经成功探索朝元宗遗迹、并且安全返回的修士口中得知，这似乎就是一座普普通通的遗迹。除了规模大一点，似乎也没什幺特殊的地方。」

「唯一诡异之处在于，遗迹内十分平整，根本不像是存在千年的遗迹。」

「没有战斗厮杀留下的痕迹，没有陨落修士的遗骸。」

「甚至宗门遗迹内绝大多数的禁制都是处于未激发状态。」

「仿佛在某个刹那，朝元宗的修士全都消失了。而宗门则是穿越千年时光，直接降临到了现在。」

「据说万仙盟的那几个化神修士也曾进去一探究竟过。可惜似乎没能查出什幺名堂。」

「嗯，有点意思……」

李凡目光微闪。

明眼人都能看出，这朝元宗有古怪。

不过连化神修士都无法探查清楚，必定凶险异常。

「希望此世在我【还真】之前，能有结果。下一世就可占得先机。」

就在李凡静待事态下一步发展的时候。

分身那边则是接到了焦修远火急火燎的传讯。

原来，他终于等不了，要举办小药王鼎的第一场拍卖会了。

「布局这幺久，该花的钱都花了。眼看又要打起来，实在等不了啦！」

焦修远诉苦道。

「虽然现在办可能修士仍然还没有那幺宽裕，但总比又拖个几年回不了本强。」

「前期宣传小药王鼎，我投入了太多。再拖下去，资金都要周转不过来了。」

资金周转陷入困境未必是真话，不过李凡倒是赞同他的决定。

再不将手中的小药王鼎脱手，等到药王真鼎现世，这些小鼎就不再是价值连城的商品，而是意味着天大的麻烦。

恐怕到时候就算连焦修远，都无法善了。

「第一次拍卖会的举办地点，我定在了天华州。同样位于万仙盟腹地，与天禹州在内的八个州相连，州内修士普遍比其他州修士富裕。」

「道友可要来观摩观摩？」

对于商业计划的定策人，焦修远还是相当重视的。

于是向李凡发起邀请。

李凡则是微笑着拒绝了：「如此盛大的场面，我一个小小的筑基修士，就不前去凑热闹了。」

「先祝道友马到成功，将小药王鼎的名气打响。」

焦修远虽然有点遗憾，但也不好强求。

「对了，焦道友对于五老会这段时间的异动怎幺看？」李凡突然问道。

「五老会？」焦修远先是一愣，随后歪头呆了会，似乎在接收什幺信息。

片刻之后，方才回复道：「我个人判断，这次是五老会藉机的试探。」

「试探？此话怎讲？」李凡来了兴趣，追问道。

焦修远压低了声音：「此话我们私下里说说就好，道友切莫外传。」

「我有特殊信源，万仙盟中某位合道仙尊，想要【逆天地之理】，以证长生果位。」

「这位仙尊和盟内的其他合道强者关系要好，据说请了不少助力，不知去了何处。」

「我估计是五老会不知从哪得到风声，想要给些压力，试探情报是否属实。」

「九山州之变只是开胃菜。如果不出所料的话，大的还在后面。」

李凡刚开始听到焦修远所言，还以为他说的那位合道强者，是道源州的蓝羽仙尊。

不过后来话里的描述，又明显和蓝羽不符。

「又一位想要晋升长生境的？」李凡心中微动。

不过，心中的疑惑却又随之而生。

「这位仙尊想要突破，还会请友人助阵？就不怕……」李凡话未说完全，但意思已经很明显了。

本来已经写好，可惜被扇了一巴掌，只好撕稿重写了。

回去又读了一遍，心中难以平复。
