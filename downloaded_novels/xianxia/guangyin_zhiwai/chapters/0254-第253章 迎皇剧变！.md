# 第253章 迎皇剧变！

第253章 迎皇剧变！展现在许青面前的，赫然是一片偌大的蓝色湖泊，水质看起来很是清澈，只不过仔细观察可以看出，这里的水都是冻状。

四周还长着一些五颜六色的植被，很是杂乱，就连这湖泊所在的洞窟之顶，也满是苔藓。

一股潮湿之意，在此地很是浓郁。

许青目光扫过一圈，无论是之前影眼的查看还是此刻他的感知，都没有在这里探到危险，但许青依旧戒备，小心的向着湖泊走去，靠近后看向湖底。

清澈的冻状湖水下，湖底清晰可见，一眼看后，许青沉默。

湖底铺满了无数的骸骨与碎石。

从骨骼去看有男有女，密密麻麻覆盖目光所及之处，仿佛在若干年前，此地经历了人间惨剧。

“这里是我曾经无意中找到，来过多次，没有遇到危险，湖底的骸骨也没有任何诡异变化。”吴剑巫在一旁，探身扫了眼，他害怕许青误解自己的善意，连忙解释。

许青点头，仔细观察后他发现这里的冻状水质，与柏大师药典内所记录的仙冻，还是存在了一些区别。

多了香气。

尤其是在这湖泊旁，来自湖水的香气笼罩全部范围，闻入口鼻之间，满口都是甜腻，初闻尚可，但闻的久了就会有一种恶心到了极致之感。

这与仙冻的描述不符。

但许青猜测在没有经历剧变前，若干年前的这里大概率是一处仙气缭绕的水池。

这里的湖水，很可能就是药典内所说的仙气所化，只不过后来经历了无法琢磨的变化，所以形成之物看似仙冻，也有不同。

算是一种变异的仙冻。

具体作用许青还不知晓，但他觉得此物应有不小的研究价值。

于是收回目光，正要再取一些。

可就在这时，许青忽然轻咦一声，仔细看向湖底。

体内命火更是在这一刻燃烧，修为全部爆发，形成了火海风暴。

一旁的吴剑巫眼睛睁大，被许青气势卷着倒退开来，但却不知发生了何事。

“啥情况！”

吴剑巫刚一开口，许青已将命火之力汇聚双眼，使得自身眼睛在这一刻好似要超越星辰，散出璀璨之芒，凝望湖底一处位置。

在那里，于骸骨与大量碎石中，有一个大石头，给了许青一种熟悉之感。

“你在看什么？这里面有什么好东西吗，可惜这湖水诡异我不敢下去，也曾想办法要去捞出里面的骸骨，但做不到……”吴剑巫注意到许青的目光，赶紧开口。

可他话语还没等说完，许青所凝望之处，那个让他感觉有些熟悉的石头，如被一只无形大手抓住，忽然挪动，向外一拽，慢慢穿梭湖水，直至被拖了出来，漂浮在许青面前。

这一幕，看的吴剑巫倒吸口气。

许青抬头仔细观察这大石块，半晌后他找到了熟悉的缘由。

那是太苍一刀的气息。

这石块上赫然蕴含了太苍一刀的神韵。

这让许青很奇异，将其收起。

影子也无声无息的归来，方才就是它过去将这大石块拿出。

与此同时，随着大石块被取出，它之前所在的位置，露出了原本藏在下面的残破石碑。

四周湖水缓缓波动间，石碑上的淤泥向着边缘散去，露出了上面一些字迹以及图案。

许青立刻看去。

“紫青上国洗仙池？”一旁的吴剑巫也有所察觉，看清了字迹，惊呼一声。

许青也很惊讶，仔细查看后，注意到石碑上的图案似是一张南凰洲的地图。

上面标注了紫青上国皇都所在，正是如今的紫土，同时也有这洗仙池的标记，在旁边还有一处标记，写着太子府。

看其位置，正是许青之前想要去的太苍道庙废墟之地。

许青凝望湖水，此地的来历已经清晰了，同时紫青上国这个名字，也再次浮现许青的记忆中。

他第一次听说紫青上国，是在紫土那座曾经的紫青国都内，陈飞源告知关于八大家族之事。

那个时候他才知道，原来在很久很久之前，南凰洲内还存在了这么一个血脉奇异的国度。

“可以与法宝共生……”

许青想到了陈飞源当时给他的感觉，明明很弱，但又很强。

许久，许青收回目光，将这里的变异仙冻又取出一些，没有滞留，转身离开了这湖泊所在之地。

回到了外界。

在山体裂缝入口外，吴剑巫一路送到这里，此刻他挫着双手，眼巴巴的看向许青，欲言又止。

“我不会告诉其他人，你放心就是。”许青大有深意的看了吴剑巫一眼，身体一晃，化作长虹远去。

只留下吴剑巫一个人，站在那里，一脸的惆怅，患得患失。

直至半晌后，吴剑巫狠狠咬牙。

“管不了那么多了，这许青虽做事情霸道，但应该也是个信守承诺之人，大不了……我短时间不回宗门了，不把宝贝儿子培育出来，决不罢休！”

吴剑巫明显被圣昀子镇压一峰之事刺激，此刻眼睛赤红，猛地转身回了自己的石窟，继续照顾那些大着肚子的凶兽。

如今天色已晚，皓月当空，凰禁在月光下，树木好似化作厉鬼，看起来无比狰狞的同时，四野有阵阵怪叫与兽吼，偶尔回荡。

许青身影穿梭在这漆黑的丛林中，身体起伏跳跃，速度很快。

吴剑巫的事，他没有对外乱说的想法。

这个乱世每个人都有自己的活法，对方虽有些怪癖，但也没有针对自己，更是带他找到了仙冻，所以许青也就懒得去理会。

他只是觉得，天下之大，无奇不有。

“但也有可能这件事里，隐藏了吴剑巫另外的目的。”许青摇头，收回思绪，在这夜色中找到了一处树洞，附近检查一番，又布置了防护之阵，这才钻了进去。

准备过夜之余，许青也将获得的仙冻取出。

此刻时间过去了一些，所以这仙冻明显少了小半，这让许青想到吴剑巫所说，此物超过一个时辰就会消失。

于是他思索一番取出一个小瓶，打开后一挥手，顿时里面的小黑虫飞出落在了仙冻上，开始吞噬。

许青仔细观察，但等了片刻直至小黑虫将这些仙冻吃完，也没有什么反应与变化，于是许青想了想，索性让这一批小黑虫去将他取来的所有仙冻都吞噬。

很快仙冻消失，许青观察许久，最终将那些小黑虫重新收入瓶子内，将这瓶子标记一下单独存放。

他准备等一段时间，让它们消化一下仙冻，再去查看是否有所不同。

做完这些，许青取出在洗仙池内获得的具备太苍一刀神韵的石块，拿在手里研究，感悟其内神韵，渐渐他眼前好像有刀影划过。

这让许青心神一震，闭目体会。

就这样，一夜过去。

第二天清晨，许青睁开眼，第一件事就是低头看向手里的石块。

“此物可让我对太苍一刀，明悟更深。”许青觉得收获极大，随后查看那些吞噬了仙冻的小黑虫。

这些昨夜吞噬了仙冻的小黑虫，在许青凭着血液牵引的仔细观察下，发现它们一个个好似醉了一样，一动不动，不是很活跃。

但气息似乎有了一些细微的改变。

这让许青有了兴趣，可如今它们不活跃，许青也看不出具体，所以他没去打扰，将它们重新放好后，又将湖泊之地所在的位置牢记在心，这才离开树洞，向着远处疾驰。

他准备去太苍道庙所在的废墟，去看看能否有机缘感悟太苍第二刀，若无法，他准备常年研究那个石块，去借此感悟。

“这石块边缘锋利，如刀锋一样，定与太苍道庙有关，极大可能是某个神像的一部分！”许青心底分析。

时间流逝，十天过去。

这十天许青始终在赶路，偶尔也收获一些毒草与毒物，同时对喝下仙冻的那批小黑虫，多次关注，但它们依旧还在沉睡。

“这么久？”许青有些诧异，若非他能感知对方还活着，都以为是不是死掉了。

“再等等看。”许青沉吟，抬头遥望前方。

他的前方，遥遥可见下坡之地，有一座古老的城池所化废墟，仿佛在岁月里永恒沉默。

此刻是这一天的晌午，阳光在苍穹浓郁，可洒落丛林穿梭一片片树叶落在许青的四周后，却化作了斑驳。

透过这斑驳的光，去看那座古老的城，沧桑岁月之意，油然而起。

同时在这个位置，还可以看到在这废墟城池的中心，存在了一座高大恢弘的神庙。

远看只能隐约见到轮廓，不太清晰，但那种古老与神秘，还是夹杂在了此城所见证的时光之中。

“到了。”许青身体跃去，走近废墟。

而就在他的身影踏入废墟半柱香的时间后。

距离这里很是遥远，隔着禁海的望古大陆上，发生了一件惊天动地，使迎皇州所有势力都内心波澜万丈的大事。

七宗联盟内的凌云剑宗，其宗内禁忌法宝，在间隔了近二百年后的今天，再次动用！

似要将七血瞳，全宗覆灭！

起因是七天前，七宗联盟对七血瞳再发调令与旨意，但血炼子依旧在拖。

直至一天前，七宗联盟最高会议的元老院，统一了决策，派去特使告知七血瞳。

决策内容，让血炼子与七个峰主，一天之内自缚回联盟赔罪，如若不来，将覆灭七血瞳，一杆弟子，不死不休！

同时告知七血瞳内所有外族外宗三天内离开，若不离开，三天后如七血瞳继续一意孤行，则天罚降临，各族余留者后果自负。

此事轰动八方，大量来访七血瞳的外族纷纷撤离，七宗联盟之人也都大量离去，整个七血瞳风雨飘摇，人心惶惶。

直至一天后，血炼子依旧没有给出答复，于是……七宗联盟震怒，凌云剑宗的禁忌法宝，瞬间爆发。

化作一道滔天血光，从七宗联盟内冲天而起，于半空形成一枚血色的种子，直奔……北方！

不是处于南面的七血瞳，而是北方！

迎皇州内，有一山一河，交叉而过，连接迎皇州东西南北，山是太司度厄山，河是蕴仙万古河。

太司度厄山连接南北，蕴仙万古河沟通西东，而它们交错之地属于迎皇州的中心部位，在那里有一个宗门。

此宗名为少司宗，与太司仙门没有直接关联，但任何大势力都知道，这少司宗就是太司仙门为遏制七宗联盟，而扶持出的爪牙。

因此宗所在之处，修建了一座大坝，阻断了蕴仙万古河中本应顺山道流淌蔓延至七宗联盟出海口的一条支流。

支流被断，七宗联盟整体灵气与异质洗刷程度，都受到极大影响。

对七宗联盟而言，那少司宗如鲠在喉，他们曾多次要求撤去大坝，但都被太司仙门干预，提出苛刻条件。

而七宗联盟若要出手针对少司宗，除非可以瞬间结束战斗，否则的话必被阻止，同时各方势力制衡下，无故开启禁忌，必被各方怀疑。

同时其他几方势力也乐于看见人族内斗，推波助澜，使得七宗联盟光景越发不堪。

而此刻，从凌云剑宗爆发出的禁忌法宝所化血色种子，其方向……正是这少司宗！

速度之快，刹那间这血色的种子就穿梭无尽范围，落入少司宗内。

七宗联盟，这是明修栈道暗度陈仓。

他们看似与七血瞳之间矛盾声势浩大，气势汹汹，可实际上这一切都是烟雾，为的就是合理开启禁忌，为的就是找一个机会，拔去喉咙的鱼骨！

如今，随着血色种子的落下，一棵诡异血树，从这少司宗内拔地而起，参天而立。

此树一晃之下，少司宗内大半修士身躯崩溃，就地死亡，鲜血被卷入树中，余者也都被重创，骇然喷出鲜血。

再晃一下，宗门轰鸣，天崩地裂，大地碎开，一处处建筑瞬间坍塌，那处大坝更是爆开四分五裂。

没了大坝的阻拦，汹涌的蕴仙河水直接覆盖而过，顺着太司度厄山道，翻腾流淌。

三晃之时，血树开，有沧桑之声，在内传出。

“少司宗杀我联盟弟子，证据确凿，灭少司全宗！”

苍穹血意滔天，形成阵法，其内出现上百身影，每一位都是气息惊人，杀意狂暴，纷纷落下！

这一切太快。

七宗联盟之前为了威慑七血瞳，一共有三个宗门开启了禁忌法宝，此刻动用的虽只是凌云剑宗，可下一瞬其他六个宗，同时都将禁忌法宝开启，一时之间整个七宗联盟的半空，风云色变。

天下震动！
