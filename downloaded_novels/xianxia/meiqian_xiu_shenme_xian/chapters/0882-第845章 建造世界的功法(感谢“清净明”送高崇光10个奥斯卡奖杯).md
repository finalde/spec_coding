# 第845章 建造世界的功法(感谢“清净明”送高崇光10个奥斯卡奖杯)

一番阅览之后，张羽便回到仙人洞天之中，开始修行这部《拔昆仑翻天神通》。

这部功法和《天昆仑移山神力》有着相似之处，那便是都和地煞引力相关。

只不过《天昆仑移山神力》是以自身爆发的地煞引力，来撕裂大地，移山填海。

而《拔昆仑翻天神通》则是能够挑动大地、天空……乃至于万事万物中所蕴含的地煞引力。

“通俗一点说，就等于是将世间万物的引力剥夺，然后汇聚到自己的手中。”

“以我的化神境界，将此功推动50级后，一经全力施展……足以将方圆万里内的地煞引力汇聚起来。”

“届时，大地失去地煞引力的束缚，轟然破碎，然后地脉升腾冲天而去。”

“大气失去束缚，同样开始剧烈运动，引得九天齐震，风云倒涌。”

这一刻，张羽似乎已经能够预览见道道大气如龙般奔天而去，一切水分迅速气化，到处都是白气升腾而起，地表不断接近真空状态。

“世间万物，都会是一片翻天覆地之象。”

“同一时刻，天地之沉重，则在我一掌之间……”

张羽心中暗道：“同样是仙门功法，几百万仙币的真传功法，果然威能要远超那些十几万、几十万仙币的真传功法。”

斩仙说道：“传功价格自然是评价一部功法最好的指标，强大的功法……宗门几乎不可能便宜出售。”

“这也能看出，从最便宜的十几万仙币的真传功法，到几百万的真传功法，再到那些上千万的战略功法，十亿起跳的镇派功法，数千亿开天功法……相互之间的差距有多大。”

张羽微微点头，自从拜入万法宗，特别是进入御法阁工作以后，他便查阅过不少功法的介绍，对于这些价格也不是第一次感叹了。

福姬说道：“哼，这些功法买到手后……说到底也就是一些图文视频，一份版权，根本就没什么成本，却要卖几百万、几千万甚至是几千亿，就算有研发成本也早就赚够了，真是一本万利的好买卖。”

斩仙淡淡道：“若不是收回研发成本后，后续的销售成本几乎为零，一些仙门功法又怎么会通过正神审批，被宗门官方授权给下界的修士?”

张羽微微点头，知道很多时候贷款、功法版权、知识版权，都是上面最喜欢用来当做奖赏的东西，因为性价比够高。

比如他曾经参加的十大联赛，对上面来说奖一部仙门功法，哪怕是战略级功法，说到底也就是几道批文的事情。

但若要奖一口战略级仙门法宝，涉及到实打实的物资，那就是另一回事情了。

当然，这是仅仅从成本上来说，若是牵扯到各方势力争斗，昆墟规划，情况自然就更复杂了。

斩仙接着说道：“掌握仙门功法的贩卖和定价，这本身就是回收仙币，道功的手段。再结合仙门功法的各类授予渠道，如培训功法、奖品功法、岗位功法……那就是一整套用来控制仙道技术流通的系统。”

“所以说整个仙门功法的定价体系，并非是完全按市场来算的。”

荒牛突然冷冷说道：“这就是严重阻滞仙道技术的上下流通，这就是仙道技术暴政。以绝对领先的仙道技术掌控暴力和资本，继而扩展权势，掌控世界。”

脑海中的交流一闪而逝，张羽的注意力仍旧集中在眼前的功法上。

虽然《拔昆仑翻天神通》威能浩大，远超军用级的《天昆仑移山神力》，但这种浩瀚的威能之下，却潜藏着不可控的缺陷。

“这门功法全功运转后，对环境的破坏太大了，若在闹市区施展，只怕一次运功便要倾家荡产。”

“也是，若不因为是这样的话，那这部真传功法的价格说不定能上到6、7百万仙币。”

摇了摇头，张羽心想这门功法最适合施展的地方，恐怕还是得到准许的施工环境这里。

心中思绪电闪，功法修行的进度却是一路突飞猛进。

毕竟以张羽如今【化神道体】和一众分身的加持，配合上羽书的传精效果， 修炼起这门功法可以说是一日千里。

短短一天的时间内，只听张羽体内不断传来阵阵轰鸣之声，如同一道道山峦起伏、地壳碰撞所发出低鸣。

而随着这股轰鸣声逐渐消散，张羽也已经将这部真传功法给推动到了50级的 水平。

在此期间，张羽购买的一条价值两万仙币的基础仙灵根也已经到货，被他塞进了丹田里。

当然，和《拔昆仑翻天神通》这种万法宗的仙门功法不同，这条仙灵根是可以由步影疏赠予的。

于是从价值2 0 0 0 0 仙币的灵根，张羽通过步影疏的渠道最后只花了1 4 0 0 0 仙币。

而此时此刻，当张羽将这部《拔昆仑翻天神通》推动到了50级之后，他也终于要来试一试所谓的妙用了。

“荒牛，按照你的说法，这部《拔昆仑翻天神通》和【倒翻昆仑】的其他八套功法一同施展……”

脑海中说话之间，张羽已经顺着洞天之桥来到了工厂区内。

只见他体内法力奔腾，整个人便如同无尽深渊一般，疯狂吸收着天地间万事万物中所蕴含的地煞引力。

轰！

整个工厂区猛地震荡了一下，紧接着所有的厂房、楼房、大地，乃至是空气、尸件、设备，全都在同一时间失去了地煞引力的束缚，便要冲天而起。

但下一刻，随着土木七绝的——运转，天昆仑移山神力轰然发动，所有被张羽吸纳的地煞引力又被——释放了回去。

而就在这一进一出之间，天地间的地煞引力似乎以张羽的身体为中心，不断循环运转，来回翻涌。

这一刻，张羽心中升出一种奇妙的感觉，似乎整个工厂区的重量都被握在他的手中。

只要他五指一伸，就能将任何一座摩天大楼也罢，灵山大岳也好，直接轻轻拔起，抛向天空。

但张羽还能感觉……

“还差了一些，还少了功法……”

思索间，张羽已经猛然推动了仙门功法《掌中昆仑》。

此刻九门同属于【倒翻昆仑】的功法一同运转之下，《掌中昆仑》也立时有了奇妙变化。

伴随着天地间地煞引力的来回循环，张羽才感觉到自己作为这个循环中心，能够越发清晰地感受到空间的存在，甚至能看到一面面空间在地煞引力的影响下，展现出各种褶皱、拉伸的景象。

而《掌中昆仑》所能覆盖的范围也是疾速扩张。

这一刻，在九部功法的一同运转下，张羽感觉整个迷境的工厂区，就像是变成了一个沙盘模型，只要他随手拨弄，就能将这类厂房、仓库、设备给轻易搬弄、移动、转向&#183;&#183;&#183;&#183;&#183;&#183;

与此同时，荒牛在张羽的脑海中说道：“这就是”世界建造武功“，最初乃是为了改造世界而被逐一创造出来的一部部功法。”

“但随着世界建设的大体完成，这些功法也被渐渐封入尘埃。”

张羽随手轻轻一放，伴随着洞天显化的切换，掌中的废旧厂房就已经被他放进了仙人洞天之中。

就这么连续地随抓随放，工厂区内很快出现了大片的空白，而张羽的仙人洞天内则被一座座工厂堆得满满当当。

张羽心中感慨，这《拔昆仑翻天神通》配合另外八门功法一同使用，简直就像是把现实世界玩成了建造游戏。

功法的缺陷被弥补，爆发出惊天伟力的同时，又具有细腻的控制力，以及控场能力，让张羽搬迁厂房像是踩蘑菇一样简单。

而张羽引发的这番动静，也很快引起了洞天内磁极、阴泉、夜星璃等人的注意。

看着赶来三人，张羽简单讲述了一下情况，便将这些残破厂区交给了她们，搜集其中可以利用的物资。

接下来的日子，有了更多的两手物资，再加上张羽的仙币持续投入，洞天内生产线、实验室的建设都可谓是突飞猛进。

只见厂房内，张羽和磁极正在一同设计法宝，他们计划将【天日煌神】进行改造，使得这件傀儡法宝能够驾驭【铸世熔炉】散逸的上阳气。

【天日煌神】是张羽在大学期间就掌握的军用级傀儡技术，能够驾驭火焰，制造高温和爆炸，并且随着火焰改变自身形态，适应种种工作。

只不过这几年来张羽的【天日煌神】迟迟没机会升级，他也没时间推动炼器技术，已经许久没有使用这傀儡了。

磁极说道：“你现在借助洞天力量，直接去搬运上阳气，这种使用方法太粗糙，也太浪费了。”

“如果能通过对天日煌神的改造，使得这件傀儡以上阳气为核心的话，恐怕能突破军用级的水准，接近仙门级法宝的威能。”

在磁极和张羽的配合下，两人不断发动【天工开物】，双方的知识反复碰撞在了一起，开始一次次的设计、改造和炼制。

张羽心道：“以我一生积攒的学识，通天的智慧，博览世界的底蕴，给我燃烧吧！天工开物，让我看看你的极限！”

磁极心道：“高达是什么东西？是我从宗门的哪里看到的吗?”

“Eva？阳电子炮？这些又是什么东西?”

“《血潮精气圣观…》？镇派功法？我还知道这里面的技术?！”

在一次次发动【天工开物】的过程中，磁极越来越感到震惊。

“我难道觉醒了前世记忆不成?”

“我的前世究竟是什么人？竟然会知道这么些…东西?”

最终，张羽和磁极不但成功完成了【天日煌神】的升级，更是将磁物极的道种【天工开物】也提升到了2级。

张羽过去和磁极的差距太大，自从得到了道种以来，一直都没什么机会升级，而这一次终于是得偿所愿。

2级【天工开物】的效用，经过张羽的一番参悟后便有所了解。

“所有翻阅过的知识，都会直接融入到【天工开物】的底蕴之中。”

“很简单却又很实用的能力。”

张羽知道，如此一来，任何只要被他和磁极看过一遍的知识，哪怕他们还没有完全掌握，也会在【天工开物】的发动过程中，直接被融入到法宝的设计图纸中。

“等于是拥有了一种限制使用范围的……超高速学习能力。”

【天日煌神】经过他们的改造，被设计成了一种全新魄儡，并命名为【上阳煌神】，一种以上阳气驱动的傀儡。

看着设计图上介绍的，被取名为阳电子炮、上阳力场、上阳震动剑的东西……

张羽心中暗道：“磁极老师的吸收能力很强啊。”

磁极说道：“【上阳煌神】已经不能算是军用级法宝，他现的动力、速度、还拥有破坏力，已经超过了一些仙门真传级法宝。”

张羽微微点头，心想着有机会的话，应该去把【上阳煌神】登记、审批一下，确认好版权。

若是有朝一日【上阳煌神】继续改造下去，成为真正的仙门级的话，那这个版权可就值钱了。

磁极接着说道：“目前生产线的改造也差不多了，已经可以实现批量制造。”

“毕竟这傀儡本就是我根据工厂区的现有材料，在你的资金助力下设计出来的。”

“以现有资源，接下来制造并维护个几十头……应该问题不大。”

听着磁极老师所说的话，张羽似乎已经能看到自己的32重分身驾驭上百【上阳煌神】所能爆发出来的惊人工作力。

特别是驾驭上阳气的傀儡弥补了张羽的一大短板，让他有了不俗的破坏力，还增强了他的高速飞行能力。

“我的工作力已经在暴涨！狂涨！疯涨！同境界下，谁还卷得过我？还有什么工作我完不成？有什么工作我完不成了？！我要为宗门一顿爆干口牙！”

考宗分身一阵大吼，便在张羽的脑海中哈哈大笑了起来。

就在张羽的傀儡军团建设时，另一边的夜星璃这些天来也一直在配合张羽，进行实验室的打造。

“七绝法骸？”

张羽说道：“土木不是我的黑历史，而是我的来时路。”

“而且我观这宗门之中，土木也大有可为啊。”

“师姐，你也该多学学土木技术了，对你未来设计法骸会有帮助的。”

张羽说道：“施怀玉有句话我觉得她说的对，君子

补仓，十年未晚也。买一只股票最好的时间是十年前，其次是现在。师姐你现在开始学也来得及。”

随着两人一起进行炼器设计，也顺利将【灵韵器心】给推动到了4级。

2级的【灵韵器心】可以选择融合方向，3级则能够最多融合3张图纸。

而4级的【灵韵器心】不但赋予了张羽、夜星璃强大的图纸学习能力，更是从原本的最高3图纸融合，变成了最高4图纸融合。

而张羽和夜星璃的第一件作品，便是经过数次合成，将万法大学土木系的七绝法骸，合成为了一枚接近仙门级的眼骸。

当然，这个过程中也少不了张羽以【天工开物】进行的辅助。

张羽将这件眼镜称为昆仑眼。

随着昆仑眼的装载完毕，张羽便感觉到体内地煞引力一阵翻涌，《天昆仑移山神力》便获得了重重强化，并在地煞引力循环的过程中，连带着推动了《拔昆仑翻天神通》和《掌中昆仑》的威能更上层楼。

反馈的种种数据被统计、综合，最终整合成了一份周遭的实时地图，完成展现了周围的地貌、地形。

靠着昆仑眼的制导能力，能够让张羽对方圆万里內的物质，施展较为精确的地煞引力控制，进行超视距的施工。

如此一来，就更不怕运动的时候造成不必要的破坏，继而赔钱了。

除此之外，在昆仑眼的作用下，配合张羽的洞天显化能力，他能更方便地连接到仙人洞天，并将自己的力量作用其中。

此刻在张羽的感应之中，仙人洞天的偏僻一角内，无数巨大的金属立方体缓缓飘起。

这是张羽特地通过步影疏的渠道，购买来的黑冥重金。

作为一种超高密度材料，黑冥重金本身的价格并不算贵，张羽花了十多万仙币就堆成了一座小山。

而此刻在昆仑眼的协调下，张羽功法运转，便将一整座黑冥重金构成的小山份量，给凝聚到了自己的掌中。

张羽能感觉到自己若是将这股力量爆发出去，便像是将一整片山川给砸出去，足以暴发出一场超级大地震。

“如此一来，不需要再像外界借力，只需要沟通仙人洞天，我就能随时爆发出巨大力量。”

而张羽知道，不论是生产线，还是实验室，这些发展都只是开始。

到现在为止，他也不过烧了一百多万的仙币而已。

随着他对此对仙人洞天的持续投资，随着磁极老师、夜星璃学姐他们的水平和道种不断提升。

未来生产线、实验室不断完善，生产出各种法宝、法骸……对他以后的能力提升都会有着巨大的帮助，也比直接花钱购买、维护市面上的法宝要更有性价比，也更安全。

“甚至以后产品质量继续提升，还能卖出去赚钱，让生产线和实验室赚钱反哺我。”

“不然老是靠工资、奖金、诈骗，还有跪舔隐忍这些办法，终究不苌久。”

“我需要现金奶牛，需要源源不断为我生产仙币的地方。”

就在张羽不停烧钱，将修行，仙研，洞天发展持续推进的时候，迷境的灵界连接也在他的计划中渐渐恢复。

而伴随着灵界的恢复，迷境中的各项工作也重新推动了起来。

只见实验区的一片战场上，一头矫健黑龙正在来回闪烁，游走在一位位化神修士的面前。

那是名为【天轨座】的炼虚尸件。

那是由无数尸皮缝合成的传送带，传送带又扭曲绞缠，最终形成了酷似黑龙的模样。

这头炼虚水平的尸件，不但力大无穷，金刚不坏，更能够洞穿虚空，和一定范围内的物质交换位置，可谓是神出鬼没，防不胜防。

此刻，天轨座倏忽出现在了图牧的面前，一道巨大的龙爪扑面抓来，似带着万千尸骨的哀嚎。

面对这炼虚水平的一抓，图牧惊惶之间，便见一只大手轻轻一捞，好像是捞鱼一样，将【天轨座】捞到了半空之中。

图牧知道，出手之人正是管委会的副会长，他的顶头上司张羽。

看着炼虚水平的尸件，竟然被张羽好像玩具一样弹来弹去，图牧的心中一阵唏嘘：“听说副会长当时虽然接下炼虚一击，却是被打得直接流产，受了重伤。”

“不过后来谢兰因就被逼着下跪赔钱，张副会长也在仙币的刺激下，垂死病中惊坐起，最终不药而愈。”

如果说这让图牧感到羡慕的话，这段时间张羽的变化则让他更感觉到惊讶。

只见空间结构一阵变化，道道火龙从张羽周身浮现。

那是稍32重分身运转《妖圣万变神经》，化作32条蛟龙，套着驾驭上阳气的【上阳煌神】，从张羽的仙人洞天里钻了出来。

伴随着32条火龙张口一吐，32道以上阳气为根基的阳电子炮已然喷涌而呼，将黑龙打得惨嚎连连，一次次瞬间换位，却又被张羽拉了回来。

看着发狂的炼虚尸件，竟然被张羽玩弄于鼓掌之中，图牧心中也不由得一阵暗惊：“虽然有谢兰因的巨额赔偿款，但这么短时间里&#183;&#183;&#183;&#183;&#183;&#183;实力竟然又有了这么大幅度的增苌，这仙人传承也未免太变态了。”

就在这时，只听张羽淡淡道：“一起出手，将这尸件镇压。”

虽然实力大涨的张羽，哪怕独自面对炼虚尸件，也已经能够稍占上风。

但想要将之彻底降服的话，只靠他一个人必然要有巨量的消耗，以及不短的时间。

可张羽自然不会做那种有手下不用的领导，随着他一声令下，众多化神修士纷纷全力出手，在他面前争取表现。

片刻后，便见张羽伸手一抓，便将【天轨座】宛如一条黑色泥鳅般抓在手中，扔进了仙人洞天里。

“走吧，接续下一处”

接下来的日子里，在张羽一边修行、发展，一边将一头头炼虚尸件收伏。

灵界逐步恢复，错乱的空间结构也——修正。

整个开发区的清理工作迅速而又稳步的推进着。

虽然还有几天就要来到第20天的约定之日，但不论是图牧还是其他化神修士，都不觉得张羽会完不成这个约定。

但就在这一切看似平静无波的日子中，某一天的夜里，管委会的副会长谢兰因疯狂联系向陆衡章。

看着这位万法宗器灵派、先天道统支持者的陆衡章，看着对方终于落下的投影，谢兰因压着心中的一丝怒火和惶恐，说道：“毁境淹区？你们疯了吗？”

看到陆衡章没有说话，谢兰因一字字道：“旧日坟场所在的小界一直是固若金汤，乃是宗门层的根基小界之一，若出现破损，引得崩塌&#183;&#183;&#183;&#183;&#183;&#183;那必然是有人暗中动了手脚！”

陆衡章淡淡道：“谢副会长，这世上哪有什么固若金汤的小界啊。”
