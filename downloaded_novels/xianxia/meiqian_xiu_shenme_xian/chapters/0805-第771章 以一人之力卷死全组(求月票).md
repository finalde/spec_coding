# 第771章 以一人之力卷死全组(求月票)

御法阁审核部的办公区内。

储忘机正在指挥着手下的外包员工们，进行一门功法的功能测试。

而祂的脑海中，也再次回想起这个月张羽号称要提高全组30%业绩的事情。

储忘机对此的评价就两个字—— 狂妄。

'你才来审核部几年？？ 就想要指挥我们所有人了？？ '

而接下来对于张羽的态度，储忘机的应对也很简单，那就是不配合。

'想要增加业绩，用的办法无非就是那些。'

'要麽更新法骸~法宝，要么主动加班，要么多找点外包。，'

在储忘机看来，想要提高工作效率，提高每月的业绩，所有方法总结下来无非就两个字——砸钱。

如果要再加两个字的话，那就是借贷。

如果让储忘机去砸很多钱来工作，甚至去借贷加班，那祂确实可以提升不少业绩。

但凭什么呢？？

'要我燃烧自己的仙道潜能，去成就你张羽的承诺？？ 去助力上台领导的功绩？？ '

储忘机不愿意做这种事情。

在祂看来，武道三组原本的工作状态就不错，像祂这种熟练的资深老员工，不拚第一，现在每个月几乎都能做到不用加班，还都能存点钱下来，真真是从容容游刃有余。

祂完全就可以这么有条不紊地积攒仙道潜力，等待升职的机会。

但如今。，这一切却被张羽搅得乱七八糟。

就在这时，一道消息在储忘机的眼前浮现。

温知微：管祂怎么来劝我求我，反正我肯定是不会配合的

储忘机知道，这是祂们几个组员小群里的消息，几名同事正在里面说着张羽的话题。

温知微是和储忘机一样的二级宗务员，储忘机知道祂和组长的关系比较近。

而自从知道了张羽要提升全组30%的业绩后，温知微便表达了强烈的不满，从今天上班开始，便一直在鼓动众人不要配合。

有人说道：'我就按照和上个月一样的效率来工作。让上面的领导明白，这张羽就是胡说八道，这业绩能这么乱提升吗？？ '

又有人说道：'千万不要让这张羽得逞，不然以后大家都没好日子过。'

温知微：'都说好了！！ 这个月谁都不准加班！！ '

就在众人在群里商量着的时候，祂们所有人的眼中都跳出了一条消息，来自武道三组的工作群，一道由张羽发出的消息。

张羽：报告组长，我已经完成了《奔云神雷掌》的审核工作，报告刚刚成功提交给了道藏

看到这条消息的众人心中冷哼一声，只觉得这个张羽太过得瑟。

储忘机心道：'直接在群里炫耀自己的业绩吗？？ 太嚣张了，而且你以为自己业绩高，大家就要听你的吗？？ '

温知微直接在小群里说道：'让祂自己去卷吧，最后欠一屁股债，我看祂能撑多久。'

但不久之后，张羽又在群里发送了一道消息。

张羽：报告组长，我已经完成了《 惊鸿照影刀》的审核工作，报告刚刚成功提交给了道藏

组长曲易之看着张羽的发言，脑海中又想起了对方在会议上的发言。

祂心中暗道：'这小子。，喜欢出风头，想表现自己，想抢功劳，以为这样做就能往上爬？？ '

'宗门，不是这样混的。'

与此同时，另一边的张羽却仍旧没有停下，还在继续审核不同的功法。

只不过第一天的四小时工作时长实在不够用，张羽刚刚只完成了两份报告，就已经迎来了下班。

但下班之后的张羽也没有停下，祂回到宿舍之后，便继续模拟各种不同的功法，这些功法都是上班时间从内部法界收集的。

而模拟完之后，就将脑海中的资料~数据~图像。，一切都通过文牍思印传递给夜星璃。

于是第二天上班后，夜星璃便立刻连上了内部法界，将张羽过去近20个小时的成果。，不断转化成书面报告。

于是在众人刚刚来到办公区内不久后，便又看到了张羽发送的第三份报告。

接着是第四份~第五份~第六份。，

如果说最开始，组内的成员看着张羽发出的报告，还是带着一种不满~看乐子~看你能撑多久的心态的话。

那随着时间来到第三天，当张羽发送的报告已经突破10份的时候，不论是武道三组的大群还是小群，都已经陷入了一片诡异的沉默之中。

当报告增长到13份的时候，张羽身上那暴涨而出的业绩威压，已经压得众人说不出话来，就好像一座大山狠狠挤在了祂们的胸口

那足足13份的报告，更是将祂们的业绩彻底碾碎，狠狠踩在了脚下。

'祂。，祂太能工作啦！！ '

'三天时间就干了我一个月的活！！ 祂的工作效率到底强到了什麽地步！！ '

'太卷了，实在是太卷啦！！ '

这一刻，张羽在所有组员的眼中已经不是寻常宗务员。

祂的每一个动作，每一个眼神似乎都是为工作而生，祂的每一次发言，每一道消息，宛如都能推动工作进度，提升项目业绩。

惊天动地的业绩威压从祂身上涌出，似是要将这世间的一切工作统统完成。

这一瞬间，和张羽站在同一个办公区的组员们只觉得自惭形秽。

明明是历经考验才拜入宗门的精英，祂们却感觉站在张羽面前的自己。，就好像是个废物，像是在整日里偷懒~摸鱼~混日子的废物。

'卷王！！ '

'这家伙是个卷王！！ '

'祂这么搞下去，是要把自己这个月的业绩提升几倍吗？？ '

储忘机脑海中无数思绪闪动，似乎已经看到上面领导目睹张羽的业绩后，开始大肆宣传，甚至开始提高各组业绩要求的画面。

'这么涨下去，上班成本又要增加了。'

储忘机感觉张羽那不断提交的审核报告，就好像是一片缓缓上涨的水面，从祂的脚踝一路涨到了祂的腰，并且还在继续往上涨，要将祂的身体彻底吞没。

'想上班少花点钱，就这么难吗??!! '

就在储忘机被张羽的绩压给压得喘不过气来的时候，祂收到了一份来自张羽的消息。

那是一份完整的审核报告，报告的内容是关于《两仪回风剑》的版权审核。

而这《两仪回风剑》正是储忘机目前正在审核的功法。

'祂。，祂把我的工作都给做了??!! '

看着其中内容的储忘机脸色一白，只觉得眼前如同一道天雷落下，震得祂心神摇曳，精神恍惚。

'我的工作被祂做了？？ 那我做什麽？？ '

又过了一天，当储忘机再次收到来自张羽的消息，看着又一门功法的审核报告时，祂只觉得脚下一软，似乎已经在这办公区里站不住了。

'要是祂能把我的工作都做了。，那我还有什么用？？ 会被上面给优化掉吗？？ '

单位的优化，部门的优化，小组的优化，宗务员的优化。，作为多年的2级宗务员，储忘机知道这一切每年都在万法宗内轮番上演。

宗门上层对于成绩~增长~创新的要求可以是无限的

而任何业绩下滑，利润下滑，又或者是上面认为没用的东西，都有可能面临被优化的风险。

至于优化后的去向？？ 除了极少部分人因祸得福之外，大部分在原岗位工作多年的宗务员因为缺乏新技能的支撑，往往会像个垃圾一样，被丢到一些像是垃圾桶一样的地方。

降薪~降级~调离。，储忘机的脑海中似乎已经浮现出无数凄凉的画面。

这一刻，储忘机只感觉张羽的身上爆发出来的绩压铺天盖地，带着不可抵挡的伟力，就要将祂从办公区内生生挤出去。

祂心中狂吼：'不要！！ 不要把我的工作给做了啊！！ '

就在储忘机心中惊怒交加的时候，祂发现收到类似消息的竟然不止自己一个。

有人惊惧道：'太恐怖了，祂把我明天的工作都给做了！！ '

有人后怕道：'张。，张羽刚刚问我接下来要做什么任务？？ 祂是什么意思？？ '

'特麽的。，'温知微惊道：'祂是要以一己之力。，卷死特麽的整个武道三组??!! '

储忘机服了祂知道自己卷不过张羽，更不想被优化，不想失去如今的工作，于是祂求到了张羽面前，希望张羽不要再碰祂的工作了。

'只要你别再碰我的工作，你可以提要求。'

看着眼前一脸认真的储忘机，张羽笑了笑说道：'那就接受我的指导吧。'

储忘机眉头一皱：'指导？？ '

只见张羽发动了无相演武，已经直接模拟出了两门武功。

而其中一门武学，正是储忘机在审核的武功。

看着自己审核的武功在张羽的手中运转，并且被融合到一门又一门其祂武功之中，储忘机脑海中顿时一片空白。

储忘机知道，靠着武一到武十的推动，负责武道审核的祂们可以模拟天下间的各种武学。

但祂从没想到张羽能在如此快速模拟一门武学的同时，还融合到各种不同的武学中。

这种武道上的资质~积累~修为。，让储忘机望洋兴叹。

张羽淡淡一笑道：'你从没有见过这门武功被这么仿真的样子吧？？ '

储忘机张了张嘴，却终究没有说话。

看着在张羽手中被放肆施展~融合的武学，储忘机终于知道了自己和张羽之间的差距有多大。

' 这门武功在我手中永远也没有在张羽手中这么光彩夺目。，'

与此同时，张羽已经拿出了文牍思印。

只听张羽淡淡道：'接受我的指导吧，你也不想看到自己的工作都被我做掉的样子吧？？ '

伴随着无数资料~数据~感悟。，齐齐涌入了储忘机的心头，顿时让祂对自己手头的工作已经有了完整的过程和结论，就好像是考试前就直接拿到了答案一样。

储忘机出神地想到：'这种感觉。，以前工作的时候从来没有过。'

祂感觉自己只要连上内部法界，很快就能完成这份审核报告。

储忘机看着眼前的张羽，祂明白只要对方愿意，便能不断提高自己的业绩。

而随着业绩的提升，祂也将彻底离不开张羽。

一旦离开张羽的支持，想要继续维持高业绩的祂，将会被拖入加班线内，在一次次付费上班中被不断消耗仙道潜力。

'这样的张羽。，'

看着张羽的背影，储忘机心中暗道：'已经可以算是我半个爹了。'

而储忘机的业绩增长很快就被其祂人看在眼里。

于是渐渐的，不断有新的组员找到了张羽这里，开始接受张羽的指导。

'羽哥，请不要动我的工作。'

'羽前辈，我想接受你的指导。'

'烛熠真君，请您教教我。'

伴随着一次次的武学模拟，张羽将心中对于各种武道的测试~参悟~感受~比较。，全都传输给了储忘机等人。

然后由祂们自己去完成所有的文书工作，将张羽的参悟整理为一份又一份的报告。

于是整个武道三组中，业绩增长的人越来越多，没有增长的人则开始惊慌，开始担忧，开始思考起接受张羽指导的事情。

。，

'不可能！！ 我怎么可能去求张羽??!! '

当看到张羽发来的消息，看着张羽竟然完成了自己的工作时，武道三组的组长曲易之心中喝道：'想要威胁我？？ 做梦！！ '

作为组长的曲易之同样是要完成审核任务的，虽然不算在组员的业绩排名表里，但祂平时完成的业绩也足以排进全组前三。

若没有这样的水平，也难以服众，难以展开管理工作。

但此时此刻，不知不觉间，当曲易之发现组里成员一个个业绩增长，甚至将祂越甩越后的时候，曲易之渐渐感觉到着急了。

'这样下去，我岂不是要成倒数几名了？？ '

想到这里的曲易之立刻感受到了极大的威胁，似乎已经看到加班线在追向自己。

而在发现越来越多组员听命于张羽后，曲易之心中更是警铃大响。

祂知道，自己绝不能再让局势恶化下去了。

于是曲易之向张羽发去消息：来我办公室一趟

但良久之后，看着张羽迟迟没有出现在自己的办公桌前，曲易之的面色微微阴沉了下来。

于是祂又发消息催促对方。

这次终于得到了回应。

张羽：正在忙工作的事情，等我忙完了再过来

看到这番回应的曲易之眉头皱起，在祂眼中，张羽这名一层爬上来的修士有城府有手段，绝不是什么愣头青。

对方如此反应的原因。，

'这家伙。，对我有不满？？ '

'祂察觉到我在拦祂了？？ '

。，

张羽在指导组员们审批功法的过程中，便发现了一件事情。

那就是在祂一次次模拟功法，在和组员们进行思维交流的过程中，祂发现每一名组员所审批的功法。，似乎在武道风格上都有所不合。

如果说是一人~两人，还能说是巧合。

但当张羽发现几乎所有人都是如此的时候，祂便明白这绝不是什麽巧合，而是有人故意为之。

而武道三组之中能够这么做，或者说有机会这么做的，便也只有一人，那便是组长曲易之。

'这个家伙，在故意对付我？？ 拖我后腿？？ '

张羽知道，要对付宗门弟子，靠讲理~讲人情。，那都是没什么用。

所以祂不打算和曲易之进行无谓的交流，祂要逼着对方来求自己。

于是看着曲易之再次发来的消息，张羽只是回道：组长，你放心好了工作要开始加速了

关闭通讯录，张羽转身便已经再次埋入了工作之中。

从这个月的月初开始，祂便没有在进行任何元婴圣体的修炼，而是全心全意地投入到了武道三组的工作之中。

上班的时候祂在模拟武道，下班了以后祂还是在仿真武道，当一名名组员找上门来后，祂更是在模拟武道。

这一刻的张羽，就好像是化为了一台分析武道功法的仪器，时时刻刻都在进行着武道分析的工作，然后将结果发送给了一位位组员。

祂就这么凭借着一己之力，将整个武道小组的业绩向前推去。

。，

看着自己的业绩排名一步步下跌，曲易之的心中也变得越来越烦躁。

'再这样下去，我这个月的业绩真的要排在最后几名了。'

想到这里，曲易之脑海中便泛起上次会议上审核部部长洛无羁的交代

祂可是当着众人的面，答应部长要全力支持张羽的工作的。

如果是大部分人的业绩都没有增长，那自然是张羽自己没有本事，胡吹大气。

但如果一个月下来，所有人的业绩都在提升，唯有祂曲易之没有增长，部长会怎么看祂？？

'恐怕会觉得我在故意扯后腿。，'

回想着洛无羁部长所说的话，曲易之明白对方是想要在3月份的考评之前，再做出一份成绩上的突破。

' 我若是被部长认为是故意坏了祂大事。，'

想到这里，曲易之便感觉到一阵头皮发麻。

'难道我要加班？？ 或者花钱提升工作效率？？ '

想到要消耗自己的仙道潜力，曲易之便又是心中一万个不舍。

思来想去，似乎接受张羽的指导，一同推动武道三组的增长，完成对部长的承诺。，这个原本不可能的选项，在这一刻却已经成了最好的结果。

'但副部长那边怎么办？？ '

就在这时曲易之目光一闪，便看到了一条消息传来。

正是洛无羁部长在御法阁的会议上，向阁主报告了审核部的近况，其中还说到了几句武道三组的业绩提升。

看到这里，曲易之眼中渐渐下定了决心。

祂来到张羽的办公桌前，开口说道：'张真君，借一步说话。'

两人来到一处修炼室，曲易之开口说道：'张真君认识陆衡章，陆副部长吗？？ '

张羽闻言心中一动，说起了对方曾经担任过自己面试官的事情。

曲易之接着开口道：'陆副部长似乎对张真君格外关注，一直让我多锻炼一下你这个新人，多磨练磨练你。'

张羽目光一凝，心中暗道：'是陆衡章想要对付我？？ ‘
