## 用法说明

- 文件中的剧情、角色、台词均来自公开搜索（百度百科 / 维基百科 / 豆瓣 / 各方媒体），仅用于个人创作灵感参考。
- 涉及的"虐点"、"名场面"用于提炼仙侠剧情节模板，不构成对原剧的转载。
- 题材标签可作为 AI 视频选题的快速过滤依据：例如 "三生三世 + 失忆"、"魔尊救赎"、"师徒禁恋"、"上古神 × 凡人转世" 等。
