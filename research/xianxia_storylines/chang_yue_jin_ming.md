## 一句话设定

世间最后一位仙门弟子黎苏苏在魔神横行、仙门尽灭的末世，发动"逆时秘术"穿越回五百年前，化身叶家千金叶夕雾，意图在魔胎澹台烬黑化前用爱救赎他、终止灭世预言；却在层层翻转中得知，自己与他的纠葛要回溯到万年前的神魔大战、上古战神冥夜与蚌族公主桑酒的旧梦。

## 完整剧情 (分卷)

### 第零卷 · 万年前 · 神魔初纪

万年之前，魔神现世，三界涂炭。十二天神率领仙族与魔神死战，几乎尽数殒落，方将魔神镇压。诸神中，上清神域的战神冥夜——本相为一条修为登顶的蛟龙——背负护佑苍生之责，于神魔大战中身受重创、神识沉眠。同一时期，墨河深处的蚌族小公主桑酒，自幼在蚌父与蚌兄桑佑的宠爱下长大，因仰慕战神风采，被卷入这场跨越神魔的爱情。

桑酒为救冥夜，舍弃蚌心化作一颗珠，融入冥夜的剑骨；冥夜醒来后却已记不全她的容颜。天欢——觊觎冥夜已久的妖族公主——与其兄天昊设局屠灭蚌族，桑酒逃过一劫，却在巨大的悲愤与失落之中入魔，最终于雷劫中灰飞烟灭。冥夜得知真相后悲痛欲绝，于墨河底以自身神力造出"般若浮生"梦境，将与桑酒共度的每一个瞬间无限循环，他自身则沉睡为一条上古魔龙，以护卫桑酒故乡。这条情债的余孽，在万年后将以"魔胎"之名重新落入凡尘。

### 第一卷 · 现世末日 · 逆时秘术

五百年后，魔神澹台烬重临，四洲三界陷入永夜，仙门尽灭。衡阳宗掌门养女、继承凤凰神血的最后一位神女黎苏苏，于景国废墟中以心头血祭"逆时秘术"，时间倒流，她的元神回到五百年前的人间——那时澹台烬尚未黑化，仍是景国一个孱弱多病、备受欺凌的质子。

### 第二卷 · 五百年前 · 质子澹台烬

黎苏苏化身叶家小女儿叶夕雾。彼时澹台烬只是景国质子，骨节嶙峋、孤身困于冷宫，唯有盛国六皇子萧凛偶尔施以善意。夕雾以"师姐"身份多次保护澹台烬：朝堂上挡刀、雪山顶挡雷、暗巷中以身相替。澹台烬看她的眼神由阴鸷转为暖意，第一次有人告诉他"你不是魔，你是个人"。

但每一次救赎将至时，命运都冷酷地把他推向魔神之路：受刑、丧母、亲眼见挚友被斩。夕雾发现自己越是靠近他、越是给他温度，他体内沉睡的魔神血就被唤醒一分——这是"以爱止劫"的诅咒。萧凛之死成为关键转折，澹台烬的最后一线人性碎裂。

### 第三卷 · 般若浮生 · 蚌珠之忆

夕雾在追查澹台烬身世时，跌入一处由上古魔龙留下的"般若浮生"梦境。她看见万年前的自己——蚌公主桑酒，与战神冥夜的相遇、相爱、生离死别。她终于明白：自己三世的元神同源，澹台烬即冥夜的"残念"投胎、亦是魔神的肉身容器；她与他的羁绊，不是从五百年前开始，而是从万年前的墨河深处。同一梦境里，叶冰裳是魔界食人妺女转世、天欢是上古妖姬重临、萧凛与冥夜剑骨同源——所有人都是旧债的延续。

### 第四卷 · 黑化反转 · 魔神觉醒

朝堂权斗、门派暗算、九嶷山阴谋三线合流，澹台烬被推到崩溃边缘。叶冰裳以庶女之恨倒戈魔界，亲手撕开夕雾"师姐"的伪装；澹台烬目睹挚爱之人原是为终结自己而来，最后一线温情成为最深的刺。他于景国朝堂之上吐血掀棺，魔神血脉彻底觉醒，黑甲披身、长发垂肩、登顶魔神大殿——"我已成魔，何必再爱"。叶夕雾陨于这一战，元神反向溯回五百年后黎苏苏体内。

### 第五卷 · 仙界沧九旻 · 五百年的搜寻

澹台烬入魔后，一缕未泯善念脱离主体，化为失忆仙人沧九旻，被上清仙门救起，收为衡阳宗弟子。沧九旻以白衣仙人之姿在幽冥川中搜寻叶夕雾的元神，找了整整五百年。直到黎苏苏发动逆时秘术、记忆与三世因果合一，他才在仙门弟子的脸上认出她。沧九旻、魔神澹台烬、上古冥夜——三魂同时浮现于他的识海，于镜花水月中互相对峙：白衣的问"为何要爱"，黑甲的答"爱即是魔"，蛟龙的低吟"我们已欠她万年"。

### 第六卷 · 大结局 · 殉道与新生

最终决战于同悲道。澹台烬以一个谎言救苍生、用一个梦满足自己——他假意全面入魔、登顶魔神大殿，反复刺激黎苏苏渡雷劫成神，借成神后的真神之力将自己与同悲道（魔神之源）一并彻底毁灭。黎苏苏识破时为时已晚，亲手以神光将爱人灰飞烟灭。

魔神身死，反噬终结，仙界回春，世间复苏。澹台烬曾以叶夕雾的身份，提前在景国皇陵为自己刻下墓碑——这是他留给她的最后一封"君取我心，我无怨"。结局五百年后，黎苏苏带着小女儿在墨河之畔玩耍，孩子的眉眼隐有澹台烬的影子；三世记忆已合一的真神黎苏苏，在月色下抬头看了一眼夜空——长月未尽，烬已成明。是开放式 BE，是国风仙侠"以爱殉道"的标准范式。

## 角色详细描述

### 1. 澹台烬 / 沧九旻 / 冥夜

| 项 | 内容 |
|---|---|
| 演员 | 罗云熙 |
| 身份 | 一魂三身：景国质子澹台烬 → 魔神澹台烬 → 仙门弟子沧九旻 → 上古战神冥夜（蛟龙本相） |
| 别名 | "魔神"、"沧仙人"、"九旻师弟"、"冥夜君" |

**外貌**：身形清瘦、肩窄腰直；质子期面色苍白、眼下乌青、唇色寡淡，长发凌乱半束于脑后，颈上常带铁链勒痕。魔神期面如冷玉，眼瞳由灰转血红再转金，长发散垂及腰，黑甲金纹、肩覆战兽纹兽吞肩。沧九旻期一袭青白仙袍、白玉簪挽发，眉目温润如初春细雨。冥夜期蓝白配色仙袍、敦煌纹样、龙鳞铠甲，眉间一点战神朱砂。

**性格**：质子期阴鸷、自卑、刺猬般敏感，唯一软肋是"被人当人看"；魔神期高傲、冷酷、毁灭欲压顶，但保留对苏苏一人的"不舍杀"；沧九旻期温润、克制、目光极远，已隐隐知晓自己是谁；冥夜期沉稳、悲悯、责任压肩，是十二天神中最孤独的那一位。无论哪一身，骨子里都是"宁负苍生、不负此心"。

**角色弧光**：从"无情丝魔胎"到"以情殉道真神"——他被设计为"必入魔"的容器，却用三身轮回证明了"魔可以不为魔"。最终他选择捏碎邪骨葬身同悲道，把"魔神"这个称号永远从三界除名。

**标志台词**：
- "你别怕，我不会再让人欺负你。"（质子期对夕雾）
- "我已成魔，何必再爱。"（魔神登顶大殿）
- "苏苏，我找了你五百年。"（沧九旻识破苏苏身份）
- "成神吧，黎苏苏，把我也烧干净。"（殉道同悲道）

### 2. 黎苏苏 / 叶夕雾 / 桑酒

| 项 | 内容 |
|---|---|
| 演员 | 白鹿 |
| 身份 | 一魂三身：现世衡阳宗女、神血最后一位神女 → 五百年前盛国叶家千金 → 万年前墨河蚌族公主 |
| 别名 | "神女"、"师姐"（夕雾对烬的自称）、"小蚌"（冥夜对桑酒的爱称） |

## 服装与造型笔记

> 由服装造型指导黄薇与团队赴敦煌采风一年完成，全剧逾 2000 套服装。整体美学以"敦煌风"为核心：高饱和雅丹色彩 + 飞天造型 + 黑白红主色调 + 流动性极强的丝带飘举感。

### 澹台烬 / 沧九旻 / 冥夜

- **质子期**：素白或月白粗布襦袍、铁链颈镣、长发凌乱半束、面颊苍白唇色寡淡；标志道具是脖颈上的勒痕与冷宫中破旧棉被。
- **魔神期**：黑色重铠 + 金色魔纹（兽吞肩、龙鳞胸甲、流云裙甲），披风内衬血红，长发垂腰。眼瞳血红时插戴一枚黑金额饰。"成神战甲"传出二十层叠穿，是全剧最重最复杂的造型。
- **沧九旻期**：青白色或月白色仙袍、白玉簪、外罩淡灰薄纱披风。
- **冥夜期**：蓝白配色仙袍、敦煌缠枝花纹、肩覆龙鳞铠甲、眉间战神朱砂。

### 黎苏苏 / 叶夕雾 / 桑酒

- **苏苏期**：高马尾束金色额饰、白红或白金法袍、外披深红披风。
- **夕雾期**：浅紫、粉白、嫩绿三套主家常襦裙轮换；双丸子头或单髻配珠花，整体糖豆色调。
- **桑酒期**：淡蓝绿水袖长裙 + 白色长发 + 金色腰带 + 蚌珠耳坠 + 琥珀色瞳，水下场景加蓝绿粒子点缀。

### 叶冰裳 / 妺女 / 天欢

- **叶冰裳**：墨青、酒红、银白三套礼服轮换，高髻配步摇，唇色冷艳。
- **妺女**：红衣大袖披散长发、瞳色泛红、眉间血泪痕。
- **天欢**：金冠 + 橙红妖艳裙裾 + 丹凤眼上挑 + 妖纹腰带，是全剧最艳丽的反派造型。

### 萧凛 / 公冶寂无 / 桑佑

- **萧凛**：白衣月华、玉冠束发、白银剑穗。
- **公冶寂无**：墨蓝道袍、剑修高束、外罩黑色雷纹短氅。
- **桑佑**：淡青水族锦袍、白色长发束高、银色蚌纹腰扣。

### 其他

- **翩然**：浅紫粉绿仙袍、灵动发饰；
- **叶清宇**：玉冠白衫 + 银甲披风（出战时）；
- **谛冕**：墨黑兽纹长袍、银鬓 + 妖王金冠；

## 文化现象

- **罗云熙"魔神黑甲"出圈**：黑甲长发金纹造型成为 2023 年 cos 圈与 AI 视频的爆款模板，"黑甲长发男 + 雪夜山顶"被海量复用。
- **白鹿一人三角**：苏苏 / 夕雾 / 桑酒三种性格反差被认为是"白鹿演技解锁"代表作，桑酒水族梦幻造型获微博美妆造型热搜。
- **陈都灵反派转型**：从校园女神到"叶冰裳 / 妺女 / 天欢"三反派的彻底转身，业内评价"看反派发疯，挺解压"。
- **敦煌风服化道破圈**：全剧 2000+ 套服装、20 层叠穿的"成神战甲"、龙鳞铠甲设计被《界面新闻》《新华网》多家媒体专题报道；"敦煌风仙侠"成为后续仙侠剧的视觉范式。
- **国际反响**：Netflix 上线首周新加坡排第二、越南 Vieon 排第一，LINE TV 同步多地区热度榜首。
- **结局争议**：BE 大结局"黎苏苏成神后亲手将澹台烬灰飞烟灭、独自带女"被部分观众评为"虎头蛇尾"、"原著党 vs 改编派"激烈对立，是该剧豆瓣评分波动的主要原因。同时也促成了"以爱殉道"的国风仙侠新审美——许多后续仙侠剧的 BE 结局直接对标本剧。
- **同人衍生**：般若浮生篇（冥夜 × 桑酒）独立衍生出《长月烬明之般若浮生》同人剧本与剧本杀，是该 IP 后续商业开发的重要支线。
