## 一句话设定

被小兰花误放出昊天塔的大魔头东方青苍，发现兰花竟是数万年前他执念所系的息山神女木槿转世；为解咒、为复活旧友、为搞清"为何看到她就心痛"，月尊与小仙女在三界打了一场绕不开的感情劫。

## 完整剧情（分卷）

### 第一卷 · 昊天塔解封（约第 1-3 集）

仙界水云天的低阶仙女小兰花，是司命上仙嗜酒时浇坏仙根的小弟子，活了一万多岁修为仍是末流。她偶然接到任务进入昊天塔——上古时期由战神赤地女子以自爆元神为代价，封印月族大军与魔尊东方青苍的天牢。塔内封印松动，小兰花指尖沾染塔身禁制时，黑色封印一道道剥落，沉睡三万年的月尊睁眼，第一次见到这个"小东西"。两人发生灵魂互换、同心咒等连串异变，小兰花被迫与月尊共享伤痛、共担生死。

### 第二卷 · 红绳契约与人间历险（约第 4-12 集）

东方青苍发现小兰花身上有他要找的"息山神女木槿"残留神识——他要靠这具神女之躯解开自己身上的诅咒、复活旧识赤地女子。他用一根红绳系住小兰花魂魄，立下"主仆契约"，把她带回月族苍盐海。两人之后又因任务下凡，混入人间江湖，演绎出大段名为"找药"实为"养小兰花"的轻喜单元——青苍亲自给她炖汤、修仙根、捏耳朵、堵她流的鼻血。"他对万物冷漠，却对她一个人格外偏心"的反差，是本剧前期口碑爆点。小兰花保留了神女的纯善天性，多次化解青苍的杀意，月尊的"七情"在她身边一寸寸长回来。

### 第三卷 · 桃花印觉醒与长珩三角（约第 13-20 集）

水云天战神长珩——天帝云中君的胞弟、真身为麋鹿的清冷仙君——一直暗中守护小兰花，他被司命告知小兰花就是他命定的"息山神女"未婚妻，但他迟迟无法点破。三人关系开始紧张：长珩礼贤温润，青苍蛮横霸道，小兰花夹在中间慢慢分得清"仰慕与感恩"和"心动"是两件事。青苍发现小兰花后颈浮现息山族的桃花印——那是当年木槿渡他飞升时种下的本源之印——前世记忆汹涌而来。"原来从一开始，要找的人就是你。"两人开始有意识地从"主仆"滑向"心仪"。

### 第四卷 · 海市与赤地女子归来（约第 21-28 集）

容昊——表面是温润仙君、内里是戴面具的"海市主"——浮出水面。他是当年赤地女子救下的盲眼孤儿弟子，对师父怀着超越师徒的执念。他在三界开海市、用神器交换激发众生贪欲、炼祟气喂凶神太岁，唯一目的是复活轮回中的赤地女子。赤地女子真身回归，但她当年自爆元神是因受天罚，每一世都要被挚爱之人背叛；她的转世谢惋卿觉醒后，与容昊、与东方青苍的旧账全部摊在桌面。月族真相也在这时揭开：东方青苍并非天生灭世魔头，他是被父亲"拔情绝爱"练业火、被命格设计成屠戮工具的孩子——所谓"屠族"另有内情，木槿当年甚至以神女之力渡过他一程。观众至此明白，所谓"魔尊救赎"的本质是"被天道选定为牺牲品的两个孩子，互相把对方救回来"。

### 第五卷 · 碎月域大决战（约第 29-34 集）

赤地女子（谢惋卿）选择再次自爆元神，以死亡终结她与东方青苍的孽缘。容昊为复师终成执念，被东方青苍亲手了结，遭天罚生生世世与赤地女子无缘。三界面临崩塌，东方青苍以月尊本源之力抵住碎月域裂缝，把小兰花以及苍生送回仙界。月族剩余被天界收编，长珩在情劫中放弃战神身份，选择云游三界做散仙。东方青苍把自己留在了黑色月光与银色裂纹的碎月域里，命簿上写着"魂飞魄散"。

### 第六卷 · 倒追碎月域与琉璃火结局（约第 35-36 集）

小兰花归位为息山神女木槿，元神再度圆满，她拒绝接受"东方青苍已不在"的结局。她追到碎月域，一袭红衣对一袭黑袍，于黑色月光下与化形将散的青苍重逢。司命暗示命簿可改：东方青苍以业火为代价，最终把"业火"炼成了"琉璃火"——从烧毁一切的诅咒之火，升级为护佑挚爱的本源之火。神女以己身渡他，命簿改写。番外里两人在苍兰殿安家，木槿怀了青苍的孩子，月尊从此学会"爱苍生，也爱一人"。

> 整剧的核心情感命题——"凭什么爱苍生就不能爱一人"——在第六卷由小兰花原话喊出，是 2022 年仙侠剧最被反复引用的台词之一。

## 角色详细描述

### 东方青苍

- **演员**：王鹤棣
- **身份**：月族月尊、苍盐海之主、三界霸主，年龄三万岁以上
- **外貌**：极长黑发（剧中多为高髻 + 全披发两种发型），剑眉、深邃丹凤眼，眼尾微挑略显冷漠；皮肤偏冷白，五官立体；标志性服饰是黑金高领大袍 / 黑红金三色礼服式月尊典藏服；战甲为繁复鎏金黑甲，需三四人协助、十几二十分钟才能穿上；战斗时常召唤业火（黑红色火焰）与紫金色雷电；中后期出现黑长披发 + 银色月族头饰造型。
- **性格**：初登场时冷酷无情、睥睨众生，因被父亲"拔情绝爱"练业火，"没有七情六欲，是个他人眼中没有感情的怪物"；霸道但不滥杀，护短到极致——他对万物漠然，却独独把小兰花放在心上；表面狂傲，内里其实是被命格选作牺牲品的孩子。
- **角色弧光**：从"为使命活着"的工具型魔尊 → 被小兰花唤醒七情 → 学会尊重 / 悲悯 / 共情 → 最终为爱打破宿命，把诅咒之业火炼成护佑挚爱的琉璃火。"爱苍生"与"爱一人"的二元在他身上完成统一。
- **标志台词 / 名场面**：
  - "小东西，你救了我。"（昊天塔解封）
  - "明明与你相识了才三个月，却好像已经相识了三万年。"
  - "这世间，所有人都恨我、怕我、仰仗我、崇拜我。唯有她，爱我。"
  - 红绳契约：以一根红绳系住小兰花魂魄，"你逃不掉了"。
  - 碎月域单膝跪地："小兰花，本座把魂还给你。"

### 小兰花 / 木槿

- **演员**：虞书欣
- **身份**：水云天司命上仙的低阶弟子；前世为息山族神女木槿
- **外貌**：长黑发常梳双环髻或半披发，发间一支桃花簪 / 绒花头饰；圆脸、大眼、樱桃唇，皮肤白皙；服装以浅绿、月白、淡粉为主，腰间垂兰花纹流苏长帔；神女觉醒后改穿一袭红衣 / 朱红披风；从"小仙女"到"息山神女"剧中共安排约 17 套造型，颜色由浅入深暗示力量觉醒。
- **性格**：天真灵动、嗜甜、易哭易笑；外显傻白甜内核却有"普世性的善良"；在被青苍威胁时仍坚持救他、不忍杀生；觉醒后变得更稳，能与神尊平视。
- **角色弧光**：低阶仙女 → 被卷入魔尊命运 → 唤醒前世神女记忆 → 主动选择"爱魔尊也爱苍生" → 倒追至碎月域救青苍。
- **标志台词 / 名场面**：
  - "大木头，你答应我，永远都不要离开我，好吗？"
  - "凭什么爱苍生就不能爱一人，我不服。"
  - 倒追碎月域，红衣乘月光剑跃入黑色月域。

### 长珩

- **演员**：张凌赫
- **身份**：水云天战神，东君幼子（天帝云中君的胞弟），真身为麋鹿
- **外貌**：长黑发束玉冠，冠侧暗藏鹿角轮廓；面如冠玉，气质清冷；服饰以月白 / 浅青 / 银线为主，衣料用夏布织就，手执檀香扇，腰悬一方苏绣兰花素帕（由江苏省工艺美术名人卢建英真制）。
- **性格**：清冷温柔、心怀大爱、内敛守礼，自幼恪守天规，被仙界倾慕；遇小兰花后才"漫长生命有了色彩"。
- **角色弧光**：恪守天规的完美战神 → 因爱抗命 → 接受"她爱的不是我" → 自请放弃战神身份云游三界做散仙，从"被命格安排的麋鹿"变成"自己选路的散仙"。
- **标志台词 / 名场面**：
  - 赠流萤石给小兰花，默默护其仙根。
  - 大结局："虽与你永世错过，但我依旧爱你如初。"

### 赤地女子（轮回身：谢惋卿）

- **演员**：郭晓婷
- **身份**：上古天界战神；为镇压月尊东方青苍，以自爆元神封印月族大军；后为救容昊逆天行事，遭天罚堕入无限轮回，每世被挚爱之人背叛
- **外貌**：上古战神造型为深红 + 黑底鎏金战袍 / 战甲，长黑发常做高马尾或大开冠髻；轮回身谢惋卿则以一袭素白、半披发，气质柔弱多舛；觉醒回赤地女子身份时改回赤红战甲。
- **性格**：身为战神时杀伐果断、孤勇悲悯；轮回中残魂化作赤灵，"喜怒无常"是天罚的副作用。
- **角色弧光**：救苍生 → 救徒弟 → 被天罚 → 轮回 → 觉醒 → 以再次自爆元神终结她与东方青苍的孽缘，是全剧最悲剧色彩的角色。
- **标志台词 / 名场面**：
  - 三生石上一笔一划刻容昊名字。
  - 觉醒后对容昊："我救你，不是要你为我活成这样。"

### 容昊（海市主）

- **演员**：徐海乔
- **身份**：表面为水云天年轻仙君，私下为戴面具的"海市主"；赤地女子昔日盲眼孤儿弟子
- **外貌**：仙君形态为浅蓝 / 雪青长袍，温润如玉；海市主形态戴半张银黑面具、一身黑衣大氅，眉眼藏在帽檐里；化形战斗时手持骨笛 / 黑色狼牙短刃。
- **性格**：温文尔雅是面具，深情执念才是内核；为复师恩"不择手段"，与魔尊行事方式异曲同工。

## 关键对白 / 名台词

> 以下台词来源以观众口碑长帖（句子控、知乎、豆瓣短评）汇总为准；具体集数为粉丝复盘的近似集，未在出现"前后"二字时以台词原句优先。

1. "小东西，你救了我。"——东方青苍 / 昊天塔解封后第一句话（约第 1 集）
2. "从现在起，你是我的，你的命属于我，你的呼吸属于我，你的心跳属于我。"——东方青苍 / 红绳契约场景（约第 5 集前后）
3. "你是什么本座一点也不在乎，本座只知道你不是无人问津的小草，你是三界之中，本座最在乎最重要的人。"——东方青苍 / 桃花印觉醒之后的剖白
4. "明明与你相识了才三个月，却好像已经相识了三万年。"——东方青苍 / 前世今生记忆叠合后的低语
5. "就算把你真身放进成千上万棵一模一样的兰花草里，本座也能一眼找到你。"——东方青苍 / 找回失踪小兰花的桥段
6. "这世间，所有人都恨我、怕我、仰仗我、崇拜我。唯有她，爱我。"——东方青苍 / 月尊独白
7. "拯救苍生是我的使命，不是你的。可我的使命，是拯救你。"——东方青苍 / 大决战前夜
8. "凭什么爱苍生就不能爱一人，我不服。"——小兰花 / 第六卷神女觉醒后的核心宣言
9. "大木头，你答应我，永远都不要离开我，好吗？" "我答应你，永远都不会离开你。"——小兰花 × 东方青苍 / 苍盐海日常
10. "大木头，虽然无法永远留在你身边，但愿你永远平安喜乐——有朋友、有兄弟、有忠诚、有人爱，再也不会孤单。"——小兰花 / 临别独白
11. "爱苍生，也爱一人。"——剧本核心命题；剧中由不同角色多次复述
12. "师父救我那天，我就该跟她一起死。"——容昊 / 海市主身份揭穿
13. "我救你，不是要你为我活成这样。"——赤地女子（觉醒后） / 对容昊
14. "命簿这种东西，是写来给人改的。"——司命星君 / 醉酒劝青苍
15. "虽与你永世错过，我依旧爱你如初。"——长珩 / 大结局放手语

> 注：第 9、第 10、第 14 句因网络流传版本不一，可能存在用字微差，统一以观众口碑长帖中复述率最高的版本为准。

## 服装与造型笔记

> 服装总设计：易小雅。剧组邀请 27 位非遗匠人，融汇 32 种非遗传统工艺、5 大非遗艺术——苏绣、漆艺、玉雕、绒花、檀香扇——头饰多采用 3D 打印（树脂等材料）实现传统造型不能完成的复杂结构。

| 角色 | 主色调 | 发型 / 发饰 | 标志道具 | 备注 |
|---|---|---|---|---|
| 东方青苍 | 黑 / 金 / 红 | 长披发 + 月族银发冠；高髻发型 | 月族战甲 / 月光剑 / 业火（黑红） / 琉璃火（金紫） | 全剧 16 套服装；黑金高领大袍是"月尊典藏"代表 |

| 木槿（前世神女） | 朱红 / 粉桃红 | 高髻 + 桃花步摇 + 流苏冠 | 息山族桃花本源、神女红绫 | 红衣 + 桃花飞瓣是 AI 仿做的高频模板 |
| 长珩 | 月白 / 浅青 / 银 | 玉冠（藏鹿角线） + 半束长发 | 檀香扇、苏绣兰花素帕、流萤石 | 夏布织衣，工艺最古典清雅 |

通用视觉语汇：
- **红绳**：贯穿全剧的契约意象，红色丝线 + 金色符文。
- **桃花瓣 / 桃花印**：息山族本源符号，落瓣 + 后颈花纹。
- **黑色月光 + 银色裂纹**：碎月域专用色板。
- **黑色锁链断裂特效**：昊天塔解封专属。
- **业火→琉璃火**：火焰从黑红 → 金紫，象征角色从诅咒到救赎。

## 文化现象

- "苍兰 CP"成为 2022 网络第一仙侠 CP；王鹤棣演艺事业由此爆发，从篮球少年人设彻底转型为顶流仙侠脸。
- 虞书欣"小兰花"贴脸演法被业界认为是仙侠"灵动派"代表，反向带动后续多部仙侠在选角时偏好"圆脸 + 大眼 + 真音演唱"配置。
- 全网二创海量，AI 视频圈最常被复刻的桥段就是"解封 + 红绳 + 桃花印"三件套。
- "爱苍生，也爱一人 / 凭什么爱苍生就不能爱一人，我不服"成为 2022 国剧十大热句之一，在抖音 / 小红书 / B 站三平台被反复使用。
- 4 亿制作成本"造东方幻境"，让国产仙侠的"卷服化道"从此成为新基线。
- 周深《余情》、詹雯婷《诀爱》、刘宇宁《寻一个你》同时登上热歌榜，本剧 OST 是当年华语影视音乐最大赢家之一。
