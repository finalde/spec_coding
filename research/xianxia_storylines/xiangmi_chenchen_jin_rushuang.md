## 一句话设定

被母亲喂下"陨丹"以断情绝爱的小花神锦觅，因好奇凡间救下天帝之子旭凤，又在天界遭遇旭凤之兄润玉的同时爱意——三方纠葛勾起亘古天魔大战。

## 完整剧情（分卷）

### 第一卷 · 花界落雪、陨丹断情

故事从四千年前一场天地变色的"花神之死"开始。先花神梓芬本应是六界第一温柔花仙，却因牵扯旧怨而难产陨落；临终前她将一颗自己性命所凝、能"断情绝爱"的"陨丹"喂入新生女儿的口中，希望女儿不识情、不知爱、不重蹈母亲的覆辙。这粒丹药从此封住了女婴口中所有的"心动"——她就是锦觅。

锦觅在被称为"花界结界"的小花海里被几位老花仙抚养长大，最常作伴的是真身为胡萝卜的散仙"老胡"。她活泼贪嘴、好奇心旺盛，却始终对外界的天界、人界一无所知。一道结界、一群花仙、一片永不凋谢的花海，是她的全部世界。

### 第二卷 · 凤鸟坠落、初识旭凤

四千年后，天界二皇子、火神旭凤一次涅槃失败，重伤跌入花界结界。锦觅本能地救下了这个浑身是火的少年，并阴差阳错变成一颗"葡萄"陪在他身边。两人因为锦觅的吃货属性、旭凤的傲娇本性，在花界打闹相处了一段日子，旭凤逐渐对这个"葡萄精"动了心。

旭凤伤愈返回天界，把锦觅一并带回。锦觅看见的天界既华丽又冷漠：恢弘的九重天、金阶玉殿、各路神仙、冷艳天后荼姚——以及那位被指婚给她的、永远微笑着的夜神润玉。润玉是天帝太微与龙鱼族公主所生的庶长子，自幼丧母、寄居璇玑宫，温润、聪慧、隐忍，几千年来在天界小心翼翼地维系自己的位置。他对锦觅一见倾心，主动提出"解除婚约"以便锦觅自由——这种近乎奉献式的好，是润玉爱情的底色。

### 第三卷 · 凡间历劫、人间副本

按照神族律法，天神成长到一定阶段必须下凡历劫。旭凤化身将军"夜神紫"，锦觅化身将军府里的小丫鬟、舞姬，与他在凡间相识、相恋。这是全剧"糖度"最高的一段，也是所谓"红尘劫副本"的扩写——他们在战乱、刺杀、宫廷算计之间相互守护，最终在一处梨花深处结成了一段人间夫妻情。

但凡间一切因果都将带回天界。陨丹在锦觅心里第一次出现裂痕，她隐约感到自己"心口空了一块"，却仍不识情为何物。同一时间，润玉在天界静静地看着这一切，他知道自己永远赶不上凤凰的火光，第一颗黑暗的种子悄悄种下。

### 第四卷 · 太微帝君的旧债与天后之恶

伴随剧情推进，太微帝君的过往被一层层揭开。原来当年花神梓芬正是他的旧爱，他为夺天帝之位戕害兄长、抛弃花神、迎娶悍妇荼姚，又对自己与龙鱼族公主所生的私生子润玉冷漠相待。润玉之母在地心冰湖被天后荼姚冰封而死——这是润玉黑化的真正源头。

天后荼姚是真正的剧中"反派之首"。她为了让亲子旭凤稳坐继承人位，连同鸟族公主穗禾一同设计：陷害水神洛霖（锦觅之父）、风神（旭凤旧友）、并把杀人的脏证一一推到旭凤头上。

### 第五卷 · 大婚刺心、陨丹破碎

剧情至此进入全剧最痛的章节。穗禾以鸟族公主身份强行嫁入凤凰殿，旭凤被迫与穗禾大婚；锦觅误信谗言，认定旭凤就是杀害父亲水神、舅父风神的凶手。在大婚当夜，她将传说中能斩断神识的"忘川水滴"幻化为一柄剑，亲手刺入旭凤的心口。

旭凤睁眼笑着接下这一剑、喝下她调的"忘情水"，留下"我不怪你"四字。剧情高潮的视觉符号在此爆发：火凤血染嫁衣、白色花瓣飘落红绸、忘川水滴化作蓝色冰刃。锦觅在旭凤倒下瞬间猛地呕出体内的陨丹，第一次"识情"——可此时，爱人已死。

### 第六卷 · 旭凤堕魔、润玉夺位

旭凤心死成魔，借魔界之力重生为新一任"魔尊"，凤翼焚天、誓要毁灭三界，尤其要对天界讨债。与此同时，润玉以"代父受过、铲除奸佞"的名义在天界发动政变，逼天帝太微让位、清算天后荼姚——他终于坐上了那把象征复仇的天帝之椅，却也彻底失去了锦觅。

剧中两位男主呈现镜像关系：旭凤外烈而内净，最终因爱成魔；润玉外柔而内冷，最终因恨登顶。两人都成了爱情的失败者。

### 第七卷 · 死劫献身、魂飞之痛

为阻止天魔大战，锦觅以"水神之力"凝出最后一击。在三界顶天的战场上，旭凤与润玉刀剑相向，锦觅纵身挡在两人之间——魔尊那一击穿过她的身体落入润玉肩头。锦觅魂飞魄散，化作旭凤眼角一滴泪。

这一段被后世剪辑无数次：白衣女子背影、红羽魔尊、银色长发夜神，三人的颜色在天空中绞在一起。锦觅"魂消"是全剧悲剧的最高点，也是国产仙侠剧"BE 美学"最早的范本之一。

### 第八卷 · 五百年后、HE 转折

时间跳跃五百年。锦觅以人界小妖之姿转世重生、不识前尘，旭凤则散尽魔气、化身一名守候在她身侧的红衣男子。剧版结局是温柔的"HE"：旭凤在她转世新婚前夕"抢亲"，重新让她记起前世种种，两人最终归隐凡间，育有一子（剧中代称"小白鹭"）。润玉则独坐九重天，成为孤独的天帝，看着六界长治久安。

剧版结局比电线原著的小说结局柔和许多——这一点在"原著党 vs 剧粉"之间形成长期争论，详见下方"衍生 / 续集"章节。

## 角色详细描述

### 锦觅 / 葡萄精（杨紫 饰）

- **真身 / 血脉**：先花神梓芬与水神洛霖之女；天界律法上属"花神后裔"，本身具备水神血脉，最终在死劫一战中觉醒水神之力。
- **核心设定**：出生时被母亲喂下"陨丹"，肉身上有一颗看不见的小丹丸卡在心口，使她对一切爱意无法识别——别人对她示爱，她只能解读为"哥哥姐姐对她好"。
- **化形**：被旭凤撞见时常以"葡萄"形态出现，因此剧粉昵称"小葡萄"。
- **性格**：天真、贪嘴、好奇、记仇但记不久；越是被压抑情感，越在每个细节里留出"裂痕"——直到刺杀旭凤的一瞬间彻底爆发。
- **角色弧光**：
  1. 花界小葡萄（无情期）；
  2. 天界宴会、人界历劫时"以为是兄妹之情"的甜虐期；
  3. 大婚刺心、识情吐丹的觉醒期；
  4. 死劫献身、魂飞期；
  5. 五百年后转世重生、被旭凤抢亲的归隐期。
- **代表镜头**：识情瞬间（陨丹自口中飞出，眼底亮起红雾）、纵身挡在旭凤润玉之间魂消的慢镜头。

### 旭凤 / 火神 / 魔尊（邓伦 饰）

- **真身 / 血脉**：天帝太微与天后荼姚之子；真身为凤凰，天界火神，原任天界二皇子、战神。
- **核心设定**：天界最具贵气的"光明之子"，骄傲、磊落、有些傲娇与口是心非。前期被天后宠溺，后期一夜失爱、堕入魔界，成为新一任魔尊。
- **武力 / 法宝**：羽剑、凤翼焚天、涅槃重生；魔尊期换为黑红长袍、玄色翼。
- **角色弧光**：
  1. 天界傲娇二皇子；
  2. 跌入花界遇到锦觅、化形葡萄期间生情；
  3. 人界将军"夜神紫"历劫期；
  4. 大婚被刺心、心死成魔的"魔尊"期；
  5. 三界大战后散去魔气、五百年后转世守候期。
- **代表镜头**：凤凰浴火涅槃（红羽展翼、金光遍野）；中剑后笑着喝下"忘情水"；魔尊登场（黑红斗篷、双瞳赤红、立于断壁之上）。

### 润玉 / 夜神 / 后任天帝（罗云熙 饰）

- **真身 / 血脉**：天帝太微与龙鱼族公主之子，真身为白龙；天界"夜神"。
- **核心设定**：温润如玉、谦谦公子的外壳下，是被嫡母（天后荼姚）冰封母亲、被父亲冷待数千年的复仇之心。整部剧最复杂、争议最大的角色，许多观众甚至把他当作"另一个男主"。
- **武力 / 法宝**：龙鳞铠、夜神之力（冰、夜、星辰）；登天帝后取得"天帝印"。
- **角色弧光**：
  1. 璇玑宫"小可怜夜神"，被嫡母与异母弟弟联手压制；
  2. 解除婚约、暗恋锦觅的"白月光"期；
  3. 与天后联手又反手清算天后的政治期；
  4. 黑化、夺位、登天帝的复仇期；
  5. 锦觅死劫后独坐九重天的孤寂期。
- **代表镜头**：童年潜入冰湖见到母尸的回忆段；登天帝后于雪夜长阶顶端独白"母后，孩儿做到了"；额头一点红痣 + 银白长发的"夜神造型"成为后续仙侠剧"病娇美人"的标杆。
- **观众反响**：润玉热度后期甚至盖过男主旭凤，"润玉粉"自称"璇玑宫"，是该剧最大的"反派抢戏"现象。

### 太微帝君 / 天帝（剧中）

- **设定**：天界最高神祇、旭凤与润玉之父。当年为夺天位，戕害兄长、抛弃旧爱花神梓芬、迎娶悍妇荼姚、对私生子润玉冷漠。
- **作用**：剧中所有"父辈罪债"的承担者；他既是旭凤被誉为"光明之子"的根源，也是润玉黑化最深的伤口。
- **结局**：被润玉以"清算前辈宿罪"为名逼宫退位，黯然让出天位。

### 天后荼姚

- **设定**：太微帝君的正宫天后、旭凤之母。强势、悍妒、护子心切。冰封润玉之母、陷害水神风神、伪造旭凤的"杀人证据"——剧中真正的"反派 BOSS"。
- **作用**：所有阴谋的核心驱动者；她对旭凤的"母爱"在剧情上反而成了害死旭凤的助力。
- **结局**：润玉登天帝后被清算，结局凄凉。

### 穗禾（王一菲 饰）

- **真身**：鸟族公主，真身孔雀。
- **设定**：旭凤指婚的政治联姻对象，外柔内刚，对旭凤痴情且偏执。剧情后段被披露为"刺杀水神风神事件"幕后黑手之一，与天后联手把脏证嫁祸到旭凤头上。
- **结局**：在锦觅刺心案真相揭穿后崩溃发疯，是剧中"反派配角"的悲剧标杆。

### 鎏英 / 卞城公主（陈钰琪 饰）

- **真身 / 阵营**：魔界卞城王之女、魔族公主。
- **设定**：六界第一"飒"的女将军气质，敢爱敢恨，与魔界暗卫暮辞 / 奇鸢相恋。
- **造型**：黑金头冠、暗紫鳞片战甲、鎏金腰封——魔族新娘装与天界新娘装形成强烈视觉对比。
- **作用**：魔界戏份的情感引擎；她与暮辞的支线 CP 是全剧"魔界最甜"的戏。
- **结局**：与暮辞成婚，暮辞为护她而陨落，她诞下女儿"卿天"，成为魔界一代女主。

### 暮辞 / 奇鸢（王仁君 饰）

- **真身 / 阵营**：灭灵族最后血脉，曾被天后荼姚收为暗卫。
- **设定**：剧中"忠诚 vs 爱情"母题的承担者；前期为天后效命刺杀风神等人，后期与鎏英重逢复爱、反向制衡天后阴谋。
- **结局**：婚后为护鎏英身亡，留下幼女卿天。

### 璇玑（夜神之妹 / 璇玑宫主）

- **设定**：润玉胞妹（一说异母妹，剧中作"璇玑公主"），与润玉同住璇玑宫，是润玉在天界唯一的亲人。
- **作用**：剧中负责"润玉柔情面"的角色——通过她，观众看见润玉黑化前后内心仍残存的家庭温度。

### 水神洛霖

- **设定**：先花神梓芬的一生挚爱，被太微帝君拆散；锦觅生父。剧情中段被阴谋杀害。
- **作用**：通过他的死亡，锦觅彻底"对天界绝望"，是大婚刺心剧情线的导火索。

### 风神

- **设定**：旭凤旧友、天界风部之首。在阴谋中与水神同时遇害。
- **作用**：让旭凤"被指为凶手"的桥梁人物。

### 火神（旭凤之父辈职位）

- **设定**：剧中"火神"职位，因旭凤血统直接由旭凤继承。
- **备注**：剧中并未显现一个"老火神"独立角色，火神一职即旭凤本人。

### 老胡

- **真身**：胡萝卜散仙，花界长大，锦觅最早的玩伴。
- **作用**：花界戏份的喜剧担当；后期负责把"花界视角"带入天魔大战。
- **梗点**："我是一根胡萝卜"是 B 站二创最常用的台词之一。

### 朱砂（剧粉爱称"风月女神"）

- **设定**：天界风月司掌管者，痴恋旭凤多年。
- **作用**：旭凤"被女配围绕"叙事的扩展，与穗禾形成"旁白对照"，凸显旭凤一直只爱锦觅。

## 服装与造型笔记

### 锦觅

- **花界期**：浅粉、浅紫、淡绿三色为主的轻纱襦裙；袖口绣花、腰间常配"葡萄"小坠；发型多为半披、点缀小花。
- **天界期**：白色长披帛 + 浅蓝色裙；少量金边，凸显"水神血脉"清冷感。
- **人界历劫期**：浅杏色襦裙 + 凡人发髻，弱化神性，强调"小丫鬟"普通感。
- **死劫前**：纯白嫁衣（与旭凤大婚时穿）；这套白衣 + 长发的视觉反复出现在魂消片段。

### 旭凤

- **天界期**：暖红主色长袍 + 金色凤纹刺绣 + 玄色腰封；发冠为金色"凤"造型。"红 + 金 + 黑"是火神的固定三色。
- **人界历劫"夜神紫"**：青灰色将军袍 + 暗银纹甲；颜色刻意压暗，与天界形成对比。
- **魔尊期**：黑红长袍 + 玄色铠甲，红色压到只剩内衫与披风缘；瞳孔变赤红，发型变张扬黑长直；这是全剧 AI 二创最高频的旭凤形象。
- **五百年后凡间**：素色直裰，重回温暖红色调，但金色被去除——象征"褪去神格"。

### 润玉

- **夜神期**：白色为主调的广袖长袍 + 银蓝缘 + 一抹紫色腰带；发色冰白长发；额头一点红色"龙鳞"印记（剧粉称"龙痣"）。
- **天帝期**：玄色 + 暗金长袍，发冠从素冠换为天帝冕旒；脸色更白、唇色更淡，强调"高处不胜寒"。
- **造型亮点**：罗云熙本人的舞蹈底子让润玉走路、转身的"长袍曳地"镜头格外有古典美感；这套白发紫鳞造型直接定义了之后多部仙侠剧（《长月烬明》等）"病娇美人"模板。

### 穗禾

- **鸟族公主期**：孔雀蓝 + 金色羽冠 + 多层飘带，肩头与裙摆做羽毛叠片造型，强调"孔雀真身"。
- **作妖后期**：色彩饱和度调高、妆容渐浓，配合角色心理失衡。

### 鎏英

- **魔族日常**：暗紫鳞片战甲、鎏金腰封、马尾束发；魔界"女将"的标准模板。
- **魔界婚装**：黑金配色头冠 + 暗纹刺绣红黑长裙，是与天界白嫁衣截然相反的视觉。

### 太微帝君 & 天后荼姚

- **太微**：玄色 + 暗金天帝冕旒长袍；后期败相后改为灰白单色，象征权位失落。
- **荼姚**：紫色 + 金色凤冠霞帔；"凤冠"刻意厚重，凸显她对凤凰之子旭凤的执念。

## AI 视频可借鉴的视觉元素

- **凤凰浴火涅槃**：暖红色羽毛风暴 + 金色背景 + 黑色长发男主，慢镜头展翼。
- **夜神龙鳞 / 额心红痣**：苍白美男 + 银白长发 + 紫色腰带 + 一点红色额痕，雪夜或月下静立。
- **花界结界**：七色花海 + 雾气漂浮 + 少女背影；适合做"无忧期"前奏画面。
- **忘川水滴 → 蓝色冰刃**：少女手心一颗水滴升空、冷光中拉成长剑；冷蓝光配红色血滴，是"刺心瞬间"的核心镜头。
- **大婚刺心**：白嫁衣 + 红凤袍 + 金色喜烛，被一柄蓝色冰剑撕开——三种颜色在 0.5 秒内对撞。
- **冰湖救母**：冰蓝水下世界 + 一具白衣女尸 + 一个穿白色少年的背影，是润玉黑化的视觉模板。
- **魔殿登顶**：黑色长阶 + 雪夜 + 单一长袍人影背身；加上独白"母后，孩儿做到了"。
- **三人决战慢镜**：白衣锦觅纵身挡在红衣魔尊与白发夜神之间，三色冲撞——"国产仙侠 BE 美学"的最高代表镜头之一。
- **五百年后凡间**：暖色滤镜、桃花 / 梨花 / 麦田，红衣男从远处缓步走来——HE 收尾标准模板。

## 文化现象

### 润玉之爆红与"罗云熙现象"

- 2018 年播出期间，润玉热度持续盖过男主旭凤；B 站、微博一度被润玉相关花絮、表情包、剪辑霸屏。罗云熙凭借润玉一角从"舞蹈演员转行的小演员"晋升一线古装男星；芭蕾出身赋予他在长袍下的姿态控制力，被认为弥补了"古装白衣少年"这一审美空缺。
- 粉丝群体自称"璇玑宫"，效仿润玉书法、绘画、跳舞特长进行二创。
- 后续多年，每当罗云熙出新仙侠剧（《长月烬明》《白发》等），媒体都会以"润玉"做对比标题，可见这一角色对其演艺路径的定义性影响。

### "润玉 vs 旭凤" CP 大战

- 剧中"灵修 CP"（旭凤×锦觅）与"玉穗 CP""玉觅 CP"（润玉相关）在播出期间形成全网讨论高潮；编剧后期为润玉加戏被"反派抢戏"舆论争议覆盖。
- 罗云熙团队曾一度"取关"剧方官博，被网友戏称"过年也不取关回来"，成为当年娱乐圈名场面。
- 杨紫凭锦觅一角再度刷新国民度；邓伦凭旭凤跻身"古偶顶流"——两人后续多次被催"再合作"，形成"灵修 CP"长尾粉丝经济。

### 收视与口碑

- 江苏卫视 / 东方卫视双台首播期间收视率"十二连冠"；爱奇艺 / 腾讯视频网络播放量同期破百亿。
- 豆瓣口碑前期 7.6，后期因"注水""支线过多"略降至 7.2；但仍被视为 2018 年仙侠剧最大赢家、《三生三世十里桃花》之后下一部"全民仙侠"。

### 演技口碑

- 杨紫 / 邓伦的"哭戏"与"识情戏"在播出期间多次登上热搜；中国日报网当年专门发文称两人哭戏"无人替代"。
- 罗云熙本人多次在采访中否认"抢戏"说法，称"每个人有自己的命运"。

## 衍生 / 续集

### 剧版结局 vs 原著结局

- **剧版**：BE→HE 转折——锦觅魂飞后五百年转世，旭凤散尽魔气、抢亲归隐人间，育有一子。润玉孤独天帝。
- **原著（电线小说）**：结局更为残酷。最大差异在润玉的人物动机——剧版的润玉对锦觅"是真爱过的"；原著中润玉对锦觅从头到尾"只有算计、没有爱"。这一点构成了"原著党 vs 剧粉"长期论战的核心。
- 剧版扩写部分包括：人界历劫"红尘劫副本"在剧中被前置到陨丹破裂前，让锦觅与旭凤多了一段完整的人间情缘；天后、穗禾、鎏英的戏份比原著更厚。

### 衍生 / 番外

- 剧方曾上线**短篇衍生剧《玉穗情深》**（一说为衍生网剧/番外），聚焦润玉的支线，但市场反响远不及正剧。
- 截至本研究文件整理时（2026-05），并无与正剧主创班底完全对应的"第二季 / 续集"立项；网传"五百年后剧情扩写""润玉番外"等多为同人创作或剧方早期宣传话术，需谨慎对待。
- 邓伦 2022 年因税务问题被列入劣迹艺人名单，此后不再公开出演新作；后续若有"香蜜续集"，旭凤角色已不可能由邓伦续演——这一点在脉脉、知乎等平台被反复讨论，是剧粉与原著粉关心的"现实层差异"。
