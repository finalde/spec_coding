## 一句话设定

灵界碧苍王、衔珠而生的凤凰沈璃，逃婚负伤跌落凡间被打回原形，被人当作"肥鸡"贩卖；凡间富商行止（即三界最后一位上古神，下凡历劫者）将其买回——一段"霸道女王 × 神隐凡商"的反向凡间历劫由此展开，背后是一场跨越三界、关乎墟天渊封印与上古因果的灭世大局。

## 完整剧情（分卷）

### 第一卷 · 逃婚坠凡（约 1–4 集）

灵界由天君主理三界秩序，碧苍王沈璃乃妖兽之王凤来与灵界二师姐琉羽所生的凤凰女君，自幼被衔珠而生、被赋"碧海苍珠"之命。她铁血善战、镇守一方，但拒绝接受天君安排的政治联姻——与拂容君（出身名门、性子风流的灵界世子）成亲。

大婚当日，沈璃于喜堂上当众悔婚，与劲敌交手负重伤后跌入凡间，灵力散尽，化作一只羽翼焦黑的小凤凰。凡间小贩误以为是只罕见"肥鸡"，将她关入竹笼带至集市贩卖。

凡间富商行云路过集市，一眼看出此鸟非凡，高价将沈璃买回宅院。沈璃白日为鸟、夜里恢复半人半妖之形，在行云的"鸡笼"里与之斗智斗气。行云外冷内热，言辞清淡却处处护短。

### 第二卷 · 凡间养鸟（约 5–11 集）

沈璃在行云身边逐渐恢复部分灵力。两人由"主仆 / 一人一鸟"的别扭相处过渡到日久生情：行云为她在后院种花、酿酒、煮粥；沈璃则替行云挡过几次匪患、教训过欺行霸市的小贩。这段凡间日常戏构成全剧"反向甜宠"的底色。

然而行云有一桩深埋的秘密——他并非凡商，而是三界最后一位上古神"行止"下凡历劫的化身，被天道授命于人世走完一段红尘劫，期满之后须回归神位、重续天秩。途中，灵界探子追迹而至，沈璃的真身被识破，凡间之约不得不止。临别夜，行云抚她焦黑的羽毛，第一次说出"我会护着你"。

### 第三卷 · 灵界归位与故旧重逢（约 12–20 集）

沈璃灵力恢复，回灵界继续担任碧苍王。其麾下副将墨方（实为前任灵尊六冥的庶子，潜伏灵界三百年）多年来默默守护、暗自钟情，但沈璃心中已为凡间那位"行云"留出位置。

行止以"灵界访客"身份再现——他是上古神，地位远高于天君，三界皆奉其为"安天定地之根"。他外表温润慵懒，实则疏离淡漠，长年遵循天道压抑情感。沈璃这才明白凡间那个被她当"行云"使唤的男人，是天地间最尊贵也最孤独的存在。

朝堂之上，沈璃身披银甲红袍登灵界大殿，对群臣说"诸位，本王回来了"——这一刻被观众称为现象级"碧苍王登朝"。两人开启"霸道女王 × 神隐忠犬"的灵界生活：沈璃在政事与征伐间斡旋，行止在她身侧默默护持。

### 第四卷 · 墟天渊与六冥之祸（约 21–32 集）

主线从"凡间甜宠"切到"灭世玄局"：上古时期，行止亲手将魔界恶神"六冥"封入"墟天渊"，以天外天连接其封印，是为三界安宁的根基。如今六冥的旧部——苻生（魑魅之王，是六冥仿照沈璃生父凤来创造出来的"替代品"）开始布局复活六冥。

苻生集齐五样开渊之物，欲借墨方之身令六冥重生。沈璃血脉中流着"碧海苍珠"，是开启墟天渊的关键。苻生以重刑拷打沈璃，将她体内的碧海苍珠剜出——剜心一场戏成为全剧最虐名场面：沈璃血色染衣、苍珠出体，行止赶到时几近迟。

墨方为护沈璃，在六冥临近现身的瞬间以肉身硬挡其致命一击身亡——三百年默默无言的副将之恋，以"非主角的牺牲"作收。

### 第五卷 · 同归与归隐（约 33–39 集 大结局）

苻生唤醒了沉睡千年的六冥与沈璃的生父凤来。为护三界苍生，行止决意将"天外天"与"墟天渊"相连——以自身神识为引，把六冥重新封死；沈璃则以本命凤凰真火做火之封印，与他同归。

最后关头，凤来代替女儿牺牲，自身化作封印之力填入墟天渊；行止虽完成封印却神识湮灭，魂魄飘散于天外。沈璃将行止的神躯安置于雪祭殿，日复一日守在身边。终于，三界众神之力合一，将行止以"雪妖"之身复活——非神非仙，亦无天道之缚。

剧终：三界恢复秩序，沈璃辞去碧苍王位，与化身雪妖的行止隐于凡间小院，种花、酿酒、看人间四时，长相厮守。其余配对皆得圆满，唯墨方独留观众心头一道意难平。

*（以上分卷依据公开剧情简介与剧评整理；具体集数为大致区段，未完全逐集核对。）*

### 主线脉络一句话提炼

- **A 线（情感）**：沈璃 ↔ 行止 = 凡间相识 → 灵界重逢 → 真身揭露 → 同生共死 → 凡间归隐
- **B 线（封印）**：行止上古封六冥 → 千年封印松动 → 苻生开渊 → 沈璃血脉成关键 → 行止以神识续封
- **C 线（血脉）**：凤来 ↔ 琉羽前缘 → 沈璃身负碧海苍珠 → 父女在墟天渊达成代际和解
- **D 线（副将之恋）**：墨方默护沈璃三百年 → 因六冥之子身份的内在矛盾 → 以肉身代死殉爱
- **E 线（朝堂）**：天君赐婚 → 拂容君退让 → 灵界政治格局转向"沈璃 + 行止"双中心

## 角色详细描述

### 1. 沈璃 / 碧苍王 / 凤凰女君

- **演员**：赵丽颖
- **身份**：灵界碧苍王、凤凰族女君；妖兽之王凤来与灵界二师姐琉羽之女；衔"碧海苍珠"而生
- **设定**：少年从军、铁血善战，是灵界少有的女性大将。霸气外露但内心赤诚；不谙情爱，初见行云时只把他当一个"啰嗦凡人"
- **情感弧线**：拒婚 → 落凡为鸟 → 与行云日久生情 → 灵界重逢直面其真身 → 为护行止以本命真火封印自身 → 大结局凡间归隐
- **战斗形象**：手持银枪"煊赤"，银甲红袍，骑战凤；杀阵时凤凰原形展翅，赤羽掠空
- **观众记忆点**：登朝那句"诸位，本王回来了"；剜心一场的血染白裙；凡间"被卖作鸡"的反差萌

### 2. 行止 / 行云 / 上古神

- **演员**：林更新
- **身份**：天地间最后一位上古神，三界奉其为"安天定地之根"；下凡化身名"行云"的凡间富商
- **设定**：表面气质温润慵懒，实则疏离淡漠，长期遵循天道规则压抑情感；偶有狡黠与"癫"态。地位高于天君，但行事低调
- **情感弧线**：被天道困局束缚 → 凡间偶遇沈璃 → 第一次为一个人破规矩 → 真身揭露后承担起"以身代劫"的因果 → 神识湮灭 → 众神之力以雪妖之身复活
- **造型形象**：上古神真身——白衣金边长袍、长发束冠、浮空、月色为背景；凡间——青衣白裳布衣富商；癫态化身——发丝散乱、神色游离
- **代表场景**：拔钉子、月夜悬空、墟天渊以神识为引

### 3. 墨方

- **演员**：辛云来
- **身份**：上任灵尊六冥的庶子，自幼以人形潜伏灵界，担任沈璃麾下副将三百年
- **设定**：寡言、忠厚、武力高强；藏着不可告人的血脉秘密；爱沈璃却始终不言
- **情感弧线**：发现自己越来越无法做"看客" → 协助查苻生 → 大结局以肉身硬接六冥致命一击牺牲
- **观众记忆点**："五对 CP 圆满，只有墨方受伤"；三百年沉默副将之恋

### 4. 拂容君

- **演员**：何与
- **身份**：灵界世家公子；曾被天君赐婚于沈璃，被沈璃当众悔婚
- **设定**：公子习气、风流不羁，初看是花花世子；后期态度转向理性大度，与沈璃由婚约阴影变为同袍战友
- **作用**：第一卷的"导火索"角色，亦在后期灵界政变中担任正面助力

### 5. 幽兰

- **演员**：李嘉琦
- **身份**：沈璃身边的贴身侍女 / 副官（凤凰族系侍）
- **设定**：娇俏、护主、嘴碎；负责剧中相当数量的喜剧节奏
- **作用**：沈璃在凡间和灵界之间的"翻译者"——既能解她的灵界包袱，也能接住她的凡间笨拙

### 6. 沈木月

- **演员**：曾黎
- **身份**：沈璃在灵界的长辈 / 守护神级人物
- **设定**：端庄、沉静、对沈璃既严厉又疼爱；持灵界古礼派立场，与沈璃自由婚恋观一度形成冲突，但最终选择尊重侄女的选择

### 7. 琉羽

- **演员**：宣璐
- **身份**：沈璃生母、灵界二师姐
- **设定**：与妖兽之王凤来跨族相恋的传奇女子；早逝；作为剧情前史角色多次以闪回 / 神识对话方式登场，是解释"沈璃为何身负碧海苍珠"的关键人物

### 8. 凤来

## 关键场景的台词功能分析

剧中部分台词承担"主题密度"——每一句都是后续情节的伏笔或回环。简要分析如下（无逐字台词转录，仅做剧作功能注解）：

- **"我会护着你"（行止 / 凡间临别）**：第一次出现于第一卷末。后续在灵界重逢、剜心、墟天渊封印三个节点反复回响——把"护"字从凡间情话升级为以神识为代价的承诺
- **"诸位，本王回来了"（沈璃 / 灵界登朝）**：开场短促有力。该句之所以成名场面，是因为它承担了从"凡间小肥鸡"到"碧苍王"的角色身份切换——同一个角色、两种相反的权力位
- **"凤凰一生只认一人"**：是凤凰族族训的角色化——把女主的"专一"从个人选择变成血脉本能
- **"与你同归，是我能想到最仁慈的结局"（行止 / 墟天渊前）**：把全剧从"灭世封印"的崇高叙事拉回"两人之事"的私语——封印的崇高性反衬出私语的稀缺性
- **拔钉子无台词长镜头**：编剧团队主动放弃台词，让光影、肢体、眼神替台词说话——是该剧少有的"反语言"高潮

## 服装与造型笔记

### 沈璃 · 三种主造型

- **碧苍王战甲**：银白鳞片胸甲 + 红色披风 + 金线凤纹束腰 + 高马尾束发 + 银枪"煊赤"。整体冷银 + 暗红配色，强调女将军的肃杀与压迫感；登朝、点兵、御敌时使用。
- **灵界常服**：高领广袖 + 暗红或玄色绣金长袍 + 凤凰金钗。色调取自敦煌壁画暖橙 / 暗朱体系，与灵界整体美学一致。
- **凡间布衣**：布料粗麻、青灰或浅赭色短襦 + 帔风披肩；有时与"焦黑小凤凰"形态切换出现，反差极大。

### 行止 · 三种主造型

- **上古神真身**：长发未束、白衣广袖 + 金线祥云纹 + 浮空背光；常出现于月夜、星河等神性场景。线条以飘逸为主，几乎不见武器。
- **凡间富商行云**：深青或月白长衫 + 玉带 + 折扇；眉目温润、姿态懒散，与上古神真身的"金白"形成视觉互文。
- **癫态 / 神识涣散态**：发丝凌乱、衣襟微敞、眼神空茫；用于行止神识受损或为护沈璃几近燃尽自身时。

### 群像造型

- **墨方**：玄色武将袍 + 银纹护腕，整体偏暗，对应其"暗中守护三百年"的身份
- **拂容君**：浅碧、银白系世家公子袍 + 玉冠
- **幽兰**：粉橙、藕荷少女襦裙 + 凤羽小簪
- **苻生**：黑紫长袍 + 金属面具 / 妆容偏阴鸷，魑魅之王的视觉符号化

### 场景美学

- **灵界**：参考重庆洪崖洞地形（多层悬崖楼阁堆叠），美术参考敦煌色调（暖橙、深绛、苍青三色为主），不走"白雾仙山"的标准化仙侠路线
- **凡间**：江南水乡 + 烟火集市 + 紫藤架小院，色调偏暖
- **墟天渊**：暗黑石壁、深红血纹、悬浮残骸，是全剧视觉最压抑的封闭空间
- **天外天**：悬浮岛屿 + 银白星河 + 冷蓝色调，与上古神身份相称

## 文化现象

### 1. 赵丽颖的"二度回归古装顶流"

继 2018《知否知否应是绿肥红瘦》之后，赵丽颖近年重心多落在现代戏与文艺片，一度被认为退出古装赛道。《与凤行》是她首次担任监制 + 古装大女主双重角色的回归之作，开播后多次登顶腾讯热度榜与微博热搜，被舆论视为"赵丽颖二度回归古装顶流"的标志性作品。"碧苍王登朝"片段在短视频平台传播极广。

### 2. 林更新意外的"封神场"

林更新此前以现代角色和综艺人设为大众熟知，本剧中其"上古神 / 行止"的清贵气质与"癫君"反差表演获得高度好评，被剪辑成"林更新封神场"系列在 B 站、抖音流传。"行止拔钉子"和"月下浮空"是其代表镜头。

### 3. 凤凰浴火与女性叙事

剧中"凤凰女君"的设定 + 沈璃以本命真火换行止寿元的情节，被视为近年仙侠剧"女性主动叙事"的代表案例：女主不止是被救者，反而是数次以自身为赌注守护三界与男主的人。对应当代女性观众的审美变化。

### 4. OST 走红与传统文化破圈

本剧 OST 阵容（张碧晨、刘宇宁、希林娜依·高、李佳薇、王赫野）在年度国剧 OST 榜上排名靠前；《如初》《世世》两曲在各大音乐平台长期高位。音乐总监杨秉音在媒体采访中提到本剧配乐"以传统民乐器为骨、流行编曲为皮"，这种"东方美学 + 现代编曲"的范式获得行业奖项肯定。

### 5. 美术体系的"非标仙侠"

灵界美术取材于重庆洪崖洞 + 敦煌色调，跳脱出仙侠剧惯用的"白雾仙山"模板，被业内视作国产仙侠剧的一次美术突破。"暖橙朱赤 + 苍青敦煌"配色后续被多个仙侠项目效仿。

### 6. "墨方意难平"

非主角的副将墨方在大结局中牺牲，成为该剧最大讨论点之一，社交平台出现大量"五对 CP 圆满，唯墨方受伤"的同人文与短视频，反向推高了角色辛云来的人气。

`凤凰女主` `女王 × 凡商` `逃婚` `凡间历劫` `小肥鸡` `上古神` `女主监制 / 大女主` `银甲战神` `碧苍王` `墟天渊` `剜心` `凤凰浴火` `政治联姻` `日久生情` `神隐者` `凡间种花酿酒` `非标仙侠美学` `敦煌配色` `HE`

## 与同类作品的差异

- **不靠虐为底**：相比《三生三世》《香蜜》以"虐"立 IP，《与凤行》主线为"反向甜宠 + 灭世主线"双轨，纯虐桥段不多
- **大女主权力差**：沈璃的"碧苍王"地位与本剧"凡商→上古神"的男主身份反差极大，是新世代仙侠"女上男下 / 女救男"的代表样本
- **凡间日常戏占比高**：种花、采药、煮酒，弱化"灭世大战"前期占比，强化人间烟火
- **主创女性向控制**：赵丽颖任监制是该剧能保留大量"小肥鸡 + 反向凡间历劫"的关键
- **美术非标**：洪崖洞 + 敦煌路线打破仙侠"白雾"惯例

## 主题分析

### "行止"的象征结构

男主名"行止"二字本身承担了主题——"行"为动、"止"为静，对应他作为天地秩序根基所必须遵循的"天道"。下凡化身"行云"则补全了第三个汉字层级：云行无定、终须散。剧中多次以"止于行"的转写呼应——例如插曲《不知返》中将这两字嵌入歌词。结尾以"雪妖"复活的安排，恰恰是把他从"行止"二字所代表的天道桎梏中放出——非神非仙、自由长存。

### "凤"与"碧"的双轨

女主血脉中既有妖兽之王凤来的"凤"（火属性、攻击性），也有灵界二师姐琉羽的"碧"（碧海苍珠 / 水属性、封印性）。她的设定本身就是一组对立——既是兵刃也是封印；既是焰也是壁。"碧苍王"这个称号正是把双轨揉成一体的产物——"碧"取自苍珠、"苍"取自妖兽血脉、"王"取自军政地位。

### "肥鸡"作为去神化的喜剧装置

凤凰在传统仙侠语境中是仅次于龙的尊贵图腾，而剧集开场直接把女主打成"焦黑肥鸡"贩卖于集市——这是对仙侠类型"无菜不仙"惯性的一次去神化反叛。该装置同时承担了三重功能：（1）让女主从一开始就处于权力低位，反衬其后期登朝的爽感；（2）让男主对她的喜爱建立在"知道她是神也仍当人对待"的人格层面；（3）以喜剧节奏稀释了仙侠剧惯有的厚重感。

### "上古神历劫"与"凡心觉醒"

上古神下凡历劫是仙侠惯用框架（参见《三生三世》《琉璃》），但本剧的差异在于：（a）历劫不再是"忘记前尘"的失忆装置，而是"主动脱去神格、亲历凡情"的伦理选择；（b）历劫期满后行止并不"回归大局"，反而以神识自毁的方式完成最后一次封印——历劫的终点不是回到神位，而是把"神"这个身份本身让渡给爱人；（c）大结局的"雪妖"复活进一步强化——他不再回到上古神位，而是接受了一个介于人神之间的低位身份。

### 大女主叙事的内置矛盾及其解法

碧苍王是灵界军政顶层，行止是上古神顶层——常规仙侠剧常因双顶层而陷入"谁救谁"的叙事僵局。本剧的解法是把救援动作分成情感与神性两条：沈璃以本命真火作"火之封印"承担情感救赎；行止以神识作"渊之封印"承担神性救赎。两人互为不可替代项，从而绕过了大女主叙事中常见的"男主功能弱化"陷阱。

## 改编与原著对照

- **原著**：九鹭非香《本王在此》是连载于晋江文学城的短篇，体量较短，主线集中于沈璃与行止凡间相识 + 灵界重逢的"双线相认"
- **剧版扩展**：编剧团队在原著基础上把后半段大幅扩写——加入墟天渊封印线、苻生 / 六冥反派线、墨方 / 拂容君副线，把短篇 IP 撑成 39 集体量
- **角色名调整**：剧版将原著中部分配角名做了调整（例如本研究文档第一版所记的"行渊 / 裴恕 / 菡萏 / 白英"等很可能是原著或前期物料中的旧名；剧集最终采用"墨方 / 拂容君 / 幽兰 / 沈木月"）
- **结局保留**：男女主 HE + 凡间归隐的核心闭环未变；牺牲配角（墨方）则是剧版扩写带来的新增牺牲点
- **女性叙事强化**：赵丽颖任监制后，剧版在"沈璃主动选择"上做了多处加戏（例如拒婚的处理方式、登朝时的身体姿态、剜心戏中眼神的处理），加大女性主体性

## 制作与主创笔记

### 监制 + 主演双重身份

本剧是赵丽颖首度担任监制。她在公开访谈中表达过两条创作意图：（1）让女主"在凡间的弱"和"在灵界的强"形成同等戏份的对照；（2）保留原著中那只"被卖作鸡"的反差萌设定不被仙侠类型惯例稀释。这两条都被剧版完整保留——前者反映在凡间养鸟段落的体量上，后者反映在第 1–2 集集市买鸡桥段的镜头时长上。

### 导演与编剧

- 邓科：以《赘婿》《一年一度喜剧大赛》之类的喜剧节奏类型见长。本剧前半段大量轻喜剧节奏（行云怼沈璃、沈璃为鸟时的一系列内心戏字幕、菡萏 / 幽兰类配角的喜剧贡献）很难脱离他的导演风格
- 九鹭非香：原著作者亲自参与剧本，是国产仙侠剧中较少见的"原作者深度参与"模式

### 美术与摄影

- 美术团队公开提到灵界设计参考重庆洪崖洞地形 + 敦煌色调
- 摄影上多用低饱和、暖色调；夜景大量使用单点光源（烛、月、灯笼）
- 战斗场面的慢动作 + 风扬披风是延续国产仙侠近年通用语法
- 凡间段落则反向使用日光 + 自然光 + 绿植，与灵界形成视觉对照

### 演员表演的两个亮点

- 赵丽颖：成功完成了"少女态 / 王者态 / 鸟态" 三种身体语言的切换
- 林更新：以往多被批评"表演单一"，本剧中"温润 / 神性 / 癫态"三种状态打开了新维度，是其演员评价的一次重要翻盘

## 受众反响与平台数据

- **腾讯视频**：开播首周即破腾讯热度纪录，全程稳居该平台年度热度前列

## 同人与二创生态

- "墨方意难平"：B 站 / 抖音上墨方相关二创视频数量在剧集结束后持续上升，是该剧最大的"非主角向"二创母题
- "肥鸡沈璃"：表情包 / 头像衍生数量极多，是仙侠剧少见的"喜剧二创"母题
- "凤凰浴火"：抖音特效模板化——大量普通用户用滤镜复刻沈璃凤凰真火片段
- "白衣行止"：林更新角色被剪辑成"上古神 BGM 集锦"系列，反向带动其后续古装项目

## 仙侠类型谱系中的位置

### 与"上古神 + 历劫"系作品的对照

| 作品 | 男主神格 | 历劫机制 | 大女主程度 | 凡间戏占比 |
|---|---|---|---|---|
| 《三生三世十里桃花》 | 天族太子 | 失忆历凡情劫 | 中 | 中 |
| 《香蜜沉沉烬如霜》 | 天族二皇子 | 失忆历劫 | 中 | 低 |
| 《琉璃》 | 天界战神 | 转世历情劫 | 中高 | 中 |
| 《沉香如屑》 | 天族储君 | 失忆历劫 | 中 | 低 |
| 《与凤行》 | 上古神（高于天君） | 主动下凡 + 不失忆 | 高 | 高 |

差异化要点：
- 《与凤行》是该谱系中少有的"男主主动选择历劫且不失忆"的作品，避免了同类作品中"失忆 + 重新爱上"的套路依赖
- 凡间戏占比明显高于同类，把"养鸟"段落写成全剧底色

### 与"凤凰女主"系作品的对照

近年的"凤凰女主"作品包括《思美人》《狐妖小红娘》等，但《与凤行》在两点上做了差异化：
- 凤凰不再是"高贵宿命"的符号，而是先被打成"焦黑肥鸡"再爬回王位
- 凤凰真火不再只是攻击工具，而是承担"封印"功能——即同一种力量同时具备攻击与守护双重属性

### 仙侠剧的女性主体性谱系

把女主从"被救"到"主动救"的谱系排列：
1. 《花千骨》：完全被救型
2. 《三生三世》：互救型
3. 《香蜜》：被救为主、关键时刻反救
4. 《琉璃》：互救 + 最终女主救男主
5. 《与凤行》：主动救援 + 主动让渡身份

## 适合改造为短视频的章节排序

如果以"先吸引、后留存、再转化"的短视频运营逻辑选材，建议如下顺序：

1. **第 1 集集市买鸡（吸引）**：3 秒钩子 = 焦黑小鸟眼神 + 集市叫卖声
2. **拔钉子（留存）**：完整长镜头需保留，无字幕浮现，让画面自己说话
3. **碧苍王登朝（转化）**：视觉冲击最强、最易出爆款封面
4. **真火换寿（情绪高潮）**：适合配合 OST 副歌
5. **大结局凡间归隐（情感收束）**：紫藤花架 + 一鸟一人 + 暖光

## 与原著标题"本王在此"的关系

原著标题《本王在此》四字直接化用了女主登朝场面的姿态语——"本王"是女主自称，"在此"是亮明权位的姿态。剧版将标题改为更诗意的"与凤行"，把着眼点从"女主权位"扩展到"两人同行"，与剧集后半段"同生共死、同归凡间"的主题闭环更呼应。剧名的字面拆解 = 与（介词，与谁）+ 凤（沈璃的本相 / 身份）+ 行（行止的名字 / 同行的动作）——即"与凤同行"也是"与凤行止"——一种巧妙的双关。
