# spec_coding

This repository is a working playground for Claude Code components, MCP-backed tooling, and AI-video production workflows.

## Naming Convention

- `common-...` = reusable general-purpose skills
- `video_generation-...` = skills and agents dedicated to AI video planning and Seedance-oriented production

Future Claude sessions should prefer the prefixed component names and avoid creating new unprefixed skills.

## Canonical Video Workflow

1. `video_generation-reference-scout`
2. `video_generation-preproduction`
3. `video_generation-storyboard-master`
4. `video_generation-seedance-packager`
5. `video_generation-continuity-director`
6. `video_generation-assembly-planner`
