# 光线 / 色调大师（light master）

审查并**直接修改** `ai_videos/{name}/` 每个 shot 的**光线与色调**：灯光贴不贴情绪、同场景换镜颜色一不一致、有没有对齐锁定色名、回忆镜能不能一眼分辨、眼睛有没有被误写成发光。发现不合理 → 当场 surgical 改 `光线:`（必要时 `渲染样式:`），不改剧情/台词/动作/运镜。

本 skill 承接原 `ai_videos__台词大师` 硬约束里的"眼睛不发光""回忆镜可辨"+ `ai_video.md` 零 hex 色彩锁定。由 `ai_videos__审查总编排` 在单镜层调用（动作之后，单镜层收尾）。

## 何时用
- 用户说"光线/色调不对""灯光不贴情绪""同场景忽冷忽暖""回忆镜分不清""眼睛发光/假红光""出现 hex/色彩没锁定"。
- `ai_videos__审查总编排` 单镜层第 5 道工序。

## 输入
- 范围：某 shot / 某集 / 整剧（场景一致性须按场景把同景镜排序）。
- 必读：被审 shot 的 `光线:`/`渲染样式:`、同场景其他镜、`scenes/{name}.md` + 角色 bible 的锁定色名、`style_guide.md` 调色/情绪光表。

## 审查标准

| # | 准则 | 不合格信号 | 怎么改 |
|---|---|---|---|
| L1 | **光线随情绪** | 灯光与此刻情绪/戏剧点脱节（压抑场却均匀亮、爆发点无强光对比） | 按情绪给光（压抑=冷调低饱和柔侧光、爆发=高反差硬光/逆光描边），参 `style_guide` 情绪光表 |
| L2 | **同场景跨镜一致** | 同一场景换镜色温/光向忽冷忽暖、主光源跳变 | 锁定该场景主光源/色温/光向，所有同场景镜对齐（除非剧情有动机的光变） |
| L3 | **色彩锁定·零 hex** | 出现颜色 hex 码；或人物服装/场景主色与锁定色名漂移 | 删 hex 改自然中文色名；服装/场景色对齐角色 bible/scenes 锁定色名 |
| L4 | **回忆镜一眼可辨** | 回忆镜与当下无视觉区分（观众分不清，shot11/12 教训） | `光线` 用"强回忆滤镜：做旧泛黄+低饱和暖褐+四角暗角+轻微胶片颗粒+柔光朦胧+边缘微虚"；`渲染样式` 加"回忆段做旧泛黄胶片质感"；同一段回忆滤镜彼此一致 |
| L5 | **眼睛不发光** | `光线`/`动作` 写"瞳泛金/瞳孔发光/瞳心泛金"，模型渲成廉价有色光斑（假红光，shot04/06/13 教训） | 删发光；能量入体改"眉心一缕微光内敛"，眼变化靠表演；`光线` 末尾补"眼里不加任何发光特效/有色光斑（不要红光、不要瞳孔发光）" |

## 工作流
1. 读 shot 的 `光线:`/`渲染样式:` + 同场景镜 + 锁定色名 + style_guide 情绪光表。
2. 过 L1–L5，列不合格项。
3. **当场 surgical 改** `光线:`（必要时 `渲染样式:`）：贴情绪、统一场景光、锁色名删 hex、回忆滤镜、删眼发光。
4. 复验：光贴情绪、同场景一致、零 hex、回忆可辨、无眼发光。

## 硬约束
- 只动**光线/色调/色彩锁定/渲染样式里的光色**，不改剧情（`剧情连贯`）、台词（`台词大师`）、动作（`动作表演`）、运镜（`运镜`）。
- 零 hex 是硬线（与 `格式契约` K9 一致；本 skill 从审美角度复核色名是否贴锁定，contract 只查有无 hex）。
- 审计写 `.audit/.../events.jsonl` 的 `validation.issue.raised`（level=light）。

- inline patch 清单 + 已落地改动 + 一句话复验结论。
