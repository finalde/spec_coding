# 武打设计大师（martial-arts combat master）

把 `ai_videos/{name}/` 里每场**打斗**设计/审查成**好看、有打击感、能传战力**的武打——并在「藏锋无外放」这类约束下，**不靠金光威能特效**也打出高手感。发现打戏空泛/无打击感/战力不清/乱用特效 → 当场把打斗写进 shot 的 `动作:`（配合 `镜头:`），三处同步。

**为什么单列**：动作表演管一般动作/表演/微表情/反应；**打戏是专门手艺**——招式、攻防回合、命中反馈、战力可视化、武打节奏、特效边界，是仙侠剧的核心卖点，需专项。本剧更有「藏锋无外放」铁律，打戏不能靠金光，难度更高，必须专项设计。

## 何时用
- 用户说：打戏/武打/动作戏/对打/出招/招式/打得不好看/没打击感/怎么体现战力/仙侠特效怎么做。
- 某 shot 含打斗（出手/交锋/受击/制敌）要设计或审查。
- `审查总编排` 单镜层、动作表演之后、运镜之前的打戏专项工序。

## 输入
- 范围：含打斗的 shot / 整集打戏。
- 必读：被审 shot 的 `情节:`/`动作:`/`镜头:`/`走位:` + 角色 bible（**修为/战力定位**、招式风格、藏锋约束）+ world.md（力量体系/外放规则）+ ai_video.md（平台合规·12.10/2026-06-18 审核 amendment）。

## 武打审查标准（每场打斗·逐拍过）

| # | 准则 | 不合格信号 | 怎么改 |
|---|---|---|---|
| W1 | **招式具体（不空泛）** | "打了一架 / 几下放倒 / 一顿拳脚 / 身手不凡"——没具体动作，模型出乱挥乱打 | 写**具体招式动作 + 时间轴**：侧身avoid、扣腕、错步、顶膝、沉肘、借力一摔、扫堂腿、点穴制动…（贴角色风格）；每拍 `起-止s {谁}{什么招}→{对方什么反应}` |
| W2 | **攻防回合·有来有往** | 单方面表演（一个人耍、对方干站挨打）；没有"谁先攻、对方怎么应" | 写成**回合**：A 出招→B 格/闪/卸/中→反击；至少 1–2 个攻防来回，敌我都有动作；群战写清主角逐个/连环处理 |
| W3 | **受击反馈·打击感** | 命中却没反应（打人像挠痒）；无物理后果 | 命中必给**物理反馈**：闷哼、踉跄后退、捂住中招处、单膝跪地、摔翻扬尘、衣袂/发丝一震、围观惊呼；打击瞬间可配顿帧/近景强化"疼"和"力" |
| W4 | **战力可视化·分级** | 看不出谁强谁弱、主角什么水平；战力忽高忽低 | 用**招式效率传战力**贴 bible 修为：弱角色挨打/笨拙；主角按设定（如刚觉醒：能制地痞但显吃力/挨一记、绝不无敌）；真高手则一招制敌、对方反应不及。战力前后一致 |
| W5 | **节奏 + 镜头配合** | 一速到底、平、不抓人 | 武打 beats **快慢结合**：起手蓄势(慢)→交锋(快·连击)→定胜(顿/慢镜)；配合 `镜头:`（交锋跟拍/快切、命中瞬间近景或顿帧、定胜定格）——打击感七分靠剪辑+定格。**镜头细节归运镜、此处只标"此拍宜近景/顿帧/跟拍"提示** |
| W6 | **特效边界·藏锋铁律（本剧核心）** | 不分剧情乱加金光/剑气/能量波/瞳孔发光/威能外放；或反过来"无外放"就把打戏拍得软绵无力 | **按本剧力量外放规则**：藏锋无外放剧——高手感靠**真·身手(快准巧)+劲力(沉、爆发)+物理副产(风压鼓衣、地面扬尘、碎石、气浪压草)+对手反应(被一股劲掀退)**，**绝不金光/瞳光/剑气/能量特效/威能光效**；外放剧才按其设定加特效。无外放≠没力量感——靠速度/精准/力反馈/对手狼狈打出"这人不简单" |
| W7 | **平台合规·安全（防拒审）** | 喷血/断肢/血腥渲染、刀剑刃口怼脸特写、真实军事符号——Seedance/Kling 视频审核高拒率（per ai_video.md 2026-06-18） | 打斗**点到为止、暴力含蓄**：不喷血断肢（最多嘴角一道血/闷哼）、兵器作远景虚化不怼刃口特写、招式以擒拿摔打卸力为主；血腥/凶器特写一律避 |
| W8 | **群战杂兵差异化·无 ref 也不许同脸（2026-06-20 教训：shot07 两喽啰同时入镜长得一个样）** | 群战有多个**无参考图的杂兵/喽啰/士兵**，prompt 只笼统写"2 喽啰/几个手下扑上"，没给各自外形——模型把同类群演渲成**同一张脸同一身形（复制人/双胞胎）**，廉价穿帮 | 每个同框无 ref 杂兵**各给差异化外形描述符**（身高胖瘦+衣色+发须+脸型+特征：甲＝瘦高尖脸鼠须、乙＝矮壮圆脸横肉带疤），在 `角色识别`+`走位`+`动作` 里**点名区分**（"瘦高喽啰甲""矮壮喽啰乙"分别做什么），并注"无 ref·自行生成·两人明显不同、绝不同脸同身形、不与有 ref 主将串脸"；`负面词` 补"喽啰同脸/群演复制人/双胞胎喽啰" |

## 「藏锋无外放」打高手感工具箱（本剧专用·W6 落地）
不靠金光，靠这些把"这人有功夫"打出来：
- **速度差**：对手看不清、主角已欺身近前 / 后发先至。
- **精准制敌**：一扣一带一沉，正中要害关节，对手瞬间脱力跪倒。
- **借力卸力**：四两拨千斤、顺势一摔，省力却利落（显内行）。
- **物理副产（非威能）**：出手带起风压鼓动衣袂、脚下碾出尘土、落地震得草石微动、一掌劲力把人**掀退**（是物理冲击不是能量波）。
- **对手狼狈 + 旁人反应**：被收拾者踉跄惊骇、围观倒抽气——侧面烘托。
- **克制的爆发**：大部分轻描淡写，关键一下突然加速加力（藏锋偶露）。

## 工作流（design → 写进 shot → 同步）
1. 读打斗 shot 的 `情节/动作/镜头/走位` + 角色 bible 战力定位 + 力量外放规则 + 平台合规。
2. **逐拍过 W1–W8**，列不合格项（cite shot/拍/准则）。
3. **把打斗写进 `动作:`**（具体招式+攻防回合+受击反馈+时间轴），并在该拍标镜头配合提示（近景/顿帧/跟拍，细节交 运镜）；战力贴 bible、特效守外放规则、暴力守合规。
4. **三处同步**：shot `动作:`(/`镜头:`提示)、`script.md` 画面动作、必要时 `情节:`。
5. 复验：招式具体、攻防有来回、命中有反馈、战力清晰一致、节奏快慢、特效守藏锋、暴力合规。

## 硬约束
- 只管**打斗设计**：不改台词（归 台词大师）、不定最终景别运镜（归 运镜、此处只给配合提示）、不裁时长（归 时长节奏，但打戏 beats 须能塞进 shot 时长、不够则提请加时长/拆镜）、不改朝向走位（归 站位朝向，但攻防相对位要和走位自洽）。
- **守本剧力量外放规则**：藏锋无外放剧严禁金光/瞳光/剑气/能量/威能特效（与 光线色调"眼不发光"、c1"藏锋铁律"一致）。
- **守平台合规**：不血腥、不凶器刃口特写（per ai_video.md 12.10 / 2026-06-18 审核 amendment）。
- 战力分级须与角色 bible + 大纲实力线一致，不擅自拔高/压低。

- inline patch 清单（shot / 拍 / 准则 / 武打改写后）+ 已落地 `动作:` + 一句话复验结论。
