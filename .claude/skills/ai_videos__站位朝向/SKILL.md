# 站位 / 朝向 / 走位大师（blocking master）

审查并**直接修改** `ai_videos/{name}/` 每个 shot 内的**空间逻辑**：谁站哪、面朝谁、视线落在哪、怎么走位。视频模型无跨镜记忆、不会自动推断说话人该看谁——**朝向必须在 prompt 里写死**，否则人物会正对镜头自言自语、或 A 背对 B 说话。发现不合理 → 当场 surgical 改（只改该 shot 的 `走位:`/`动作:`/`台词:`，不动剧情措辞），并三处一致。

本 skill 承接原 `ai_videos__台词大师` 的 C 系列准则（站位朝向走位），与 `ai_video.md` 12.4-D 的 S 系列空间项对齐。由 `ai_videos__审查总编排` 在单镜层调用。

## 何时用
- 用户说"站位/朝向/走位不对""对着空气说话""背对着说话""谁看谁没写清""内心独白眼神乱飘"。
- `ai_videos__审查总编排` 单镜层第 2 道工序（紧接台词、先于运镜）。

## 输入
- 范围：某 shot / 某集 / 整剧。
- 必读：被审 `shots/shotNN.md`（`走位:`/`动作:`/`台词:` 三字段）、角色 bible（体型/身份决定俯仰）、`scenes/{name}.md`（场景方位/机位）。

## 审查标准（每句在画对白逐条过）

| # | 准则 | 不合格信号 | 怎么改 |
|---|---|---|---|
| C1 | **说话人朝向写死** | `走位:`/`动作:` 没写说话人面朝哪、头/身朝向，模型让人物正对镜头或随机朝向，像对空气说 | 写明说话人此刻**身朝/面朝/视线方向**（朝跪着的兄长俯视、转向东侧王妃、回头望镜外画外人…） |
| C2 | **对谁说=视线落点** | 多人同框却没写"A 看着 B 说"，眼神涣散无交流 | 每句在画对白写 `{说话人}面向/看着{对象}说`；对象在画→眼神落其身/脸，对象画外→标明朝画外某方向。⚠ **慎用「偏头」**——模型常把"偏头"渲染成"转头背对对方"（shot09 教训：主角背对弟弟说话）；要正脸对着对象就写"转身正脸朝向X、与X面对面、绝不背对"。⚠ **内心独白/失神镜把视线写死一个具体落点**（如"目光垂落身前地面、不乱飘、不看任何人"），否则眼神乱瞟（shot10 教训）。 |
| C3 | **相对位置 + 互相朝向** | 只写各自站位、没写彼此关系朝向（谁在谁前/侧/上方、谁俯谁仰） | `走位:` 用世界坐标写清相对位（前景/侧位/中轴），并写**相对朝向**（裴昭立于跪着的知秋前方俯睨、沈婉侧身偏离知秋避视） |
| C4 | **OS / 画外 / 内心独白朝向** | 内心独白却让人物开口对人说；画外音被安到在画脸上 | 内心独白=不对口型、视线可虚焦/望向别处（不与人对视）；画外音说话人不在画→在画人物**不对口型**、按真实反应朝向（听画外声转头望声源方向） |
| C5 | **防角色重复** | 同一在画沉默/被动角色被多次具名+多位置短语，模型渲染成两个同人（实测 shot03 出现两个裴知秋） | 同一角色每镜**只具名一次**（首次在 `走位` 定位），其余用"他/兄长/向他"；必要时 `走位` 首句写"全画面共 N 人各一名、互不重复、{关键角色}只一个" |
| C6 | **参考图绑定消歧（多人同框）** | 多人物同框 shot 上传多张选角脸图，却没把「哪张参考图＝画面哪个位置的人」写死，模型串脸/换头/融人（2026-06-18 选角供脸：脸由选角图承载，文字职责＝绑定） | `角色识别 / 参考图:` + `走位:` 里逐个写死「参考图↔画面位置↔识别标签」（如 画面左·裴知秋＝参考图A·清瘦病气少年；画面右·裴昭＝参考图B·锦衣骄纵少年），让每张脸图各就各位 |
| C7 | **跪/受责/下位者身体锚定权威，只转头应对旁人（2026-06-19 教训：shot03 主角跪着背对父亲）** | 一个跪着受责/下位的人，在场有**居高/权威方**（高座上的家主、判罚者、上位者）；某镜他要回应**侧旁**的人（兄弟/旁人）时，prompt 把他**整个身体转向旁人**，于是**背/侧对了权威方**——既失礼又突兀，跨镜还和上一镜的跪向打架 | **跪姿/下位者身体与跪向锚定权威方写死**（如"身体始终朝北跪向高台上的父亲、受责跪向不变、绝不背对"）；要应对侧旁人**只写"转头/转视线"**（"仅头颈转向东看裴昭、身体不转开"），**严禁整身转开背对权威**。跨镜该下位者**跪向/身体朝向与前一镜保持一致**（在权威未离场前不变），只有头与视线随对话对象移动 |
| C8 | **跨镜朝向/站位连贯（同一对峙/同一场景内）** | 同一段对峙跨多镜，某人的**站位或身体朝向在镜与镜之间无故跳变**（上镜面北、下镜整身朝东），或**和 `scenes/{name}.md` 站位锚冲突**（锚定"母妃居西"却放东） | 以 `scenes` 站位锚 + 上一镜为准，**保持每人世界朝向/站位跨镜稳定**；确需变（走位移动）要有动作交代（走过去/转身的 beat）。审一个镜必带看**相邻镜的同人朝向**是否承接 |
| C9 | **镜内站位连贯（一个 shot 内多 beat / 反打不许换位）（2026-06-20 教训：shot06 主角"抱"老人→恶霸挤进中间→老人又到旁边）** | 一个 shot 有多个动作 beat 或过肩反打，prompt 只写了**起始站位**没写"全程不变"，模型把每个 beat / 每次反打**当独立画面重新摆人**，于是三人相对位置在镜内乱跳（A 在 B 前→下一秒 C 挤到 A B 中间→再下一秒又换）；含**贴身/护人被渲染成搂抱**（"挡在身前+护在身后侧"被渲成抱着） | 多人 shot 的 `走位:` **开头钉死"全程相对位置固定不变·反打/换 beat 也不许换位/挪位"** + 从左到右列死每人定位（谁居中/谁在谁身后半步/谁在正前方）；`动作:` **每个 beat 都复述该不变关系**（"地痞始终在正前方、老者始终在主角身后半步")；护人写**"横臂虚拦、不搂不抱、不接触对方身体"**，别让模型脑补成拥抱 |
| C10 | **在场/离场锚定（谁该留、谁该走，逐 beat 写死）（2026-06-20 教训：shot08 道谢的老者混进落荒而逃的人群）** | 某 beat 有人离场/逃跑（地痞逃、群演散），prompt 没写清**哪个具名角色留下、哪个离开**，模型把**本该留场的人也卷进逃跑人群**（该留下道谢的老者跟着地痞一起逃），或凭空补出成群逃跑路人 | `走位:`/`动作:` 对每个具名角色写死此 beat 的**在场/离场状态**——"只有 X 逃出画、Y 没逃、留在原地{做什么}"；明确**画面无其他逃跑人群**（除非剧情要）；`负面词` 补"{该留角色}逃跑 / 路人成群逃散 / {该留角色}混在逃跑人群" |

## 朝向写法落点（三处一致，模型才不让人对空气说话）
`走位:`（相对位置 + 相对朝向）→ `动作:`（该 beat 谁转向谁、视线移动）→ `台词:`（每句标 `面向{对象}` / `望向{方向}`）。

## 工作流
1. 读 shot 的 `走位:`/`动作:`/`台词:` + 角色 bible + 场景方位。
2. 逐句在画对白过 C1–C10，列不合格项（cite shot/句/准则）。**必带核对**：① 场内有无居高/权威方→下位者身体是否锚定它（C7）；② 与相邻镜同人朝向/站位、与 `scenes` 站位锚是否连贯（C8）；③ 多人/多 beat/反打镜→镜内相对站位是否钉死"全程不变"（C9）；④ 有人逃/离场的 beat→该留的具名角色是否写死"不逃、留场"（C10）。
3. **当场 surgical 改**：在三字段写死朝向/视线/相对位；全员无口型镜按 C4 处理；防重复按 C5。
4. 复验：每句在画对白都有明确"谁面向谁、视线落点"；无背对说话、无对空气、无同角色重复。

## 硬约束
- 只动**空间/朝向/视线/走位**，不改台词措辞（台词归 `台词大师`）、不改景别运镜（归 `运镜`）、不改剧情（归 `剧情连贯`）。
- 与 `运镜` 冲突时（如 camera 要偏头反打 vs 本 skill 要正脸不背对）：**朝向硬约束优先**（防背对/对空气是底线）。
- 内心独白/画外 OS：在画人物**不对口型、嘴唇不动**（rule 12.4-F 口型契约）。
- 审计写 `.audit/.../events.jsonl` 的 `validation.issue.raised`（level=blocking）。

## 输出
- inline patch 清单（shot / 原行 / 准则 / 改写后）+ 已落地改动 + 一句话复验结论。
