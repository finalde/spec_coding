### Design decisions
- SlackNotifier owns the WebClient instance; injected token/channel validated in __init__.
- PipelineResult is frozen=True to prevent mutation after construction.
- Block Kit message includes a "View Pipeline" button only when a URL is provided.
- STATUS_EMOJI dict maps status strings to Slack emoji for easy extension.
- Full type hints on all methods and attributes (Python 3.10+ union syntax).
