# 白话文大师（vernacular master）

把 `ai_videos/{name}/` 里**每一句会被听到的话**（角色对白 + 内心独白 OS + 系统提示音 + 旁白 + 小说原文里的对白）改成**真人会脱口而出的地道中文大白话**。标准只有一条总纲：**念出来像活人说话，不像写出来的、翻译出来的、念稿子。** 发现书面/拗口 → 当场 surgical 改该句（不动剧情、不动结构），三处同步。

**为什么单列**：台词大师管台词全维度（说人话/因果/信息/声口/情绪/长度/人称/节奏），白话只是其中一两条；实战中"白话不够"反复出现（EP1/EP2 多轮），需要一个**只盯白话、标准更狠**的专项，台词仍书面时单独过它。

## 何时用
- 用户说：对白不够白话 / 像书面 / 像翻译腔 / 像念稿 / 太文 / 再口语些 / 把对白改活 / "用白话文大师过一遍"。
- 某句被指"突兀 / 别扭 / 不像人说的"。
- `审查总编排` 单镜层、紧跟台词大师之后的白话加强工序（可选）。

## 输入
- 范围：某 shot / 某集 / 整剧。
- 必读：`script.md`/`dialogue.md`/`shots/*.md` 的台词字段 + 角色 bible（声口/身份/关系）。

## 白话审查标准（每句逐条过；任一条不过＝改）

| # | 准则 | 不合格信号 | 怎么改 |
|---|---|---|---|
| B1 | **朗读测试（硬标准·总闸）** | 默念一遍——要换气才念完 / 拗口绕嘴 / 舌头打结 / 像在念课文 = 不合格 | 拆短句 + 加停顿（`…`/`——`）+ 换口语；念顺为止 |
| B2 | **去文言·去半文四字格** | 文言虚词句式（吾/汝/尔/乃/之/矣/焉/哉/岂/何不/安能/莫非…耶）；半文四字格言（潜伏待显/蓄势待发/暗藏锋芒/落得下场/不容置喙/一锤定音当台词念） | 一律换现代口语；身份/年代靠用词分寸+称谓体现，不靠"说文言" |
| B2b | **单字文言/书面动词带具体宾语 → 双音节白话** | 单音节文言动词直接带宾语、读着像电报缩略（离裴家/归府/赴京/出府/居此/弃城/携刀/食肉/着锦）——口语没人这么省，连系统/旁白也不行（"离裴家，太突兀，至少也是离开裴家" 2026-06-20 反馈） | 补成白话双音节：离→离开、归→回/回到、赴→去/前往、出→走出/离开、居→住、弃→丢下/扔下、携→带、食→吃、着(穿)→穿、视→看、闻→听、寻→找、启→打开、取→拿、置→放。即 `离裴家→离开裴家`、`归府→回府里`、`出府→走出府门`。⚠ 像"离家出走/回家"这种已成词的双字组合不算，盯的是**单字动词+具体/专名宾语**的硬缩略 |
| B3 | **去对仗·去格言·去排比腔** | `X的是…，Y的是…` / `不劳…` / `先…不急…` / `我的X自己Y，我的Z自己W` 这类对称句、格言腔、工整排比 | 改成真人散句（有长有短、不工整）；决意/狠话也用大白话，不堆修辞 |
| B4 | **书面词→口语词（见词典）** | 句里有书面/公文/古白话词（将/其/此/或许/为何/无需/即可/前往/二人/皆/亦/方才/莫要/岂能/落脚/根基/族籍…） | 按下方「书面→白话替换词典」逐个换口语 |
| B5 | **去翻译腔·拆别扭定语** | 长定语从句套娃、"做一个…的人"、"对…进行…"、"当…的时候"、被字句堆叠、主语+一长串修饰才到谓语；**`能+动词短语+的+名词`** 这类把动作硬塞进定语里修饰名词（找个**能练身子的**地方、找个**能落脚的**去处、买件**能御寒的**衣裳——念着绕、不像口语，2026-06-20 反馈） | 拆成中文短句、主谓靠前、定语拆出来单说；`能V的N` → 把动作拆成独立小句（找个能练身子的地方落脚 → **找个地方落脚，把功夫练起来**；找个能御寒的衣裳 → 买件衣裳，好御寒） |
| B6 | **小词短句+口头语** | 句子干巴、没生活气、没语气 | 多用"行/算了/没准/这会儿/咋/呗/呢/啊/哟/嘛"等口语小词，情绪靠语气词+停顿带。⚠ 慎用出戏网络梗/赌气小词（卧槽/我偏不/哼/切）——生死决绝重场显轻佻；重场要"冷而决"不"赌气" |
| B7 | **称呼·声口地道** | 称呼不贴身份关系（地痞叫"官爷"、平民对王爷用错敬称）；谁说都一个腔、不像那个人会说的话 | 称呼按真实身份关系（街头混混→"几位大爷/好汉"，不叫官）；声口贴人（痞子有痞气、老者怯弱、王者威重但仍说白话） |

## 朗读测试（核心方法·每句必做）
默念这句话——**想象一个真人在那个处境下，会不会原样这么说出口**。会＝过；觉得"书面/像旁白/像翻译/念稿/没人这么讲"＝改。越关键的句子越要短、越要像话。

## 书面 → 白话 替换词典（常见，按需扩充）
| 书面/古白 | 白话口语 |
|---|---|
| 将…（将其/将之） | 把…（把它/把他） |
| 其 / 此 / 该 | 他·它 / 这 / 这个 |
| 二人 / 诸位 / 皆 / 亦 | 俩人·两个人 / 各位·大伙 / 都 / 也 |
| 或许 / 莫非 / 岂能 / 焉能 | 说不定 / 难道 / 哪能 / 怎么能 |
| 为何 / 缘何 / 何不 | 为啥·干嘛 / 怎么 / 干嘛不 |
| 无需 / 不必 / 即可 / 方可 | 不用 / 用不着 / 就行 / 才行 |
| 前往 / 离去 / 归来 / 落脚 | 去 / 走 / 回来 / 待·安顿下来 |
| 离X / 归X / 赴X / 出X（单字动词带专名宾语） | 离开X / 回到X / 去X / 走出X（离裴家→离开裴家） |
| 根基 / 修为(可留setting) / 族籍 | 底子·根底 / 修为(术语保留) / 这个姓·家谱上的名 |
| 落得…下场 / 沦为 | 落个…样 / 变成·成了 |
| 不容置喙 / 一锤定音 | （别当台词念，写动作）/ 一句话定死 |
| 潜伏待显 / 蓄势待发 | 先藏着 / 憋着劲 |

| 找个能练身子的地方落脚（"能V的N"别扭定语） | 找个地方落脚，把功夫练起来 |

## 工作流（review → 直接改）
1. 读范围内全部会被听到的句子（对白+OS+系统+旁白+小说原文对白）+ 角色 bible。
2. **逐句过 B1–B7**，每句先做朗读测试；列不合格项（cite shot/句/准则）。
3. **当场 surgical 改写**：只改措辞、只让它更像人话；不动剧情/结构/人称/朝向/锁定串/时长。
4. **三处同步**：shot 的 `台词:` 行、`dialogue.md`、`script.md` 改后一致。
5. 复验：每句念出来都像活人说的；无文言/对仗/翻译腔/书面词残留；称呼声口地道。

## 硬约束
- **只动"说话方式"**：不改剧情逻辑（归 剧情连贯）、不改因果/信息/人称/节奏的实质（归 台词大师）、不改朝向走位（归 站位朝向）、不改景别运镜（归 运镜）。改完语义不变、只更口语。
- **setting 专名/术语保留**（武神躯/丹田/宗门/玉佩/系统/voice_id 等），不要为"白话"把设定词也换掉。
- 改写后仍是中文、零 hex；不引入 world 未有的设定；不剧透后续 arc 专名。
- 节奏（字数÷时长 ≤5）以 台词大师/时长节奏 为准；白话改写若把句子改长，连带自检不超节奏。

## 输出
- inline patch 清单（shot / 原句 / 准则 / 白话改写后）+ 已落地改动 + 一句话复验结论。
