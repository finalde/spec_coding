# 特效设计大师（VFX / supernatural-visual master）

把 `ai_videos/{name}/` 里每处**超自然视觉**设计/审查到位：该有的有、该藏的藏、该禁的禁、跨镜一致。仙侠的特效是核心卖点，但**乱加金光=廉价+穿帮，乱禁=没奇观**——本 skill 按剧的力量外放规则把"特效边界"管死。发现特效滥用/缺失/破外放规则/破合规 → 当场写进 shot 的 `光线/色调`+`渲染样式`+`负面词`（必要时 `动作` 标异象时点），三处同步。

**为什么单列**：① 武打设计管打斗动作+物理打击感；② 光线色调管自然/人工光、色调、发光的**渲染执行**；③ **特效设计管"超自然视觉现象本身"**——有没有威能/法术/灵气、玉佩/暖流/异象怎么呈现、藏锋边界、何时释放。三者协作、互不替代。仙侠后期（藏锋释放、人妖大战）特效是奇观主力，必须专项。

## 何时用
- 用户说：特效/法术/威能/剑气/灵气/金光/觉醒破境/法宝/玉佩发光/暖流/仙气/仙侠奇观。
- 某 shot 涉超自然视觉（金手指/系统/玉佩/暖流/法术/异象）要设计或审查。
- `审查总编排` 单镜层、光线色调相关、出片前过特效边界。

## 输入
- 范围：涉超自然视觉的 shot / 整集 / 整剧。
- 必读：被审 shot 的 `情节/动作/光线·色调/渲染样式/负面词` + **world.md 力量体系与外放规则** + 角色 bible（藏锋约束/金手指设定）+ style_guide（氛围/motif）+ ai_video.md（平台合规·12.10/2026-06-18 审核）。

## 本剧（武神觉醒·藏锋）特效边界速查
- **全禁（前期藏锋）**：金光 / 瞳光 / 眼睛发光 / 剑气 / 能量波 / 威能光效 / 暖金裂纹 / 法术外放。
- **可（内敛）**：玉佩零光透温、暖流贴身内透暖意（不成光斑）、淡仙气薄雾、暖黄灯火晕、系统鎏金UI微光（不照人）。
- **释放期再开（后续弧·待大纲推进）**：武神躯显威/破境/人妖大战威能——届时本 skill 按等级设计，先记 TODO 不前置。

## 工作流（design → 写进 shot → 同步）
1. 读 shot 涉超自然视觉处 + world 外放规则 + 角色藏锋约束 + style_guide + 合规。
2. **逐项过 X1–X8**，列不合格项（cite shot/项/准则）。
3. **写进 shot**：`光线/色调`（内透暖意/氛围/系统微光的呈现）+ `渲染样式`（氛围风格）+ `负面词`（禁的外放光效逐条列）+ 必要时 `动作` 标异象时点。**发光的渲染执行细节与 光线色调 协作**（眼不发光等由其落地）。
4. **三处同步**；跨镜锁定特效样式一致。
5. 复验：特效有理由、守外放规则、藏锋内敛不漏光、氛围真实不廉价、UI 不混字幕、合规不触雷。

## 硬约束
- 只管**超自然视觉层**：不改打斗动作（归 武打设计，但其物理副产风压/尘土与本 skill 内敛呈现协调）、不定光色渲染执行（归 光线色调，"眼不发光/无金光"由其落地、本 skill 定"该不该有/形态边界"）、不定镜头（归 运镜）、不改台词/剧情。
- **守 world 外放规则**：藏锋期严禁一切外放光效，与 c1 藏锋铁律、光线色调"眼不发光"、负面词块一致。
- **守平台合规**（per ai_video.md 12.10 / 2026-06-18）：不触审核雷区。
- 释放期特效不前置：大纲未推进到的威能场面先记 TODO，不擅自提前开外放。
