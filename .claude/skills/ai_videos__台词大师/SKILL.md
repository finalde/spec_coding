# 台词大师（dialogue master）

审查并**直接修改** `ai_videos/{name}/` 每行**台词的措辞质量**，标准是「一个普通观众听着会不会觉得假、空、绕、不像人话、没共鸣」。发现不对 → 当场 surgical 改写（只改该行，不重写整集），三处同步：shot 的 `台词:` 行、`dialogue.md`、`script.md`。

**单一职责（拆分后）**：本 skill 只管**台词措辞 + 台词节奏（字数/时长）**。其余维度已各自独立——朝向/视线/走位→`站位朝向`；景别/运镜→`运镜`；动作/表演→`动作表演`；情节链/动机/爽点/系统互动→`剧情连贯`；跨集序列→`全剧序列`；格式/字段/字幕/锁定/hex→`格式契约`。整体顺序串跑由 `ai_videos__审查总编排` 编排（本 skill 是单镜层第 1 道）。

## 何时用
- 用户说：对白太浅/不像人话/像喊口号/缺因果/声口雷同/太文言/念不完；"审台词""把对白改深""润色"。
- `ai_videos__审查总编排` 单镜层第 1 道工序。

## 输入
- 范围：某 shot / 某集 / 整剧。
- 必读：该集 `script.md`/`dialogue.md`/`shots/shotNN.md` 的 `台词:` 字段；角色 bible（声口/动机）；`world.md`（设定/系统）。

## 台词审查标准（每行台词逐条过）

| # | 准则 | 不合格信号 | 怎么改 |
|---|---|---|---|
| D1 | **说人话** | 像标语 / 书面口号 / 文绉绉硬邦邦 / 没人会这么说话；**规整书面长句、像念稿 / 演讲稿**（赌约 / 宣言写得太工整对仗）；**自指用第三人称**（"这废物也不稀罕" 听着像在说别人，shot14 教训） | 改成**真人脱口而出**的口语、有停顿、有语气起伏、有称呼（嘲讽 / 隐忍 / 独白贴当下情绪：嘲讽可"区区废物，也敢大放厥词？行啊——"）；**说自己就用「我」**，别用"这废物/此人"自称除非刻意自嘲（2026-06-17 EP2 全镜优化教训） |
| D1b | **白话文·禁古语** | 文言虚词/句式（吾 / 汝 / 尔 / 乃 / 之乎者也 / 矣 / 焉 / 哉 / 岂 / 何不 / 何曾 / 安能 / 莫非…耶）、半文半白拗口腔、纯古风格言；仙侠/古装题材也不例外（见 ai_video.md 2026-06-16「台词与小说原文一律用白话文」amendment） | 一律改现代白话口语；身份/年代差异只靠用词分寸、语气、称谓体现，不靠"说文言"。覆盖台词 + 小说原文段 + prompt 正文对白/旁白 |
| D2 | **因果交代** | 只下结论不讲缘由（"裴家不养废物"→为什么？） | 补前因后果："你丹田碎了→修不了武→裴家以武立命→容不下废人" |
| D3 | **信息量** | 一句废话、不推进剧情也不揭示动机/stake | 让每句话推进情节、揭示人物关系/动机/利害，或埋钩 |
| D4 | **角色声口** | 谁说都一样、不贴身份性格（bible §角色定位/气质） | 按该角色身份/性格/处境调措辞（嚣张少主≠威严家主≠软弱母亲）；声口差异在**白话范围内**用词分寸实现，不靠说文言（见 D1b） |
| D5 | **情绪真实** | 平、假、用力过猛、不符合此刻处境 | 贴合此刻情绪与关系；恨里有由、护里有怯 |
| D6 | **长度合宜** | 一律砍到≤15字导致干瘪；或啰嗦拖沓 | 该交代因果/情绪的关键台词可长（自然口语），废话砍掉 |

## 时长 / 台词节奏审查（每个有台词的镜都要过——防"时间太短念不完、语速飙")

> 时长的**整体合理性裁决**（台词念得完 + 动作演得开 + 不赶不拖 + 拆镜）归 `ai_videos__时长节奏`（PA1–PA6）。本节是台词侧的字数÷时长自检，发现念不完优先精简台词；要不要加时长/拆镜由 时长节奏 仲裁。

台词字数必须配得上 shot 时长，否则配音被迫加速、演员像抢词（实测 shot09 台词长、时长短、语速过快）。

- **估算基准**：自然中文台词 ≈ **4 字/秒**（情绪/停顿/独白镜更慢，约 3 字/秒；平述快切镜至多 5 字/秒）。
- **判定**：一镜**总台词字数 ÷ 时长(秒) 应 ≤ 5**，目标 ≈ 4；多说话人镜把各句字数**相加**再除。> 5 字/秒 = 不合格。标点不计字。
- **怎么改（按剧情优先级择一）**：① 加时长（守 ≤15s 上限）；② 精简台词（砍废字、留因果与信息量，D3/D6）；③ 拆镜（一镜拆两镜各给够时长 + 连续性 token）。⚠ 加时长须连带 `动作表演` 把动作 beats 铺满新时长。
- **改后三处自洽**：shot 的 `时长:` 字段、`台词配音 prompt` 各句 `时长目标`、台词字数 互相对得上（各句 `时长目标` 之和 ≤ shot 时长，留转场/停顿余量）。

## 工作流（review → 直接改）
1. 读范围内 `script.md`+`dialogue.md`+`shots/*.md` 的台词字段 + 角色 bible + world.md。
2. 逐行台词过 D1–D6、逐镜过台词节奏，列不合格项（cite shot/行/准则）。
3. **直接改写**不合格台词（surgical：只改该行，不重写整集；保留 shot 结构/锁定串/时长，时长按节奏审查可调）。改写要：说人话 + 白话 + 给因果 + 贴声口 + 有信息量。
4. **三处同步**：shot 的 `台词:` 行、`dialogue.md`、`script.md` 改后一致。
5. 复验：台词像真人对话、白话无文言、有因果、声口贴身、字数÷时长 ≤ 5。

## 硬约束（台词层）
- **`台词:` 字段内严禁任何字幕信息**（字体/位置/颜色/硬字幕/软字幕…），label = `台词:`（不是 `台词 / 字幕:`）；字幕由用户后期自加（格式由 `格式契约` 把关）。
- **台词正文必须留在 `台词:` 字段（不可剥离，否则成片无台词声音）**：视频模型（Seedance）从 `台词:` 文本生成台词配音，删正文 = 成片没台词声音（shot05 教训）。全员无口型镜的正解 = 保留正文 + 每句标 `画外配音 voice-over`/`内心独白 OS` + `嘴唇不动不对口型` + 系统句标 `无任何在画人物对此口型`，而非删文。
- **台词配音块 voice_id 全剧锁定**：同角色跨集复用同一 voice_id（音色一致性铁律）；系统（鎏金对话框）保留配音、`时长目标` 配足整句（follow-up 041 option B）。
- 改写后台词仍是中文、零 hex；不引入 world.md 未有的设定；不剧透后续 arc 专名（专名时序由 `剧情连贯`/`全剧序列` 把关）。

## 输出
- inline patch 清单（shot / 原行 / 准则 / 改写后）+ 已落地的台词改动 + 一句话复验结论。
