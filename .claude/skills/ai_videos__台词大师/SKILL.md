# 台词大师（dialogue master）

审查并**直接修改** `ai_videos/{name}/` 每行**台词的措辞质量**，标准是「一个普通观众听着会不会觉得假、空、绕、不像人话、没共鸣」。发现不对 → 当场 surgical 改写（只改该行，不重写整集），三处同步：shot 的 `台词:` 行、`dialogue.md`、`script.md`。

**单一职责（拆分后）**：本 skill 只管**台词措辞 + 台词节奏（字数/时长）**。其余维度已各自独立——朝向/视线/走位→`站位朝向`；景别/运镜→`运镜`；动作/表演→`动作表演`；情节链/动机/爽点/系统互动→`剧情连贯`；跨集序列→`全剧序列`；格式/字段/字幕/锁定/hex→`格式契约`。整体顺序串跑由 `ai_videos__审查总编排` 编排（本 skill 是单镜层第 1 道）。

## 何时用
- 用户说：对白太浅/不像人话/像喊口号/缺因果/声口雷同/太文言/念不完；"审台词""把对白改深""润色"。
- `ai_videos__审查总编排` 单镜层第 1 道工序。

## 输入
- 范围：某 shot / 某集 / 整剧。
- 必读：该集 `script.md`/`dialogue.md`/`shots/shotNN.md` 的 `台词:` 字段；角色 bible（声口/动机）；`world.md`（设定/系统）。

## 台词审查标准（每行台词逐条过）

| # | 准则 | 不合格信号 | 怎么改 |
|---|---|---|---|
| D1 | **说人话（含·禁对仗金句/文艺比喻）** | 像标语 / 书面口号 / 文绉绉硬邦邦 / 没人会这么说话；**规整书面长句、像念稿 / 演讲稿**（赌约 / 宣言写得太工整对仗）；**对仗排比的"金句"+ 文艺比喻**（2026-06-20 教训："找个落脚的地方，也找条往上走的路"＝排比+"往上走的路"比喻，像编剧写的漂亮话、不是真人脑子里的念头；**OS 尤其要 raw、具体、不抽象**——内心独白是念头不是台词金句）；**自指用第三人称**（"这废物也不稀罕" 听着像在说别人，shot14 教训） | 改成**真人脱口而出**的口语、有停顿、有语气起伏、有称呼（嘲讽 / 隐忍 / 独白贴当下情绪）；**把比喻/金句降成具体大白话**（"找条往上走的路"→"找个宗门落脚、安心练功"）；**说自己就用「我」**，别用"这废物/此人"自称除非刻意自嘲（2026-06-17 EP2 全镜优化教训） |
| D1b | **白话文·禁古语（但白话≠出戏俗词）** | 文言虚词/句式（吾 / 汝 / 尔 / 乃 / 之乎者也 / 矣 / 焉 / 哉 / 岂 / 何不 / 何曾 / 安能 / 莫非…耶）、半文半白拗口腔、纯古风格言；仙侠/古装题材也不例外（见 ai_video.md 2026-06-16「台词与小说原文一律用白话文」amendment）。**反向也防：过俗/现代/出戏的口语词**（2026-06-20 教训：仙侠里"练身子""身子骨"出戏） | 一律改现代白话口语；身份/年代差异只靠用词分寸、语气、称谓体现，不靠"说文言"。**白话但贴境**：把出戏俗词换成又口语又不跳戏的说法（"练身子"→"调养根基/把功夫练起来/运功"、"这身子"→"这副病躯/这副底子"）。覆盖台词 + 小说原文段 + prompt 正文对白/旁白 |
| D2 | **因果交代** | 只下结论不讲缘由（"裴家不养废物"→为什么？） | 补前因后果："你丹田碎了→修不了武→裴家以武立命→容不下废人" |
| D3 | **信息量** | 一句废话、不推进剧情也不揭示动机/stake | 让每句话推进情节、揭示人物关系/动机/利害，或埋钩 |
| D4 | **角色声口** | 谁说都一样、不贴身份性格（bible §角色定位/气质） | 按该角色身份/性格/处境调措辞（嚣张少主≠威严家主≠软弱母亲）；声口差异在**白话范围内**用词分寸实现，不靠说文言（见 D1b） |
| D5 | **情绪真实** | 平、假、用力过猛、不符合此刻处境 | 贴合此刻情绪与关系；恨里有由、护里有怯 |
| D6 | **长度合宜** | 一律砍到≤15字导致干瘪；或啰嗦拖沓 | 该交代因果/情绪的关键台词可长（自然口语），废话砍掉 |
| D7 | **人称关系正确（当面对话用对你/他——2026-06-19 教训：shot02 父亲当面对主角说话却说"除他族籍"）** | 说话人**当面对在场的某人**说话，指代对方却用**第三人称「他/她」**（"今日起，除他族籍"——明明在对儿子说，却像在跟别人讲他），不像真人对话；或同一对话里**人称前后不一致**（这镜对他说"他"、下镜又"你"，shot02↔shot08 裴霆一处"他"一处"随你") | 判定受话人=谁：① **当事人就是受话人**（当面说给他听）→指代他用**「你/你们」**（"今日起，除你族籍，送你出府"）；② 当事人**不在场 或 说给第三方/众人听**（宣告、背后议论、当事人已转身离场）→才用「他/她」（shot10 裴昭冲离场背影说"有他哭的"＝对）。**复核全场同一对受话关系的人称一致**（跨镜也要一致）。OS/旁白指代他人用三人称正常 |
| D8 | **禁角色口头禅 / 签名重复句（2026-06-20 教训：EP3 主角把 bible「锋，藏着」当口头禅复用、读着像喊口号念稿）** | 给角色安一句反复出现的**签名口头禅 / 定调金句**（"锋，藏着""力量，藏好""废物哥哥又来讨打了"…），跨镜/跨集重复念；或单镜里硬塞一句对仗式 slogan 当记忆点 | **一律去掉口头禅**。同一意思每次**换不同的大白话说法**，贴当下处境临场说；角色辨识靠声口（用词分寸/语气/称谓/语速，见 D4）+ 造型，**不靠重复同一句台词**。bible 不再设「标志台词/口头禅」字段；真要复现某情绪，重写一句新的口语，别复刻原句 |
| D9 | **信息边界·伏笔别写满（角色别说超出他当下认知的话——2026-06-20 教训：EP3「从头来过·这一次不一样」要观众没见过的前情/级别才懂；「还差一样东西、那才是关键」让刚开练的主角像全知报答案）** | 角色说出**他此刻不可能知道的结论**：笃定某伏笔/点破谜底（"差的那样才是关键"＝替编剧报答案）、拿一个**观众&角色都没见过的"上一次/之前"**作比较（"这一次不一样"）、或台词**依赖观众没看过的前情/数值**才成立 | 只让角色说**他当下能感知的**：把"确定结论"降成 **felt 不安 / 疑惑 / 说不上来**（"总觉得还差点什么，又说不上来"）；伏笔留 authorial、台词只埋"觉得不对劲却抓不住"；要交代的前情若观众没看过，就别让台词依赖它——改用观众已知的信息（如"废了丹田的人"而非"从武人三级跌下来"）讲明白 |
| D10 | **长串规则/科普台词拆短句（别一句塞满多条信息——2026-06-21 教训：EP4 S3 测资执事一句塞下「七品名＋四档分档规则」，逗号分号串成一长条、拗口难懂、听一遍记不住）** | 一句话里塞进**多条并列信息/规则/枚举**（一口气报完等级表＋各档分档＋条件），靠逗号/分号串成一长条；念着绕嘴、观众一遍抓不住重点 | **拆成数个短句、一句一层意思**（规则/科普类按「①做什么 → ②有哪些/枚举 → ③怎么分/条件」分段）；每句独立成句、能单独念顺；信息确实多就**加时长/拆镜**（守节奏 ≤5 字/秒、≤15s），别压成一长句硬塞 |
| D11 | **承接镜台词禁复述已完成的过渡动作（2026-06-23 教训：EP3 shot3 首帧已阖目、台词却以「闭上眼…」开头，首尾帧一接像"闭眼再蹦出来闭一次"）** | 该镜是**首尾帧承接镜**（首帧＝上一镜末帧），台词（尤 OS/旁白）却**复述刚在上一镜做完、已定格在首帧上的过渡动作**（闭上眼 / 睁开眼 / 坐下 / 站起来 / 转过身 / 抬起头…），与画面"已完成态"打架 | 台词从**交接帧的已到达状态**往下说，删掉对已完成动作的复述（"闭上眼，把心神沉进丹田…"→"把心神慢慢沉进丹田…"）。是否承接镜、首帧是什么状态，由 `ai_videos__运镜` M8 给出（本规则与 M8 承接判定协同）；只在承接镜适用、硬切镜不受限 |

## 自然口语进阶（学真实中文影视台词的"说人话"——2026-06-18 反馈进化）

> 反馈来源：用户指出"救他的是这具身子，弃他的也是这一家人""这族籍，不劳谁动手，我自己除""先收着，不急着让人看见"仍**书面/拗口**，要求台词大师多学真实中文影视/短剧台词。落地准则（参影视编剧通则：简练、生活气息、人物化、朗读可过）。审台词时**叠加在 D1 上从严**：

- **朗读测试（硬标准）**：每句默念一遍——**要喘气才念完、或拗口绕嘴 = 不合格**，必拆短/改口语。**适用所有发声单元——不止角色对白，还包括内心独白(OS)、系统提示音、旁白**（2026-06-19 教训：shot9 系统"潜伏待显"半文突兀、OS"进骨头里了"别扭，因只盯对白、漏审 OS/系统行）。系统/旁白可"冷/简/clinical"，但仍须是**自然中文**：禁半文四字格言（潜伏待显 / 蓄势待发 / 暗藏锋芒…）、禁生造拗口短语，改大白话（"暂时潜伏、外人看不出来"）。OS 同角色声口、同朗读标准。
- **拆对仗、去格言**：禁 `X的是…，Y的也是…` / `不劳…` / `先…不急…` 这类**对称句、文绉句、格言腔**；改成真人会脱口而出的大白话。
- **小词短句 + 口头语**：多用"行""算了""没必要""再说""跟我没关系""这会儿""轮不到你"等口语小词；情绪靠语气词/停顿带，不用工整修辞。⚠ 慎用**轻佻/赌气小词**（"我偏不""哼""切"）——在生死/决绝的重场里显**出戏、像赌气小孩**（2026-06-18 教训：果决选择写"我偏不"出戏）；重场要"冷而决"不"轻而赌气"，改"我不走这条路""我不认这命""这条老路，我不走"。
- **对白要饱满、像真人对话（别全程 OS）**：一场戏不能只有主角内心独白（OS）、配角各甩一句就完；关键冲突要**有来有往**——主角在该开口处开口回话、反派/家人接话，凑成真的对话。审到"通篇 OS、对白稀薄、人物不交流"时，把部分 OS 改成在场人物间的**对白往来**（仍守声口 + 节奏 ≤5 字/秒，busy 对话镜不够念就加时长/拆镜、交 `时长节奏` 仲裁）（2026-06-18 EP1 反馈）。
- **生活动词替书面动词**：除族籍→退出/不认这个姓；收着→压着/藏着；不劳动手→不用你们动手；配不上→不配。
- **断句留气口**：长句拆短句 + 停顿（`…` / `——`），重点放慢、简单加快。
- **简练**：越关键越短；不堆砌、不让一个人长篇自说自话。
- **删填充/无价值句**：承接前镜的复述（"连她都别开了脸"——S3 已演过沈婉避脸）、空泛感慨/凑情绪的废句（"现在闹、现在求，都没用"）若**不推进情节、不揭示新信息、不揭示动机**，直接删——别为凑情绪硬留（2026-06-18 EP1 教训）。
- **禁指代不清的口语黑话**：像"玩这套""这一套""那一套"这类**"这/那 + 套/事/茬"指代含糊**的说法，观众不知道具体指什么——改成**说清楚那件事/那种状态**（"玩这套"→"再这么窝囊地活着""再跟他们这帮人耗"）（2026-06-18 EP1 教训）。

| 书面/拗口（✗） | 口语改写（✓） |
|---|---|
| 救他的是这具身子，弃他的也是这一家人 | 我替他挨了那一掌，换来的却是全家拿我当废物 |

## 红果官方台词法则（首要平台·2026-06-18 反馈进化，源：红果短剧官方《短剧编剧第一课》台词打磨篇）

> 红果/番茄是首要投放平台，台词按其官方标准从严。叠加在 D 与「自然口语进阶」之上：

- **杜绝"水台词"**：无信息量的重复、寒暄、纯情绪铺垫一律删——每句要么给信息、要么给情绪、要么推剧情，三者占其一。
- **台词推动剧情（红果核心）**：好台词不止承载信息/情绪，更要**用一句话引出下一个冲突或转折**。审每句必问："这句把剧情往前推了吗？带出下一个钩了吗？"推不动的=水词，删或改。
- **简洁有力**：越关键越短；冗长啰嗦会拖慢节奏、观众划走。
- **声口贴人设（带职业/身份感）**：身份/年龄/性格/职业各有说法，贴真实生活语言但经提炼（强化 D4）。
- **开场黄金三秒**：开场台词即切入突发事件、迅速立核心悬念，不做冗长铺垫。
- **钩在抉择处**：段落/集尾收口在关键抉择、对峙处**暂停、不急着给答案**，留悬念。
- **改尴尬台词法**：模仿爆款短剧的真实台词语感，不原创生造腔调（呼应「自然口语进阶」朗读测试）。

## 时长 / 台词节奏审查（每个有台词的镜都要过——防"时间太短念不完、语速飙")

> 时长的**整体合理性裁决**（台词念得完 + 动作演得开 + 不赶不拖 + 拆镜）归 `ai_videos__时长节奏`（PA1–PA6）。本节是台词侧的字数÷时长自检，发现念不完优先精简台词；要不要加时长/拆镜由 时长节奏 仲裁。

台词字数必须配得上 shot 时长，否则配音被迫加速、演员像抢词（实测 shot09 台词长、时长短、语速过快）。

- **估算基准**：自然中文台词 ≈ **4 字/秒**（情绪/停顿/独白镜更慢，约 3 字/秒；平述快切镜至多 5 字/秒）。
- **判定**：一镜**总台词字数 ÷ 时长(秒) 应 ≤ 5**，目标 ≈ 4；多说话人镜把各句字数**相加**再除。> 5 字/秒 = 不合格。标点不计字。
- **怎么改（按剧情优先级择一）**：① 加时长（守 ≤15s 上限）；② 精简台词（砍废字、留因果与信息量，D3/D6）；③ 拆镜（一镜拆两镜各给够时长 + 连续性 token）。⚠ 加时长须连带 `动作表演` 把动作 beats 铺满新时长。
- **改后三处自洽**：shot 的 `时长:` 字段、`台词配音 prompt` 各句 `时长目标`、台词字数 互相对得上（各句 `时长目标` 之和 ≤ shot 时长，留转场/停顿余量）。

## 工作流（review → 直接改）
1. 读范围内 `script.md`+`dialogue.md`+`shots/*.md` 的台词字段 + 角色 bible + world.md。
2. 逐行台词过 D1–D10、逐镜过台词节奏，列不合格项（cite shot/行/准则）。**逐行＝每个发声单元**：角色对白 + 内心独白(OS) + 系统提示音 + 旁白都要过（别只盯对白）。**每句必问**：① 朗读顺不顺、像不像人话（D1/朗读测试）② 当面对话人称对不对（D7：该用「你」还是「他」）。
3. **直接改写**不合格台词（surgical：只改该行，不重写整集；保留 shot 结构/锁定串/时长，时长按节奏审查可调）。改写要：说人话 + 白话 + 给因果 + 贴声口 + 有信息量。
4. **三处同步**：shot 的 `台词:` 行、`dialogue.md`、`script.md` 改后一致。
5. 复验：台词像真人对话、白话无文言、有因果、声口贴身、字数÷时长 ≤ 5。

## 硬约束（台词层）
- **`台词:` 字段内严禁任何字幕信息**（字体/位置/颜色/硬字幕/软字幕…），label = `台词:`（不是 `台词 / 字幕:`）；字幕由用户后期自加（格式由 `格式契约` 把关）。
- **台词正文必须留在 `台词:` 字段（不可剥离，否则成片无台词声音）**：视频模型（Seedance）从 `台词:` 文本生成台词配音，删正文 = 成片没台词声音（shot05 教训）。全员无口型镜的正解 = 保留正文 + 每句标 `画外配音 voice-over`/`内心独白 OS` + `嘴唇不动不对口型` + 系统句标 `无任何在画人物对此口型`，而非删文。
- **台词配音块 voice_id 全剧锁定**：同角色跨集复用同一 voice_id（音色一致性铁律）；系统（鎏金对话框）保留配音、`时长目标` 配足整句（follow-up 041 option B）。
- 改写后台词仍是中文、零 hex；不引入 world.md 未有的设定；不剧透后续 arc 专名（专名时序由 `剧情连贯`/`全剧序列` 把关）。
