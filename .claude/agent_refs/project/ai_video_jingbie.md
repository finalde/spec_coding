# 景别 reference — 五级景别阶梯 + 切换铁律（AI 短剧通用）

> AI 短剧镜头语言的 **景别（shot scale）** 权威定义与切换规则。跨项目通用，是 `ai_videos__运镜` M3、`stage5_分镜` 拆镜出 shotlist、`stage6_prompt` 写 `镜头:` 字段的取词依据。
> 项目级 `style_guide.md § 镜头语言关键词字典` 的景别表须与本表对齐（同名同义）；项目可在本五级上加细分（如「大特写」「中近景」），但五级骨架不变。

---

## 1. 五级景别阶梯（由远及近）

| # | 景别 | 取景范围 | 主要用途（戏剧功能） |
|---|---|---|---|
| 1 | **远景** | 交代整体环境、场地全貌；人物占画很小或不入画 | 立空间、定地点、压迫感 / 渺小感、开场定调、段落转场 |
| 2 | **全景** | 人物全身 + 所处空间 | 交代人物与环境关系、走位调度、群像站位、动作全貌 |
| 3 | **中景** | 人物膝盖以上 | **日常对话标配**；兼顾表情与上半身肢体语言 |
| 4 | **近景** | 人物胸口以上 | 表情交流、情绪递进、对话中的强调与反应 |
| 5 | **特写** | 脸部 / 手脚 / 细节物体 | 放大情绪、关键动作、关键道具；钉住观众注意力（反应、顿悟、杀机的那一下） |

> 递进方向：**远 → 全 → 中 → 近 → 特**（由环境到情绪、由疏到密）。每跨一级是「一档」。

---

## 2. 核心切换铁律

**① 绝不同机位、同景别直接硬切。**
相邻两镜若机位（角度/方位/焦段）几乎不变、景别又落在同一级（如中景直切中景、近景直切近景），观众读作「跳了一下 / 穿帮」而非「切了一个镜头」。这是景别切换的头号失败模式，**比"大全景直切大特写"更隐蔽、更常见**。

避法（任一即可，优先级由高到低）：
- **换景别**：相邻镜至少差一档（中→近 / 近→特 / 全→中）。
- **换机位**：景别同级时必须明显换角度/方位/高度（正面中景 → 过肩中景 / 平视 → 俯视），且不越轴（M1）。
- **若两者都改不动** → 说明这两镜本就该并成一镜（M7 视觉无增量），合并它。

**② 景别要有节奏、不硬跳。**（＝ M3）
- 一般段落：景别**递进或有动机**（远→中→近 / 反应切特写），不要无理由大跨级（远景直切大特写）。
- 大跨级**可以**，但要么**有强动机**（猛地推到特写制造冲击）、要么**用动作或视线引导**（角色低头→切其手部特写）做"看点交接"。
- 连续多镜**禁同一景别原地踏步**（全程中景念对白＝呆板）；用景别变化给对话分轻重。

**③ 景别配 beat、不滥用特写。**
- 反应 / 顿悟 / 关键动作 / 关键道具 → 特写或近景。
- 群像 / 站位调度 / 动作全貌 → 全景或中景。
- 交代环境 / 开场 / 转场 → 远景。
- 特写是"强调"货币，**滥用则贬值**——一段戏只在最该钉住的那一下用特写。

---

### 2.4 推荐切换组合（实战配方，相邻镜首选这几招）

每招都是"差一档 + 视觉层层递进"的安全切法，照配即不跳戏：

| # | 配方 | 前一镜 | 后一镜 | 为什么好用 |
|---|---|---|---|---|
| 1 | **全景 → 中景** | 全景交代整体环境与氛围 | 直接切中景，从大环境收缩到人物主体 | 视觉层层递进、由疏到密，观众看着舒服不跳戏 |
| 2 | **中景 → 近景** | 中景展示人物整体姿势与状态 | 切到脸部 / 手部等关键动作的近景 | 衔接丝滑、增加画面层次感，把注意力从"整体"导向"细节" |
| 3 | **近景 → 特写** | 近景给人物关键动作 | 直接切入细节特写 | 一步锁住观众情绪，把那一下钉死 |
| 4 | **对话正反打** | 角色 A 一个角度的中景 / 近景 | 角色 B 另一角度的中景 / 近景，来回切 | 给对抗 / 交流张力。⚠**两人景别尽量错开**：一个偏近景、一个偏中景，避免画面呆板重复（也天然不违铁律 ①） |

> 配方 1–3 是同一条"由远及近层层递进"链的相邻段（全→中→近→特），逐档收缩；配方 4 是双人对话的专用错档法。这四招覆盖了绝大多数顺切场景；非常规跨级（远↔特等冲击切）才需另写动机。

## 3. 单镜内的景别切换（Seedance 多 take shot）

一个 15s shot 内允许多次景别变化（ai_video.md 镜头字段：默认每 15s 至少 3 次景别切换，单 take 静止＝失败模式）。同样守铁律 ①②：

- 逐段写 `镜头:`：`0–3s 全景仰拍 / 3–6s 推近中景 / 6–9s 切近景反应 / …`，每段标景别 + 运动 + 时间窗。
- 段内相邻景别同样**至少差一档或明显换机位**；推/拉/移让景别变化连续过渡（递进式），硬切处用动作/视线引导。
- 首末帧反差（M9）天然要求首、末景别或机位有显著 delta——景别推/拉是最有效的反差杠杆。

---

## 4. 在流程里的落点

- **stage5 分镜**：拆镜出 `shotlist.md` 时，每镜「景别+运动」列须从本五级阶梯取词；**逐对检查相邻镜不同机位同景别硬切**（先情绪→据情绪定景别→再保证相邻不撞档）。
- **stage6 prompt**：`镜头:` 字段景别词出自本表（经 `style_guide.md` 景别字典），shot 级不新造词。
- **运镜审查（M3 + 本 reference）**：单镜判景别是否配 beat；跨镜逐对判 ①同机位同景别硬切 ②大跨级是否有动机/引导 ③连续同景别无变化。发现 → 当场改 `镜头:`（必要时 `走位:` 机位关系），优先改后镜顺承前镜。

*(建立来源：用户反馈 2026-06-27「五级景别定义 + 绝不同机位同景别硬切」。)*
