# 世界观锁定 — 女帝退婚后悔了

本档为 `nvdi_tuihun_houhuile` 项目的世界观锁定文。stage-6 后续所有 `arc_outline.md` / `characters/*.md` / `scenes/*.md` / `episodes/ep01/**` 文件的剧情、地理、势力、修炼物理规则均以本文为唯一事实源。

> **v0.1 (ep01 启动版)**：仅锁定 ep01 接旨场必需信息，随剧情推进 surgical 扩档。

## 1. 时代背景

架空古风王朝 · 女帝在位 · 朝堂势力以世家与宗师为主。
- 女帝年号 / 朝代名: **待定** (ep02 触及朝堂时再锁)。
- 朝堂礼制: 接旨需跪礼 · 三品以上世家由内监上门宣旨 · 五品以下需亲赴宫门接旨。
- 婚约制度: 皇家婚约由女帝亲诏立 / 撤, 解除需诏书形式公示。

## 2. 势力图 (ep01 最小版)

- **皇室 (女帝直辖)**: 朝堂中枢, 宣旨内监 (c4_太监) 为女帝近侍。
- **陈国公府**: 老牌世家, 国公爵位 (一品), 陈国公 (c3) 朝堂老臣身份, 与女帝过往关系密切 (致使五年前指婚)。三公子陈凡 (c1) 为本府嫡幼子。
- **其他世家 / 宗门**: ep02+ 引入。

## 3. 修炼体系 (待定)

ep01 仅暴露陈凡"装废五年"事实, 未涉及具体修为体系。
- **占位方向 (待 ep02 锁)**: 倾向 武修 + 修真混合 (古风短剧常见), 段位由低到高 e.g. 后天 → 先天 → 宗师 → 大宗师 → 圣者。
- 陈凡真实修为: 装废五年, 真实段位至少 先天 顶层 (ep01 锁), 上限留待 ep02+ 揭穿。

## 4. 关键地理

- **京都 (无名)**: 王朝都城, 陈国公府所在地, 皇宫所在地。
- **陈国公府**: 京都内, 一座大型世家府邸。本剧 ep01 接旨场为府内正厅。
- **皇宫 / 朝堂**: ep02+ 触及, 当前仅"陈国公需亲自进京请罪" 隐含其方位与威严。

## 5. 时间线 (lore)

- **5 年前**: 女帝赐婚陈凡。陈凡同期开始装废 (出于何种动机, ep02+ 揭穿)。
- **ep01 当前**: 女帝下诏退婚, 陈凡内心狂喜, 五年装废至此终结。
- **后续 arc**: 陈凡 真实身份逐步显露 → 朝堂势力震动 → 女帝 见识真容后悔婚 → 陈凡 已不愿。
