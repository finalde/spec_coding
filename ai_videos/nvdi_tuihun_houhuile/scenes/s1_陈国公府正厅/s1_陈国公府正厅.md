### 命名 / 导入约定 (供导入功能按文件名归位)

**实测**: jimeng / 即梦等出图工具**不**用 prompt 首行做下载文件名, 而是取 **prompt 正文** (`主体:` 行) 的前段, 如 `jimeng-2026-06-01-3688-陈国公府正厅 朝北视角，相机自南厅门方向…png`。故归位**不能**靠「首行 key = plate_id」, 而靠正文里**唯一可靠的「方位词」**。

- **方位段是路由键**。folder 命名 `bg{N}_{方位}_{描述}`, 其**方位段** (`朝北`/`朝南`/`朝东`/`朝西`/`高位俯瞰`/`案前`) 必出现在 `主体:` 行开头 → 必进下载文件名。导入+重命名功能提取 `bg\d+_` 之后的**第一段** (= 方位) 做子串匹配, 命中即归位到该 folder。
  - ⚠️ **只匹配方位段, 不匹配描述段**: 描述段 (如 `厅门`/`长案`/`东侧`) 会作为相机走位词散落在别的朝向文件名里 (如「朝北…自南**厅门**方向」含 `厅门`), 匹配描述段会串。方位段两两互斥, 不串。
- **场景视频**: 文件名含 `陈国公府正厅` 但**无方位词** (如 `s1_陈国公府正厅.mp4`) → 归位到 scene 根 folder `scenes/s1_陈国公府正厅/` (覆盖全朝向的源视频, 后续所有朝向图与 shot 视频的 reference)。
- **归位后自动重命名**: 导入器把 PNG move 进对应 bg folder 后, rename 步骤按父 folder 名改名 → `bg{N}_{方位}_{描述}.png`, 与 prompt / folder 同名。
- 六张朝向图均以 `s1_陈国公府正厅.mp4` 为共同参考源, 保证跨图材质 / 配色 / 时辰光源一致。**纯背景, 不含人物。**
