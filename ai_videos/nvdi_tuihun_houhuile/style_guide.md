## 镜头语言关键词字典 (rule #12.4 镜头取词契约)

### 景别
| 关键词 | 含义 |
|---|---|
| 远景 | 全身 + 大量环境 |
| 全景 | 全身 + 适量环境 |
| 中景 | 腰部以上 |
| 中近景 | 胸部以上 |
| 近景 | 肩部以上 |
| 特写 | 面部 / 物件细节 |

### 运动
| 关键词 | 含义 |
|---|---|
| 锁机位 | 完全静止机位 |
| 推 | 镜头向被摄主体逼近 |
| 拉 | 镜头远离被摄主体 |
| 摇 | 机位不动, 镜头水平/垂直转向 |
| 移 | 机位平行于主体水平/垂直移动 |
| 跟 | 机位跟随主体移动 |
| 升降 | 机位垂直升起/下降 |

## 调色 · 光位 (ep01 接旨场)

- 主色调: 沉香木 + 诏书 + 地砖 + 烛光
- 时辰: 日间 (上午接旨) · 4500K-5000K 顶左侧斜射, 透过府厅高窗格栏漏光
- 氛围: 庄重压抑 + 仪式感 + 单方面 reveal (观众随陈凡内心 OS 看到反差)

## 字幕规范 (rule 12.4 三选一: 内嵌硬字幕 / 后期软字幕 / 默剧)

默认 **后期软字幕**: shot prompt 的 `台词 / 字幕:` 字段写出对白与字幕时间窗, 视频生成阶段不烧字幕, 由 user 在 CapCut / 剪映 后期加入。
- 字幕字体: 思源宋体 / 古风衬线
- 字幕位置: 画面下 1/6, 居中
- **内心独白** 用斜体区分自外部对白。

## Motif (本剧专属视觉符号)

- **诏书展开 motif**: 朱红诏书在镜头前缓缓展开, 烫金字若隐若现 (用于 ep01 cold open + 后续 fakeout 回放)。
- **陈凡 reveal motif**: 外人视角看陈凡是含胸低头废柴姿, 镜头转到陈凡正脸/特写时眼底锐光一闪即收 (单方面 reveal motif, 多次复用)。
- **女帝缺席 motif (ep01 only)**: 全集女帝不出场, 仅以诏书 / 字幕 / 太监口述存在, 强化"她看错了人"的悬念铺垫。
