## 1. 修炼境界体系

本世界沿用经典 8 阶修真阶梯：**练气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘**（终阶取「大乘」，刻意与 baseline 中 `wode_moni_changsheng_lu` 的「渡劫」收束错位，避免季终节拍撞车）。每一大阶内部分 9 层小阶段（如练气 1 层至练气 9 层），层间为线性灵气累积；大阶之间则需以「关」过渡 — 关上有大变，过则脱胎换骨，过不去则修为停滞或反噬。该 9 层 + 8 阶结构是 ≥ 5 本基线共有的修真公共资产，可放心沿用。

- **练气**：吐纳引气入体，灵气在经脉中循行；寿元延至 90–100 岁，可强身辟谷。九层后期可凝气成形，能御小型法器。
- **筑基**：以心火淬体、灵气筑成丹基，肉身骨骼初炼如玉；寿元延至 200 岁，能御中型飞剑，运法力可数十里行走。标志能力 — 御物驱风。
- **金丹**：丹基孕成内丹，气海中一颗灵丹凝形；寿元延至 400 岁，肉身可硬抗下品法宝；标志能力 — 一念出剑，剑可破甲。
- **元婴**：金丹破开，元婴破丹而出，气脉与神识独立成形；寿元延至 800 岁，元婴外出可独立行动；标志能力 — 神识攻杀，元婴出窍。
- **化神**：元婴神化，气与神合一，可短暂窥见天地法则；寿元延至 1500 岁；标志能力 — 法则映照，可短驻虚空。
- **炼虚**：神化之体进一步虚化，肉身灵气可逆向化为「虚体」过山穿石；寿元延至 3000 岁；标志能力 — 化虚遁形，分身可立。
- **合体**：神与虚合，肉身可与天地灵气短暂合而为一，调动方圆百里灵气如臂使指；寿元延至 6000 岁；标志能力 — 借天地之力一击。
- **大乘**：超脱凡境之极，万法皆通而万法皆空；寿元理论无尽，实战寿元仍以「关」之代价不断削减；标志能力 — 一念可裂虚空，但每用一次寿元剧损。

## 2. 修炼物理规则

**通则（所有修士共用）**：灵气是有形之物，可吸纳、可储存、可外放；修炼是消耗时间、灵药、灵石、机缘的累积过程，每过一阶必经一关，关上有「劫」（雷劫 / 心劫 / 形劫 / 因果劫），渡之即升，渡不过即死或废。

**寄生分歧（仅 裴知秋 一人）**：自 ep01「焚寿罗盘」嵌入胸前那一刻起，主角 裴知秋 的每一次境界突破额外承担两项代价 —

1. **寿元代价**：「焚寿罗盘」共 24 格，每升一阶（含层间小突破中具有标志意义的关键一次），罗盘上转黑 1 格，代表本应延寿的「关」之红利被反向吞噬 — 不仅不延寿，反而扣 N 年寿元（N 由当前阶位决定，越高阶扣得越狠）。罗盘 24 格转黑之日，便是 裴知秋 寿命兑尽之时（理论刻度对应大乘终关，剧情上落在 ep60 季终）。
2. **记忆代价**：每升一阶，系统额外抽走 裴知秋 一段 episodic memory（具情节、有人物、有情绪的完整记忆段）。系统并非随机取，而是按「情绪显著性」从高到低、由温至烈逆向取 — 温柔的、深爱的、被珍视的先被吃掉；憎恨的、痛楚的、复仇驱动的留到最后。剧情对应 — 主角越升越强，越想不起为什么要复仇；只有「恨」最不肯走，是最后一根撑着他走完 24 格的脊梁。

寄生系统对外不可见：旁人看 裴知秋 只觉「天才横空出世、但身骨虚浮、神情时有恍惚」，无人能察觉他每升一阶都在燃烧自己 — 这是本剧最大的视觉孤独感来源。

## 3. 三方势力 + 朝堂格局（四方）

xianxia baseline 沿用「正派联盟 / 散修盟 / 魔门」三分格局（≥ 5 本基线共有，安全公共资产），本剧在此基础上 **额外引入「朝堂 · 太师府」作为第四方** — 凡俗朝堂上的修真势力，提供「应砚之」作为正派 + 朝堂双线背叛者的政治根脉，让 ep17 后期主角对应砚之的清算可同时打在「正派宗门权位」与「凡俗朝堂势力」两条线上，是本剧政治深度的支撑层。

### 3.1 正派联盟（三宗）

三宗共同奉「悬剑盟约」为正派最高公约 — 每六十年于「澹台宗」山顶共议天下大势，三宗掌门轮值盟主。三宗之间表面联盟，内里互相牵制，赤霞门尚剑、九寰阁尚智、澹台宗尚静，三家路数不同、长期角力。

- **赤霞门（剑宗）**
  - 立派之祖号「赤霞剑祖」，传说在晚霞燃尽前的一瞬以一剑断天，开宗立派。
  - 现任掌门即五大背叛者之一 **卫长烛**（前世 裴长砚 的同门师弟，最早叛主之人）。门内分外门 / 内门 / 嫡传三层。
  - 内部政治：嫡传弟子党与外门长老党的代际冲突；卫长烛靠「长烛烧不熄」的虚伪人设维持权位。
  - 标志视觉：山门为悬于绛红色断崖之上的剑形大殿，每至落日，整宗如赤色火炬；门人着红黑剑袍，腰悬一柄不出鞘的赤霞剑。

- **九寰阁（藏书 + 阵法宗）**
  - 立派之祖以九重圆环阵镇压一次域外妖患，于云海中筑九重圆环玉璧大殿为门。
  - 现任阁主非背叛者群 — 是个超脱具体派系党争的学究型长老。藏书阁掌一界阵法 / 古籍 / 灵符的源头。
  - 内部政治：藏书阁与阵法堂的资源分配争夺；阵法堂偏实战、藏书阁偏考据。
  - 标志视觉：九重圆环形玉璧大殿，悬于云海之间；门人着月白长袍，手执玉简或竹简，发束墨玉冠。

- **澹台宗（御剑冷静派）**
  - 立派之祖号「澹然真人」，以「水静则鉴」立宗 — 御剑而不求快，求静。
  - 现任宗主非背叛者群 — 是位寡言冷静的女性掌门，与主角无直接恩怨。
  - 内部政治：与赤霞门的「快剑 vs 静剑」长期道争；门内严禁结党。
  - 标志视觉：山顶青石高台 + 一柄横置长剑，云海漫过台沿；门人着青灰长袍，腰悬细身御剑。

### 3.2 散修盟 — 流烛盟

「流烛盟」是散修群体的松散自治组织，表面与三大正派宗门平起平坐，实则结构上更接近一处「散修议事场」 — 各路无门无派的修士以盟约相约互不相杀、共享坊市与交易、有事议事大堂集议。盟主之位每十年轮换，名义上无实权。

- **现状**：盟内长期由数位「元老」实际把持，其中即 **戚归砚**（背叛者 #3，散修身份 + 暗投魔门）与 **池洇**（背叛者 #4，杀手长老）。盟主之位形同虚设。
- **内部政治**：散修个体主义强，结盟松弛；任何「以盟之名做大事」的提议都会被否决；但小圈子结党极多。
- **标志视觉**：盟门已破败 — 一道半坍的石牌坊上「流烛盟」三字被风雨侵蚀；议事大堂内一根永远燃着的「流烛」烛火越烧越短，象征散修群体「无门无派、自燃自灭」的宿命。门人着土褐麻布、草绳腰带、不戴宗门徽记。

### 3.3 魔门 — 忘川教

「忘川教」是本剧母题载体 — 它的核心功法《偿岁真言》以「他人之寿元」为偿付货币，本质上即是本剧寄生系统的源头。该教隐于「忘川河」（一条永远在雾中流淌的黑色之河）河谷的黑色石塔群中。

- **立教之始**：传说立教之祖以一己之力盗了某位上古大乘真君的「岁数」为修，活了千年；其传承即《偿岁真言》。教中以「偿岁」为最高交易方式 — 弟子之间互相买卖寿元、记忆、命数。
- **现任教主**：**言息**（背叛者 #6，最终 BOSS）— 三长老 **阮惘** 为副。教内有「偿岁堂」（功法传承）、「忘心堂」（洗忘他人记忆）、「石塔堂」（守教派门阵）。
- **内部政治**：教主言息以「偿岁契约」绑死所有长老 — 每位长老的寿元都已抵押给教主，背叛即寿尽；这是言息维持权位的方法。也正因如此，反叛言息只能从「外人」身上找突破 — 这是 裴知秋 后期能策反阮惘的剧情底层逻辑。
- **标志视觉**：忘川河边的黑色石塔群，雾色永驻；门人着深紫黑长袍、骨白点睛、面绘金纹；教中所用法器多为骨白与漆黑配色。

### 3.4 朝堂 · 太师府（凡俗朝堂上的修真势力）

本剧另立「朝堂 · 太师府」为修真势力的第四方 — 既非宗门、亦非散修、亦非魔门，而是 **凡俗朝堂上的修真世家**。太师府世代有修真者出仕，修真兼任太师之职，掌国家典制兼控修真界与朝堂的双向沟通通道。

- **现任太师**：即 **应砚之** 之父 — 是一位筑基后期 / 金丹初期的世家修士，于凡俗朝堂任太师，于修真界以「世家系」自居，与三宗保持距离、与流烛盟有交易、与忘川教暗中通信。
- **应砚之 的政治根脉**：应砚之以「太师之嫡子」身份继承家系修真传承，前世为 裴长砚 挚友、亦是正派宗门内的世家代表；背叛之后被父亲举荐入朝堂任要职，成为「正派 + 朝堂」双线人物。这一身份让本剧后期对其清算可同时打掉他的「正派权位」与「朝堂权位」两条线，是本剧政治深度的支撑层。
- **内部政治**：太师府作为世家系，与三宗的「宗门系」长期道争 — 宗门系强调「修真者超脱凡俗」，世家系强调「修真者亦应入世任事」。这一道争是本剧朝堂线的核心张力。
- **标志视觉**：太师府位于凡俗京城之中，为一座层叠青砖灰瓦的大宅；正门悬「太师」匾额，宅内有暗藏的修真灵脉与小型法阵；府中下人皆凡人，唯应氏家系修真者出入。

## 4. 地理与时代背景

本剧时代基调为**中古修真 + 朝堂并立**，无现代元素，无穿越，无外来灵魂 — 主角 裴知秋 即为本世界本土修士，前世名为 裴长砚（ep17 揭穿）。地理上以「澹江洲」为主舞台，三宗山门 + 流烛盟 + 忘川教 各居一方，朝堂在凡俗京城之中，整体形成「四方地理 + 一座京城」的格局。

### 4.1 主要地名

- **澹江洲**
  - 裴知秋 本世出生地。一片江水冲积平原，灰白苇草满洲，远山隐约。江名「澹江」，水清而静，地势平缓。
  - 与澹台宗同享「澹」字 — in-fiction 解释为澹台宗立派之祖即出自此洲，故以洲名为派名。
  - 此处为本剧第一阶段（ep01–ep15）的主要居所。

- **落雁渊**
  - 澹江洲西陲一道深陷的山渊。渊口狭窄，渊底深邃，渊底散布大量白色与灰色的鸟骨（传说为远古飞禽集体坠亡之地），与丛生的灰白苇草形成强烈视觉。
  - **ep01 系统觉醒地** — 裴知秋 自渊底苏醒，焚寿罗盘 自胸前浮出。

- **栖梧崖**（旧名「无寿崖」）
  - 前世 裴长砚 的坐化处 — 一座白梧桐林环绕的悬崖，秋季黄叶满地。崖上常年有冷风。
  - 旧名「无寿崖」直接昭示「寿元尽于此」的命运暗示；前世 裴长砚 渡劫被剑刺穿、寿元终结即在此崖。
  - 死后该崖被三派联手以「不祥」为名改名「栖梧崖」（凤栖梧桐之意），用美名掩盖原意。
  - **ep17 揭穿名义改写、ep49 主角回归**。

- **忘川河谷**
  - 北边迷雾常年不散的一道河谷，忘川教的石塔群即隐于此。河水黑色，无生灵。
  - 凡俗修士轻易不入；河谷入口有忘川教守护石塔阵。

- **凡俗京城**（不另立地名，剧中只称「京城」/「都」）
  - 朝堂 · 太师府所在；与三宗山门、流烛盟、忘川教共构「四方一城」的格局。
  - 应砚之 长居于此。

### 4.2 三派 + 朝堂地理布局

粗略地理（≥ 2 shot 复用前不另立 scene 文件，本档作为地理参考）：

- **赤霞门** 居澹江洲东南方一处绛红色悬崖之上。
- **九寰阁** 居澹江洲北方云海之中。
- **澹台宗** 居澹江洲西部山顶之上。
- **流烛盟** 无固定山门，盟会议事大堂分散于澹江洲各小镇坊市间，但有一处「主堂」位于澹江洲中部。
- **忘川教** 居澹江洲北边迷雾河谷（忘川河边），与九寰阁同属北方但互不交通。
- **朝堂 · 太师府** 居凡俗京城，地理上不属修真界主舞台，但通过应砚之家系与三宗、流烛盟、忘川教均有牵连。

### 4.3 时代基调

中古修真 + 朝堂并立。无现代器物（无电、无机械、无玻璃）；无穿越者；无外族外神。主角是本世界土生土长的修士，其「重生」是寄生系统提供的本世界内时间循环，并非跨世界灵魂转移。这一设定让本剧的「寄生代价」具有真正的悲剧重量 — 主角失去的记忆与寿元都是「这一具肉身、这一份命」的真实损耗，没有外挂、没有重来。

## 5. 寄生系统 in-fiction lore（全貌 — 作者必须懂全部，观众只看片段）

本节为 stage-6 ep01+ 的 prompt 与 dialogue 必须背诵的「真相全貌」 — 观众在每集只能看到一个片段，作者必须知道完整链路，才能在 ep01 就埋下后期能落实的种子。

### 5.1 ep01 现象层（观众所见）

ep01 0:00–0:03：前世 裴长砚 于栖梧崖（旧名「无寿崖」）渡劫，雷劫第三道劈下时，背后被剑刺穿。

ep01 接 0:30：本世 裴知秋（7 岁练气体）苏醒于澹江洲西陲的「落雁渊」渊底，胸前一道白光浮起 — 一面 24 格的青铜罗盘嵌入皮下，盘面浮出红字「焚寿罗盘 · 已激活 · 寿元 -1 / 修为 +1」。

观众视角：这是一件系统类法器，自动生成。

### 5.2 ep17 揭穿（罗盘的来历）

ep17 主角通过「归砚镜」（师父闻砚清遗物）回看前世死亡现场，看到罗盘其实是 **师父 闻砚清 的遗物** — 闻砚清在前世 裴长砚 渡劫被剑刺穿的那一瞬，从远处一掌印过来、将罗盘从自己识海中送入濒死的 裴长砚 体内。罗盘是「契约残片」的物化 — 表面是一件法器，本质是一份契约。

观众视角：师父原来不是叛徒，反而是想救他。

### 5.3 ep30 揭穿（契约的源头）

ep30 主角通过「归砚镜」拼回更多前世记忆，得知罗盘之契约源自 **忘川教 的核心功法《偿岁真言》** — 该功法以他人寿元为偿付货币，是修真界的禁忌之术。主角现修的功法《残忆经》正是被改写过的《偿岁真言》盗版 — 但改写者不是别人，是 **前世的 裴长砚 自己**。

观众视角：原来主角自己就是源头。

### 5.4 ep49/50 揭穿（前世的真相）

ep49 归砚镜拼回最后几段记忆，主角看到 — **前世的 裴长砚 自己创造了《残忆经》**。他以《偿岁真言》为底本，反向改写，企图制造一份「反吞」契约 — 让 忘川教教主 言息 修习时反吞己身寿元与记忆，以一人换一教。这一计划被五大背叛者（卫长烛 / 应砚之 / 戚归砚 / 池洇 / 阮惘）联合察觉，他们在栖梧崖渡劫之夜围杀 裴长砚 — 雷劫为天劫，剑光为人祸；契约在反噬中失控，反过来寄生于其本身。

ep50（付费节点 3）：主角彻底接受「我即系统之源」的事实 — 寄生系统不是外来神迹，是他亲手设计的反噬武器，用错对象、反噬自身。

观众视角：主角原来是受害者亦是加害者，命运闭环。

### 5.5 ep60 兑现（最后一格转黑）

ep60 季终：主角击败 言息，焚寿罗盘最后一格自动转黑 — 因为这是「大乘」终关，按契约 24 格之约，全部代价兑现。代价的最后一笔是 **关于 容漪 的最后一段共同记忆** — 主角在战胜 言息 的那一瞬，看着 容漪 站在面前，但她在他眼中从「恋人」变成「陌生人」 — 他不再认识她。

季终钩：《偿岁真言》的残片寄居于 容漪 体内（ep28 揭穿她是忘川教安插的「活体记忆库」） — 寄生契约虽于 裴知秋 处兑尽，但已转生到她身上。第二季的契约从这里开始。

## 6. 戚归砚 dual-affiliation（双重身份明文）

**戚归砚** 在本剧中具有 **明确的双重身份** — 这是 ep17 之前作者必须暗埋、ep17 之后必须揭开的关键设定：

- **公开身份**：流烛盟 元老（散修身份） — 表面上是散修群体中的资深长老，与 裴知秋 在 ep05 散修盟交涉时即以「友善长辈」姿态出现，对主角主动示好。
- **真实身份**：忘川教 暗线 — 与 言息 早有「偿岁契约」，是 忘川教 安插在 散修盟 内部的卧底。其在 ep17 之前对 主角 表面亲近，实为受 言息 之命搜集「焚寿罗盘」情报、确认主角是否为前世 裴长砚 转生。
- **背叛节点**：ep17 揭穿前世 裴长砚 名义改写时，戚归砚 的双重身份同步暴露 — 主角通过「归砚镜」回看前世，发现这位「散修元老」其实是当年围杀自己的五人之一。
- **后期清算**：因戚归砚 同时挂在 流烛盟 与 忘川教 两条线上，主角对他的清算可双线打击 — 既打掉 流烛盟 元老党，又斩断 忘川教 在 散修盟 内的眼线。这一双线设计是本剧中后期（ep30–ep45）剧情张力的重要支柱。

stage-6 character bible `characters/c8_戚归砚/c8_戚归砚.md` 必须明文记录这一 dual-affiliation；ep01 倒叙 0:03–0:30 montage 中，戚归砚 的「reasons-to-trust → reasons-to-die」对照中，其「reason-to-trust」必须是「流烛盟元老身份」，其「reason-to-die」必须暗示「他是忘川教的人」 — 这一对照是 ep17 揭穿的视觉根脉。
