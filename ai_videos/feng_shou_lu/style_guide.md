# 《焚寿录》视觉风格指南（style_guide.md）

> 本风格指南是《焚寿录》的视觉契约 — 镜头取词 / 调色 / 光影 / 转场 / 字幕 / 寄生升级 motif / 负向锁定 共同构成统一视觉语言。每个 shot prompt 在 `渲染样式:` 和 `负向:` 字段从本文件 verbatim 粘贴；任何字段调整须更新本文件，不在 shot 级别新造词（per ai_video.md rule #12.4 v4 「镜头取词契约」）。

---

## 1. 整体视觉锚（视觉北极星）

《焚寿录》的视觉北极星是 **「灰青底色 + 寄生紫签名 + 寿命红代价」三元结构**：通篇基调以水墨灰青为底，主角寄生觉醒态独占寄生紫与寿命红两条专属签名色——前者是「系统介入」的视觉标记，后者是「代价兑现」的视觉收据。任何 shot 离开本三元结构都属于错调，stage-6 validator 会以此为基准 grep。

整体审美锚点：**亚洲俊男靓女写实 + 影视级真人质感 + 9:16 竖屏单镜 ≤ 3 人主体铁律**。禁止 AI 通用脸 / 西方审美面孔 / 二次元 / 塑料皮肤 / 玩偶感。所有渲染样式必须能与《琉璃》《长月烬明》《苍兰诀》《沉香如屑》四档实景仙侠剧的视觉记忆并行不撞档。

本剧不强调高速御剑或大规模群战（弱项规避，per angle-visual_style §2.3）；强调 **「升级 → 寿减 → 记忆缺失 → 复仇推进」** 的私人化、内压式镜头语言。每一镜的「主体定义 / 光线 / 节奏」三件套必须能向「焚寿罗盘」总叙事钩回——任何与寄生代价无关的视觉装饰都属于冗余。

### 2.3 调色板使用守则

- 任意单 shot 不得同时使用 ≥ 3 张派系调色板；越界即视觉混乱。
- 主角寄生觉醒态出场时，所在派系调色板降饱和 30%，让寄生紫 + 寿命红浮出为视觉主点。
- 朝堂 · 太师府调色板与正派联盟调色板可同帧（应砚之 dual-affiliation 戏份），但朝堂沉檀木为底、正派银白为辅。
- 焚寿罗盘 cold light 出现时，画面整体色温向 6500K 冷漂移 0.3s，背景失色作为「系统介入」的视觉提示。

### 4.1 时间段配比

- **晨**：重生灰晨为主（主角觉醒戏 / 散修期日出）；色温 5000K，肤色微虚。
- **日**：仙气（正派）/ 散修土褐 / 灵田晨光为主；中段亮度，对话戏稳定光位。
- **暮 / 月夜**：月夜冷峻 / 朝堂烛影；4000–4500K，反派对峙首选。
- **夜**：魔门紫黑骨白 / 寄生 aura；3500K 以下，全片寄生戏份 60% 落在夜或室内夜。
- **焚寿罗盘冷光** 不绑定时间段，是系统介入的瞬时光位，任何时段均可叠加。

## 5. 服装 + 道具 visual grammar

### 5.1 主角双形态契约（production hook）

| 形态 | 出现时机 | 体态 | 肤 / 瞳 / 唇 / 发 | 服饰 hex | aura |
|---|---|---|---|---|---|
| **重生弱体 state A** | ep01–ep05 + 任何回忆 | 微瘦 + 含胸 + 步伐迟（病初愈） | 肤色虚浮 / 瞳暗褐 / 唇色淡 / 发褐黑微乱 | 麻布长袍 + 草绳腰带 | 无 aura；自身散弱光 |
| **寄生觉醒 state B** | ep05+ 系统首次激活后 | 直立 + 步轻 + 微飘起（修为内涌） | 瞳由暗褐 → 寄生紫 / 唇色冷紫 / 发漆黑束半发 | 同款麻布外披 + 内衬暗纹中衣 | 黑紫细丝自指尖溢出 + 寿命红计数器在系统弹窗 |

**关键契约**：两 state 必须共享 face-differentiator — 主角候选为「左眼下方 0.3cm 灰青胎记」（state A 半隐 / state B 在寄生紫调下显现为冷紫色斑）。stage 4 character bible 定稿，stage-6 validator 强制扫描「两 state 同人识别」。

### 5.2 寄生升级 motif — 5s 三拍序列（production hook）

每次升级触发，时长固定 5s，作为可复用 motif。shot prompt 复用「寄生系统升级 motif v1」标签，byte-stable 取词。

| 拍 | 时长 | 视觉 | 音 / 字 |
|---|---|---|---|
| 拍 1 — 系统提示 | 0–1.5s | 金字弹窗自下浮起 `叮——任务完成 / 修为 +1 阶` + 寿命红计数器跳动 `寿命 -1 格` | 内嵌硬字幕黑底金字 |
| 拍 2 — 寄生 aura | 1.5–3.5s | 主角双瞳由暗褐 → 寄生紫一闪 / 黑紫细丝自指尖逆向缠手腕 / aura 由暖橙转冷紫 / 背景失色 0.3s | 顿挫节奏（静默 0.5s） |
| 拍 3 — 寿命流失 | 3.5–5s | 主角嘴角血迹一线（薄）/ 鬓边骤白一缕 / 皮下青脉浮现 0.5s / 字幕「代价已扣除 — 记忆 1 段」淡入淡出 | 内嵌硬字幕红字 |

motif 在 ep01–ep05 各出现 1 次（per qa.md 5 集 detail batch）；后续每集出现频率 1–2 次为佳，避免审美疲劳。

### 5.3 服饰 / 道具 fabric · silhouette · 签名继承规则

- **正派联盟**：素白 / 月白 / 浅青长袍 + 银线云纹绣边；男性束发玉冠，女性盘髻簪银钗。布料为细绫 / 薄缎，悬感强。signature 道具：长剑（赤霞门）/ 玉璧 + 简册（九寰阁）/ 青石腰牌（澹台宗）。
- **散修江湖（流烛盟）**：麻布 / 粗葛 / 土褐外披 + 草绳腰带；男女通用宽袖短打、行脚草鞋。signature 道具：流烛灯（盟纹灯笼，烛火将尽）/ 散修玉牌（粗石材质）。
- **魔门（忘川教）**：黑色长袍 + 深紫暗纹刺绣 + 骨白领口 / 袖口；半截黑纱面巾。布料粗厚有重量感，长袍拖地。signature 道具：黑色寿牌（用以「偿岁」契约登记）/ 黑石塔形吊坠 / 忘川河碎冰刃。
- **朝堂 · 太师府**：沉檀木色官袍 + 御金腰带 + 朱漆冠饰 + 玉笏；男性面留短须，女性盘髻插翠钗。布料为厚锦 / 暗花缎。signature 道具：太师印 / 玉笏 / 朝靴。

- **戚归砚 dual-affiliation**：表层服饰为流烛盟散修土褐打扮（per §2 调色板 #2），内衬一抹暗纹仅在风吹翻领时一闪；signature 道具：双牌——明面流烛盟玉牌 + 内袋忘川教黑石塔吊坠。揭穿戏份（ep17）后服饰可整体切换为忘川教调色板 #3。

### 5.4 道具继承铁律

- 焚寿罗盘 全剧唯一专属，禁止重设 / 双胞 / 复刻；任何「罗盘进入画面」必填 §2.2 焚寿罗盘 cold light 光位。
- 角色 signature 道具一经 character bible 锁定，shot prompt 不得在 shot 级别新造 / 替换。
- 道具配色必须从所属派系调色板取色，不得跨派系混用。

## 6. 字幕 / 台词 + 标题卡 typeface + safe area

### 6.1 三选一契约（per ai_video.md rule #12.4 + 本剧 §6.2 默认锁定）

每个 shot 的「台词 / 字幕:」字段须明确声明三选一之一：

- **内嵌硬字幕**：字幕作为像素直接烧入视频帧，AI 渲染时已可见。
- **后期软字幕**：AI 渲染产物不含字幕，由人类操作员在剪映 / Premiere 阶段叠加。
- **默剧**：本镜无台词、无字幕；以镜头语言 + 表情承载叙事。

### 6.3 字体规范

| 用途 | 字体 | 字号 | 颜色 |
|---|---|---|---|
| 中文主字幕 | 方正粗黑简体 | 32–36 pt | 白 + 黑描边 |
| 中文系统弹窗 | 宋体 / 楷体（保古风感） | 24–28 pt | 金字主 + 银边 |
| 中文寿命计数器 | 宋体加粗（fallback 方正粗黑） | 28–32 pt | 寿命红 + 黑描边 |

| 英文（YouTube Shorts post-v1） | Helvetica Neue Bold | 同步主字幕 | 同主字幕 |

### 6.4 位置 + 安全区

- **常规字幕**：y ≈ 1100–1180（9:16 最底部）；左右安全区距帧边 ≥ 60 px。
- **系统弹窗**：y ≈ 200–400（顶部至上 1/3 线）；居中弹出。
- **集头标题卡**：满屏居中 0.8–1.2s 淡入 + 1.2s 停留 + 0.8s 淡出。
- **集尾 cliffhanger 钩字**：y ≈ 600（黄金视觉中心）；闪烁 / 跳动动画。
- **抖音封面安全区**：底部 240 px + 左右各 60 px + 顶部 200 px（账号头像 + 标题位）— 任何关键视觉元素不得落入该区。
- **每屏字幕最短停留**：≥ 1.5s（保证可读）。

### 6.5 标题卡 typography

ep01 集头标题卡示例：

- 主标题：`第一集 · 落雁渊` — 楷体 56 pt 月白 + 寄生紫描边；居中 y ≈ 600。
- 副标题：`焚寿罗盘 · 觉醒` — 宋体 28 pt 金字 + 黑描边；居中 y ≈ 720。
- 背景：落雁渊 渊底全景慢推 1.5s，叠加 30% 黑色半透层保证字可读。

### 8.2 三选一字幕字段在 §6 的语义引用

`台词 / 字幕:` 字段的三选一不在 shot 级别决策，而是按 §6.2 类别表查表填写：

- 凡涉及系统弹窗 / 集头标题 / 集尾 cliffhanger / 黄金钩台词 → 强制 `内嵌硬字幕`，并附完整字模板（颜色 + 字号 + 位置 from §6.3 + §6.4）。
- 常规对话 → 默认 `后期软字幕`，prompt body 只写出台词文本，标注「post 阶段叠字」。
- 纯视觉戏 → `默剧`，字段值为 `默剧（无台词）`。
- 任何 shot 违反 §6.2 类别表（例：把系统弹窗写为后期软字幕）= stage-6 validator 拒收。

### 8.3 Kling vs Seedance 选用规则

- ≤ 10s 单 take：两者皆可；有角色 turntable reference 优先 Kling image-to-video；无 ref 优先 Seedance text-to-video。
- 11–15s 单 take：优先 Seedance 1.0 Pro；走 Kling 须 mid-seam frame 切片 + 二次拼接。
- 升降镜 > 1 画面或环绕 > 180°：Seedance 优先。
- 寄生升级 motif（顿挫 + match-on-eye）：建议 Seedance（hard cut + temporal beat 控制更稳）。
- 任意 shot 同时产出 `shotNN_kling.md` + `shotNN_seedance.md` 两文件（per ai_video.md FR-9 / FR-10），人类操作员择优渲染。

## 11. 9:16 竖屏构图铁律

- 每镜 ≤ 3 人主体（多于 3 人主体即视为构图过载，须拆 shot）。
- 推镜 / 拉镜占主，横向运镜禁（per §3.2 禁忌行）。
- 主体居 y 中线，重点信息（脸 / 道具 / 系统弹窗）落 y ≈ 200–1000 安全区。
- 抖音 / 红果封面安全区：底部 240 px + 左右各 60 px + 顶部 200 px 留给账号头像 / 标题 / 互动按钮，关键视觉元素不得落入。
- 字幕避让区：y > 1000 永不放主体脸或关键道具。
- 标题 / 封面帧（per FR-9 cliffhanger shot）显式标注 `cover_frame: true`，构图须满足封面安全区 + 主体可识别度（即 1/4 缩略图依然能辨主角脸）。
