# 裴知秋 · 男主 / 重生者 (本世)

## 角色定位

《焚寿录》男主角 / 第一视角。前世名「裴长砚」（ep17 揭穿），上古剑修大乘修为，于栖梧崖（旧名「无寿崖」）渡劫被五大背叛者围杀剑刺穿死亡 — 实为自设《残忆经》反吞契约困住忘川教时被察觉而遭围杀 (ep49 / ep50 揭穿)。本世重生为澹江洲穷童裴知秋，7 岁练气体于落雁渊渊底苏醒，胸前嵌入 24 格青铜「焚寿罗盘」— 系统每升一阶吞噬 1 年寿元 + 1 段 episodic memory，按情绪显著性自温至烈逆向取，温柔记忆先失、仇恨最后留。

剧中位置贯穿 ep01–ep60 全 60 集，每集必出场。与师父闻砚清的关系是「以为被叛 → 真相是被救」的最大 reversal；与主女主容漪是「盟友 → 忘川教记忆备份 → 共苦盟约 → 季终共同记忆全失」的悲剧弧；与五大背叛者（卫长烛 / 应砚之 / 戚归砚 / 池洇 / 阮惘）+ 最终 BOSS 言息是 60 集逐一清算的复仇靶标。段位档从练气 1 阶（ep01）线性推到大乘成（ep60），24 ticks 寿元 ledger 在 ep60 兑尽。

## 性格 / 动机

- **核心动机**：以罗盘 24 格烧尽前的有限寿元手刃六个背叛者（卫长烛 / 应砚之 / 戚归砚 / 池洇 / 阮惘 / 言息），并查清寄生系统的真正源头 — 直到 ep50 真相揭穿「我即系统之源」后，动机进化为「以最后一搏为容漪与天下了断这场契约」。
- **表层人设**：一个含怯虚浮、神情时有恍惚的「天才横空出世」穷童 — 外人看是神童早慧，无人察觉他每升一阶都在燃烧自己。
- **反差点**：state A 麻布草绳的弱体形态下藏前世大乘剑修千年识海；state B 寄生紫瞳冷峻杀意下藏对容漪未失却必将失去的温柔记忆。
- **关键弱点**：(1) 寄生系统按情绪显著性自温至烈逆向取记忆 — 越升越强，越想不起为什么要复仇；(2) 焚寿罗盘 24 格是硬倒计时，ep60 必至兑尽；(3) 容漪是「忘川教 planted memory backup」(ep28 揭穿) — 既是盟友也是契约延续的活体载体，主角无法两全；(4) 自己即系统设计者 (ep50 揭穿) — 复仇靶标最后一刀必须落在自己。
