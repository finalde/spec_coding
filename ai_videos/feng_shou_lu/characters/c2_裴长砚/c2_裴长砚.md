# 裴长砚 · 男主前世名义档案 (archived · 前世人物)

> **Archive 标记**: 本档为 archived 前世人物记忆载体, 仅在 ep01 cold open / ep08 倒叙 / ep17 残页揭名 / ep49 + ep60 镜中对视 五个 ep01-范围-外的时点出场。不进入 60-ep 累计 ledger; 不主动推动剧情。一句话锁定与本世「裴知秋」共享 face-differentiator (左眼下方 0.3cm 灰青胎记) 以让 AI 在回忆 / 倒叙 / 镜中显形镜头中识别为同一人前世。

## 角色定位

裴长砚是本剧主角 裴知秋 的前世名义档案 — 渡劫期剑修, 师承九寰阁 / 赤霞剑系传承的剑修长辈 闻砚清 (师徒共「砚」字), 为前世剑宗渡劫前辈, 修长清隽的成人男子。前世末期, 他以《偿岁真言》为底本反向改写出《残忆经》, 企图制造一份反吞契约困死忘川教教主 言息。计划被五大背叛者 (卫长烛 / 应砚之 / 戚归砚 / 池洇 / 阮惘) 联合察觉, 在栖梧崖 (旧名「无寿崖」) 渡劫之夜围杀于雷劫第三道劈下的瞬间 — 背后被剑刺穿, 寿元终结。其反吞契约在反噬中失控, 由师父 闻砚清 从远处一掌将契约残片 (即「焚寿罗盘」前世原型法器) 送入濒死的 裴长砚 体内, 转化为本世主角 裴知秋 胸前所嵌的 24 格青铜罗盘。裴长砚 即是本剧寄生系统的真正源头, 也是 ep01 cold open / ep08 倒叙 / ep17 残页揭名 / ep49 归砚镜拼回 / ep60 镜中对视 五个时点的形象载体, 不在本世以「裴长砚」名活动。

## 性格 / 动机

- **核心动机**：以一己之寿元换一教之灭 — 设计反吞契约困死 忘川教教主 言息, 以《残忆经》为反向改写之底本, 企图让 言息 修习时反吞己身寿元与记忆。
- **表层人设**：剑宗渡劫期前辈, 师承 闻砚清, 沉静清隽, 不结党, 不张扬, 与师父共「砚」字传承。
- **反差点**：天才设计师 → 被自己契约反噬 — 反吞计划被五叛察觉, 渡劫之夜雷劫 + 剑光双重夹击, 契约在反噬中失控, 寿元 + 记忆全数转入本世重生的 裴知秋, 自身寿元终结于栖梧崖。
- **关键弱点**：信任未察觉的盟友 — 五大背叛者 (卫长烛同门师弟 / 应砚之挚友 / 戚归砚掌教师弟 / 池洇亲传弟子 / 阮惘护道女剑修) 全部从信任内核突破, 渡劫之夜围杀于无寿崖。
