# 阮惘 · 忘川教三长老 / 魔门叛者 #5

## 角色定位

阮惘 是 主角前世（裴长砚）的护道女剑修，曾于前世共战、在「无寿崖」（旧名「栖梧崖」前称）下救前世主角于剑光雷劫之间 — 后被忘川教「忘心咒」翻策，被言息以「偿岁契约」绑死寿元，入魔门为三长老（隶偿岁堂副手 / 忘心堂主掌）。本世 ep12 九寰阁玉璧大殿，伪装游学者偶遇主角，于玉璧倒影中察觉主角双瞳寄生紫一闪；ep24 以「前世曾救你」为由请求结盟，主角婉拒，阮惘留下针对忘川教的信物（伏笔 ep40 / ep44）；ep30 主角金丹突破时失去「阮惘救命崖下伤口形状」的记忆 — 寿命代价兑现的视觉锚之一；ep43 主角率乌泽+容漪奇袭忘川教边界石塔，阮惘亲自坐镇；ep44 临死自陈「忘心咒翻策」+ 留下针对言息的密钥（伏笔 ep55 化神二品殊死起 + 密钥启动）+ 半截黑纱面巾掉落露全脸 → 右眼角 三点纹烫印特写。她是六叛者中唯一「被吃过的人 / 被翻策的受害者 + 叛者」二重身份载体，配音为 storyteller 6-voice matrix「命数感」一档（低沉 + 慢速 + 尾音下沉）。

## 性格 / 动机

- 核心动机（前世）：护道女剑修 — 出于剑道与情义守护前世主角；崖下救主时是「救人不计代价」的纯粹一击。
- 核心动机（被翻策后 → 本世）：在忘心咒约束下表面执行忘川教教旨，但内心始终留有前世剑修底色 — 以「偿岁契约绑死寿元」的命数感行走于教中，知反叛必死却不曾真心归服。
- 表层人设：忘川教三长老 / 偿岁堂副 + 忘心堂主掌 — 冷静、规矩、命定，半截黑纱面巾遮口含言不外露。
- 反差点：在主角眼中是「前世曾救我之人 + 本世魔门长老」的双面人；在言息眼中是「被忘心咒翻策、寿元已抵押」的工具；在自己眼中是「被吃过、所以认得另一个被吃的」的同病者 — ep24 求结盟的真正动机是「我也是被吃过的人」。
- 关键弱点：寿元已抵押给教主，反叛即寿尽 — 这是言息维持权位的根本约束（per world.md §3.3）；阮惘 的全部行动都在「在抵押期内偷偷留下反言息的种」框架内。
- 沉默承担：从未在主角面前自陈「忘心咒翻策」的具体过程 — 把这份痛压在「命数」二字下，直到 ep44 临死才一次性吐出 + 递密钥。

## 标志台词或口头禅

1. 「我也是被吃过的人。」（ep24 求结盟 + ep30 救命崖下回响 两次同句；沉缓含怅，是她对主角寄生代价的隐性共鸣 + 翻策真相的暗扣 — 命数感配音 storyteller matrix #6）
2. 「忘心咒不会饶过任何长老。」（ep44 临死自陈 + 递密钥时；慢速尾音下沉，是她对言息控教方式的最后一刀 + 给主角的实战情报）

## 弧光（前世护道 → 翻策 → 长老 → 临死密钥）

- **起（前世）**：主角前世（裴长砚）的护道女剑修，与前世主角共战；崖下救主时留下伤口形状记忆 — 这是后期 ep30 主角金丹突破时被消除的记忆。
- **拐（前世末期）**：被忘川教「忘心咒」翻策，被言息以「偿岁契约」绑死寿元，入教。前世剑修的瞳色 `#3a2818` 暗褐保留至今，是她未被完全吞噬的最后视觉证据。
- **起（本世 ep12）**：九寰阁玉璧大殿，伪装游学者偶遇主角；玉璧倒影显寄生紫双瞳，阮惘察觉主角身上有「另一个被吃的」气息 — 这是她本世第一次确认主角即前世故人转生。
- **拐（本世 ep24）**：以「前世曾救你」为由求结盟，主角婉拒；阮惘留下针对忘川教的信物（伏笔 ep40 第二操作员事件 / ep44 临死密钥）— 信物是一枚未刻字的黑石小印，主角在 ep40 阮瑶死后才开始读懂其含义。
- **拐（本世 ep30）**：主角金丹突破时失去「阮惘救命崖下伤口形状」的记忆 — 寿盘 15/24 黑；这是主角对阮惘的认知第一次崩塌（不知她曾救过自己，故 ep43 奇袭时下手无顾虑）。
- **拐（本世 ep43）**：主角率乌泽+容漪奇袭忘川教边界石塔，阮惘亲自坐镇；表面以三长老身份御敌，实则故意按局让出几道防线 — 让主角直入她所在之塔。
- **拐（本世 ep44 死亡）**：临死自陈「忘心咒翻策」+ 留下针对言息的密钥（伏笔 ep55 化神二品殊死起 + 密钥启动 + 罗盘 24/24 名义耗尽）+ 半截黑纱面巾掉落露全脸 → 右眼角 三点纹烫印灼光一闪特写 + 偿岁契约反噬的暗血一线 `#a82c2c` 从颈侧渗出；死前最后一句「忘心咒不会饶过任何长老。」
- **终 ep44**：死。死后她的密钥 + 信物在 ep55 启动 — 阮惘 是六叛者中唯一「死后仍以情报形式继续在剧中作用」的角色，与言息击败链条直接相关。
