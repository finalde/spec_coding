# 闻砚清 · 主角前世师父

## 角色定位

闻砚清是男主前世 裴长砚 的入室师父 — 剑宗渡劫后期高人，与弟子共「砚」字传承，是六叛之外唯一未叛之人，却被前世主角误判为叛徒之一。剧中以「archived 倒叙人物」语义出场：ep01 final-15s cliffhanger 剪影首现 → ep08 归砚镜首次激活时镜中倒映陌生男子身影（未点名）→ ep15-17 残页揭旧名「裴长砚」并缝合前世/本世 timeline → ep47 遗笔显形 → ep49 镜中拼回完整记忆。前世已死于栖梧崖（旧名「无寿崖」）外围，本世以遗物（归砚镜 + 玉印 + 残页 + 遗笔）持续作用于剧情。Midpoint Reversal 的情绪核心 — Δ between false memory (被误判为叛徒) and truth (实为护道) = ep17 emotional payload。

## 性格 / 动机

- 核心动机：明知契约结局仍守护前世弟子 裴长砚 — 选择以「默许 + 远程印物」的方式护道，而非阻止或揭发。
- 表层人设：剑宗高辈静修师父，言少而稳，门内门外皆敬。
- 反差点：在前世主角眼中，他是渡劫之夜「未现身相救」的可疑者之一；在剧情真相层，他是渡劫之夜从远处一掌印过来、将归砚镜化为契约残片送入濒死弟子体内的唯一护道者。Midpoint Reversal 的全部情绪重量都压在这一层 Δ 上。
- 关键弱点：寿元已尽 + 力有不逮 — 渡劫之夜见五叛围杀已无法亲身出手，只能以「契约残片」远程救一程，自身寿元同夜抵尽。
- 沉默承担：从不向弟子明言改写《残忆经》的代价已被自己看穿；以「默许」代「劝阻」，把选择权留给弟子。

## 弧光（起拐终 + 本世遗物层）

- **起（前世）**：剑宗渡劫后期高人，收 裴长砚 为入室弟子，共「砚」字传承 — 师徒共字是后期 ep15-17 残页揭穿前世名义的视觉根脉。
- **拐（前世末期）**：察觉弟子 裴长砚 反向改写《残忆经》以设反吞契约（per world.md §5.4），却选择默许 — 因弟子托付在先，且明知改写的初衷是困住忘川教而非谋私。这一「默许」是闻砚清最核心的道德抉择，也是 ep47 遗笔显形时观众接收到的情绪 payload。
- **终（前世死亡，渡劫之夜未演场景，由 ep17 残页揭穿）**：渡劫之夜远观 — 见五叛围杀 裴长砚 已无力亲身出手，从远处一掌印过来将归砚镜化为契约残片送入濒死弟子体内（per world.md §5.2）。与弟子同夜陨于栖梧崖外围，寿元抵契约代价。
- **本世遗物层（倒叙 / 镜中显形）**：ep01 cliffhanger 剪影首现 → ep08 归砚镜首次激活，镜中倒映陌生男子身影（未点名）→ ep15 旧名「裴长砚」从遗物匣残页显出 → ep17 师父契约残片揭穿，前世/本世 timeline 缝合（per world.md §5.2）→ ep47 闻砚清遗笔在归砚镜中显形，揭「师父明知契约结局仍替主角守住前世」→ ep49 归砚镜拼回完整记忆。
