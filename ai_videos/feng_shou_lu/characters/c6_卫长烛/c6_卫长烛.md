# 卫长烛 · 赤霞门掌门 / 正派叛者 #1

## 角色定位

卫长烛是男主前世 裴长砚 的同门师弟，最早叛主之人 — 渡劫之夜第一个递剑刺向师兄，五大背叛者中最早动手者（per world.md §3.1 + arc_outline 前世期）。本世为赤霞门掌门，剧中身位是「正派联盟首席宗门掌门」+「靠『长烛烧不熄』虚伪人设维持权位」的高站位反派 (world.md §3.1)。剧中位置贯穿 ep01 倒叙 montage 第一人 → ep04 朝霞门弟子下山招收童子 → ep06 主持外门收徒（主角认出此世师兄即前世第一叛徒）→ ep10 第一付费节点 face-slap（赤霞门掌门当众被揭出当年递剑画面，跪江岸，主角失母容貌记忆，神秘人剪影第二次出现）→ ep15 主角升筑基失「卫长烛递剑那一笑」记忆（伪温情记忆消除 = 仇恨纯化）→ ep19 朝堂联名弹劾被夺掌门印 → ep53 旧部反水回归，朝霞门残部以身续主角灯火。配音风格按 stage-5 storyteller 6-voice matrix 锁定为 **礼训式**（高位 + 礼训式 + 用「门规」「正道」托词）。

## 性格 / 动机

- **核心动机**：以「长烛烧不熄」的虚伪正派人设维持赤霞门掌门权位 — 前世第一个递剑叛主，本世以掌门身份延续这套「礼训正道」表演；ep10 跪江岸瞬间是这套人设崩坏的临界点。
- **表层人设**：赤霞门高位掌门，端正礼训，每句开头带礼度词（「师弟」「师侄」「掌门」），字尾微下沉，不带情绪烈度；门内门外皆敬其「灯火不熄」的正派典范。
- **反差点**：礼训式拱手抚剑鞘的端正姿态下藏前世递剑那夜的笑；笑容端正而含虚伪 — 这正是 ep15 被「伪温情记忆消除」吞掉的那一笑，主角失去后从此对他只剩纯化的恨。
- **关键弱点**：(1) 权位 — 掌门印是命根，ep19 被夺即权位崩塌；(2) 旧账 — 左眉上 1cm 旧剑痕是前世递剑之夜在自己眉上留下的，本世自己亦不知由来，ep10 face-slap 当众揭穿；(3) 旧部 — 朝霞门残部对其忠诚有限，ep53 反水回归即倒戈以身续主角灯火。

## 标志台词或口头禅

2. 「你不该回来。」 — ep10 face-slap 跪江岸瞬间，颤抖含愕（礼训表演崩坏的临界一句）

- **起（前世师兄期）**：前世主角 裴长砚 的同门师弟，表面端正礼训 — 共修剑道，递剑共修笑容是 ep01 倒叙 montage 中观众第一个看到的「reason-to-trust」frame。

- **起（本世 ep04）**：赤霞门掌门，朝霞门弟子下山招收童子；以「师弟之礼，烛火不熄。」的礼训式拱手出场，门徒簇拥。

- **拐（本世 ep19）**：应砚之朝堂联名弹劾朝霞门 + 卫长烛被夺掌门印，权位崩塌；从此挂在朝霞门残部线上苟延。
