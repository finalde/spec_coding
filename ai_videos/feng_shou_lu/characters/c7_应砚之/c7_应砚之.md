# 应砚之 · 正派 + 朝堂双线背叛者 #2 (朝堂太师之嫡子)

## 角色定位

《焚寿录》六大背叛者第 2 号 — 横跨「正派世家系」与「凡俗朝堂 · 太师府」两条线 (per world.md §3.4 朝堂 · 太师府)。前世为 裴长砚 挚友, 二人共修同读, 文气兄弟相称; 背叛后被父亲 (现任太师, 筑基后期 / 金丹初期世家修士) 举荐入朝堂任要职, 成为「正派 + 朝堂」双线人物 — 这一身份使本剧后期对其清算可同时打掉「正派宗门权位」与「凡俗朝堂权位」两条线, 是本剧政治深度的支撑层。

剧中位置: 前世 ep01 cold open 倒叙 montage 中作为第 2 / 第 3 个 betrayer 出现 (reason-to-trust = 文气兄弟共修笑容; reason-to-die = 朝堂俯首执玉笏的冷算); 本世 ep12 世家系背景透露; ep18 现身, 主角升筑基失去朝堂春日阳光记忆; ep19 朝堂上奏弹劾朝霞门, 致卫长烛被夺掌门印; ep20 第二靶标 — 朝堂上斩, 同集《偿岁真言》vs《残忆经》镜像揭示; ep54 应砚之遗书被人投至主角揭朝堂第七叛伏笔 (次季 ep01 ep01 引子)。段位档: 金丹 (本世 ep12-ep18 期间) → 元婴 (ep18 出场期) → ep20 死亡 (朝堂上斩)。

## 性格 / 动机

- **核心动机**：以家族世家系修真传承 + 朝堂太师权位为根基, 联合「正派宗门 → 朝堂」双重路径篡夺修真界与凡俗朝廷的双向话语权。前世背叛 裴长砚 的根本动机是: 察觉 裴长砚 自设《残忆经》反吞契约对 忘川教 不利, 联合其余四位背叛者出手围杀, 以「为天下道义」为表层托词, 实为家族借此交易换取忘川教的延寿契约与朝堂势力扶持。
- **表层人设**：朝堂太师之嫡子, 端正文气, 持玉笏拜礼, 朝堂书面感发言, 文言夹白话, 字间含抑扬 — 一个典型「世家系修真者亦应入世任事」的代言人。
- **反差点**：表层端正文气 vs 内里家族权位野心 + 早已与忘川教暗通消息; 文言朝堂书面感的体面 vs 上奏弹劾朝霞门时直接夺卫长烛掌门印的政治冷算; 「故人之名」的怀旧口吻 vs ep20 斩前对主角微讽含文「家父知道我会回来」的家族算尽。
- **关键弱点**：(1) 太师府家系是公开的, 应砚之 行动必须借家族名义 — 主角 ep20 以「裴知秋」之名上朝堂当面斩之, 直接斩在朝堂上, 让世家系无可推诿; (2) 朝堂书面感口语模式 = 凡俗法度认证, 主角以「国法」之名反向制衡; (3) 父亲太师虽筑基后期 / 金丹初期, 但凡俗朝廷修士不可滥用大法, 应砚之 单兵作战时无家族修真援护; (4) 与忘川教的暗通消息是把柄 — ep54 遗书揭出朝堂第七叛即来源于此。

## 弧光

**起 (前世)**：主角 裴长砚 前世挚友, 二人共修同读, 文气兄弟相称, 同游栖梧崖 (旧名「无寿崖」) 论道。**拐 (前世末期)**：察觉 裴长砚 自设《残忆经》反吞契约威胁忘川教与世家系交易, 联合卫长烛 + 戚归砚 + 池洇 + 阮惘 + 言息 围杀 裴长砚 于栖梧崖渡劫之夜; 背叛后被父亲举荐入朝堂任要职, 成「正派 + 朝堂」双线人物。**起 (本世 ep12)**：世家系背景透露 — 朝堂太师之嫡子, 主角隐约觉察「这人不止是宗门系」。**拐 (本世 ep18)**：现身, 主角升筑基失去朝堂春日阳光记忆 (双倍寿耗 -2 年, 罗盘 12 / 24); 应砚之 以「太师门客」伪装出现, 持玉笏拜礼但目光含算。**拐 (本世 ep19)**：朝堂上奏弹劾朝霞门, 联名上奏 — 致卫长烛被夺掌门印; 朝堂线推动正派宗门权位重洗, 主角被推至风口浪尖。**拐 (本世 ep20 第二靶标)**：应砚之斩, 主角上朝堂以「裴知秋」之名当面斩之; 同集《偿岁真言》vs《残忆经》镜像揭示 — 魔门功法即主角功法源头, 应砚之 临死自陈家族与忘川教暗通消息。**拐 (本世 ep54)**：应砚之 遗书被人投至主角, 揭朝堂第七个叛徒 (次季 ep01 伏线 — 家族系仍持续影响朝堂线)。**终**：死后家族系不灭, 朝堂线继续埋伏笔进入次季。
