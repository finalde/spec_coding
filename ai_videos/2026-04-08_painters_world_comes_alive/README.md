# The Painter's World Comes Alive

这个文件包用于 `45s / 9:16` 的短视频项目《画中世界苏醒》。

## 使用顺序

1. 先生成 `charactors/` 里的角色参考图
2. 再生成 `backgrounds/` 里的背景参考图
3. 再生成 `frames/` 里的每段开始帧和结束帧
4. 最后按顺序生成 `scene/scene_01.md`、`scene/scene_02.md`、`scene/scene_03.md`

## 目录结构

- `charactors/painter.md`
- `charactors/painted_hand.md`
- `backgrounds/studio.md`
- `backgrounds/river_valley.md`
- `frames/scene_01_begin.md`
- `frames/scene_01_end.md`
- `frames/scene_02_begin.md`
- `frames/scene_02_end.md`
- `frames/scene_03_begin.md`
- `frames/scene_03_end.md`
- `scene/scene_01.md`
- `scene/scene_02.md`
- `scene/scene_03.md`
