# 被家族除名，看我武神归来之武神觉醒

> 古风玄幻 · 男频爽剧 · 竖屏短剧（红果 / 番茄 · IAA 免费+广告）

## 项目概要

裴王府嫡长子裴知秋，曾是一国惊艳的天骄，却因丹田被毁沦为废人，遭家族除名、被胞弟裴昭当众羞辱。至暗一刻，"武神觉醒系统"降临——他不再向高坐其上的王爷父亲低头，当众自请脱离裴王府，随即觉醒人间巅峰的**王体资质**，紧接着显化万古唯一的**武神躯**异象，惊呆满堂天骄。一道剑修宗门的目光自暗处投来：武神躯，百年未现……

全剧 **80 集**，每集约 **3 分钟**竖屏快切（约 25–35 镜），每集强 cliffhanger。本仓库当前已详化 **第 1 集（EP1）**，其余集见 `arc_outline.md` 一行大纲，按需逐批解锁。

## 使用说明（出片流程）

本仓库交付的是 **AI 生成提示词与制作规划**，不是成片视频。推荐流程：

1. **角色立绘**：打开 `characters/cN_中文名/` 下每个角色档的「Seedream 立绘 prompt」段，到 Seedream 生成该角色立绘 PNG（竖屏 / 4K）。立绘是后续所有镜头的人物一致性锚点。
2. **逐镜出片**：打开 `episodes/ep01/shots/shotNN.md`，每个文件含三个代码块——
   - `起始帧` / `结束帧`：可选，用 Seedream 生成镜头首末静帧，帮图生视频锁定首尾。
   - `视频 prompt`：复制整段，粘贴到 **Kling（图生视频，上传角色立绘作参考）** 或 **Seedance（文生视频）**，A/B 出片。
3. **拼接**：按 `episodes/ep01/shotlist.md` 顺序拼接各镜，配 `episodes/ep01/dialogue.md` 的字幕。
4. **上架**：按 `episodes/ep01/publish.md` 填平台标题 / 简介 / 标签 / 封面。

世界观见 `world.md`，统一视觉风格见 `style_guide.md`。
