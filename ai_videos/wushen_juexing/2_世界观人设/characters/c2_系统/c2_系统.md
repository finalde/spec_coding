# 系统 · 金手指 · 非实体

## 性格 / 动机

- **核心动机**: 推动 / 测试主角抉择 (二选一 motif), 给关键提示与撑腰, 引导主角藏锋成长 → 终向人妖大战。
- **表层人设**: 无情绪的冰冷裁定者, 不寒暄、不解释、不安慰 (**日常基线**)。
- **反差点 (破例口子·2026-06-21)**: 冷调是常态; **唯独在「重大破境 / 觉醒 / 入境」等关键成长节点, 系统会破例多给一两行刻度 readout + 一句克制的恭喜 / 撑腰**——仍是合成冷腔、不寒暄不絮叨, 只在这种节点透出"一分暖意 / 一分郑重", 作为主角孤立处境里唯一「外援」的反差高光。**非关键节点 (指路 / 提示 / 二选一) 一律保持 EP1/EP2 的冰冷零情绪**, 此口子不滥用。
- **关键弱点 (设定)**: 只给选项与提示, 不替主角动手; 力量仍需主角自己藏锋苦修。

## 弧光

(EP1 冰冷登场, 抛出二选一) → 主角选硬刚, 系统授武神躯 → 全程冷语撑腰 / 提示 / 指引 (情绪零起伏, 但提示愈发关键) → 人妖大战段成为主角决策的关键信息源。

## 关键场景 (EP1)

- EP1 / 中段 — 系统降临, 鎏金对话框淡入, "检测到宿主" + 二选一 (仅主角视角可见)
- EP1 / 后段 — 主角选【二】硬刚离家, 系统授武神躯提示 (UI 文字浮现, 无人物发光)

## 说话风格模板 (声口锚)

类型: **极简短句、无人称、冰冷利落** (日常基线)。从不寒暄、不自称、不解释多余信息; 只下结论、给选项、报提示, 句子能多短就多短。
- 典型口吻示范 (日常): "检测到宿主。" / "二选一。" / "力量，藏好。"
- **关键节点破例示范 (破境/觉醒/入境·2026-06-21)**: 先冷 readout 再一句克制恭喜, 如 "丹田重塑。经脉已通。已入武人一级。" → "废丹之身重上武道——可喜。" (仍合成冷腔、短句、无絮叨人称, 只透一分郑重/暖)。
- 禁忌: 日常镜不说情绪化长句, 不寒暄安慰, 不用「我」「你」絮叨人称, 不带语气词 (啊 / 呢 / 吧); 提示音也忌繁复音效。**破例口子仅限重大成长节点, 且恭喜须克制 (一句为限)、不滥用、不变话痨。**

re-paste `style_guide.md § 负向锁定` + UI 专属:
