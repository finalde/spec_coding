补充约定：

## 4. 情绪 → 光线 表

> shot 的 `光线 / 色调:` 字段按本镜情绪 beat 从此表取光位 / 色调；同场景跨镜须一致（ai_videos__光线色调 审查依据）。

| 情绪 beat | 光线 / 色调配方 |
|---|---|
| **压抑**（受不公 / 隐忍 / 蓄势） | 冷调低饱和 + 柔侧光；明暗层次平缓，主体半藏阴影，色温偏冷，不打高光 |
| **爆发**（硬刚 / 觉醒一瞬 / 出手） | 高反差硬光 + 自然逆光；强侧逆光勾轮廓，背景压暗，主体边缘一道冷亮描边，反差拉满 |
| **回忆 / 闪回** | 做旧泛黄 + 暗角 + 胶片颗粒；整体降饱和偏暖褐，四角压暗（vignette），叠轻微颗粒，一眼可辨「这是过去」 |
| **藏锋**（武神躯内敛 · 平时常态） | 无任何发光特效；眉心 / 眼底**一缕极淡微光内敛**（不外放、不泛金、几不可见），其余如常人，靠表情 / 眼神演「藏」而非发光（见 §5 藏锋 motif + §6 眼不发光铁律） |

跨表通则：

- 同一场景内连续镜的光位 / 色温须连贯，不得忽冷忽暖（除非剧情有明确光源切换，如点灯 / 日落）。
- 回忆镜与现实镜务必**一眼可辨**：回忆=泛黄做旧颗粒暗角；现实=本场景正常调色。

## 5. 镜头词典 / motif

### 5.1 镜头语言词典（`镜头:` / `运镜:` 取词契约）

**景别**（本剧用 7 级细分；骨架＝ `agent_refs/project/ai_video_jingbie.md` 五级阶梯，`中近景` / `大特写` 为本剧两个细分插档，**切换守同一铁律：相邻镜绝不同机位同景别硬切**）

| 关键词 | 含义 | 五级骨架归属 |
|---|---|---|
| 远景 | 全身 + 大量环境 | 远景 |
| 全景 | 全身 + 适量环境 | 全景 |
| 中景 | 膝盖–腰部以上 | 中景 |
| 中近景 | 胸部以上 | 近景（细分·偏松） |
| 近景 | 肩部以上 | 近景（细分·偏紧） |
| 特写 | 面部 / 物件细节 | 特写 |
| 大特写 | 眼 / 局部极近 | 特写（细分·极近） |

**运动**

| 关键词 | 含义 |
|---|---|
| 锁机位 | 完全静止机位 |
| 推（slow push-in） | 镜头向被摄主体逼近 |
| 拉 | 镜头远离被摄主体 |
| 摇 | 机位不动，镜头水平 / 垂直转向 |
| 移 | 机位平行主体水平 / 垂直移动 |
| 跟 | 机位跟随主体移动 |
| 升降 | 机位垂直升起 / 下降 |

运镜通则：关键情绪 / 反转 / 眼神瞬间（藏锋微光乍现、觉醒一瞬、reaction）用**缓慢推近至面部 / 眼睛特写**放大冲击；运镜要缓、控速、稳、不抖（保 Kling 稳定性）。9:16 竖屏构图，主体居中偏上留情绪空间。

### 5.2 全剧 motif（本剧专属视觉符号）

- **藏锋 motif（核心）**：裴知秋的武神躯**无外放**——外人 / 全景视角看他是落魄废少、含蓄低调；镜头转到他正脸 / 特写时，眉心或眼底**一缕极淡微光内敛一闪即敛**（绝不泛金、不外放、不发光特效），观众随特写察觉「此人不简单」，旁人不察。可多次复用，是「暗中成长 / 实力将显未显」的统一符号。
- **武神躯·力沉而不张**：出手 / 觉醒时不堆夸张金光（违 §2 负向），改用**自然逆光 + 高反差硬光 + 环境反应**（地面微震、衣袂气劲、薄雾被气劲推开）表现力量，力量「沉」而非「炫」。
- **玉佩 motif（长线伏笔）**：贴身玉佩在关键情绪点入画特写，温润内敛微光（非发光特效），关乎身世 / 武神躯本源——早期亮相、长线揭示。
- **系统 / 金手指 motif**：系统一切交互（绑定 / 提示 / 二选一 / 发奖）= **在画悬浮 UI 对话框 / 面板**（网文系统流既视感）——半透青蓝底 + 鎏金描边圆角框 + 冷白框内字，选项分行可带【】按钮高亮，主角与观众都看得见。发奖走「送大礼」仪式感（弹「恭喜宿主获得：武神躯」贺词框 + 光门 / 礼匣自框炸开把奖励加身）。声口冷冽极简。`情节:`/`走位:`/`台词:`/`光线:` 各字段须点明对话框位置与框内字，让模型当画面元素渲染（rule 12.4，wushen_juexing/025）。
- **妖族压迫 motif（迟现 · 前期铺垫）**：妖族很后面才正式登场；前期只靠**神秘 + 压迫**——边疆异动、暗处黑影 / 剪影、妖瘴墨绿边缘、细作侧影，不给正脸，留终极威胁悬念。

## 6. 眼不发光铁律

> 全剧硬约束，凌驾于任何「酷炫」诉求之上。违反即 ai_videos__光线色调 审查 blocker。

- **禁瞳泛金 / 禁瞳孔发光 / 禁眼睛发光特效。** 任何角色（含主角武神躯、觉醒一瞬、系统加身、妖族）眼睛**都不发光**——`光线 / 色调:` 与 `动作:` 字段不得写「瞳孔发光 / 瞳泛金光 / 眼放金光 / 眼底亮起光芒 / 眼冒光」等。
- 主角武神躯的「不凡」靠**藏锋 motif**（§5.2：眉心 / 眼底一缕极淡微光**内敛**、表情眼神演绎）+ 高反差自然逆光表现，**不是**靠瞳光。「微光内敛」= 极淡、几不可见、不外放的神采，**不等于**发光特效。
- 此律与 §2 负向「不要 多余发光特效 / 不要 夸张金光」一致：力量与觉醒用光位 / 描边 / 环境反应表现，不用眼睛 / 体表发光。

**卡图美术（你在 Kling 做图时参考）**：
