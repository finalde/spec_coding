# 破庙

## 场景定位
EP3 核心场景：荒废多年的山野小神庙（荒废神庙·夜雨 → 天明）。**功能**：EP3「破庙·初运武神躯·入武人一级」全集独角戏的唯一空间（裴知秋避雨独处 + 系统 UI + 玉佩 prop + 运功觉醒破境）。室内、孤寂、藏锋底色，与 EP2 热闹集市长街成冷清对比。出场：EP3 S1–S9（bg1_庙内_神像前·夜雨）、S10（bg2_庙门·天明收尾）；可后续复用为「破败山野/独处闭关」通用场景。

## 关键变化态 / 分区
- **庙内夜雨态**（S1–S8·bg1_庙内_神像前）：青灰冷调、雨幕成帘、破顶漏光、潮气薄雾——避雨 + 系统降临 + 玉佩异动 + 运功觉醒（运功段开气血暖金内透、S7 破境一记克制外放）。
- **庙内雨歇态**（S9·bg1_庙内_神像前）：雨势渐歇、漏雨转稀、天光微亮一线——收功、回归藏锋内敛。
- **庙门天明态**（S10·bg2_庙门）：转 bg2 庙门，门外晨光暖、庙内冷暗冷暖对冲——出庙启程钩尾。
