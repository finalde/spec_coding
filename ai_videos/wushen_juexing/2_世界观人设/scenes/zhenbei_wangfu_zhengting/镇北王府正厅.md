# 镇北王府正厅

## 场景定位

镇北王府正厅，北疆门户一座以练武为尊的王侯世家府邸的中心议事厅，也是镇北王 裴霆 升座议政 / 家族议事 / 接待边军将领 / 处置族务的礼制空间。此厅是**镇边王侯的尚武厅堂**——高座威重、甲胄仪仗陈列、玄铁器物小面积点缀；但建筑取唐宋营造法式木构（暖木原木色木柱础 + 铺作斗栱华栱出跳、举折坡屋顶、直棂槛窗、覆斗 / 平棊藻井、梁架唐宋风彩画，疏朗大气、去明清繁缛），室内更陈山水 / 书法挂轴字画、博古架青瓷 / 汝窑瓷器、古琴、铜香炉燃香、盆景松石、落地素绢屏风等唐宋文人厅堂清雅陈设（文武相济）；沿梁柱列柱与高台两侧悬满暖黄唐宋样式宫灯（纱罩宫灯 / 莲花灯 / 羊角灯，竹木骨架糊纱、流苏垂坠）、室内通透柔光、淡仙气薄雾，整体走古装仙侠剧古风真人实拍，而非冷硬 documentary 写实。剧中功能：(a) 前几集主场——家族重武 + 偏爱二公子致**长期不公待主角**、**主角离家前的当众对峙**多在此发生；(b) 后续 ep 作 家族议事 / 边军议事 复用。出场集数：前几集多镜使用 + 预期后续多次复用，满足 rule 12.3 立档阈值（≥ 2 shots 复用）。bg 代号见 §背景图系统（对齐 `world.md` 三-地域与场景 bg 代号清单）。

### 生成流程（image-first → 各方向 image→image）

> ⚠ **平台能力约束**：AI 平台**不支持「上传 mp4 → 生成图片」**——上传视频只能出视频、上传图片只能出图片。故各方向静帧用**图生图（image→image）**得到：以一张全局建场图作 reference + 该方向 prompt 出图（2026-06-19 改：image-first 取代「从 mp4 抽帧」为主路径——抽帧出来的方向图质量偏低、机位/锐度不稳，改用图生图各方向单独出图，质量更高、跨图材质/配色/光源更一致）。

1. **出全局参考底图**：用本档「步骤一 · 背景图 seed prompt（Seedream 立绘）」（text→image）生成一张**最具代表性的全局建场图**（覆盖空间结构 + 材质 + 配色 + 光源），存为本 scene 的参考底图。
2. **逐方向 image→image（主路径）**：逐方向打开 `bg{N}_{方位}/{plate_id}.md`，用其方向 image prompt + **上传步骤一全局底图作 reference** → 图生图 → 生成该方向 bg PNG，存回该 folder。参考图锁**材质/配色/光源/风格一致**，prompt 驱动该方向的**机位/构图/主体**（prompt 必须写全该方向几何，单靠底图不足以转向）。
3. **步骤二 walk-through 视频**仍要生成（见下），但**用途 = 后续 shot 的视频 reference**（出 shot 时上传给 Kling/Seedance），**不再是方向背景图的来源**。
4. **抽帧（🧭 截取方向背景图）= 兜底**：某方向 image→image 反复转不过去（如朝南 reverse / 俯瞰），或想快速取一张打底，可在 walk-through mp4 上按截帧时点（见「背景图系统 index」表）抽一张**作图生图的 reference**（含正确机位）再精修。`bg6_座前_虚化背景` 也可从 #5 长焦段（13.5s，景深最虚）抽帧打底。
