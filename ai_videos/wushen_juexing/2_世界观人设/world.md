## 一、力量体系

### 1.1 人族修为境界（七阶 · 由低到高）

| 阶 | 境界 | 实力坐标（说人话的强弱锚） |
|---|---|---|
| 一 | **武人** | 刚开脉的练武学徒，凡人之上、家丁护院水平。**前身裴知秋丹田破碎前曾达武人三级**；丹田碎后修为尽废、跌回未入境（废人）。 |
| 二 | **武者** | 真正踏入武道，能御气发力，一名武者可压十数壮汉；边军普通士卒的门槛。 |
| 三 | **武师** | 一方教头 / 镖局总镖头 / 县镇豪强级；气血外放、拳风裂石。 |
| 四 | **武宗** | 一郡一城的顶梁，宗门长老、王府高阶将领多在此境。 |
| 五 | **武王** | 镇守一方的强者，万军中取上将；**镇北王 裴霆＝武王境**（已是人族顶尖少数、一方镇守之主）。 |
| 六 | **武皇** | 人族顶端的少数巨擘，顶级宗门掌门 / 一国镇国之柱级别。**裴霆毕生所望＝二公子裴昭能登武皇、超越自己这一代的武王。** |
| 七 | **武神** | 战力天花板，万年难出其一。**世人皆以为只是传说——没人相信真有人能修到武神**（正因如此，主角裴知秋的终点＝武神，全剧最大的"不可能"逆袭）。**武皇→武神是一道死关**：纵到武皇之巅也破不了，须**第二宝**这种近乎不存在的机缘与本源呼应方能跨过（见 §四·双宝合一）——这正是它沦为"传说"的根因。**故考核等公开场合的境界科普只点到武皇为止、刻意不提武神（常人不知其存在）。** |

> 晋阶不是单纯堆数字：武宗→武王 之间隔着「质变关」，多数武者一生困死在武师 / 武宗。境界差一阶即可碾压，差两阶近乎无解——这是后期人妖大战压迫感的换算尺。
>
> **每境分九级**（一级最低、九级巅峰；如「武人一级」最弱、「武人九级」近临武者关）。跨大境界（武人→武者→…）须破「境关」。**主角穿越得武神躯、于破庙首次运转后正式踏入「武人一级」**（从最底重修——前身丹田碎、修为尽废，如今从头来过；但武神躯之路直指武神，每一步皆比常人快、稳、无瓶颈）。后续升级由系统冷冽提示（如「已入武人一级」），是观众可感的成长爽点刻度。

### 1.2 妖族等级（六阶 · 与人族对位）

| 阶 | 等级 | 说明 |
|---|---|---|
| 一 | **妖兽** | 未开灵智的凶兽，靠本能厮杀，边疆最常见的兽潮主体。 |
| 二 | **妖兵** | 初开灵智、能成队列作战的妖卒，妖族军阵的基层。 |
| 三 | **妖将** | 统兵的妖族悍将，单体即可屠一队边军。 |
| 四 | **妖帅** | 一域妖军的统帅，运筹兼勇武，对位人族武宗—武王。 |
| 五 | **大能长老** | 妖族元老级巨擘，**对位人族顶宗掌门 / 武皇**；妖王座下多名，是前期「暗处压迫」的真正源头。 |
| 六 | **妖王** | 妖族唯一至尊，**对位人族武神**；统御全族，与人族多年恩怨的核心，**全剧很后面才正式登场**。 |

| 人族 | 妖族 | 对位说明 |
|---|---|---|
| 武者 | 妖兵 | 基层对杀，数量决定胜负。 |
| 武师 | 妖将 | 一对一互有胜负，妖将通常更悍。 |
| 武宗 | 妖帅 | 一城一域的对峙线。 |
| 武王 | 妖帅（强）/ 准大能 | 镇北王 裴霆（武王境）守边疆，主要顶住的就是妖帅级压境。 |

> 妖族整体「同阶略强于人族、且基数恐怖」，故人族靠境界尖端（武皇、武神）+ 边军体系撑住战线。这层差距是后期「为何非要一个武神不可」的剧情地基。

### 1.4 藏锋规则（写进力量体系 · 防 AI 乱加特效）

**主角的武神躯与一切人族 / 妖族力量，外放表现一律「藏锋」——无任何外溢光效。** 全剧渲染铁律：

- **无金光、无瞳光、无外放威能、无气劲可见光柱 / 光环 / 符文 / 元素特效。** 修为高低不靠发光体现。
- **力量只以三样落地**：① 体内**暖流**（角色自述 / 内心 OS 描述的体感，画面不可见）；② **微表情 + 眼神**（眼神骤锐即敛，靠表演不靠发光）；③ **物理结果**（拳风扫起尘土、地砖龟裂、对手被震退——结果可见，力量本身不发光）。
- **眼睛不发光**：任何「瞳泛金 / 瞳孔发光 / 异色眸自发光」一律禁止。眼神变化只用表演（收缩 / 锐利 / 平静）。
- 妖族强者的压迫感同样**靠氛围**（温度骤降、飞鸟噤声、黑影、血腥气、对手本能战栗）而非自发光特效铺垫。

> 根因：仙侠题材若不写死，视频 / 图像模型默认给修真角色乱加金光 / 瞳光 / 元素光柱。本规则要在每个 shot 的 `光线/色调:` 与 `负向:` 落地复用（详见 `style_guide.md` 负向锁定块）。

### 1.5 资质（灵根）等级 + 测资灵石 + 宗门弟子分档（考核弧 EP4 新立 canon · 借宗门招收考核向观众揭示）

> **资质（灵根）≠ 修为境界（§1.1）。** 境界（武人→武神）是**当下修到的层次**；资质（灵根）是**天赋——决定修炼快慢与能走到境界梯多高的上限**。宗门招收新人，用一方「测资灵石」按手测资质定档，资质越高、宗门给的弟子档越高、抢得越凶。

**① 资质七档 + 测资灵石特效（每档反应不同·法器发光、非人物体放——藏锋铁律照旧）**

| 资质档 | 灵石特效（法器读数） | 宗门弟子档 |
|---|---|---|
| **废品** | 死寂、毫无反应 | 不收 |
| **下品** | 石面浮起一层微光、明灭不定 | 外门 |
| **中品** | 稳定柔光自石心晕开 | 外门（好的可争内门） |
| **上品** | 明亮光华 + 缕缕灵气环石而绕 | **内门**（顶宗争抢、可当场直招内门） |
| **玄品** | 光华盛放、灵气成束环柱不散 | 内门精英（顶宗抢破头·近核心） |
| **王品** | 灵气漩涡 + 石身轻鸣震颤 | **核心**（长老亲点） |
| **帝品** | 光华暴涨、冲天光柱 + 异象（灵气漩涡 / 虚影） | 核心 / 亲传（万中无一·绝顶） |

> 灵石所有发光都是**法器对资质的读数**，人物周身无金光无瞳光（§1.4 藏锋照旧落每个相关 shot 的 `光线/色调` 与 `负向`）。

**② 宗门弟子分档（招收即定档·待遇/资源天差地别）**

- **外门**：人数最多、资源最薄；下品 / 中品资质的去处。
- **内门**：宗门重点培养、资源倾斜；上品资质方能入，顶宗为抢内门苗子开价争夺。
- **核心 / 亲传**：长老 / 掌门亲收，王品 / 帝品资质或特殊机缘者，举宗之力栽培；顶尖者更被立为**「宗门圣子」**（一宗最受重视、资源/名师尽倾的天才招牌）。

**③ 残缺资质（主角的伏笔机关）**

资质可以**「高却残」**——读数直冲**帝品**、光华却**缺一块 / 裂一道**（如前身丹田碎留下的根基旧伤）。世人共识：**残缺资质强行修炼，根基迟早反噬，轻则废、重则提前殒命，故纵是帝品也无人敢收。** 唯一解＝有特殊机缘补上那一缺口——**主角已得武神躯、正好暗补这一块，但世人不知**，仍当他是个「资质帝品却必死」的残废（藏锋：真底牌不露）。

**④ 修为境界梯向观众揭示（考核背景点名·只点名不展机制·武神不揭）**

借宗门招收考核的世界观铺垫，向观众点出修为境界梯，但**只点到武皇为止**（§1.1：武人→武者→武师→武宗→武王→武皇）——**武皇已是「几百年一出」的遥不可及，武神境界刻意不提：世人不知其存在、只当传说（呼应 §1.1 阶七），留作标题悬念/暗线**；再借各宗实力坐标（顶宗长老多在武宗 / 武王、掌门更高）+ **镇北王裴霆自陈毕生困于武王境、唯望二公子裴昭登武皇**，把境界压迫感坐实，让观众早早建立"力量坐标"；**只点名、不展开晋阶机制**（机制待主角真修炼时再揭）。

**⑤ 招收大会的性质 + 皇室/天骄/长老到场动机（EP5 立 canon · 补逻辑闭环 follow-up 2026-06-22）**

> 用户指出两处逻辑硬伤：(1) 测资招的是**资质苗**（潜力、未成战力），皇室要的是即战力对抗妖族，为何来招？(2) 各宗**已成名天骄已入宗门**，为何出现在招新大会？下面把大会性质一次锁死、兼顾两条线：

- **大会＝区域性「宗门招收大会」（不止招新）**：① 各宗设台**招收新苗**（测资收徒·主角/裴昭线）；② 各顶宗/大宗**带自家已成名的得意弟子（天骄）到场亮相**——目的＝撑场面亮招牌（顶宗弟子越强，越引好苗投奔）、替宗门物色尖子新苗、年轻一辈同场**扬名/较劲/结识**（**天骄不是来被招，是来代表宗门 + 自我扬名**——解逻辑2）；③ 各宗由**带队长老**领队压阵（顶宗长老多武宗/武王境·§2.2），坐镇旗台、定夺直招内门等大事。
- **皇室（大梁朝廷·萧）借此关注、记名人才（非栽培·非强征·非拉拢）**：因边疆妖患吃紧，借「各宗天骄 + 新苗齐聚」之机**留意并记下**有潜力的英才——成名天骄留个心、有潜力的新苗记个名；但**人才始终由各自宗门培养、朝廷不插手栽培**，待他们在宗门成才，**人妖战时朝廷再倚重 / 征召他们并肩抗妖（人族共同大义·非即战·非强征剥夺·非资助拉拢）**——解逻辑1。靖安公主亲临，亲自留意未来战力、并引人妖战火背景。
- **宗门来大会＝纯为本门，不为讨好皇室（修订·废前稿错误定位）**：各宗到场动机全是**江湖向**——招新苗、带天骄撑场亮招牌、让年轻一辈同场扬名较劲、招揽好苗（②）；**与皇室之间没有"争体面 / 换扶持"的关系**（朝廷只在场关注记名，宗门各忙各的，天骄弟子主体始终属宗门、朝廷不挑走、只战时倚重）。**〔follow-up 2026-06-22：作废前稿"宗门博朝廷扶持 / 挣体面 / 结好互利"的错误定位——宗门无须、也不会在皇室面前争体面。〕**
- **落地**：EP5 群英录由「各宗带队长老压阵 + 天骄代表宗门亮相」框起；前向钩（皇室来意）一律写**关注 / 记名、日后倚重抗妖**，**禁"挑苗子马上送去打妖族"的即战表述、禁"朝廷栽培"、禁"宗门在皇室前争体面"**。带队长老本集 EP5 S4 旗台压阵群像亮相（无台词）；**三主要宗门长老已各建独立角色卡·定名（c16 云鹤真人·天行宗 / c17 玄微子·玄天阁 / c18 商无极·万象门·2026-06-23）**，中小宗（青松/铁剑等）长老仍作背景群像不单建卡；个人戏留第三幕大比/第四幕秘境。

## 二、势力划分

### 2.1 镇北王府（起点 · 边军 · 镇守边疆对抗妖族）

- **定位**：北疆门户的镇守势力，以练武为尊的钟鸣鼎食之家 + 一支边军（镇北军）。**非小人反派**——是真刀真枪顶住妖族的人族屏障。
- **王爷 裴霆**：镇北王，武王境强者，常年镇守边疆、对抗妖族压境。因家族重武 + 偏爱二公子，对废了丹田的主角长期不公待之，但其本身是镇边强者，非阴险小人。**人设钉（2026-06-21）**：其冷待长子源于对方废丹田、无法习武、与「振兴镇北王府」的理想无缘的**实用主义失望**，**非打心底厌恶亲子，不会落井下石讥讽**（刻薄嘲讽归裴昭）。
- **与主角关系**：起点，非终极反派。主角因不公而离家，**不回头打脸**；日后人妖大战中与父辈自然同台（这是把「家族起点」勾连「人妖终战」的主钩）。
- **家族结构（同母嫡亲一家）**：镇北王 裴霆（父）+ 镇北王夫人 沈婉（母）+ **大公子 裴知秋（主角·嫡长子）** + **二公子 裴昭（同母嫡次子·受偏爱的天才）**。主角与裴昭是**同母嫡亲兄弟**；亲爹亲娘都因重武、偏爱有天赋的二公子，冷落了废了丹田的亲长子大公子（"连亲生父母都嫌弃"的至暗底色）。镇北王夫人沈婉＝软性偏袒者（偏护二公子、避视长子）。
- **力量坐标**：裴霆 武王境（毕生望二公子裴昭登武皇、超越自己）；府中将领、教头 多在 武师—武宗；镇北军士卒 武者级为主。

### 2.2 宗门体系（成长舞台 · 中段）

- **太虚剑宗（主角将去 · 落魄剑宗）**：曾经辉煌、如今没落的剑道宗门，门庭冷落、弟子凋零。主角离家后入此宗，**藏锋**一步步成长。暗线人物 **凌虚子**（太虚剑宗高人，察觉主角异样、未识破武神躯）在此。**canon 锁定：主角的宗门 = 太虚剑宗。**
- **顶级主力宗门（人人都想去·实力排位顶尖）**：**天行宗**（剑道称雄）与 **玄天阁**（御法/法修第一）——两大顶宗并列顶尖，弟子云集、资源鼎盛，是天下武者趋之若鹜之地；长老多武宗/武王境、掌门更高。与太虚剑宗形成「落魄 vs 鼎盛」的强烈反差。
- **一流 + 中小宗门（略提·背景走过场）**：两大顶宗之下，**万象门**等一流大宗排得上号；再往下是数家中小门派（如 **青松派 / 铁剑门**，收下品 / 中品资质的外门弟子凑数）。与顶宗 + 太虚剑宗同处一界。**宗门之间竞争、有摩擦不和，但非生死仇敌**（区别于妖族的终极对立）。
- **宗门实力排位（EP4 借围观点出·canon）**：**天行宗 · 玄天阁（两大顶宗）＞ 万象门等一流大宗 ＞ 青松派 / 铁剑门等中小门派 ＞ 落魄太虚剑宗（连年垫底·将除名）**。
- **与主角关系**：成长舞台。主角在落魄太虚剑宗藏锋，对照顶宗的繁盛，循序渐进逆袭（不强调打脸）。

### 2.3 妖族（终极对立 · 迟出场 · 神秘压迫基调）

- **结构**：**单一妖王 + 多名大能长老**（详见 §1.2）。与人族有**多年恩怨历史**，欲争天下 / 灭人族。
- **出场节奏铁律**：**很后面才正式登场。** 前期（前几集 ~ 中段）**只靠神秘感 + 压迫感铺垫，不给妖族正脸 / 不给妖王全貌**：
  - **边疆异动**：兽潮、烽台告急、镇北军调动的零星信报。
  - **暗处黑影**：远景一闪而过的黑影 / 剪影，不交代是谁。
  - **细作 / 暗影**：潜伏人族的妖族细作、边疆暗影，制造「敌在暗处」的不安。

### 2.4 百年前妖族大战 · 太虚剑宗落魄 · 双玉佩之秘（暗线 backstory · 迟揭 · follow-up 2026-06-21）

> **作用**：一次性把「太虚剑宗为何落魄」「妖王/妖患」「双宝合一」「凌虚子 / 玉佩」四条悬空 canon 落地并对齐时间线。**埋法铁律**：前期只埋不揭——凌虚子此刻顶多说到「大师兄妖王之战后失踪、他不知细节」，绝不点破「双玉佩合一＝成武神钥匙 / 主角那半正是另一半」；真相留到季末/临武皇再挑明。

- **百年前 · 妖族大战（妖王封印之役）**：约百年前，妖王率妖族倾巢南下、人族死战。彼时**太虚剑宗尚是辉煌的剑道大宗**，倾宗参战、**死伤惨重、师傅辈（掌门的同辈长老们）大半战死**。当时太虚剑宗**掌门为武皇境中期**强者——力不能杀妖王，唯**燃尽自身、以命为引，勉强将妖王封印**（封印非永绝、只是镇压）。临终前，掌门将**一枚玉佩**（半枚螭龙云纹古玉·与封印/破妖王之秘相关）托付给座下**大弟子**。
- **大师兄 · 持半玉佩远遁（百年杳无音讯）**：那位大弟子＝**凌虚子的大师兄**。受掌门临终托付，他**携半玉佩独自远遁**——一为护宝（玉佩是封印与日后了结妖王的关键、绝不能落妖族手），二为寻**彻底了结妖王之法**（封印终非长久之计）。自此**百年杳无音讯、生死不明**，连留守的凌虚子也寻他不见。【师兄"某些原因"＝受命护宝 + 寻灭妖之法而远遁失踪·留作长线·可后续重现】
- **太虚剑宗 · 百年衰落**：大战后元气大伤、师傅辈凋零、镇宗之宝（半玉佩）随大师兄远去，太虚剑宗一蹶不振、弟子流散，终沦为**连年垫底、将被除名的落魄剑宗**（§2.2）——外人只道它"不行了"，无人知其曾以一宗之力为人族封过妖王。**凌虚子**＝当年大战的**幸存弟子**（掌门的小弟子、大师兄的师弟），剑仙级修为、寿元绵长，百年来隐于落魄宗中、**落魄醉酒、藏锋扫迹**（酒剑仙）。
- **双玉佩＝双宝合一的"第二宝"具体化（对齐 §四·双宝主线）**：所谓**第二宝＝这对"半枚螭龙云纹古玉"合一**——师兄持的那半 ＋ **主角裴知秋伴生武神躯而来的这半**，两半顺纹相合、续成整条螭龙整句符箓，方能与武神躯本源呼应、助主角**破武皇→武神死关**、并成为**镇杀已破封妖王**的钥匙。主角那半的来历（为何随武神躯/系统而来、与师兄那半如何成对）＝**深线、不过早揭**。
- **当下（EP4）引信对齐**：百年封印**渐弱**，妖患复起——EP4 S3 街头说书「妖潮连破烽燧、边军节节败退、朝廷急着招人指望年轻一辈顶上去」正是**封印松动、妖族resurging**的前兆（迟出场基调照旧）；**妖王终将破封** ＝ 第二季「人妖大战」的引线。**时间线**：百年前封妖王 → 百年太虚衰落 → 当下封印渐弱·妖患起（第一季背景）→ 第二季妖王破封·寻师兄那半玉佩·双宝合一·武神临世·镇妖王终章。
