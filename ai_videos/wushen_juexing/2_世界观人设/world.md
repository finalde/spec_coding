# 世界观 · 武神觉醒

> 阶段 2 产物（`ai_videos__全流程编排` / stage2_世界观人设）。对齐 `1_立项/concept.md`：成长向仙侠（不打脸、不归来），穿越绝境 → 硬刚离家得武神躯 → 入落魄太虚剑宗藏锋成长 → 终在人妖大战中成为人族关键。
>
> 本档锁定全剧 lore 骨架：力量体系（人族境界 + 妖族等级 + 对位表 + **藏锋规则**）/ 势力划分（镇北王府边军 / 宗门 / 妖族）/ 地域与场景 bg 代号清单 / 妖族迟出场基调 / canon 锁定。**敏捷原则**：只先把前几集需要的世界观落死；宗门腹地 / 妖域等场景待出场再立档。

## 一、力量体系

### 1.1 人族修为境界（七阶 · 由低到高）

| 阶 | 境界 | 实力坐标（说人话的强弱锚） |
|---|---|---|
| 一 | **武徒** | 刚开脉的练武学徒，凡人之上、家丁护院水平。前身裴知秋丹田破碎前止步于此。 |
| 二 | **武者** | 真正踏入武道，能御气发力，一名武者可压十数壮汉；边军普通士卒的门槛。 |
| 三 | **武师** | 一方教头 / 镖局总镖头 / 县镇豪强级；气血外放、拳风裂石。 |
| 四 | **武宗** | 一郡一城的顶梁，宗门长老、王府高阶将领多在此境。 |
| 五 | **武王** | 镇守一方的强者，万军中取上将；**镇北王 裴霆约在武王—武皇之间**。 |
| 六 | **武皇** | 人族顶端的少数巨擘，顶级宗门掌门 / 一国镇国之柱级别。 |
| 七 | **武神** | 传说之境，人族战力天花板，万年难出其一；**主角裴知秋的终点**。 |

> 晋阶不是单纯堆数字：武宗→武王 之间隔着「质变关」，多数武者一生困死在武师 / 武宗。境界差一阶即可碾压，差两阶近乎无解——这是后期人妖大战压迫感的换算尺。

### 1.2 妖族等级（六阶 · 与人族对位）

| 阶 | 等级 | 说明 |
|---|---|---|
| 一 | **妖兽** | 未开灵智的凶兽，靠本能厮杀，边疆最常见的兽潮主体。 |
| 二 | **妖兵** | 初开灵智、能成队列作战的妖卒，妖族军阵的基层。 |
| 三 | **妖将** | 统兵的妖族悍将，单体即可屠一队边军。 |
| 四 | **妖帅** | 一域妖军的统帅，运筹兼勇武，对位人族武宗—武王。 |
| 五 | **大能长老** | 妖族元老级巨擘，**对位人族顶宗掌门 / 武皇**；妖王座下多名，是前期「暗处压迫」的真正源头。 |
| 六 | **妖王** | 妖族唯一至尊，**对位人族武神**；统御全族，与人族多年恩怨的核心，**全剧很后面才正式登场**。 |

### 1.3 人妖对位换算表（力量对比 / 压迫感打底）

| 人族 | 妖族 | 对位说明 |
|---|---|---|
| 武者 | 妖兵 | 基层对杀，数量决定胜负。 |
| 武师 | 妖将 | 一对一互有胜负，妖将通常更悍。 |
| 武宗 | 妖帅 | 一城一域的对峙线。 |
| 武王 | 妖帅（强）/ 准大能 | 镇北王 裴霆守边疆，主要顶住的就是妖帅级压境。 |
| 武皇 | 大能长老 | 人族顶宗掌门级才能与妖族元老抗衡。 |
| **武神** | **妖王** | 唯一能正面镇压妖王的存在——**这是主角的终极坐标**。 |

> 妖族整体「同阶略强于人族、且基数恐怖」，故人族靠境界尖端（武皇、武神）+ 边军体系撑住战线。这层差距是后期「为何非要一个武神不可」的剧情地基。

### 1.4 藏锋规则（写进力量体系 · 防 AI 乱加特效）

**主角的武神躯与一切人族 / 妖族力量，外放表现一律「藏锋」——无任何外溢光效。** 全剧渲染铁律：

- **无金光、无瞳光、无外放威能、无气劲可见光柱 / 光环 / 符文 / 元素特效。** 修为高低不靠发光体现。
- **力量只以三样落地**：① 体内**暖流**（角色自述 / 内心 OS 描述的体感，画面不可见）；② **微表情 + 眼神**（眼神骤锐即敛，靠表演不靠发光）；③ **物理结果**（拳风扫起尘土、地砖龟裂、对手被震退——结果可见，力量本身不发光）。
- **眼睛不发光**：任何「瞳泛金 / 瞳孔发光 / 异色眸自发光」一律禁止。眼神变化只用表演（收缩 / 锐利 / 平静）。
- 妖族强者的压迫感同样**靠氛围**（温度骤降、飞鸟噤声、黑影、血腥气、对手本能战栗）而非自发光特效铺垫。

> 根因：仙侠题材若不写死，视频 / 图像模型默认给修真角色乱加金光 / 瞳光 / 元素光柱。本规则要在每个 shot 的 `光线/色调:` 与 `负向:` 落地复用（详见 `style_guide.md` 负向锁定块）。

## 二、势力划分

### 2.1 镇北王府（起点 · 边军 · 镇守边疆对抗妖族）

- **定位**：北疆门户的镇守势力，以练武为尊的钟鸣鼎食之家 + 一支边军（镇北军）。**非小人反派**——是真刀真枪顶住妖族的人族屏障。
- **王爷 裴霆**：镇北王，武力不凡（约武王—武皇之间），常年镇守边疆、对抗妖族压境。因家族重武 + 偏爱幼子，对废了丹田的主角长期不公待之，但其本身是镇边强者，非阴险小人。
- **与主角关系**：起点，非终极反派。主角因不公而离家，**不回头打脸**；日后人妖大战中与父辈自然同台（这是把「家族起点」勾连「人妖终战」的主钩）。
- **家族其他**：受偏爱的**弟弟**、偏袒幼子的**母妃**。
- **力量坐标**：裴霆 武王—武皇之间；府中将领、教头 多在 武师—武宗；镇北军士卒 武者级为主。

### 2.2 宗门体系（成长舞台 · 中段）

- **太虚剑宗（主角将去 · 落魄剑宗）**：曾经辉煌、如今没落的剑道宗门，门庭冷落、弟子凋零。主角离家后入此宗，**藏锋**一步步成长。暗线人物 **凌虚子**（太虚剑宗高人，察觉主角异样、未识破武神躯）在此。**canon 锁定：主角的宗门 = 太虚剑宗。**
- **顶级主力宗门（人人都想去）**：**天行宗** 与 **玄天阁**——两大顶宗，弟子云集、资源鼎盛，是天下武者趋之若鹜之地。与太虚剑宗形成「落魄 vs 鼎盛」的强烈反差。
- **其它宗门（略提）**：另有数家中小宗门，与顶宗 + 太虚剑宗同处一界。**宗门之间竞争、有摩擦不和，但非生死仇敌**（区别于妖族的终极对立）。
- **与主角关系**：成长舞台。主角在落魄太虚剑宗藏锋，对照顶宗的繁盛，循序渐进逆袭（不强调打脸）。

### 2.3 妖族（终极对立 · 迟出场 · 神秘压迫基调）

- **结构**：**单一妖王 + 多名大能长老**（详见 §1.2）。与人族有**多年恩怨历史**，欲争天下 / 灭人族。
- **出场节奏铁律**：**很后面才正式登场。** 前期（前几集 ~ 中段）**只靠神秘感 + 压迫感铺垫，不给妖族正脸 / 不给妖王全貌**：
  - **边疆异动**：兽潮、烽台告急、镇北军调动的零星信报。
  - **暗处黑影**：远景一闪而过的黑影 / 剪影，不交代是谁。
  - **细作 / 暗影**：潜伏人族的妖族细作、边疆暗影，制造「敌在暗处」的不安。
- **基调**：温度骤降、飞鸟噤声、血腥气、对手本能战栗（藏锋规则同样适用——**不靠自发光特效**）。
- **与主角关系**：终极宿敌。主角武神躯的意义最终落在「人妖大战中成为人族关键力量 / 救星」。
