# 剧集大纲 —《被家族除名，看我武神归来之武神觉醒》

全剧 80 集，每集约 **1 分半（~90s）** 竖屏快切，每集强 cliffhanger（follow-up 022：每集 ~90s，对白不删、内容超出顺延下一集）。本轮 **EP1 已详化（14 镜/90s）、EP2 已详化（14 镜/~89s·藏锋版：厅内收束+凌虚子远眺 6 镜 + 离府赶路/系统初引/玉佩生疑/城门末镜钩 8 镜）**（见 `episodes/ep01/`、`episodes/ep02/`）。**EP3–80 为一句话大纲（follow-up 029）**，逐批 stage-4 regen 解锁详化。

## 三幕拍点（季度骨架）

| 集区 | 幕 / 功能 | 拍点 |
|---|---|---|

| EP3–15 | 第一幕·起 | 离府求生 → **测资质石(王体爆表·超过裴昭·各大宗门疯抢)** → **偏选落魄太虚剑宗** → 入门**速成剑修、超越前辈** → 随师姐云清越下山历练（第15集钩） |

| EP31–50 | 中点反转 + 第二阶段 boss | 追查真相、与裴霆交锋、妖族暗通款曲浮现（~40集全剧枢纽身世大反转）、绝地反杀（第50集钩：幕后+妖族阴谋揭开） |
| EP51–70 | 节节高 payoff + 决战序章 | 打脸昔日轻视者、红颜/兄弟线收束、武神躯完全体、妖族大战、终极反派对决（第65集生死钩） |

> 强留存钩集：第 7 / 15 / 23 / 25 / 30 / 40 / 50 / 65 集放章节级大反转 / 钩子兑现（红果语境=追剧留存峰，非付费墙）。前 3 集（72 小时高热窗口）钩子密度最高，直接决定全剧分账。
> 必要情节锚（用户指定）：测资质石·超弟·宗门疯抢（EP6–8）→ 选落魄剑宗（EP9）→ 速成剑修超越前辈（EP11–13）→ 随师姐历练（EP15–16+）。
> 长线钩（follow-up 030/032）：① **藏锋武神躯**——EP2 系统暗授但不显、众人看不出，威能留到 **EP25 当众首度爆发被识破**（"众人才发现"大反转）。② **石碑"短命"误判**——EP7 测资质石碑箴言赞天赋却警"短命"、末句未显碑先碎；师傅师姐众人皆信他将死、态度反复横跳显小人嘴脸，**EP23 死期已过人没死、众人打脸**；唯系统知他本是命定武神、"短命"纯属莫须有。③ **师姐云清越**正直却起初小看这废物新人，被一次次打脸方改观。

## 分集一行大纲（EP1–80，每集一句）

### 第一幕·起（EP1–15）

- **EP1** ✅ 已详化 — 穿越醒来即被当众除族；至暗一刻系统早现、解释两条路的结果给「继续苟活/觉醒武神」二选一，选【觉醒武神】激活武神系统，有底气硬刚镇北王裴霆、戳穿胞弟裴昭、看透偏袒的生母沈婉（回忆补救弟致残/送蛮荒城），决心与镇北王府断绝。
- **EP2** ✅ 已详化（14 镜/~89s·藏锋不显）— 当众断绝走出；系统**私下暗授武神躯**（仅主角可见的对话框、当集不公开显化、众人看不出），主角刻意藏锋；离场宣言"站上武神之位、让所有人都仰起头看我"被众人当作废物放狂言而嗤笑（戏剧反讽）；凌虚子远眺隐隐生疑却看不透（宗门暗线）；**离府赶路、系统初引（远离是非·静处淬体·前路机缘）、断裂家纹玉佩忽透微温生疑（身世/传承暗钩）；城门暗藏按刀黑影锁定背影（末镜钩·引 EP3 城门截杀）**。**武神觉醒威能留待后续"众人才发现"。**
- **EP3** — 孤身出府，旧仆暗助卷细软；城门口遭裴昭派的杀手截杀，他竟莫名一招震退（武神躯暗劲、连他自己都未尽明、外人只当废物突然会了点拳脚），扬长而去（钩：杀手回报"那废物…会武功了？"）。
- **EP4** — 荒郊野岭系统派下首个修行任务；裴知秋撞见太虚剑宗采药的师姐**云清越**遇险，出手相救、结下善缘（埋师姐线）。
- **EP5** — 镇上裴府姻亲当街仗势欺人，裴知秋出手打脸扬名；耳闻"测灵大典"将启、各大宗门齐聚招新，决意前往。
- **EP6** — 测灵大典开幕、天骄云集；弟弟**裴昭**携天阶资质高调亮相、被数家宗门追捧，当众羞辱来路不明的兄长。
- **EP7** — 裴知秋按上**测资质石碑**——碑面箴言先**盛赞天赋**：王体资质爆表、万古难遇（远超裴昭、全场哗然·天骄被打脸）；笔锋一转又显**警语**"此等逆天之资必伴大磨难、寿元不永…"（埋"短命"钩）；**末句truth尚未显现、石碑已轰然碎裂**（双层钩：点破他乃命定武神的末句永远没露出来）。【强钩】
- **EP8** — 各大宗门为王体资质的裴知秋**疯抢**、争相招揽；却又因碑上"短命"警语而态度摇摆、压价观望（小人嘴脸初显），裴昭趁机嘲他"短命废物"；裴府脸色铁青（钩：他会选谁？）。
- **EP9** — 众望所归之际，裴知秋偏偏**选了连续垫底九年、险被除名的落魄太虚剑宗**——满场愕然嗤笑，唯剑王长老柳鹤鸣、师姐云清越动容（风骨+凌虚子暗线+武神躯与剑道相契）。
- **EP10** — 入太虚剑宗，宗门凋敝、弟子寥寥；裴知秋分得最差洞府、遭老弟子排挤；师姐云清越虽正直、却也信他"短命"、打心底**不看好这废物出身的新人**、言语间多有小看。
- **EP11** — 系统+武神躯逆运周天，裴知秋一日千里、速成剑徒；剑王长老惊觉其修炼速度异于常人。
- **EP12** — 藏剑阁参剑，裴知秋悟性惊人、几日参透前辈数十年未通的剑诀，**超越前辈**，宗门震动。
- **EP13** — 排挤他的老弟子设擂羞辱，裴知秋以剑徒之身连越数阶、当众挫败，立威外门；众人惊其修行之速、却又念及"短命"而态度反复（惜才又疏远）。
- **EP14** — 裴昭买通太虚败类下毒手陷害，被将计就计反杀打脸；裴府第一桩阴谋告破。
- **EP15** — 宗门大比在即（垫底十年即除名的生死关）；掌门凌虚子现身点拨，师姐云清越邀他**一同下山历练**积累实战（第一幕收束钩）。

### 第二幕·承（EP16–50）

- **EP16** — 师姐带裴知秋下山历练，初入江湖、接宗门悬赏任务。
- **EP17** — 历练遇邪修害命，裴知秋剑气初成、一剑破敌，护一城百姓。
- **EP18** — 与师姐联手剿匪、配合渐生默契；裴知秋剑势已逼剑师。
- **EP19** — 撞见裴府商队仗势害民，裴知秋出手打脸昔日轻视他的裴府管事。
- **EP20** — 历练入一处妖气古迹，初见**妖族**踪迹（伏笔加深），得一缕远古剑修传承。
- **EP21** — 裴昭得知废物兄长成了王体天骄入了剑宗，妒恨交加，向裴府与三公主进谗。
- **EP22** — 三公主**萧绛雪**当年退婚转嫁裴昭，今闻裴知秋飞黄、心思浮动，裴昭危机感顿生。
- **EP23** — 历练归宗：石碑所断的"死期"竟已悄然过去、裴知秋好端端没死，先前断他必死、对他反复横跳的众人当场愕然打脸；一身惊人修为震动太虚、宗门士气大振，师姐云清越对他彻底改观。【钩兑现】
- **EP24** — 宗门大比开打，太虚多年垫底；裴知秋一人挑梁、连胜强敌。
- **EP25** — 大比关键局，裴知秋以低阶之身**降维碾压**宿敌天骄、资质无法解释；剑仙掌门凌虚子失声——那是绝迹百年的**【武神躯】**！全场哗然（**武神躯·众人才发现**·藏锋大反转），同役**助太虚摘掉垫底耻辱、保住宗门**。【强钩】
- **EP26** — 太虚声名鹊起，裴知秋被推为新一代领军，引来更大势力与天骄觊觎挑衅。
- **EP27** — 裴府察觉裴知秋崛起威胁，裴霆暗中布局打压；母亲沈婉态度微妙松动。
- **EP28** — 裴知秋入大城历练，邂逅红颜（女配线起），卷入拍卖会夺宝风波。
- **EP29** — 夺宝牵出神秘势力，裴知秋以武神躯硬撼强敌、名动大城。
- **EP30** — **章节大反转**：查到当年裴府之祸（致原主丹田碎的旧案）另有黑手，矛头隐指裴府内部。【强钩】
- **EP31** — 顺线追查，裴知秋与裴府正面交锋，当面质问裴霆、不欢而散。
- **EP32** — 裴昭联合外宗天骄围杀裴知秋，反被打得溃不成军，威信扫地。
- **EP33** — 裴知秋境界连破、晋阶高阶剑修，剑势惊动隐世前辈。
- **EP34** — 妖族异动频现、修行界暗流；太虚百年前抗妖旧账被重提。
- **EP35** — 随宗门驰援妖患边城，裴知秋初与妖族强者交锋、险胜立功。
- **EP36** — 救边城时与师姐/红颜情感线推进；身世之谜再现线索（断裂玉佩/血脉）。
- **EP37** — 裴府勾结某大宗门设死局除他，师姐云清越为护他身受重伤。
- **EP38** — 裴知秋怒而爆发、武神躯进阶，破死局、重创仇敌、护住师姐。
- **EP39** — 养伤参悟剑道更进，与师姐情愫暗生；宗门倾力栽培。
- **EP40** — **全剧枢纽大反转**：身世与武神躯渊源全揭，幕后大黑手浮出水面。【强钩】
- **EP41** — 真相之下立誓复仇雪冤，目标直指幕后大势力。
- **EP42** — 裴霆父子背后靠山浮现，与妖族暗通款曲（反派升级）。
- **EP43** — 裴知秋单刀赴会，当众揭穿裴府/反派一桩大阴谋、朝野震动。
- **EP44** — 反派狗急跳墙，挟持裴知秋在乎之人（师姐/红颜/母亲沈婉）相胁。
- **EP45** — 雷霆反击救出人质，重创第二阶段 boss、威震一方。
- **EP46** — 大战后声望登顶，各方或拉拢或忌惮；旧仇头目现身放话。
- **EP47** — 赴险境寻终极传承，武神躯再进化、逼近武道高阶。
- **EP48** — 妖族大举来犯，修行界仓促结盟；太虚与裴知秋成中流砥柱。
- **EP49** — 联盟内有内鬼通妖，裴知秋揪出叛徒、力挽狂澜。
- **EP50** — **大反转**：幕后黑手与妖族的惊天阴谋全盘揭开，裴霆父子沦为弃子。【强钩】

- **EP54** — 裴知秋连破强敌、晋阶武皇/剑帝级，实力跃升新台阶。
- **EP55** — 终极反派试探性出手，身边人接连遇险，大战将至。

- **EP57** — 妖族圣子/王者登场，与裴知秋宿命对决初交手、平分秋色。
