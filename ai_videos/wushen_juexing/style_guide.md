## 一·B、美术方向（电影级唯美仙侠写实 + 反卡通铁律，follow-up 007）

场景不做 100% 朴素写实——在**照片级真实质感打底**之上叠加华美/仙侠元素，让地板、墙壁、灯火、氛围都**漂亮 fancy**。**这是仙侠剧，观感漂亮 > 古建考据**（follow-up 011）。

- **暖黄宫灯 / 灯笼（主提亮，follow-up 011）**：室内**高雅黄色宫灯 / 灯笼成列高悬**、玉阶两侧立地宫灯、廊下灯笼成串，**暖黄灯火盈盈、满堂生辉**——这是把场景提亮、显华贵的主要手段，整体偏**明亮暖调**。
- **地面**：磨光纹样青砖 / 碾玉装石板，光可鉴人、映暖黄灯火与天光的淡倒影。
- **墙壁**：障壁画 / 织锦壁衣 / 碾玉装彩画 / 描金线脚 / 轻垂纱幔，华美不素白。
- **fancy 仙侠氛围**：鎏金描金、琉璃、玉阶生辉、暖金流光、极淡薄雾 + 丁达尔光束（**克制**）。
- **整体调色**：暖黄灯火 + 米白天光 + 描金 + 朱红 + 天青点缀，**明亮暖调、华美考究**。
- **放宽古建考据（follow-up 011）**：唐宋营造法式仅作**风味参考**，不必与真实王府古建一模一样、明清禁令放宽——**以美学与仙侠观感优先**。
- **bg 背景板须高清 + 精准镜头机位（follow-up 012）**：场景 bg prompt（seed + 各方向 plate）须声明 4K 超高清/高分辨率 + 精准镜头机位（视角方位 / 水平视平线 / 标准镜头约35–50mm等效 / 透视稳定不畸变 / 构图位置精准 / 预留人物站位空间）——bg 是 shot 背景参考板，镜头/位置不精准会导致各 shot 透视/站位对不齐。
- **仍反卡通**：影视级真人实拍 / 照片级写实质感**始终主导**，漂亮 fancy 但不是 卡通 / 动画 / anime / CG 渲染感 / 塑料感 / 扁平光 / 过曝过饱和；零 hex 自然色名；prompt 不点名任何影视 / 游戏 / 真人 IP。

## 二、调色（明亮通透为主 + 暖人物，自然色名）

> follow-up 006：王府基调改为**明亮**，不再阴暗。

> follow-up 009：色彩要**明亮**，不要整体墨绿/暗红/冷暗；带点**仙侠气**。

- **主色调（明亮，follow-up 009）**：**明亮朱红 + 描金 + 天青 / 青碧透亮 + 月白 + 米白暖金天光**。**避免**整体墨绿、暗红、深紫沉调与大面积压暗。（注：「残血暗红」仅限觉醒/武神躯裂纹 motif；「深紫描金蟒纹」仅限裴霆身份色——均非整体调，保留。）
- **仙侠气**：青碧透亮、淡金流光、暖金天光、月白仙气、极淡薄雾 + 丁达尔光束（克制）；照片级写实主导、反卡通。
- **整体基调**：**明亮通透、大气雍容**。尤其裴王府正厅——高窗天光满堂、空间高敞明亮。保留淡青雅致底子，但**显著提高亮度、减少压暗**。
- **环境底色**：天青米白暖天光为主，通透而**明亮**；纵深处自然柔和渐暗即可，不做大面积压暗。
- **人物光**：主角与师父的高光帧叠**暖金光束**，形成"明亮环境 + 暖人物"的层次；肤色白皙通透。
- **局部压暗按需**：戏剧性的压暗 / 低调光只用于**确有必要的情绪镜**（如伏地至暗、末镜暗处注视），不作全场默认。

## 六、字幕规范

- 形式：**全部台词字幕一律后期叠加，视频不烧任何字幕 / 文字（follow-up 033）**。shot `视频 prompt` 不写「内嵌硬字幕 / 硬字幕 / 白底黑边」等烧字措辞；`台词:` 只留说话人+台词原文+语气+口型/OS 注（供对口型与配音），`渲染样式:` 末尾加「画面不烧任何字幕文字（台词后期另加）」。剧名卡 / 「下一集」转场卡字样亦后期叠加（prompt 只留黑屏转场）。
- 字体调性（后期叠字用）：方正粗黑 / 思源黑体，白底黑边，位于画面下三分之一居中。
- **系统文字例外**：以**在画对话框内嵌字**呈现（鎏金描边圆角悬浮框，是画面 UI 元素、非台词字幕），配清脆「叮」提示音（follow-up 025）——属画面元素，不在"不烧字幕"之列。
- 音频契约：v1 视觉优先，不生成 TTS / 配乐；台词只作字幕，不下发为音频 prompt。
