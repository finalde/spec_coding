# 裴王府正厅

## 场景定位

裴王府的核心殿堂，镇国王府的门面所在，EP1 绝大多数镜头（S01–S29）发生地。除族羞辱、穿越醒来、系统二选一、断绝走出、王体+武神躯觉醒均在此。基调**大气、恢弘、高敞、明亮**——面阔多间的高大殿堂、丹墀玉阶、满堂天光，既显镇国王府的雍容威仪，也为觉醒大爽点留足空间纵深。北端丹陛之上设家主高座，中央青砖跪礼区，两侧列宾客席。戏剧性的压暗、冷峻由各 shot 的 `光线 / 色调:` 按需叠加；场景基底始终高敞明亮。

## 关键变化态

- 默认态（S01–S20）：日间高窗天光满堂 + 满堂暖黄宫灯灯火盈盈、明亮暖调、华美 fancy；肃穆威仪由陈设、华灯与人物表演承载，而非靠压暗。
- 觉醒态（S21–S27）：灵气青白光柱自厅心冲霄、九柱呈环 + 暖金裂纹光，金光满殿、与高窗天光交映，恢弘更甚。
- 出走态（S28–S29）：镜头转向南面三开格子大门 / 门匾，天光自门外大片涌入，背影没入光中。

## 站位锚 / 朝向约定（全集 byte-identical 引用，follow-up 028）

> 正厅内人物方位与朝向恒定，shot 的 `走位:` 一律遵此，杜绝「主角背对王爷」之类穿帮：
> - **裴霆（镇北王）** 恒居 **北端家主高座**（最高、最尊）。
> - **裴知秋** 跪 / 立于 **中轴中央跪礼区**，**面朝北（朝向裴霆 / 高座）**——受审、对峙、宣告皆面向王爷，**切忌背对王爷**。
> - **裴昭** 多在 **东侧（东列长案）**；**沈婉** 多在 **西侧（西列长案）**；**厅门在南**（出走口，EP2 走出用）。
> - **正反打机位**：① 看裴霆威仪 → `bg1_朝北`（机位在南、主角背影前景 + 裴霆正面高座）；② 看主角表情（硬刚 / 宣告）→ `bg3_朝南`（机位在北 / 高座侧反打、主角正脸 + 南面厅门逆光为背景）；③ 主角独镜 / 系统镜 → `bg5_中`（中轴俯瞰）。两人对话用 ①②反打交替，保持轴线一致（180° 不跳轴）。
