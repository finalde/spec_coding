# 风格指南 — 《魔尊归来》

## 渲染样式锁定（follow-up 001 强制）

> 全剧渲染**必须**为影视级真人写实，**不允许**任何动画 / 国漫 / 插画 / 工笔 / 二次元 风格。Seedream 立绘 + Kling 视频 + Seedance 视频三管线的"风格" / "光线/色调" / "避免" 段全部按本节锁定。

### 正向关键词（每份 prompt 必含 ≥ 3 个）

```
影视级真人写实 / cinematic / photorealistic / live-action / 4K HDR / 8K
仙侠古装真人剧造型 / 实拍剧照风 / 现代电影级灯光
真实皮肤质感 / 真实布料质感 / 真实毛发质感
真人演员 / 三庭五眼东方面孔
```

### 负向关键词（每份 prompt 必含全部）

```
anime / cartoon / illustration / chibi / manga
国漫 / 插画 / 工笔 / 水墨写意 / 二次元
CGI 3D render / 塑料皮肤 / 玩偶感 / 低多边形
painterly stylization / 卡通色 / 荧光色
```

### 类比参考（live-action only）

类比《琉璃》（成毅 / 袁冰妍）、《长月烬明》（罗云熙 / 白鹿）、《苍兰诀》（王鹤棣 / 虞书欣）、《沉香如屑》（杨紫 / 成毅）、《与凤行》（赵丽颖 / 林更新）等真人电视剧的剧照画质 — **这些剧本身都是 live-action 真人剧**，类比时引用它们的演员造型 / 服化道 / 镜头语言 / 调色，不引用它们的 IP 动画版。

### Seedream 立绘四段式（per FR-26）

```

[细节] {面貌 + 服装 + 配色 hex + 姿态}

[参数] 9:16 / 4K
[负向] 不要 anime / 不要 cartoon / 不要 illustration / 不要 chibi / 不要 manga / 不要 国漫 / 不要 插画 / 不要 工笔 / 不要 水墨写意 / 不要 二次元 / 不要 CGI 3D render / 不要 塑料皮肤 / 不要 卡通色 + 项目通用负向（不要现代服饰 / 不要西方面孔 / 不要文字水印 / 不要多余手指 / 不要血腥过度）
```

### Kling video prompt 渲染样式行

每镜 Kling prompt 在 `光线/色调:` 行末尾追加：

```
；渲染样式: 影视级真人写实 + cinematic + 4K HDR
```

并在文件末尾追加（如无负向区，则新增 `负向:` 行）:

```
负向: anime / cartoon / illustration / 国漫 / 插画 / 工笔 / 水墨写意 / 二次元 / CGI 3D render / 塑料皮肤
```

### Seedance video prompt 渲染样式行

每镜 Seedance prompt 在 `光线/色调:` 行末尾追加 `；渲染样式: 影视级真人写实 + cinematic + 4K HDR`，并在 `避免:` 段补加上述全部负向关键词。

**严禁出现** "避免: 写实毛孔皮肤瑕疵" 这种反向写实表述（这是把"写实"误判为要避免的内容；本剧明确要求写实皮肤质感）。

## 美学定位

**传统仙侠主框架 + 黑金沉郁主色调**

两套美学的张力：
- 传统仙侠（白衣山门、剑气漂浮、青绿仙气、玉冠云海）= 五大宗主 / 正道盟视觉表层
- 黑金沉郁（黑袍魔气、雷劫沉云、星河金芒、玄黑长阶）= 男主沧冥 / 魔域 / 反英雄视觉

**核心约束（per spec FR-21）**：每集必须有 **≥ 1 镜两套美学同框**——白衣伪君子 vs 黑袍魔尊在同一帧出现，视觉语言本身即在替观众戳穿"伪"。

## 9:16 竖屏构图规范

### 角色站位 3 档

| 档位 | y 坐标范围 | 用途 |
|---|---|---|
| 上 1/3 线 | y ≈ 240-480 | 男主威压 / 反派俯瞰 |
| 中心 | y ≈ 480-720 | 中性叙事 |
| 下 1/3 线 | y ≈ 720-1050 | 角色绝境 / 凡间烟火 |

### 抖音封面安全区

- 主体（脸部）置 y ≈ 300-500
- 大字标题 8-12 字置 y ≈ 800-950（背景留出黑底）
- 文字安全带 y ∈ [160, 1050]
- 不要把脸部放在 y > 700 区域（会被字幕挡）

### 铁律（不可破）

1. 任何 shot 必须明确填 `比例: 9:16`
2. 不允许大幅横向运镜（9:16 不耐看）
3. 单镜头中角色不超过 3 人主体（9:16 容不下 4 人正脸）

## 字幕规范

- 字体：中文方正粗黑简体 + 英文 Helvetica Neue Bold
- 颜色：白色 + 黑色描边（保证 9:16 任意背景可读）
- 位置：y ≈ 1100-1180（最底部）
- 字号：32-36 pt
- 时长：每屏字幕 ≥ 1.5 秒（保证扫读）
- 双语 (YouTube Shorts)：上中文 + 下英文，间距 8 pt

## 字符编码与数字格式

- 中文标点全角（，。！？；：（））
- 英文标点半角
- 数字：阿拉伯数字（"60 集"，"ep01"，"shot01"）
- 修为术语保留汉字 + 英文同位语（YouTube Shorts 字幕：练气 Qi Refining、金丹 Golden Core、渡劫 Tribulation）
