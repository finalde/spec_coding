## Prompt

白月清 · 紫霄宫主。雪白锦绣广袖袍玉冠束发，浅紫云纹绣领，长发披腰俊朗含笑，桃花玉佩，左手负后。

半身立绘，正面构图，三庭五眼东方面孔，剑眉星目，鼻梁笔挺，薄唇含笑；看似三十的俊朗温润如玉。

服装：雪白锦绣广袖袍 (#f5f5f0)，浅紫云纹绣领 (#9070a8)，腰悬桃花玉佩 (#ffd0c0)，外披浅金羽氅 (#c8a85a)。

姿态：双手负后含笑而立，眼角余光斜睨（暗示阴狠），头微抬。

光源（"双面光线"为本剧关键约束）：
- 正面 60% 主光（雪白仙气 #f5f5f0 + 浅金 #c8a85a），照亮含笑的脸
- 背面 40% 暗紫阴影 (#1a1a3a)，从右侧斜后方斜插，让背影染暗紫魔意
- 视觉读法：正面是"君子"，背面已经是"伪"

背景：紫霄宫山门，云海雾气 (#d0e8f0)，飞檐蓝瓦 (#5a8a98)，桃花飘落 (#ffd0c0)。

风格：影视级真人写实 + 4K HDR cinematic + 仙侠古装真人剧造型 + 8K 写实人像摄影 + 真实皮肤布料质感 + 实拍剧照风（live-action）。类比《琉璃》刘学义柏麟 / 《长月烬明》庞宜之 的"清谪反派俊朗"质感。

画幅：9:16，4K。

不要 anime / cartoon / illustration / chibi / manga，不要 国漫 / 插画 / 工笔 / 水墨写意 / 二次元，不要 CGI 3D render / 塑料皮肤 / 玩偶感 / 低多边形 / painterly stylization / 卡通色 / 荧光色，不要现代服饰，不要西方面孔，不要文字水印，不要多余手指，不要赛博朋克，不要明显血腥，不要全黑主调（保持白衣表面），不要全暗（双面光线必要），不要看起来过四十的老态（保持看似三十的俊朗温润）。
