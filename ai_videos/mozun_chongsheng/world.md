# 世界观 — 《魔尊归来》

## 世界格局

**三界对峙**:

1. **仙界 / 天庭** — 万古以来权位之顶，与本剧主线关联弱（仅卷六出现一次"天音封印解除"的远景）。
2. **人间 · 中州** — 主舞台。仙凡共处的中央陆地，宗门林立，凡人与修士比邻而居。本剧 60 集事件 90% 发生在此。
3. **魔界 · 沧冥魔域** — 男主前世据点。位于人间西北雪山深处的"九幽之地"，一座黑金大殿悬于黑色星河之上。男主沧冥归位后将魔界与人间通路重启。

## 男主前世据点 · 沧冥魔域

地理：人间西北雪山深处，"九幽之地"。

特征：黑色星河悬天，一座黑金大殿悬于星河之上；魔域无昼夜，仅冷月为光；殿前有"魔气长阶"自雪山地表延伸到大殿。

ep01 镇压前：男主沧冥据魔域统御群魔。
ep01 镇压后：魔域天柱崩塌，群魔散溃，魔气长阶被五道盟封印。
ep59-ep60 沧冥归位：天柱重立，魔气长阶重现。

## 关键道具与机制

### 1. 系统（男主独占金手指）

男主转生后体内浮现的"系统"。三轨混合：
- **任务模块**: 派发"诛伪君子 / 救凡人 / 揭露阴谋"任务，奖励 = 修为提升 / 神识强化 / 魔功秘籍。
- **双修机缘**: 与三女主之一关键节点双修，触发修为大关突破（金丹→元婴 = 主女主；化神→合体 = 副 1；大乘→渡劫 = 副 2）。
- **丹药机缘**: 药王谷女医师所炼丹药为另一关键阶级跃迁辅助。

**系统 UI**（视觉锁定，per style_guide.md）：
- 配色：文字 + 半透明面板 + 边框
- 字体：中文宋体 / 楷体
- 动画：弹入 0.3s + 停留 1.5s + 淡出 0.3s
- 文案模板：「叮——任务发布: ...」「叮——修为 +1 阶: 练气大成」「叮——魔功获得: 黑息引」

强度：中等加成（系统给任务 / 提示 / 机缘，男主仍需自己阳反击）。

### 2. 双修机缘（系统第二轨）

男主修为关键大关突破由系统触发"双修任务"，与三女主之一在心意相通时双修破境。本剧"双修"用蒙太奇 + 转场 + 第二天满阶画面表现，**不出现裸露 / 直白色情画面**——视觉表达靠光影 + 古风意象（红绳缠手 / 莲花共开 / 灵泉双沐 等）。

### 3. 丹药（系统第三轨）

药王谷女医师苓夭夭炼制：
- **聚气丹**（练气期）— ep10 助男主练气大成
- **金丹**（金丹期）— ep20 男主金丹自成
- **化神丹**（化神期）— ep30 化神突破
- **大乘丹**（大乘期）— ep43 大乘突破
- **渡劫保命丹**（渡劫期）— ep49 渡劫期助男主分担雷劫

## 核心反讽（项目主题）

"正道盟" ≠ 正道；"魔尊" ≠ 邪。
正道是伪君子，魔尊是反英雄。
但魔尊也并非全无瑕疵（ep50-ep55 Midpoint Reversal B：男主前世曾误伤一名正派女弟子）——
最终的"正"，是承认旧污后选择继续清算伪君子的勇气。
