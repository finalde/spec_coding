# perf_0104 · 羞辱难堪·强度3·内敛·肢体（后退半步扶桌维持体面）

> 强度3 的肢体承重态：被当面戳破后下意识后退半步、手扶桌沿稳住，身形随即僵直绷起强维持体面——退缩与僵直同框，是难堪从身体外化的一拍。载体在肢体，面部维持克制。

```
上身先不自觉地向后缩退半步、重心一虚、似要躲离对方的逼视；脚跟落定后一只手立即探向身侧桌沿、指尖按住边缘借力稳住发虚的身形；随即刻意把退势收住、肩背重新挺直、下巴端平、整副身形僵直绷起强装镇定；与此同时视线避开对视向斜下方躲、眼睑低垂；眉内角极轻上提、嘴角下沉又抿平。先退后僵、靠扶桌撑住——退缩的本能与强维持体面的僵直同在，即难堪在身体上的读法。
```

## 配套镜头

- 景别：近景偏中（须容纳后退半步、探向桌沿的手与僵直的上身，人物占画 60–65%）。
- 运镜：镜头静止、轻微手持晃动（失控感外化），不切，让「后退—扶桌—僵直」一镜可见；reaction shot 特写。
- 色调：冷调、略不安的侧光（难堪、失血感），无暖光介入。
