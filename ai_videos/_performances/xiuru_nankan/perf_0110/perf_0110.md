# perf_0110 · 羞辱难堪·强度3·内敛·面部（红转白色变·悲惧混核）

> 本情绪面部判别器的标本条：脸色由红转白的色彩骤变 + 眉内角上提 + 嘴角下拉三者同框，正是「悲（嘴角下拉）+ 惧（眉内角上提）混合」这一无单一 FACS 内核的面部呈现。载体专写面部，视线下移仍作核心信号在底。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
面色在几秒里由一阵涌起的泛红迅速褪向发白（充血到失血的色彩骤变、血色一点点抽走）；眉内角明显向上提起、眉间微皱（惧与悲的混合痕迹），眉峰不动；嘴角随之向下拉沉、下唇微微外翻又被抿住（悲的下拉）；上睑轻抬一道（惧的瞪痕）；视线在色变间向斜下方悄悄躲开、避开对视、眼睑半垂。脸由红转白、眉内角提、嘴角下拉同时成立——悲惧混合的面部内核，配上视线躲闪下移这一核心信号，即难堪的面部读法。
```

## 配套镜头

- 景别：近景偏特写（面部吃满「红转白」的色变与眉嘴的悲惧混合，人物占画 65%+）。
- 运镜：镜头静止、轻微手持晃动（失控感外化），不切，让色彩骤变与微表情全程可见；reaction shot 特写。
- 色调：冷调、略不安的侧光（难堪、失血感），无暖光介入。
