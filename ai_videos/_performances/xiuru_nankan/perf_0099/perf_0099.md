# perf_0099 · 羞辱难堪·强度1·内敛·眼神（gaze aversion 微表情）

> 本情绪强度下限的种子条：被点破一瞬，视线一闪躲开对视、向斜下方溜走又强行收回——一个几乎一帧的 gaze aversion 痕迹，是羞耻核心信号最克制的呈现。强度1 微表情易被模型丢，默认起始帧模式（峰值静帧兜底），见 `## 起始帧表情`。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
正与对方平视的眼神在被点破的一瞬间忽地向斜下方溜开、避开对视一闪、随即又强行抬回平视假装无事；这一躲一回之间眼睑极轻一垂、上睑微眨一次掩饰；眉内角几不可察地一提一落（泄漏的窘）、嘴角极淡一沉又压平。全程身形不动、表情维持平静——唯一被读出的就是那一瞬视线的躲闪下移，是羞耻最早从眼神泄漏的痕迹。
```

## 配套镜头

- 景别：近景偏特写（眼部吃满那一瞬视线的躲闪与收回，人物占画 65%+）。
- 运镜：镜头静止、轻微手持晃动（失控感外化），不切，让「一躲一回」的微动全程可见；reaction shot 特写。
- 色调：冷调、略不安的侧光（难堪、失血感），无暖光介入。
