# perf_0108 · 羞辱难堪·强度2·内敛·肢体（无地自容低头）

> 强度2 的「无地自容」起手：不抗争、直接垂头——头一点点低下、视线钉死在地面、双肩往里缩、后退半步想退出对方的视野。与「强忍难堪」相反，这一支放弃了维持体面，只剩躲避。载体在肢体姿态。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
头一点点低垂下去、下巴收向胸口、不再试图抬眼对视；视线钉死在身前的地面、眼睑半阖；双肩明显往里缩、上身略含、整个人想把自己收小；脚下不自觉地后退半步、重心后撤、像要退出对方的视野；眉内角微提、嘴角下沉抿着不出声。没有任何抗争或强撑的动作——只有垂头、缩肩、退步与钉地的视线，一味把自己藏起来，是无地自容的躲避读法。
```

## 配套镜头

- 景别：近景偏中（须容纳垂下的头、内缩的肩与后退的身形，人物占画 60–65%）。
- 运镜：镜头静止、轻微手持晃动（失控感外化），不切，让「垂头—缩肩—退步」全程可见；reaction shot 特写。
- 色调：冷调、略不安的侧光（难堪、失血感），无暖光介入。
