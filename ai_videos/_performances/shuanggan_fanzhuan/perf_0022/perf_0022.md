# perf_0022 · 爽感反转·强度4·外放·复合（扬眉吐气联动）

> 外放复合的旗舰前序条：把面（挑眉得意笑）、身（肩背打开）、步（半步上前占位）熔成 ONE 向外辐射的 Psychological-Gesture——三通道同向、同时朝「向上、向外、占地盘」打开，合成「我赢了，这局面归我」的扬眉吐气整体。Radiating 质感（由核心向外辐射、free flow）对应外放张扬。house-style 精品微表情派：强度4 是「痛快的一声短笑 + 身体占位」，舒展而有底气，不是咧嘴蹦跳的浮夸。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
三通道在同一拍里同向、同时朝「向上、向外打开」联动成一个向外辐射的整体动作：面——双眉猛挑、嘴角大幅咧开、颧肌上推顶起下睑眼角眯弯（眼周必须参与的真得意笑），随这股舒展仰头送出一声短促的笑（「哈」一声即收，不拖成傻笑）；身——肩胛后收、胸廓打开、上身随仰头微微后展，整个人从「缩着」变成「撑开」；步——一只脚松快地半步上前、把身位往对方一侧推进占位，重心稳稳压下。三处力道一致地朝「外放、占地」走，合成一个由内向外辐射的得意 icon，而非面笑而身僵的拼贴。
```

## 配套镜头

- 景别：近景偏中（须同时容纳上半身与半步上前的身位推进，人物占画 60–70%，肩部以上及上身入镜）。
- 运镜：镜头静止或极缓 push-in，让「仰头短笑—肩背打开—半步上前」的同向辐射在一镜内被读成一个占位瞬间；亦可极轻低角度仰拍一档强化「居高、得意」。
- 色调：可极淡偏一档暖（扬眉吐气=暖意上来），或维持中性白光；不压暗、不冷调，保住舒展的明亮。
