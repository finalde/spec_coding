# perf_0015 · 爽感反转·强度1·内敛·面部（冷笑打脸·微表情）

> 强度1 = 痕迹 / 微表情：只露一两个核心 AU、低强度、单区域。本条是「单侧冷笑」的最小可辨样本——只动一边嘴角的内收上提一瞬，面部其余完全不参与。单侧不对称是蔑视在 FACS 里唯一规范的签名，必须只动一边。微表情常被模型丢失，故默认「起始帧」模式（峰值静帧兜底），见 `## 起始帧表情`。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
单侧嘴角（仅一边）极淡地向内收紧并极轻上提一瞬、随即抚平；另一边嘴角完全不动，左右明显不对称——这一边动、一边静的失衡正是「冷笑」的读法；面部其余区域全程不动：眼周不眯、颧肌不堆、下巴不抬、不出声。情绪只在单边嘴角掠过一瞬即收回，像一句没说出口的「就这？」。
```

## 配套镜头

- 景别：近景偏特写（嘴角与下半脸吃满，单侧不对称唯一可辨的景别，人物占画 65%+）。
- 运镜：镜头完全静止；任何运动都会盖过痕迹级单侧微表情。也可配极缓升格慢镜定格于冷笑那一帧。
- 色调：冷调低饱和（优越冷处理外化），平光，避免阴影在另一边嘴角制造伪不对称。
