# perf_0017 · 爽感反转·强度2·内敛·面部（冷笑打脸）

> 强度2 = 轻度、可辨但不外放。本条把单侧冷笑从「一瞬」延展为「定住」：单侧嘴角上扬并保持，配下巴缓抬 2–3° 形成俯视。这是冷笑打脸子类型在面部载体上的标准件——蔑 AU12+AU14 的单侧不对称（只动一边嘴角）+ 抬下巴俯视，收掉真喜的眼周堆叠。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
单侧嘴角（仅一边）向内收紧并上扬、停住约一拍保持住，另一边嘴角完全不动——左右明显不对称的定格冷笑；下巴缓缓抬起约 2–3°、视线随之略微俯下看向对方，形成由上而下的俯视；眼周不眯、颧肌不堆（非真笑）、不出声。整体冷而稳——一边上扬的嘴角加微抬的下巴，传达「我早料到」的俯视优越。
```

## 配套镜头

- 景别：近景（嘴角不对称与下巴抬角度均需可辨，人物占画 65%+）。
- 运镜：镜头静止，或低角度仰拍主角配合下巴上抬强化俯视；亦可升格慢镜定格于单侧冷笑保持那一拍。
- 色调：冷调或中性低饱和，平光，避免阴影制造伪不对称。
