# perf_0024 · 爽感反转·强度5·外放·复合（扬眉吐气顶端·开口大笑）

> 本情绪强度轴顶端（外放天花板）：仰头开口大笑、张臂、俯视全场，全幅满开的得意辐射。强度5 在真笑（眼周参与的得意笑）基础上**招募开口与下颌松落**——嘴张开、下颌落下，笑出声。house-style 精品微表情派：强度5 也走「电影感的真实开怀大笑」（一个赢家由衷的痛快），而非土味浮夸的咧嘴瞪眼、夸张前仰后合。承重点是「由内向外辐射的满开」读作真实而有底气的得意，不是表演式的狂笑。Radiating 质感（核心向外全幅辐射、free flow）对应外放顶端，也是本情绪的 A/B 种子点（大笑易过火）。

## 锁定文本块


## 配套镜头

- 景别：近景偏中（须容纳仰头大笑的面部、张开的双臂与挺起的上身，人物占画 60–70%，张臂时可让出两侧空间）。
- 运镜：可由静止转极轻 push-in / 低角度仰拍，镜头随人物仰头张臂而微微上扬，强化「居高、满开、俯视全场」的得意；不切，让「仰头—张臂—俯视」的满幅辐射一镜到底。亦可极轻升格定格在张臂峰值半秒，把「赢家亮相」坐成定格。
- 色调：可较测试台基准偏一档暖、提一档亮（扬眉吐气顶端=暖意与光彻底打开），或维持中性白光到底；不压暗、不冷调，保住开怀的明亮与辐射感。
