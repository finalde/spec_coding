# perf_0025 · 爽感反转·强度2·内敛·呼吸（云淡风轻不屑回击）

> 本情绪以「呼吸」为唯一载体的极简条：把那声标志性的「嗤」之以鼻——鼻腔轻吐一口气——外化成可见的呼吸动作，是「云淡风轻不屑回击」最克制的写法。承重点是把听觉的「嗤」（不屑的气音）翻译成**可见的胸口/鼻翼的呼吸位移**，而非靠表情。house-style 精品微表情派：强度2 是「懒得理你」的一口气，几乎不动、随即平静，单侧嘴角的不屑只留极淡一丝。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
鼻翼极轻一张、从鼻腔短促地向下吐出一口气（把「嗤」之以鼻的气音外化成一次可见的鼻息），气流送出时胸口随之微微一沉、肩头几不可察地落一档；与此同时只有一侧嘴角向内极淡地收一丝（不屑的痕迹，幅度小到近乎没有、眼周不参与）；这口气吐尽后立刻恢复平静的呼吸基线、面部回到无表情，全程视线平放不特意去看对方。读作「懒得多费一句、嗤一声就过」的云淡风轻不屑，靠一次可见的鼻息与胸口微沉承载，而非靠表情用力。
```

## 配套镜头

- 景别：近景（须同时容纳鼻翼/胸口的呼吸位移与面部，人物占画 60–70%，肩部以上入镜，胸口起伏需在框内可辨）。
- 运镜：镜头完全静止；任何运动都会盖过「鼻息—胸口微沉」这一档极小的呼吸位移。
- 色调：冷调低饱和（疏离、不屑的外化），不介入暖光；平光，避免阴影伪造表情。
