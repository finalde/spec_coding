# perf_0019 · 爽感反转·强度3·内敛·肢体（云淡风轻不屑回击）

> 本条以肢体优越姿态承重、面部冷处理为辅：说反转台词时不看对方、低头整理衣袖、说完转身离开。「不看对方 + 整理衣袖」是云淡风轻子类型的标志签名——用「我连看都懒得看你」的身体语言把优越感外化，面部只保留单侧 AU14 的极淡冷意（只动一边嘴角）作底色，不抬调。

```

```

## 配套镜头
