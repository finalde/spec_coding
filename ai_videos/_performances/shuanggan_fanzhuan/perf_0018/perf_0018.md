# perf_0018 · 爽感反转·强度3·内敛·复合（冷笑打脸·三拍标杆）

> 本情绪「先怔后冷笑」三拍套路的标杆条，也是内敛冷处理的中段校准基准。复合载体 = 面部冷笑 + 头颈下巴俯视 + 眼神收放联动成一个统一的优越图标（冷笑 + 抬下巴俯视 + 视线最后才给一眼），用 `mffr: Molding` 把数个区域塑成一个「我赢了且你不配」的整体姿态——这不是 suppression 三层（无真情绪外泄/压制冲突），而是优越感的一体化外化。单侧不对称（只动一边嘴角）是冷笑的硬签名。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。按 timed beats 分三拍。

```
0–0.5s 怔：被挑衅瞬间瞳孔一缩、面无表情、全脸不动地静默半秒（台词前的静默帧）。
0.5–2s 冷笑：单侧嘴角（仅一边）向内收紧并上扬、定住，另一边嘴角完全不动——左右明显不对称的冷笑；眼周不眯、颧肌不堆（非真喜）。
2–5s 俯视：下巴缓缓抬起约 2–3°、视线由上而下扫向对方并停住，单侧冷笑保持不变；不出声、动作极简。
三拍连成一个统一的俯视优越图标——怔是蓄势、单侧冷笑是签名、抬下巴俯视是落点。
```

## 配套镜头

- 景别：近景（面部冷笑、下巴抬角与视线落点均需可辨，人物占画 65%+）。
- 运镜：镜头静止，或低角度仰拍主角配合下巴上抬强化俯视；可在「冷笑—抬下巴」段用升格慢镜，并定格于俯视那一帧。
- 色调：冷调或中性低饱和，平光，避免阴影制造伪不对称。
