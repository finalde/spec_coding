# perf_0016 · 爽感反转·强度2·内敛·眼神（云淡风轻不屑回击）

> 强度2 = 轻度、可辨但不外放。本条用眼神承重：眼皮微抬斜睨一眼即移开、配单侧轻挑眉与极淡的单边嘴角，把「你不配我多看一眼」的优越感全压在眼里。单侧——只挑一边眉、只动一边嘴角——是蔑视不对称签名的显化，绝不可两边对称。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
眼皮微微上抬、目光斜睨向对方方向看一眼、停留不足一秒即平淡移开收回正前方，像看过一件无关紧要的东西；同时单侧眉毛（仅一边）极轻上挑后落回，另一边眉不动；单边嘴角极淡地一收，另一边不动、左右不对称。全程面部基线平静、不出声、下巴不抬——优越感只用「斜睨一眼即收」与单侧挑眉传达，不给对方第二眼。
```

## 配套镜头

- 景别：近景（眼部与上半脸吃满，斜睨与单侧挑眉可辨，人物占画 65%+）。
- 运镜：镜头静止或极缓升格慢镜，让「斜睨—移开」一瞬全程可见；亦可定格于斜睨那一帧。
- 色调：冷调或中性低饱和，平光，避免阴影盖过单侧不对称。
