# perf_0021 · 爽感反转·强度3·外放·面部（扬眉吐气大笑）

> 外放风格的面部标杆条：把「得意大笑」收在面部一区里演满，是真笑与假笑的校准样例。判别承重点是 **眼周（颧肌）必须参与**——颧肌上提把下睑顶起、眼角堆出笑纹、眼缝眯弯；只咧嘴不眯眼即落入假笑/土味。house-style 精品微表情派：强度3 是「真心得意的一笑」，灿烂但不咧到失真。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
双眉同时上挑、眉骨抬高把额头牵出一道舒展的横纹（挑眉的「赢了」声明）；嘴角向两侧大幅咧开、上排牙露出、嘴角带笑意上提；与此同时颧部肌肉用力上推、下眼睑被顶起鼓成卧蚕、眼角堆出放射状笑纹、眼缝因此眯弯收窄——眼睛跟着一起笑，这是真心得意而非皮笑肉不笑的关键；头可极轻后仰一档承住这股舒展。整张脸朝「向上、向外打开」走，是「我赢了、痛快」的灿烂得意，而非客套微笑。
```

## 配套镜头

- 景别：近景（面部吃满，人物占画 60–70%，肩部以上入镜；眼周笑纹与咧嘴须同框可辨）。
- 运镜：镜头静止或极缓 push-in，让「眉挑—咧嘴—眼眯」三处同时绽开的得意在一个连续镜头里被读成一个灿烂瞬间。
- 色调：可较测试台基准极淡偏一档暖（扬眉吐气=封冻打开、暖意上来），或维持中性白光到底；不压暗、不冷调，避免削掉笑的明亮感。
