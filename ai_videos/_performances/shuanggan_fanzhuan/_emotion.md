# 爽感反转 / 扬眉吐气（shuanggan_fanzhuan）

功能主轴：**爽感**（逆袭 / 打脸 / 复仇 / 阶层跨越——长期被压、被轻视者在反转那一刻把局面翻过来）。

灯光氛围（检验视频）: 明亮高调、主光偏正、轻微冷暖对比（敞亮、扬眉吐气）

## 情绪定义

「爽感反转」不是单纯的喜悦，而是**得意 + 蔑视的复合优越态**：人物在反转瞬间同时投射两件事——「我赢了」（喜的 AU6+AU12 得意）与「你不配」（蔑的 AU12+AU14 单侧冷笑）。判别核心是 FACS 里唯一规范的**单侧不对称**表情——蔑视：AU12+AU14 单边嘴角内收上提，构成那个标志性的「冷笑」。与真喜（必含 AU6 颧肌堆起、眼周眯弯）相比，冷处理变体常**收掉 AU6**，只留嘴角与抬下巴的俯视。

短剧实战的标志套路是「**先怔后冷笑**」三拍：① 怔——被挑衅瞬间瞳孔一缩、面无表情约 0.5s（台词前的静默帧）；② 单边嘴角上扬冷笑；③ 缓抬下巴俯视对方。配合「说反转台词时不看对方、整理衣袖」的优越姿态。这是打脸/复仇赛道的爽点承重墙，与「压抑隐忍」互为前后段——压得越紧，这里释放越爽。

## 风格二分

本情绪是**释放/优越型**情绪，无 suppression 三层模型。两种风格是同一优越感的不同外化幅度：

- **内敛**（冷处理微表情）：单侧 AU14 冷笑 + 缓抬下巴 + 视线不给对方，全程不抬调、动作极简。FACS 原型 = 蔑 AU12+AU14 单侧 + AU14 维持。关键 beat：嘴角只动一边、下巴 2–3° 缓抬、眼神最后才扫过对方一眼即收。
- **外放**（张扬大笑）：AU6+AU12 得意大笑 + AU2 挑眉 + 俯视，幅度全开。FACS 原型 = 喜 AU6+AU12（强度 5 招募 AU25/26 开口笑）。关键 beat：仰头出声笑、双眉上挑、身体打开占位。

## 子类型

| 子类型 | 优越感配方 | 标志签名 |
|--------|-----------|----------|
| 冷笑打脸 | 蔑 AU12+AU14 单侧 + 抬下巴俯视 | 单边嘴角冷笑、不看对方、整理衣袖 |
| 扬眉吐气大笑 | 喜 AU6+AU12 + AU2 挑眉 | 仰头出声笑、双眉上挑、身体打开 |
| 云淡风轻不屑回击 | 蔑 AU14 极淡 + 平静基线 | 几乎不动、轻挑单眉、说完即转身 |
