# perf_0023 · 爽感反转·强度4·内敛·复合（冷笑打脸旗舰 PG）

> 内敛风格的旗舰复合条：把面（单侧冷笑）、手（整理袖口）、头（抬下巴俯视）熔成 ONE 冷处理的优越 icon——三通道同向、同时、低幅地朝「向内收紧、向下俯视、不给对方」走，合成「你不配，我都懒得多看」的冷处理优越。判别承重点是 FACS 唯一规范的**单侧不对称**——只在**一边**嘴角内收上提构成那个标志性冷笑（另一边不动），眼周不参与、不眯弯（区别于真喜的眼笑），冷而不暖。Molding 质感（向内塑形、bound flow）对应内敛冷处理。house-style 精品微表情派：强度4 是「克制到位的优越」，单边冷笑幅度小而准，不咧、不挑衅。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
三通道在同一拍里同向、同时、低幅地联动成一个「冷处理优越」的整体动作：面——只有一侧嘴角向内收并上提、咬肌微鼓拉出一道不对称的单边冷笑（另一侧嘴角纹丝不动，明确的左右不对称是冷笑的签名），眼周不参与、不眯弯、不堆笑纹（收掉真喜的眼笑，冷而不暖）；手——同时不慌不忙地低头整理一下衣袖袖口、指尖把袖缘抚平，一个「你不值得我正眼相待、我先打理自己」的优越姿态；头——随即下巴缓抬 2–3°、视线由这个微抬的角度从上往下斜落到对方身上俯视，眼神最后才扫过对方一眼即收回。三处力道一致地朝「收、抬、不给」走，合成一个向内塑形的冷处理优越 icon，而非三个孤立小动作的拼贴。
```

## 配套镜头

- 景别：近景（须同时容纳面部、整理袖口的手、抬下巴的颈头，人物占画 60–70%，肩部以上及手部入镜）。
- 运镜：镜头静止；让「单边冷笑—理袖—抬下巴俯视」的同向收紧在一个连续镜头里被读成一个整体冷处理动作；亦可极轻低角度仰拍一档，把「抬下巴俯视」的居高感坐实。
- 色调：冷调低饱和（冷处理优越的疏离外化），不介入暖光；理袖口处可极轻一束侧光勾出袖缘质感，不破冷调基线。
