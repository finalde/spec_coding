# perf_0026 · 爽感反转·强度3·外放·肢体（扬眉吐气·甩袖指人）

> 外放风格以「肢体」为主载体的条：得意不靠脸演满，而靠**身体的占地与方向**——转身甩袖、抬手一指、步伐放开走向对方，把「扬眉吐气」写成可见的肢体外放，面部的挑眉笑意只作辅。承重点是肢体动线的「放开、推进、指向」读作得意而有底气的占位，挑眉笑意为辅而非主。house-style 精品微表情派：强度3 是「痛快地一甩袖、抬手指你」的舒展，干脆有底气，而非张牙舞爪的浮夸。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
肢体为主导地朝「放开、推进、指向」打开成一个得意的占位动作：先一个利落的转身、手臂顺势把衣袖向外一甩（甩出干脆的弧线、一个「不放在眼里」的潇洒）；随即抬起一只手、伸出食指朝对方一指（「就是你、看清楚了」的指认）；同时步伐放开、不疾不徐地迈步走向对方、重心稳稳压着推进、上身挺展肩背打开。面部为辅：双眉随这股舒展上挑、嘴角带着得意的笑意上提（笑意辅佐、不抢肢体的主戏）。全程力道朝外放、占地、指向走，读作一个赢家干脆有底气的甩袖指人与推进，舒展痛快，而非张牙舞爪的浮夸。
```

## 配套镜头

- 景别：近景偏中（须容纳转身甩袖、抬手指认与迈步推进的肢体动线，人物占画 60–70%，上身及手臂入镜，甩袖/指人时让出动作空间）。
- 运镜：镜头可极轻跟摇或随人物迈步极缓后撤一档，承住「走向对方」的推进；亦可极轻低角度仰拍把「居高、得意」坐实；不切，让「转身甩袖—抬手一指—迈步推进」的肢体外放一镜到底。
- 色调：可较测试台基准极淡偏一档暖（扬眉吐气=暖意上来），或维持中性白光；不压暗、不冷调，保住舒展明亮。
