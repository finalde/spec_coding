# perf_0020 · 爽感反转·强度3·内敛·眼神（冷笑打脸）

> 本条用眼神承重冷笑打脸：由下往上缓抬眼皮锁定对方、眼神不眨定住三秒、配单侧嘴角极淡冷意。AU7 下睑收紧让眼神更冷更具压迫感，AU14 单侧嘴角内收（只动一边）保留冷笑的不对称签名。眼睛「锁住不放」的定力是这一条的爽点——把「我盯着你看你慌」的优越感全压在不眨的眼神里。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
眼皮由下往上缓缓抬起、视线自下而上抬至锁定对方、下睑微微收紧使眼神更冷更聚；锁定后眼神不眨、稳稳定住约三秒、不游移不躲闪，像把对方钉在原地；同时单侧嘴角（仅一边）向内收一丝极淡冷意、另一边不动、左右不对称；不出声、面部其余不动。优越感全靠「缓抬眼—锁定不眨」的眼神定力承重，单侧冷笑只作底色。
```

## 配套镜头

- 景别：近景偏特写（眼部吃满、下睑收紧与不眨定力唯一可辨的景别，人物占画 65%+）。
- 运镜：镜头静止，或低角度仰拍主角配合眼皮上抬强化「由下往上锁定」；可升格慢镜定格于锁定不眨那一帧。
- 色调：冷调低饱和，平光，避免阴影制造伪不对称或盖过眼神。
