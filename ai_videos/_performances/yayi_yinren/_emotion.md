# 压抑隐忍（yayi_yinren）

功能主轴：**痛感**（被背叛 / 被压迫 / 被剥夺 / 被不公对待时，情绪上涌却强行压住）。

灯光氛围（检验视频）: 冷调低饱和、柔和侧光、略压暗（窒息感）

## 情绪定义

「压抑隐忍」不是悲伤 / 愤怒的弱化版，而是一种**冲突态**：内里有强烈情绪（悲、怒、屈）正在上涌，而人物正用可见的努力把它压住。表演的核心张力来自「上涌」与「压制」的对抗——观众看到的是**情绪在泄漏、而抑制肌在拦截**。这正是 FACS 微表情研究里「强忍 / suppression」的机制：被掩盖的情绪从控制力较弱的上半脸（眼 / 眉）先泄漏，而下半脸（嘴）被刻意维持。

在短剧里这是痛感赛道（重生 / 复仇 / 追妻 / 赘婿前期）的承重情绪：受辱不发作、含泪不落、强压怒火——压得越紧，后续「爽感反转」释放时越爽。

## 三层模型（本情绪所有「内敛压抑」条目必含）

1. **泄漏的真情绪 AU**（低强度 / 单区）——悲的 AU1+AU4（眉内角上提 + 眉间皱）或怒的 AU4+AU7（压眉 + 下睑紧），只露一点、只在一个区域。
2. **可见的抑制肌**——AU23/AU24（双唇收紧 / 用力相压）、AU17（下巴上推抗住下垂的嘴角）、吞咽、控制下颌。**抑制的"努力"本身是可见的**，这是「忍」的实体。
3. **上下脸冲突**——眼 / 眉泄漏，而嘴强行平复（或反之）。两区域的不一致 = 「强忍」的读法。**眼睛先泄漏**，所以眼神是本情绪最高信号的载体。

## 风格二分

- **内敛压抑**（本情绪主体）：bound flow + sustained——压住、不放。
- **外放爆发**：压抑**失守**的临界 / 决堤瞬间（suppression breaks），是本情绪强度轴顶端（强度5）的形态，不是另一种情绪。

## 强忍子类型（抑制肌签名不同，各成独立 entry）

| 子类型 | 上涌的真情绪核 | 抑制肌签名 |
|--------|----------------|------------|
| 强忍泪水 | 悲 AU1+AU4+AU15 | AU17 下巴上推 + 吞咽 + 眨眼抑制锁泪膜 |
| 强压怒火 | 怒 AU4+AU7 | AU23/AU24 双唇相压 + 咬肌收紧 + 鼻翼张 |
| 强装镇定 | 惧 / 慌 AU1+AU2+AU5 | 维持平静基线 + 偶发 micro-leak 破绽（手抖 / 吞咽 / 眼神飘） |
