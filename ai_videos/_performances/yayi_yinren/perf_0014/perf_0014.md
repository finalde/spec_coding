# perf_0014 · 压抑隐忍·强度3·内敛·复合（强装镇定签名）

> 「强装镇定」子类型的签名条。与忍泪/忍怒的关键区别：抑制肌不是某一组肌肉用力，而是 **刻意维持的一条「平静基线」本身**——上涌的是惧/慌（眉抬、瞪眼的惊惧痕迹被强压下去），人物努力扯出一个不到位的淡定表情把它盖住；而**破绽从控制力最弱处不自主泄漏**（一次不合时宜的吞咽、指尖极轻发抖、眼神飘移半秒又拉回、眨眼频率不自然）。镇定是「装」出来的：维持的平静线 + 控制不住的 micro-leak。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。三层读法：①泄漏=惧/慌（眉微抬、上睑一闪的瞪、被迅速压下）②抑制肌=刻意维持的平静基线（努力放松面部、扯出淡定）③上下脸冲突=表面镇定 vs 控制弱处漏出的不自主破绽。

```
主体是一条刻意维持的平静基线：面部努力放松、扯出一个不太到位的淡定神情、嘴角勉强端平、视线刻意稳住平视前方，像在告诉所有人「我没事」。但底下压着的是慌——眉曾微微一抬、上睑一闪的瞪（惊惧的痕迹）旋即被强压回平静。镇定是装的，破绽从控制力最弱处不自主漏出：喉头一次不合时宜的吞咽、指尖极轻地发一下抖、眼神飘移半秒又被硬拉回平视、眨眼频率变得不自然（忽快忽顿）。这些 micro-leak 不是表演给人看的，是压不住而溜出来的——维持的平静线越稳，漏出的小破绽越说明「装」。无泪、无咬肌鼓起、无下巴上推（那些是忍泪/忍怒）；签名落在「假镇定基线 + 不自主的吞咽/手抖/眼神飘」这一组。
```

## 配套镜头

- 景别：近景偏特写（捕捉假镇定下的破绽微动——吞咽、指尖抖、眼神飘半秒，人物占画 65%+；若收指尖入画可略放宽至胸上）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让「稳住的脸」与「溜出的破绽」在同一镜头里对照，破绽才有反差。
- 色调：冷调低饱和（压抑/不安外化），无暖光介入；平光避免阴影制造伪表情，让破绽是表演给的而非光给的。
