# perf_0008 · 压抑隐忍·强度3·内敛·肢体

> 本情绪「以肢体为主通道」的标杆条：把情绪挤进手——手部的「用力」是抑制的出口。冲突在「手上积聚的力（攥紧、捏变形）」与「上半身刻意维持端正不动」之间，面部仅作泄漏副层。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
一只手缓慢收拢、攥紧身侧衣料（或桌沿），手指逐节收紧、指节因用力而发白，前臂肌肉绷起；另一种处理：一只手把纸杯/易拉罐缓慢捏到变形。攥力持续积聚不松（情绪被挤进手）；而上半身刻意维持端正、肩与头不动、坐姿不塌——身体上下分裂：手在用力，躯干强装无事。面部仅极轻眉间一皱（怒/屈的痕迹）泄漏，双唇抿成平线极轻相压控住。手的「持续攥紧」即「忍」的实体。
```

## 配套镜头

- 景别：近景（须同时容纳攥紧的手与端正的上半身，人物占画 60–65%，肩部以上及手部入镜）。
- 运镜：镜头静止；让「手积力 ↔ 躯干不动」的分裂全程并置可读。
- 色调：冷调低饱和（窒息感外化），无暖光介入。
