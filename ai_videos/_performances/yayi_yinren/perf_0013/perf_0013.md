# perf_0013 · 压抑隐忍·强度3·内敛·面部（强压怒火签名）

> 「强压怒火」子类型的签名条。与忍泪的根本区别在**上涌的真情绪核是怒、不是悲**，因此抑制肌签名也完全不同：没有泪、没有下巴上推托嘴角，而是 **双唇用力相压成一条直线 + 咬肌（下颌角）收紧鼓起 + 鼻翼微张**。三层模型：怒的泄漏（压眉+下睑紧、眼神凌厉）＋抑制肌（唇压+咬肌+鼻翼）＋上下脸冲突（眼在喷火、嘴被死死压住不发作）。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。三层读法：①泄漏=怒（压眉+下睑收紧+凌厉直视）②抑制肌=双唇相压+咬肌鼓起+鼻翼张 ③上下脸冲突=眼喷火、嘴压死。

```
眉头向下压、眉间挤出川字（怒的压眉），下睑用力收紧使眼神变凌厉、直视正前方寸步不让；这股上涌的是怒、不是悲——眼里没有泪膜、只有压着的火。抑制肌签名清晰：双唇用力相压、抿成薄薄一条直线（不是噘、不是垮，是横向压死）；下颌角的咬肌随之收紧、鼓起一小块、太阳穴侧绷住；鼻翼随压着的呼吸微微一张一合（怒的呼吸从这里泄漏）；整张脸紧绷、颈侧青筋隐现但克制着不发作。嘴绝不张、不喊；下脸被咬肌与唇压锁死，上脸的眼在喷火——这上下脸的对抗即「强压怒火」的读法。无泪、无下巴上推（那是忍泪），签名落在咬肌+唇压+鼻翼这一组。
```

## 配套镜头

- 景别：近景偏特写（咬肌鼓起、唇压直线、鼻翼张同框可辨，人物占画 65%+）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让紧绷的「压而不发」全程可见，逼出窒息的对峙感。
- 色调：冷调低饱和（压抑感外化），无暖光介入；可让侧脸下颌角处微有阴影勾出咬肌轮廓（但不引入方向硬光破坏控制变量，仅靠表演本身）。
