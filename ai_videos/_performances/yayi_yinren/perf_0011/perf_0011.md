# perf_0011 · 压抑隐忍·强度5·外放·复合（压抑失守 → 决堤）

> 本情绪强度轴顶端：不是另一种情绪，而是「压抑」的**失守瞬间**——前面所有内敛条目压住的那股上涌，在这一条决堤。表演的承重点是「压住 → 决堤」这个**转折本身可见**，而非崩溃后的哭脸；house-style 走精品微表情派，强度5 也要演成「电影感的真实崩塌」，不是土味嚎啕浮夸。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
开场半秒仍在压：双唇用力相压成一条直线、屏住呼吸、下巴绷住、肩端着——还是「忍」的残留姿态；随即抑制肌一齐松脱，转折在这一瞬可见：双唇骤然失守、嘴角猛地下垮成深倒八字、嘴随之张开（不是嚎，是被冲开）、第一声抽泣从喉头顶出令上身一震；几乎同时眼眶蓄着的泪决堤、成串顺颊滑落，眉内角猛挑、眉间皱成川字，闭眼一下又睁开已满是水光；最后肩塌、颈一软、上身前倾或向下瘫坐一档，呼吸碎成抽气。全程读法是「堤坝崩开」——先有压住的痕迹、再有压不住的决口，而非一上来就放声大哭。
```

## 配套镜头

- 景别：近景偏特写（崩塌瞬间的面部全程吃满，人物占画 65%+；前倾/下沉时可让出一点头顶空间）。
- 运镜：可由静止转极轻手持跟焦，镜头随人物上身下沉而微微下坠，强化「人塌下去」的失重感；不切，让「压住—决堤」的转折一镜到底。
- 色调：起手维持内敛的冷调低饱和，决堤瞬间可极淡转一档微暖（失守=封冻松动）或维持冷调到底（窒息感不松手）——两种皆可，按所在剧情段的底色定。
- 声音：环境音在转折前一瞬抽空留白，第一声抽泣顶出时回声，放大决堤的听觉冲击。
