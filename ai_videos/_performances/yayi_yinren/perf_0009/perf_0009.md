# perf_0009 · 压抑隐忍·强度3·内敛·复合（旗舰 PG）

> 本情绪的旗舰复合条：一个把眼/面/呼吸/肢体熔成 ONE coherent beat 的 Psychological-Gesture icon——四通道同向、同时、低幅，合成「全身都在压住」的整体。Molding 质感（向内塑形、bound flow）对应内敛压抑。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
四通道在同一拍里同向、同时、低幅地联动成一个「全身向内压住」的整体动作：眼——泪膜在睑缘成形、盈而不落，眨眼极慢锁住泪膜；面——下巴极轻上推、双唇相压抗住正在下垂的嘴角，眉内角极轻上提一闪；呼吸——缓慢深吸后把吐气拉长、压住胸腔不让起伏；肢体——一只手悄然收拢攥紧衣角、指节因用力发白。四处力道一致地朝「收、压、向内」走，没有任何一处放出来——这是一个整体被向内塑形（molding）的克制 icon，而非四个孤立小动作的拼贴。
```

## 配套镜头

- 景别：近景（须同时容纳手、上身、面部，人物占画 60–70%，肩部以上及手部入镜）。
- 运镜：镜头静止；让四通道的「同向收紧」在一个连续镜头里被读成一个整体动作。
