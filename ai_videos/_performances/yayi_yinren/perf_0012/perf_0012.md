# perf_0012 · 压抑隐忍·强度3·内敛·复合（强忍泪水签名旗舰）

> 「强忍泪水」子类型的签名旗舰条。三层模型齐备且抑制肌签名必须**一眼可辨、不可与忍怒/装镇定互换**：上涌的真情绪核是悲，抑制肌签名是 **下巴明显上推 + 喉头吞咽一次 + 眨眼刻意抑制锁泪膜**。复合载体把眼/嘴/喉/眉四区联动成一个「含泪不落」的整体。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。三层读法：①泄漏=眉内角上提的悲 ②抑制肌=下巴上推+吞咽+眨眼抑制 ③上下脸冲突=眼在垮、嘴被托平。

```
泪膜在睑缘明显成形、亮汪汪蓄住却不滑落；下巴明显向上推顶（把往下垮的嘴角硬托回平、下唇微噘），这一下巴上推是最显眼的「忍」动作；喉头随之滚动、清晰地吞咽一次压住哽咽；眨眼被刻意抑制——睁着不眨以靠表面张力锁住泪膜，偶尔一次极慢的半阖又睁开把泪压回；眉内角上提、眉间极轻皱（悲的泄漏）而眉峰不动；嘴在下巴的上推下维持一条勉强的平线、不下垮也不张。上脸（眼/眉）在垮、下半脸被下巴硬托住——这上下脸的对抗即「强忍泪水」的读法。无泪滴落、无嚎、无咬肌鼓起、无鼻翼张（那是忍怒不是忍泪）。
```

## 配套镜头

- 景别：近景偏特写（眼部泪膜与下巴上推同框可辨，人物占画 65%+）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让「泪膜成形—不落」与「下巴上推—吞咽」的全过程可见。
- 色调：冷调低饱和（窒息感外化），无暖光介入。
