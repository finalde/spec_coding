# 柔情 / 深情（rouqing_shenqing）

功能主轴：**欲望**（爱情 / 占有 / 关系——情感向对方收拢、整体语气放软）。

灯光氛围（检验视频）: 暖调柔光、低对比、浅景深氛围（亲密、放软）

## 情绪定义

「柔情 / 深情」是注意力向一个人收拢、防御卸下的**放软态**。与本库多数痛感 / 冷感情绪相反，它不是情绪在泄漏或被压制，而是**面部肌张力主动松弛 + 注视成为核心载体**。FACS 原型是**轻 AU6+AU12**（颧肌轻堆、嘴角微斜上拉的真笑底色）叠加**整体面部放松**；但真正承载「深情」的不是嘴而是**眼睛——gaze（注视）是这个情绪的核心**：眼神聚焦放软、瞳孔微湿、视线先落对方唇 / 眼再缓移开。FACS 不编码视线方向，所以这是少数需要 oculesics（注视行为学）词汇而非纯 AU 的情绪。

在短剧里它是欲望主轴的承重 beat：追妻线的破冰、双人同框的对视、表白前的欲言又止都靠它。它是少数**非痛感、非冷感**的情绪——呼吸放慢、说话带气声、语气整体放软是它的统一基调。

## FACS 原型与标志 beat

- **原型 AU 核**：轻 AU6+AU12（颧肌轻堆、嘴角微上扬的真笑底色）+ 面部放松（额、眉、下颌松开）。注意 AU6 在否是真情 / 假笑判别器——深情必含 AU6 的眼周柔化。
- **核心载体 = 注视（gaze）**：眼神聚焦放软、瞳孔微湿（泪膜微光非泪）、视线先落对方唇 / 眼再缓移开。视线行为压过面部表情成为最高信号。
- **标志套路**：指尖将对方碎发别到耳后；呼吸放慢、说话带气声、语速放缓。
- **配套镜头**（entry 选填 `配套镜头:` 字段建议——可写暖调，但**检验视频测试台仍中性白光**，二者不冲突）：暖黄色调（压抑→新生的色彩转向）、逆光柔光、浅景深、中近景双人同框。

## 风格二分

- **内敛**（克制深情 / 欲言又止）：情感在涌但话压在喉头——视线落对方又移开、唇微动又收回、想触碰的手停住。注视是承载体，张力来自「想说 / 想靠近而克制」。
- **外放**（炽热表白 / 靠近拥抱）：情感外溢——身体主动靠近、伸手拥抱、直视不避、声音里带颤的暖意。不是另一种情绪，是同一收拢的外放释放。

> 三层抑制模型只对「欲言又止」这一类内敛条部分适用（涌起的暖意 + 压住的话语 + 想触碰而停的手 = 上下脸/手冲突），但这里的「压」是含蓄而非痛感压制，签名是**柔化的迟疑**而非抑制肌发力；故按 gaze 核 + 放软 beat 写，仅在欲言又止条注明克制层，不强套抑制肌签名。
