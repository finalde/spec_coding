# perf_0116 · 外放愤怒·强度3·外放·复合（拍案而起 · 拍桌起身）

> 拍案而起子类型的复合基准条：一掌拍桌 + 猛地起身 + 挥手 + 颈侧青筋暴起。怒通过一连串爆发性动作往外宣泄（Radiating），是「动作宣泄」的标志套路——区别于狠戾「缓慢摔杯」的冷收，这里是热的、突发的、放出来的动作。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
一掌猛地拍向桌面、发出一声闷响、上半身随这一掌前压；紧接着双腿一蹬、猛地从坐姿站起、椅子被带得后退；起身的同时一只手往侧下方狠狠一挥（甩开 / 指向对方）；颈侧的青筋随用力鼓起一道、下颌前送；眉头压下、瞪眼、下睑收紧，呼吸变粗、胸口起伏。拍桌、起身、挥手三个动作一气呵成往外炸——怒借这串爆发动作整个宣泄出去，即「拍案而起」的复合读法。
```

## 配套镜头

- 景别：近景起、随起身松到中近景（吃满拍桌与起身的纵向爆发，人物占画 60%+）。
- 运镜：快速 push-in 配合拍桌一震；轻晃手持随起身上抬跟摇，让三个动作的连贯爆发可见。
- 色调：硬光高对比、略偏暖（火气、逼迫）。
