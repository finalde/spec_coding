# 外放愤怒（waifang_fennu）

功能主轴：**痛感 / 爽感**（被背叛 / 被压迫激起的当众爆发，或反击施压时的宣泄式发作）。

灯光氛围（检验视频）: 硬光高对比、略偏暖（火气、逼迫）

## 情绪定义

「外放愤怒」的 FACS 内核是**怒（anger）= AU4+AU5+AU7+AU23 全力**（压眉 + 瞪眼 + 下睑收紧 + 双唇抿压），并在峰值招募 **AU25 张口吼**。区别于「狠戾·阴鸷」(`henli_yinzhi/`) 的"**压**"，外放愤怒是把怒火**放出来**：呼吸变重、胸口起伏、上前逼近、拍桌 / 挥手、声调拔高、青筋暴起、眼神发直。它不是内收威胁，而是向外宣泄。

短剧里它是痛感 / 爽感赛道的爆发承重 beat，常配**快速 push-in**（怼脸推进）、轻晃手持、硬光高反差。

**与「狠戾·阴鸷」的区分（关键）**：狠戾是**冷的、压的、威胁性的**（向内收的狠——指节发白、咬肌收紧、由下往上抬眼、不眨眼凝视、冷声不抬调、缓慢摔杯）；外放愤怒是**热的、放的、宣泄性的**（向外爆的怒——呼吸重、胸口起伏、声调拔高、拍桌挥手、青筋）。同样源自怒，狠戾把它收进威胁的低气压，外放愤怒把它炸成宣泄的高声浪。

## 三层模型（仅本情绪「临界·将发未发」低强度内敛变体含简化形）

外放愤怒**主体是外放**——本就是放出来的怒，绝大多数 entry 不走 suppression。唯一例外是低强度的**「将发未发」临界内敛变体**（与狠戾 / 压抑交界）：怒已上涌、抑制肌（AU23 双唇相压 + 咬肌）还压着、即将失守。此变体含简化三层（泄漏的 AU4+AU7 + 抑制肌 AU23/咬肌 + 一触即发的临界），其余外放 entry 直接用怒原型 AU 全力，不含三层。

## FACS 原型 AU 组合与关键 beat

- **怒核（全力）**：AU4 压眉（川字纹）+ AU5 上睑提瞪眼 + AU7 下睑收紧眼神凌厉 + AU23 双唇抿压。
- **峰值张口**：AU25 唇分 / 张口吼（强度5）。
- **外放标志 beat（区别于狠戾的"压"）**：呼吸变重、胸口起伏、上前逼近、拍桌 / 挥手、声调拔高、青筋暴起、眼神发直。
- **配套镜头**：快速 push-in（怼脸推进）、轻晃手持、硬光高反差。

## 风格二分

- **外放**（本情绪主体）：free flow + sudden + strong weight（Punch/Slash）——怒火放出来，呼吸重、上前、拍桌、拔高声调。
- **内敛**（低强度「将发未发」临界变体）：bound flow——怒已上涌但抑制肌还压着、一触即发，是与狠戾 / 压抑的交界形态，仅占强度轴低端。

## 子类型（爆发形态不同，各成独立 entry）

| 子类型 | 形态签名 |
|--------|----------|
| 爆发怒吼 | AU25 张口吼 + 声调拔高 + 胸口剧烈起伏（声浪宣泄） |
| 拍案而起 | 拍桌 / 猛起身 + 挥手 + 青筋（动作宣泄） |
| 逼近质问 | 上前逼近压迫 + 眼神发直 + 压低又重的质问（逼问宣泄） |
