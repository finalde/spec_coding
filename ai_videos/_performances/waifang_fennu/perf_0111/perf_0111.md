# perf_0111 · 外放愤怒·强度1·内敛·面部（怒微表情痕迹）

> 本情绪强度轴最低端的微表情种子条：怒只在脸上一闪而过——眉间川字纹瞬聚瞬散、下睑收紧一瞬，旋即抹平。是「将发未发」之前的更早一刻，怒火刚冒头就被自己摁回。微表情易被模型丢，故默认配 `## 起始帧表情`，按 `_testrig.md` 与 perf_0120 做 A/B 种子点。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
眉间极快地一聚、竖出一道川字纹后立刻松开（半秒内的一闪）；与此同时下睑收紧一瞬、眼神短促地一沉发直；鼻翼几不可察地张了一下、一口气略微吸重；随即整张脸刻意抹平、恢复平静。全程不出声、不张口、不前倾——怒只在眉眼间走过一道痕、又被自己摁回去，瞬聚瞬散的微表情即「怒火刚冒头」的读法。
```

## 配套镜头

- 景别：近景偏特写（眉间与下睑吃满川字纹的瞬聚瞬散，人物占画 65%+）。
- 运镜：快速 push-in（怼脸推进）后定住；轻晃手持，让一闪的怒痕全程可辨。
- 色调：硬光高对比、略偏暖（火气、逼迫）。
