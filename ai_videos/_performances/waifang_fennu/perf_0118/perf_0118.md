# perf_0118 · 外放愤怒·强度4·外放·复合（逼近质问 · 怼脸拔高）

> 逼近质问子类型的高强度条：逼到怼脸的近距、声调一句句拔高、眼神发直地钉死对方质问。比 perf_0115（基准）更近、更响、更失控半档——逼迫已带破音的拔高（区别于狠戾「不抬调的冷质问」，这里是热的、声浪往上冲的逼问）。高强度条默认起始帧模式（峰值怒容纯文本易僵），见 `## 起始帧表情`。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
逼到几乎贴脸的近距、头前探把脸送到对方跟前、眼神发直地钉死对方不许躲；眉头死死压下、眉间深川字纹，瞪眼、下睑绷紧；嘴随一句句质问越张越开、声调一句比一句往上拔、最后一句带出破音；下巴随每句往前一送一送、唾沫星子几乎喷出；呼吸又重又急、胸口剧烈起伏，颈侧青筋鼓起。逼近、怼脸、声调一路拔高地连珠质问——怒借逼问的声浪整个往外冲、把对方逼到无处退，即「逼近质问」高强度的外放读法（强烈而真实，不是浮夸喊叫）。
```

## 配套镜头

- 景别：近景偏特写、随逼近压成怼脸（面部与前探的头同框，人物占画 70%+）。
- 运镜：快速 push-in（怼脸推进）配合人物逼近；轻晃手持，让一句句拔高的逼问张力可见。
- 色调：硬光高对比、略偏暖（火气、逼迫）。
