# perf_0120 · 外放愤怒·强度5·外放·复合（爆发怒吼 · 张口炸开）

> 本情绪强度轴顶端的怒吼旗舰：张口怒吼到底（AU25）+ 胸口剧烈起伏 + 声浪炸开，面、气、声三通道全量招募熔成 ONE 爆发的怒。house-style 精品微表情派：强度5 怒吼纯文本最易合成成土味咆哮、僵脸干嚎，故默认配 `## 起始帧表情`，视频只承担从峰值帧出发的吼气与胸口起伏插值，写成电影感的真实爆发。按 `_testrig.md` 与 perf_0111 做 A/B 种子点（峰值怒纯文本是起始帧增益的核心验证点）。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
嘴张到最大、下颌彻底落下、喉咙整个打开吼出一声（口型方而宽、能看见声带绷紧）；眉头死死压到最低、眉间深竖纹，瞪眼瞪到极限、上下眼白都露出来、眼神发直冲向前方；颈侧与额角的青筋随这一吼全鼓起来、面部因充血而涨红；胸口随吼声剧烈一鼓、整个上身往前送、双肩端起；呼吸在吼与吼之间又急又深地大口换气、鼻翼大张。怒火顺着这一声吼整个炸出去、声浪把脸往前推——张口到底的爆发即「爆发怒吼」顶端的外放读法（强烈、真实、电影感，不是浮夸干嚎）。
```

## 配套镜头

- 景别：近景偏特写、随吼略松（吼到极限的口型与涨红的脸吃满画面，人物占画 65%+）。
- 运镜：快速 push-in（怼脸推进）配合怒吼炸开；轻晃手持随吼震微跟摇，让张口爆发可见。
- 色调：硬光高对比、略偏暖（火气、逼迫）。
