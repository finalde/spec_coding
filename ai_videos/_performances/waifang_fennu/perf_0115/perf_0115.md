# perf_0115 · 外放愤怒·强度3·外放·肢体（逼近质问 · 上前一步）

> 逼近质问子类型的肢体基准条：上前逼近一步、用身体压向对方、眼神发直地钉住。怒通过整个身体往前推的「逼」放出来（区别于面部条的表情承重）——是「向外宣泄」最直白的身体动作。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
上身先向前一倾、随即跨上前一步、整个身体明显逼向对方、把距离一下压到很近；下巴随之前送、头略前探、眼神发直地钉住对方不放；肩膀端起、双肩前压，重心整个压上去带出逼迫的体势；眉头压下、下睑收紧，呼吸变重。怒不是停在脸上而是用整个身体往前推过去——上前一步、把人逼到跟前的体势即「逼近质问」的肢体读法（热的、放出来的逼，不是冷站着的威胁）。
```

## 配套镜头

- 景别：近景起、随上前压成更近的怼脸（人物占画 65%+并持续放大）。
- 运镜：快速 push-in（怼脸推进）配合人物上前；轻晃手持，让逼近的压迫体势可见。
- 色调：硬光高对比、略偏暖（火气、逼迫）。
