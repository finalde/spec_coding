# perf_0119 · 外放愤怒·强度4·外放·复合（拍案而起 · 猛拍暴起）

> 拍案而起子类型的高强度条：双掌重重砸桌、猛地暴起、大幅挥臂、青筋暴起。比 perf_0116（基准）更猛、更大幅、更接近失控——是动作宣泄轴临近顶端的爆发。高强度条默认起始帧模式（峰值动作纯文本易僵），见 `## 起始帧表情`。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
双掌一齐重重砸向桌面、桌上的东西被震得跳起、上身随这一砸猛地前压；几乎同时双腿一蹬、整个人猛地暴起站直、椅子被狠狠带翻往后；起身的惯性带着一条手臂大幅往侧上方一挥、像要把眼前的东西扫开；颈侧与额角的青筋随暴发力鼓起、太阳穴突突跳；眉头死死压下、瞪眼、下睑绷紧，嘴张开吼出一声、呼吸又重又急、胸口剧烈起伏。砸桌、暴起、挥臂连成一记爆发——怒被这股暴发力整个甩出去，即「拍案而起」高强度的外放读法（强烈而有重量，不是浮夸摔打）。
```

## 配套镜头

- 景别：近景起、随暴起松到中近景（吃满砸桌与暴起的纵向爆发，人物占画 60%+）。
- 运镜：快速 push-in 配合砸桌一震；轻晃手持随暴起上抬猛跟摇，让连贯爆发可见。
- 色调：硬光高对比、略偏暖（火气、逼迫）。
