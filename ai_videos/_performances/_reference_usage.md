# 引用样例 — shot 如何「引用 + 按剧情融入」表演库（FR-10 v2）

短剧 shot **引用**表演库条目、按本镜剧情**融入**，**不照抄**整段。以 `perf_0003 · 压抑隐忍·强度3·内敛·眼神（强忍泪水）` 为例，用于一个 shot（女主苏晚当众受辱、强忍不发作的反应镜头）。

> 注：实际引用应在被引 entry `validated` 后。`perf_0003` 当前 `pending_review`。

## 引用契约（标注 + 融入）

1. **标注**（shot.md 的 `## Shot context`）：加一行
   `表演库参考: perf_0003 (压抑隐忍·强度3·内敛·眼神) — 用于 苏晚 当众受辱强忍`
   多角色/多 beat 可多行。这行让"更新表演库后一键重生成本 shot prompt"可机检。
2. **融入（非照抄）**：把 entry `## 锁定文本块` 的**物理动作要点**改写进 shot 的 `动作:` / `表情:`——保留"写物理动作不写情绪名"的内核与关键肌肉动作，但按本镜的**角色 / 机位 / 时长 / 剧情语境**重新措辞。**不要**把 entry 的检验视频 prompt 整段粘进来。

## 被引用的 entry（库里的通用锁定块）

```
眼眶下缘极淡泛红、泪膜在睑缘成形但不滑落；上睑微抬强压、眨眼频率刻意放慢以锁住泪膜；视线先垂向斜下方一瞬、随即强行抬回平视；眉内角极轻上提（忧的痕迹）而眉峰不动；喉头一次不易察觉的吞咽。全程嘴部保持平静、不参与——上下脸的不一致即「强忍」的读法。
```

这是 generic 的（无任何剧目专名、无机位、无时长）。

## 融入后的 shot prompt（按本镜剧情改写，非照抄）

```text
01集07镜视
表演参考: perf_0003 (压抑隐忍·强忍泪水)
人物: 苏晚
场景: 宴会厅·众人围观
镜头: 近景偏特写，正面，镜头静止
动作: 0–2s 苏晚立于厅中承受当众羞辱，眼眶下缘极淡泛红、泪膜在睑缘成形将盈而不落；2–4s 上睑微抬强压、眨眼放慢锁住泪膜，视线先垂向斜下方一瞬随即强抬回平视，迎向画外讥讽她的方向；4–5s 眉内角极轻上提而眉峰不动、喉头一次不易察觉的吞咽，全程嘴角抿平不参与——上下脸的不一致即强忍。
台词 / 字幕: （无台词）
光线 / 色调: 冷调低饱和
比例: 9:16
时长: 5s
```

## 对比要点

| | 库里的 entry（通用） | 融入后的 shot（本剧专用） |
|---|---|---|
| 主体 | 无名演员 | 苏晚 |
| 机位/时长 | 测试台 5s | 本镜近景特写、按 0–2/2–4/4–5s 拆 beat |
| 语境 | 无 | "承受当众羞辱""迎向画外讥讽方向" |
| 物理动作内核 | 泪膜成形不落 / 眨眼抑制 / 喉头吞咽 / 上下脸冲突 | **逐项保留**，只换措辞与节拍 |

- 物理动作内核（泪膜成形不落、眨眼抑制、视线强压回、喉头吞咽、上下脸冲突）**逐项保留**——这是被验证过会 land 的写法。
- 但措辞、beat 拆分、主体、语境都按本镜重写。这是「融入」不是「照抄」。
- shot 的 `表演库参考:` 标注记录了用了 perf_0003——更新表演库后，点 shot 页面的「🎭 按表演库重生成」可重新融入（见 FR-17）。
- 库条目永远 generic（无剧目专名）；剧目专名（苏晚 / 宴会厅）只在 shot 里，不回灌库。
