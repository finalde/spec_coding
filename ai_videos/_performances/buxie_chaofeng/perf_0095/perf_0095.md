# perf_0095 · 不屑 / 嘲讽·强度4·外放·复合（嗤笑嘲讽）

> 外放高强度旗舰：出声嗤笑 + 单边冷笑 + 抱臂后仰配嘲讽台词，四载体联动把"我根本看不上你"放到最外放。一贯居高临下、无前置怔，不是被挑衅后的反击，从一开始就嗤笑着碾压。强度4 默认起始帧模式（峰值静帧兜底），见 `## 起始帧表情`。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
鼻腔与喉头先短促地"嗤"出一声出声的笑、随即半声轻笑泄出、鼻翼一张一收；单侧（仅一侧）嘴角向内收并大幅斜向上挑、挂出明显的单边冷笑、不露齿、另一侧嘴角平直不动——左右不对称是核心；双臂抱在胸前、上身明显后仰、头后扬、下巴高抬呈开阔的俯视；眉外梢上挑、目光自上而下睥睨着对方。配嘲讽台词时唇形随话动、单边斜挑始终不松。整体出声嗤笑、单边冷笑、抱臂后仰合成最外放的不屑——不对称的单边冷笑是核心读法，对称大笑则失真。
```
