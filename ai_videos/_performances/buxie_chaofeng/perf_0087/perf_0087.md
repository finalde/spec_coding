# perf_0087 · 不屑 / 嘲讽·强度1·内敛·面部（单侧微痕）

> 本情绪最低强度的单侧微表情标杆：只有**一边**嘴角极淡内收上提一瞬即收，留下不对称的冷蔑痕迹——无需被挑衅触发、一开始就懒得正眼看。强度1 默认起始帧模式（微表情易被丢，峰值静帧兜底），见 `## 起始帧表情`。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
仅左侧嘴角极淡地向内收并斜向上提一瞬、随即收回平静，右侧嘴角全程不动——一边动一边不动的不对称是唯一可读的痕迹；不露齿、不咧嘴；眼睑不变、目光平稳偏冷未聚焦在对方身上；下巴位置不变。整张脸几乎静止，只那一侧嘴角的单边斜挑泄出"懒得理会"的冷意——对称才是笑，单边才是看轻。
```

## 配套镜头

- 景别：近景偏特写（单侧嘴角的不对称吃满画面，人物占画 65%+，据画面优势侧）。
- 运镜：镜头静止，不切、不晃，让单边斜挑那一瞬的不对称痕迹全程可辨。
- 色调：中性偏冷、清晰平光（居高临下的清醒），无暖光介入。
