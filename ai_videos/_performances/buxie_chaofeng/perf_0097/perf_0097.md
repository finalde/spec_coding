# perf_0097 · 不屑 / 嘲讽·强度3·外放·复合（四载体联动旗舰 PG）

> 本情绪 playground 旗舰条（PG）：单侧蔑笑 + 挑眉 + 上下打量 + 鼻腔嗤声四载体联动，把不屑的全部标志签名一次性串成一段连贯表演。一贯居高临下、无前置怔，从一开始就以审视的姿态俯视、看轻对方——四个标志按一气呵成的次序自然流出而非被挑衅触发。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
先以半垂的偏冷目光自下而上把对方从下到上扫量一遍、在脸部停顿评估半秒；随即眉外梢上挑、下巴极轻上抬呈居高临下；几乎同时鼻腔短促喷气发出一声轻"嗤"、鼻翼一张一收；伴着嗤声单侧（仅一侧）嘴角向内收斜向上挑、挂出单边蔑笑、不露齿、另一侧嘴角平直不动——左右不对称是核心；最后目光不屑地从对方身上移开、不再看回。全程松弛不费力，上下打量的评估、挑眉的俯视、鼻腔的嗤、单边的蔑笑顺次连成一段——评估再丢弃、嗤之以鼻即"看轻"的读法。
```

- 运镜：镜头静止，不切、不晃，让上下打量—挑眉—嗤声—单边蔑笑的顺次联动全程可见；可定格在单边斜挑那一帧。
