# perf_0098 · 不屑 / 嘲讽·强度4·外放·复合（嗤笑转身甩袖）

> 外放高强度收束旗舰：嗤笑出声后转身甩袖、不屑再多看对方一眼——以"懒得多给一秒"的离场动作把看轻推到顶。一贯居高临下、无前置怔，转身离场不是赌气而是嫌对方不值得停留。强度4 默认起始帧模式（峰值静帧兜底），见 `## 起始帧表情`。A/B 种子点之一（与 perf_0087 配对验证单侧不对称在双渲染器上是否 land）。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
鼻腔与喉头先短促"嗤"出一声出声的轻笑、鼻翼一张一收；单侧（仅一侧）嘴角向内收斜向上挑、挂出明显的单边冷笑、不露齿、另一侧不动——左右不对称是核心；随即下巴上抬、头一偏，身体转身侧过、一手随转身甩动袖子/外套一甩，干脆地把对方撇在身后、转身后视线不再回看对方一眼；全程肩颈松弛、动作不带气急的发力而是嫌弃的随手。整体出声嗤笑、单边冷笑、转身甩袖合成"不值得我多看一眼"的外放离场——单边斜挑是核心读法。
```

- 运镜：镜头静止，不切、不晃，让嗤笑—单边冷笑—转身甩袖不回看的离场全程可见。
