# perf_0093 · 不屑 / 嘲讽·强度3·外放·呼吸（嗤笑嘲讽）

> 嗤笑嘲讽的呼吸载体标杆：鼻腔轻"嗤"一声从鼻孔短促喷气 + 嘴角单边斜挑——出声的鼻腔嗤是外放的标志发声。一贯居高临下、无前置怔，不是被挑衅后的反击，是一开始就嗤之以鼻。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
鼻腔短促地喷出一口气、发出一声轻"嗤"、鼻翼随之微微一张一收；几乎同时单侧（仅一侧）嘴角向内收斜挑、另一侧不动、不露齿，呈不对称的冷蔑；眉外梢上挑、下巴极轻上抬、目光偏冷下俯扫向对方；肩头随嗤气极轻一沉。全程松弛不费力——鼻腔的嗤气与单边斜挑同时发生，把"不值一提"出声地丢出来，即外放嘲讽的读法。
```

## 配套镜头

- 景别：近景（鼻翼一张一收的嗤气 + 单边斜挑吃满画面，人物占画 65%，据画面优势侧）。
- 运镜：镜头静止，不切、不晃，让鼻腔嗤气与单边斜挑同步发生的全程可见。
- 色调：中性偏冷、清晰平光（居高临下的清醒），无暖光介入。
