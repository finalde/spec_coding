# perf_0030 · 崩溃失控·强度3·外放·肢体（无声崩溃）

> 「无声崩溃」子类型在肢体通道上的标杆：失守主要由身体承担——肩塌、背靠墙缓缓滑坐、双手垂落。LMA = 彻底 giving-in-to-weight（向地心引力交出体重），与隐忍的 held-weight（撑住体重）相反。面部只带极轻悲意（嘴角下拉），重量全压在肢体的「撑不住、往下沉」上。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
两肩先一点点塌下、像被抽掉了支撑、脖颈随之前垂；身体重量交给背后的墙、脊背贴着墙面缓缓向下滑坐、膝盖随之弯折、整个人沿墙沉下去；原本垂在身侧的双手彻底松开、无力地垂落、手指松散不再握；嘴角向下牵拉、面部只有极轻的悲意。全程不出声——失守由身体承担：是肌肉一处处交出体重、撑不住地往下沉，不是受控地坐下。
```

## 配套镜头

- 景别：近景偏中景（须容纳肩、上身与沿墙下沉的轨迹，人物占画 60–70%）。
- 运镜：手持跟焦、随人物沿墙下沉同步下移，环境音抽空只剩极轻的呼吸抽泣；可在尾段拉远全景显孤立。
- 色调：冷调低饱和，无暖光介入。
