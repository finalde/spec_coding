# perf_0031 · 崩溃失控·强度3·外放·复合（无声崩溃旗舰）

> 「无声崩溃」子类型的旗舰复合条：把眼/面/肢体/呼吸四通道熔成一个递进式失守的整体——流泪 + 嘴颤 + 肩抖 + 扶墙滑坐联动，四通道同时失守但全程不出声。Radiating 质感（向外辐散、free flow）对应外放，与 perf_0009（内敛 Molding 向内塑形）成镜像。动作按递进式崩溃写成累加到 5 秒的 timed beats：不是四个动作齐发，而是一处先垮、带着下一处一起垮。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。动作为递进式累加，按拍点描述。

```
0–1s：眼眶涨满、眨眼一次后泪膜溢出睑缘、第一行泪无声滑落，眉内角上提；
1–2.5s：嘴角被向下剧烈牵拉成倒八字、下唇外翻发颤、抖动幅度渐大，但唇不张开、不出声；
2.5–4s：两肩塌下并开始无声抽动、一只手伸向身侧的墙撑住、身体重量交给墙面；
4–5s：脊背贴墙缓缓向下滑坐、人沿墙沉下去、泪持续滑落、肩抖不止。四通道一处带一处地先后失守、辐散开来（radiating），合成一个「全身一起垮下去却始终没出声」的无声崩溃整体，而非四个孤立动作的拼贴。
```

## 配套镜头

- 景别：近景（须同时容纳泪、嘴、肩与扶墙滑坐，人物占画 60–70%，必要时近景偏中景以容下沉轨迹）。
- 运镜：手持跟焦、随人物沿墙下沉同步下移，环境音抽空只剩呼吸抽泣；可在尾段拉远全景显孤立。
- 色调：冷调低饱和，无暖光介入。
