# perf_0032 · 崩溃失控·强度4·外放·复合（嚎啕大哭·从无声转出声的临界）

> 「嚎啕大哭」子类型的入口条，也是全情绪招募张口 AU（唇分 + 下颌松落）的临界标杆：从无声崩溃跨过门槛、第一次张口出声——上身前伏、呼吸抽泣成串。FACS 在悲核三联之上招募 AU25/26，整张脸下半部彻底失守。intensity≥4 默认配「起始帧」模式（嚎啕峰值纯文本最易合成成僵硬怪脸），见 `## 起始帧表情`。动作按递进式写成累加到 5 秒的 timed beats，演出「无声 → 出声」那一下决堤。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。动作为递进式累加，按拍点描述。

```
0–1s：悲核到位——眉内角上提、眉间皱拢、嘴角下拉成倒八字，泪已在滑落，唇仍紧抿、还在无声边缘；
1–2s：抿不住了——双唇被向下扯开、下颌松落张口，第一声哭出声、气从喉咙冲出；
2–3.5s：上身随这一声前伏、肩猛地耸起，张着口的哭声断续成串、每一声都带着抽气；
3.5–5s：呼吸彻底乱成抽泣——吸气一抽一抽地往里倒、呼气带着哭腔送出，肩随每次抽泣大幅起伏，泪横流不擦。从无声崩溃跨过门槛、第一次张口出声并辐散开来（radiating），是决堤那一下。
```

## 配套镜头

- 景别：近景（须容纳张口、前伏上身与起伏的肩，人物占画 60–70%）。
- 运镜：手持跟焦、随人物前伏下沉同步运动，环境音抽空只剩呼吸抽泣与哭声；可在尾段拉远全景显孤立。
- 色调：冷调低饱和，无暖光介入。
