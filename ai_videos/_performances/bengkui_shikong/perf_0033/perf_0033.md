# perf_0033 · 崩溃失控·强度4·外放·呼吸（嚎啕大哭）

> 强度4 嚎啕的「呼吸通道」条：哭的主信号走在**抽噎的换气节律**上——一吸一顿一串抽泣，肩随呼吸大幅起伏，哽咽把句子切成断续的气音。house-style 精品微表情派：嚎啕纯文本最易合成成「张大嘴定格」的僵脸，故默认配 `## 起始帧表情`（峰值静帧兜底），视频只承担从峰值帧出发的抽噎换气插值，写成电影感的真实抽泣、不是土味干嚎。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
猛吸一口气后喉头一顿、紧接一串短促抽噎的换气、气音断断续续把哽咽切成碎句；双肩随每一次抽气大幅耸起又塌落、上身随呼吸节律一抽一抽地起伏；眉内角用力上提、眉间拧皱、嘴角向下垮成倒八字、唇微分露出哭腔；泪顺着抽气一波波涌出滑落；鼻翼随抽噎张缩、偶尔一次吸气过猛呛住、声音断了半拍再续上。失控的呼吸节律带着整张哭脸一抽一抽——抑制肌已经守不住、悲随着每一次换气往外冲。
```

## 配套镜头

- 景别：近景（吃满抽噎换气带动的肩与上身起伏，人物占画 60–70%）。
- 运镜：手持跟焦随人物下沉、随每一次抽气的塌肩微微跟动；不切，让抽噎节律的连续性可见。
- 色调：冷调低饱和、孤立感外化；环境音抽空只剩呼吸抽泣，拉远全景显孤立。
