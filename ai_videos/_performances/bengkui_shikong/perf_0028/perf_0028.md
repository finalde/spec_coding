# perf_0028 · 崩溃失控·强度2·内敛·呼吸（崩溃前临界）

> 临界条：把「崩溃前」做在呼吸通道上。与「压抑隐忍」的「压住呼吸不让起伏」相反——这里呼吸节律已经乱了、压不太住了：吸气一顿、声线先抖、胸口起伏开始失稳。演的是「正在失守的递进」（递进五拍的第二拍「声音先抖」），不是稳定地压住胸腔。区别全在「节律已乱 vs 节律被压平」。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
一次吸气吸到一半突然卡顿、喉头哽了一下没吸顺；想把气压平，但接下来的呼气带出一声极轻的发抖气音、声线先于面部抖了出来；胸口起伏由原本的均匀变得忽深忽浅、节律开始失稳，肩随这失稳的呼吸极轻地一耸；眉内角极轻上提、眼眶下缘淡红将起。不是把胸腔压住不动（那是隐忍），而是已经压不住——呼吸的节律先一步失守、抖了出来，是崩溃前临界的递进。
```

## 配套镜头

- 景别：近景（须同时容纳胸肩起伏与面部，人物占画 60–70%，肩部以上入镜）。
- 运镜：手持微跟焦、随人物极缓下沉，环境音抽空只剩呼吸与那一声发抖气音；可在尾段极缓拉远暗示孤立。
- 色调：冷调低饱和，无暖光介入。
