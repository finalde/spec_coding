# perf_0027 · 崩溃失控·强度1·内敛·眼神（崩溃前痕迹）

> 本情绪强度轴的起点，也是 A/B 种子点：与「压抑隐忍 perf_0001」同为强度1 眼神微表情，但读法相反——perf_0001 是「掠过一瞬旋即被压回」（防线还守得住），本条是「目光开始守不住、涣散失焦」（防线已在裂、崩溃前的前兆）。差异全在「涣散失焦 vs 压回」一念之间，须演出「正在失守」而非「稳定压住」。强度1 微表情常被模型丢失，故默认配「起始帧」模式（峰值静帧兜底），见 `## 起始帧表情`。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
眼眶下缘一瞬浮起极淡泛红、泪膜未成形；目光先有焦点、随即焦点散开——眼神不再聚在一处而是涣散失焦地向前虚望约一秒，像突然失去抓住外界的能力；眉内角极轻上提（悲的痕迹）后不再抚平、留在原处轻颤；眨眼一次后睑缘没有收回原来的紧度。情绪不是被压回去，而是开始从眼底漏出来——这一瞬是崩溃前防线开始裂开的前兆，不是稳住。
```

## 配套镜头

- 景别：特写（眼部占画主体，涣散失焦的目光唯一可辨的景别）。
- 运镜：手持微跟焦、随人物极缓下沉，环境音抽空只剩极轻的呼吸；可在尾段极缓拉远一点点暗示孤立。
- 色调：冷调低饱和，平光，避免阴影制造伪表情。
