# 委屈 / 装可怜（weiqu_kelian）

功能主轴：**欲望 / 痛感**（追妻 / 萌宝赛道高频——示弱求怜，把情绪向外展示以唤起对方愧疚或保护欲）。

灯光氛围（检验视频）: 柔光偏暖、眼神光点亮泪膜、略俯柔（楚楚、示弱）

## 情绪定义

「委屈 / 装可怜」是受了不公而**示弱求怜**的情绪：核心 FACS 是**悲 AU1+AU15+AU17**——AU1（眉内角上提）+ AU15（嘴角下拉、倒八字）+ **AU17（下巴上推、噘嘴 / 下唇前突）**，其中 **AU17 是委屈的标志 AU**（与单纯悲伤的区别就在这口噘起的不甘）。标志套路是「含泪仰望」：微仰头看对方、睫毛挂泪、嘴角下撇但忍着不哭、声音发软变小、攥衣角。

这个情绪的**方向性是向外的**——这是它与本库相邻情绪的关键分界（见下）。在短剧里它是欲望 / 痛感主轴的承重 beat，追妻线、萌宝线高频，靠它把观众和剧中对象的怜惜 / 愧疚拉满。

## 关键区分（落库铁律，三组各成判别器）

1. **委屈 vs 装可怜**（同情绪内的展示性差异）：
   - **真委屈**：无意识流露，情绪自己漏出来，**眼神不瞟向对方**——委屈是被情绪淹没、来不及表演。
   - **装可怜**：带**对对象 / 观众的展示性**（表演性的可怜），**眼神会瞟向对方确认效果**、动作幅度放大、噘嘴更夸张。瞟视确认这一眼是装可怜的判别签名。
2. **委屈 vs 压抑·强忍泪水**（方向性分界，跨情绪）：
   - **委屈 = 示弱求怜，向外**——要让对方看见我受了委屈（哪怕真委屈也是「被看见」的姿态）。
   - **强忍泪水 = 压住不让人看见，向内**（见 `yayi_yinren/` 强忍泪水子类型）——努力不让眼泪掉、不让人发现。
   - 两者都「忍着不哭」，但委屈的忍是**含泪仰望求怜**（泪挂着给人看），强忍泪水的忍是**下巴上推 + 吞咽锁住泪膜不让滑落**（藏起来）。

## FACS 原型与标志 beat

- **悲—委屈 AU 核**：AU1（眉内角上提）+ AU15（嘴角下拉倒八字）+ **AU17（下巴上推、噘嘴/下唇前突，委屈标志 AU）**。
- **标志套路「含泪仰望」**：微仰头看对方、睫毛挂泪膜、嘴角下撇但强忍不落、声音发软变小、攥紧衣角。
- **配套镜头**（entry 选填 `配套镜头:` 字段建议）：略俯拍营造弱小感、柔光大特写、眼部泪光高光点。

## 风格二分

- **内敛**（真委屈强忍）：委屈在涌但压着不发——眉内角上提、嘴角下撇即被噘起的下唇抗住、睫毛挂泪不落，**不瞟对方**（无意识流露）。
- **外放**（放大的委屈 / 撒娇装可怜）：委屈被主动放大展示——噘嘴幅度大、声音拖软、跺脚 / 拽衣袖、**眼神瞟向对方确认效果**（表演性）。外放在此天然偏向「装可怜」，但二者由「是否瞟视确认」判别，而非纯由强度判别。

> 三层抑制模型只对「真委屈强忍」内敛条部分适用（涌起的悲 AU1+AU15 + 噘嘴抗住的 AU17 抑制层 + 嘴角下撇与强忍的上下脸冲突）；但委屈的「忍」目的是**求怜地撑住**而非**藏起来**，与强忍泪水的抑制肌目的相反，故签名写「含泪仰望撑住」不写「吞咽锁泪膜」。

## 子类型（展示性 / 求怜姿态不同，各成独立 entry）

| 子类型 | 签名（含展示性判别） |

| 真委屈隐忍 | 悲 AU1+15+17 无意识流露、**不瞟对方**、睫毛挂泪不落 |
