# perf_0077 · 委屈 / 装可怜·强度3·内敛·面部（真委屈隐忍·委屈核旗舰）

> 本情绪 FACS 核（眉内角上提 + 嘴角下拉倒八字 + 噘起下唇抗住）的标杆条：嘴角向下垮的同时被噘起的下唇顶住、强忍不哭。委屈区别于单纯悲伤的判别签名全在这口不甘的噘嘴——嘴角想垮、下唇却撑着不让它彻底垮下去。真委屈隐忍：眼神不瞟对方、情绪自己漏出来。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
眉内角明显上提、内端向上聚（委屈起头）；嘴角向下拉成倒八字、却在垮到一半时被向前噘起的下唇顶住——下巴随之上推、下唇前突，嘴角想彻底垮下去而被这口噘嘴撑着不松，这一「下撇被噘起抗住」的对抗正是委屈区别于单纯哭丧的读法；眼眶下缘泛红、泪膜成形不落；视线垂向斜下方、不抬眼看人；下巴极轻发颤但咬住不出声。整张脸是「想哭、又把哭噘回去」的不甘。
```

## 配套镜头

- 景别：柔光大特写（面部吃满嘴角下撇与下唇噘起的对抗，人物占画 65%+）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让噘嘴抗住下撇的微动全程可见。略俯拍营造弱小感。
- 色调：柔光偏暖；眼神光点亮泪膜、略俯柔（楚楚、示弱）。
