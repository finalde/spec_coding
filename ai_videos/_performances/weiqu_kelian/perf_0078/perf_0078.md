# perf_0078 · 委屈 / 装可怜·强度3·内敛·肢体（真委屈隐忍）

> 肢体载体条：委屈从脸延伸到身体——低头、攥紧衣角、肩微塌、声音发软变小。攥衣角是委屈的标志小动作（无处安放的不甘攥在手里）。真委屈隐忍：动作是无意识的自我安抚、不是做给对方看的放大展示（与外放装可怜的拽对方衣袖 / 跺脚相对）。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
头低下、下巴向胸口微收；双肩极轻向内塌、整个人缩小一圈；一只手在身侧悄悄攥紧自己的衣角、指节收力、把布料攥出褶（无处安放的委屈攥在手里）；眉内角上提、嘴角下撇被噘起的下唇抗住；若开口、声音发软变小、尾音往下沉、像不敢大声；视线垂着、不抬眼去看对方——这些是自己缩起来安抚自己的无意识动作、不是放大给谁看。整体是把委屈往身体里收的隐忍。
```

## 配套镜头

- 景别：近景偏特写收到胸肩（攥衣角的手与塌下的肩同框，人物占画 65%+）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让攥衣角与塌肩的收缩可见。略俯拍营造弱小感、强化缩小感。
- 色调：柔光偏暖；眼神光点亮泪膜、略俯柔（楚楚、示弱）。
