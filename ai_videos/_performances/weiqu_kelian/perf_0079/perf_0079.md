# perf_0079 · 委屈 / 装可怜·强度2·内敛·复合（楚楚可怜求助·含泪仰望标杆）

> 「含泪仰望」标志套路的标杆条：微仰头看对方、睫毛挂泪、嘴角下撇忍着不哭。委屈方向性向外（示弱求怜）的最清晰样本——泪挂着是要让对方看见、抬头是把委屈递向对方。区别于强忍泪水（向内压住藏起、努力不让人发现）：此处的「忍」是含泪仰望地撑住给人看、不是吞咽锁住藏起来。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
头极轻向上微仰、下巴略抬、眼睛由下往上望向对方（含泪仰望的起手）；眼眶盈起一层泪膜、睫毛上挂着将落未落的泪光、随仰头的角度更显得亮；眉内角上提；嘴角下撇、被向前噘起的下唇抗住、忍着不让哭出来；视线落在对方身上、是把委屈递出去求对方看见——这一份「抬头给人看」正是委屈向外示弱求怜（而非向内藏起）的读法。整体是柔软无害、把自己的委屈摊开求怜的姿态。
```

## 配套镜头

- 景别：柔光大特写（仰起的脸与挂泪的眼同框，人物占画 65%+）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让含泪仰望的角度与挂泪可见。略俯拍营造弱小感、与人物微仰头形成「以下望上求怜」的对位。
- 色调：柔光偏暖；眼神光点亮泪膜、略俯柔（楚楚、示弱）。
