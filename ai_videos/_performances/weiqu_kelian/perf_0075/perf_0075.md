# perf_0075 · 委屈 / 装可怜·强度1·内敛·面部（真委屈隐忍·微表情）

> 本情绪强度轴起点：眉内角极轻上提、下唇极淡噘起一瞬即收的委屈痕迹。委屈标志 AU 的最小可辨样本——委屈区别于单纯悲伤就在这口不甘的噘嘴一闪。真委屈隐忍条：情绪无意识漏出，眼神不瞟向任何人。微表情常被模型丢失，故默认「起始帧」模式（峰值静帧兜底），见 `## 起始帧表情`。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
全脸维持平静、不抬调；眉内角在半秒内极轻上提一瞬（委屈的痕迹）；下唇极淡地向前噘起、下巴随之极轻上推一下、随即抚平——这一闪而过的噘嘴正是「委屈」区别于单纯难过的读法；嘴角不明显下拉、不出声；视线垂向斜下方、不抬起去看任何人，情绪是自己漏出来的、不是做给谁看。整张脸唯一变化只在眉内角与下唇那一瞬的不甘。
```

## 配套镜头

- 景别：柔光大特写（面部吃满，眉内角与下唇的微动唯一可辨的景别，人物占画 65%+）。
- 运镜：镜头完全静止；任何运动都会盖过痕迹级微表情。略俯拍营造弱小感。
- 色调：柔光偏暖；眼神光点亮泪膜、略俯柔（楚楚、示弱）；平柔光避免阴影制造伪表情。
