# perf_0076 · 委屈 / 装可怜·强度2·内敛·眼神（真委屈隐忍）

> 强度2 单拍：睫毛挂泪膜不落、视线垂下不瞟人的眼神委屈。真委屈隐忍条——泪虽挂着但情绪向内不展示、眼神始终不抬起去确认对方反应（与装可怜的瞟视确认相反，也与强忍泪水的「锁住吞下」相反：此处泪是挂着的、未藏起）。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
眼眶下缘极淡泛红、睫毛上挂起一层将落未落的泪膜；视线垂向斜下方、停在那里不抬起、也不偏头去看任何人——这一份「不瞟对方」正是真委屈（无意识漏出）的读法；眉内角极轻上提（委屈的痕迹）；眨眼放慢、每眨一下睫毛上的泪光晃一晃却始终不滑落；嘴角极淡下撇但保持闭合不出声。泪挂在眼里给情绪一个出口、却没有抬眼讨怜的动作——委屈在向内涌、不在向谁展示。
```

## 配套镜头

- 景别：柔光大特写（眼部吃满挂泪的泪膜与垂下的视线，人物占画 65%+）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让睫毛挂泪「成形—不落」全程可见。略俯拍营造弱小感。
- 色调：柔光偏暖；眼神光点亮泪膜、略俯柔（楚楚、示弱）。
