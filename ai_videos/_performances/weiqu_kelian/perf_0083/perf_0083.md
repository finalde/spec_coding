# perf_0083 · 委屈 / 装可怜·强度4·外放·复合（撒娇装可怜·放大的委屈）

> 强度4 放大的委屈：泪眼汪汪贴近对方、噘嘴摇手臂、反复瞟视讨怜。比 perf_0082 更进一档——动作更黏更密、泪盈到眶满、整个人贴上去摇晃讨怜、瞟视确认从一记变成反复几记。仍守在「装可怜」不到「决堤」（顶端 perf_0085 才扑住对方哭）。高强度条默认起始帧模式（峰值静帧兜底），见 `## 起始帧表情`。


## 配套镜头

- 景别：近景收到胸肩（汪汪的泪眼、摇对方手臂的动作、反复瞟视同框，人物占画 70%+）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让贴身摇晃与反复瞟视可辨。略俯拍营造弱小感。
- 色调：柔光偏暖；眼神光点亮泪膜、略俯柔（楚楚、示弱）。
