# perf_0084 · 委屈 / 装可怜·强度4·内敛·眼神（真委屈隐忍·与强忍泪水分界参照）

> 强度4 濒临：泪在眶中打转、强撑不落、视线避开人。本条是与 `yayi_yinren/` 强忍泪水的跨情绪分界参照——两者都「泪在眶中打转、强撑不落」，但读法相反：**强忍泪水向内藏**（下巴上推 + 吞咽锁住泪膜、努力不让人发现）；**本条委屈向外求怜的撑**（泪是挂着给人看的、是示弱、只是这条选了避视的真委屈版本——避开对方的眼不是要藏起泪、是委屈到不敢直视、泪仍明明白白盈在那里被看见）。verify 重点：渲出「示弱求怜的撑」而非「藏起来的锁」。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
眼眶盈满泪、泪在眶中打转、几欲溢出却撑着不大颗滑落；眉内角高高上提、内端向上聚（委屈到顶）；嘴角下撇、被向前噘起的下唇顶住、下巴极轻上推抗着不哭出声；视线避开对方、垂向斜下方或偏开一点——是委屈到不敢直视、而非把泪藏起来：泪仍明明白白盈在眼里、晃着光、被看得见（向外示弱、不向内藏）；睫毛带颤、眨眼一次把将溢的泪逼回但泪膜立刻又盈满。整体是泪盈满眶、强撑着求怜地不让它掉、把委屈撑给人看的临界忍。
```

## 配套镜头

- 景别：柔光大特写（眼部吃满盈满打转的泪与避开的视线，人物占画 70%+）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让泪盈满—打转—被逼回的临界张力全程可见。略俯拍营造弱小感。
- 色调：柔光偏暖；眼神光点亮泪膜、略俯柔（楚楚、示弱）。
