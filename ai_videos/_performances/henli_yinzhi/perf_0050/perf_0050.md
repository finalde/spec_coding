# perf_0050 · 狠戾 / 阴鸷·强度2·内敛压抑·呼吸（阴鸷冷狠）

> 强度2 呼吸载体：鼻腔缓出一口长气、胸口几不起伏、压住不动声色——以呼吸的克制外化「能量被压住反而更冷」。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
鼻腔极缓地、长长地呼出一口气、气息绵长而克制（像把一股冲到喉口的怒气压着慢慢放掉）；呼气时鼻翼仅极轻一张、胸口与肩几乎不起伏（刻意压平呼吸幅度、不让情绪在身上显形）；下颌咬肌随呼气一鼓一定；眉头维持轻压、下睑收紧、目光平钉正前不漂、不眨眼；全程不抬声、面无表情。靠一口压得极慢极平的长气把狠意按在体内——以呼吸压住不动声色即「阴鸷」的读法。
```
