# perf_0042 · 狠戾 / 阴鸷·强度3·内敛压抑·肢体（阴鸷冷狠）

> 强度3 肢体标志 beat：指节捏至发白、缓慢将杯放下而非砸、不抬声——「能量被压住反而更冷」的肢体签名（指节发白却不挥拳、缓慢放杯而非砸杯）。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
手握杯、五指缓缓收紧直到指节泛白、手背青筋微凸（攥住的力却不挥出）；手臂极慢地下沉、把杯轻轻放到桌面、收力到几乎无声（缓放非砸）；放下后手指仍不松、停在杯壁上；同时眉头轻压、下睑收紧、目光平钉对面，下颌咬肌一鼓；全程肩颈僵直、动作刻意放慢、不抬声。攥白的指节配上无声放杯——压住不挥的狠即「阴鸷」的读法。
```
