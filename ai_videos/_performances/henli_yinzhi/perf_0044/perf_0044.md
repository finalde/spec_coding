# perf_0044 · 狠戾 / 阴鸷·强度2·内敛压抑·面部（黑化冷笑）

> 强度2 黑化冷笑面部：单边嘴角冷意上提、眼神凌厉、声线压平——单侧蔑笑 + 眼神凌厉的签名，痛感打底而非得意。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
单侧嘴角极慢地向上、向内勾起一道僵硬的冷笑、另一侧不动（不对称的蔑）；笑只到嘴角、眼睛不参与——下睑收紧、目光凌厉地平钉正前，眼神里没有暖意；眉峰轻挑半分又压回；声线刻意压平、若开口则字字压扁不抬调。嘴在笑、眼在冷的上下脸割裂，加一丝痛感打底的僵——冷意上提的不对称冷笑即「阴鸷·黑化」的读法。
```

## 配套镜头

- 景别：近景（嘴角不对称冷笑与凌厉眼神同框，人物占画 60–70%）。
- 运镜：镜头静止或极缓 push-in；不切不晃，让单侧嘴角上提与眼神不暖的割裂可见。
- 色调：低调硬光、侧逆光半脸阴影（per emotion）。
