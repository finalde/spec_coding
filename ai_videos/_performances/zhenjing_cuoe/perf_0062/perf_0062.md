# perf_0062 · 震惊错愕·强度1·内敛·眼神（怔住定格）

> A/B 低端种子：纯眼神、瞳孔放大一闪即收，无任何头身或口型参与。与 perf_0060 高端做强度两端的起始帧增益对照——验证瞬时微表情在低端最易被纯文本丢。默认配起始帧。

## 锁定文本块

> 可逐字嵌入 shot prompt 的 `表情:` / `动作:` 字段。

```
眼神原本平稳，瞳孔忽然向外一放、黑眼珠中心猛地一大、上睑随之极轻抬起一线；这一放大仅一闪、不足半秒便收回原状、眼神定在一点上微微发空；眉只是几不可察地一动、眉间始终舒展不皱。头不动、嘴不动、肩不动——只有瞳孔的一放一收记录下被瞬间击穿的那一点，纯眼神的极弱怔住读法。
```

## 配套镜头

- 景别：特写（眼部吃满、瞳孔放大的一闪卡在画面正中，人物占画 70%+）。
- 运镜：快速 push-in 怼脸后瞬间定格，把瞳孔骤放的微表情卡住。
- 色调：中性偏冷、略高对比（瞬间击穿感的清晰），无暖光介入。
