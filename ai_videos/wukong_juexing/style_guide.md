# 风格指南 — 《悟空觉醒》

本文档是 5 个镜头共享的视觉与语言锚点。所有 `prompts/shotNN_*.md` 中 `光线/色调:` 与 `镜头:` 字段使用的色彩、光线、运镜词汇必须出自本文件；任何超出本文档词汇表的表达式均为风格违规。

## 1. 调色盘（palette）

| 角色 | 命名 | Hex | 用途 |
|---|---|---|---|
| 主调 1 | 暖橘金（落日金辉） | `#F2A65A` | 悟空逆光勾边、碎石蛋金光、环境光 — **仅作光线 / rim，不作金属表面色** |
| 主调 2 | 血暮赤（血红暮色） | `#9B2D20` | 地平线 1° 处的暮色残光、冠带、血污 |
| 阴影 | 深岩青（冷青阴影） | `#2E5C6E` | 山岩阴面、悟空甲胄阴影、整体冷背景 |
| 高光天 | 紫罗兰夜霞（暮转夜过渡） | `#3A2A55` | 星空与暮色之间的过渡带 |
| 极光点 | 星辰白 | `#E8E6D9` | 星点、瞳孔金光中心、白光爆破核 |
| 中性 | 山岩灰褐（毛色基底） | `#5C4A3A` | 岩面、皮革、金属基底、悟空浅褐色短毛基色 |
| 装饰金底 | 紫金冠 / 金甲 / 金箍棒底色 | `#6B4226` | 金属表面紫金底色（非反光高光） |
| 装饰金高光 | 紫金 / 金甲鎏金高光 | `#C9A96E` | 金属表面鎏金亮部反光 |
| 毛色逆光 | 浅褐色短毛在金光勾边时呈现 | `#8A6A3A` | **仅在 rim light 区域出现，不为基色** |
| 黑底 | 全片最暗 | `#1A2A30` | 暗部最低位（不使用纯黑 `#000`） |
| 立绘背景浅 | Seedream 工作室背景渐变浅端 | `#E0E0E0` | **仅用于 `characters/ref_images/main_seedream.md` 立绘背景，不进电影画面** |
| 立绘背景深 | Seedream 工作室背景渐变深端 | `#707070` | **仅用于立绘背景，不进电影画面** |

**对比策略**：teal-and-orange 偏中式文人画。冷端不滑向工业青，压向深岩青 `#2E5C6E`；暖端不滑向霓虹橙，压向落日金 `#F2A65A`。冷部饱和度 ≤ 35，暖部饱和度 55–75。最暗处用深岩青压暗到 `#1A2A30`，禁用纯黑 `#000`。

## 2. 镜头语言（lens & framing）

| 项 | 取值 |
|---|---|
| 画幅 | **9:16 竖构图（硬约束 — 全片所有镜头）** |
| 等效焦距偏好 | 大特写 35mm、广角揭示 24mm、中景 50mm；不使用长焦压缩 |
| 景深 | 浅景深为默认（f/2.0–2.8 等效），主体清晰背景虚化；仅 Shot 02 山巅广角揭示改深景深 |
| 耀光 | 横向条形耀光（anamorphic-style flare）— 仅在金光爆发瞬间触发 |
| 构图 | 主体置于 9:16 中央上 1/3（手机观看习惯），头顶留 8–12% 安全区 |
| 视差 | 前景元素（碎石碎屑 / 飞絮）始终存在 — 制造层叠厚重感 |

## 3. 光线状态词汇（lighting state vocabulary，6 锁定 token）

供 Kling / Seedance prompt 直接拼装，**逐字使用**以下词条；不允许同义改写：

- `暮色魔幻时刻顶光`：太阳已沉于地平线下 1°，仅余暖橘金天光，无直射阳光
- `星空环境光`：上半天球深紫罗兰至深岩青渐变，星点分布稀疏戏剧化
- `金色边缘逆光`：主体后方 30°–45° 高度的暖橘金强光，勾出毛发与甲胄边缘 (rim light)
- `点状装饰光`：散落于地面 / 岩缝的微小暖光点 — 用于打破暗部
- `体积光丁达尔束`：金光爆发瞬间穿透扬尘 / 雾气，呈可见光锥
- `冷蓝补光`：从画面侧上方注入低强度冷蓝（来自 `#2E5C6E` 偏冷一端），避免暗部死黑

**禁用光线词汇**：正午顶光、纯白补光、霓虹彩色光、舞台追光、五颜六色 LED 光。

## 4. 色调 / 调色（color grading）

- LUT 取向：teal-and-orange 偏中式文人画
- 对比度：高（暗位 `#1A2A30`，亮位 `#E8E6D9`）
- 饱和度：选择性 — 冷部低饱和（≤35），暖部高饱和（55–75）
- 颗粒：轻 35mm 胶片颗粒全片叠加，避免数字 AI 生成的「塑料感」

## 5. 运镜（motion patterns，5 锁定 token）

| Token | 取值范围 | 用途 |
|---|---|---|
| `极缓推近 dolly-in` | 每秒 ≤ 2% 画幅 | 碎石蛋特写，紧张前压感 |
| `慢速广角揭示 reveal pan` | 4–5 秒横扫 30° | Shot 02 山巅广角揭示 |
| `轨道环绕 arc` | 180° 半弧, ≤ 6 秒 | Shot 04 持棒站立全身展示 |
| `手持轻微浮动 handheld micro-sway` | ±1° 抖动 | Shot 03 怒吼蓄力中景，加强在场感 |
| `垂直升降 crane-up` | 向上 1.5–2 米 | Shot 05 收尾，与开场推近呼应形成回环 |

**禁用运镜词汇**：whip-pan、360° 旋转、夸张鱼眼变形、第一人称 POV、希区柯克变焦（保留概念但本片不用）。

## 6. 转场偏好（transitions，2 锁定 rule — 白闪过渡已剔除）

- **默认 hard cut（硬切）** — 适配 Shorts 节奏；用于所有未指定的镜间转场
- **匹配剪辑 match cut** — Shot 04 → Shot 05 的金箍棒姿态匹配；Shot 05 末帧到 Shot 01 起帧的隐含回环

**禁用转场词汇**：金光闪白过渡（v1 因 AI 不可控剔除，原 R03 提议），交叉溶解 dissolve、推拉转场 push、3D 旋转转场、运动模糊过渡。

## 7. 美术风锚点（register anchor phrases）

每个 shot prompt 在 `场景:` 或 `风格:` 字段中必须出现下列锚点之一（推荐 `黑神话·悟空美术风` 整词）：

- `黑神话·悟空美术风`（推荐）
- `Black Myth: Wukong 写实质感`
- `weighty mythic realism 厚重神话感`

**禁用美术风词汇**：anime、cel-shading、二次元、戏曲红黄妆、京剧脸谱、86 版西游记、Q 版、低多边形、卡通。

## 8. 默认 / 进阶参数（per-shot 文件不必显式声明）

- Kling `cfg_scale`：默认 0.5（用户可在 Kling UI 内 0.3–0.7 微调）
- Seedance `seed`：每镜显式声明 `seed: -1`（随机）；用户欲复现可改为固定整数
- Kling `static_mask_url` / `dynamic_mask_url`：v1 不使用
- Seedance `camera_fixed`：本片所有运镜皆运动镜头，统一 `camera_fixed=false`

## 9. 角色一致性铁律（与 characters/main.md 协同）

每个 shot prompt `角色:` 字段下**字节级**粘贴 `characters/main.md` 的锁定描述符块（自 `【孙悟空 · 觉醒态 · 锁定描述符 v1】` 起，至 `禁用 卡通线条、cel-shading、二次元大眼、低多边形。` 止）。任何文字变更构成 v 升级，需通过 follow-up 流程触发下游重生成。
