## 出场场次

5 个镜头全部出场（覆盖率 100%）。Shot 01 仅以「破碎石卵 + 金光」前景出现，悟空本体在 Shot 02 山巅广角揭示首次现身；Shot 03 怒吼蓄力中景；Shot 04 持金箍棒环绕全身亮相；Shot 05 在金光淡出尾帧仅以剪影或不可见状态参与（与 Shot 01 帧 0 构图字节级相同，仅光照状态翻转）。

## 锁定描述符（block — 字节级粘贴源）

```
【孙悟空 · 觉醒态 · 锁定描述符 v1】
身份：齐天大圣孙悟空，刚自石卵中破壳而出的觉醒瞬间，气质介于野性灵猴与天界战神之间。

面貌：猴脸，颧骨高耸，眉骨突出，鼻梁短而上挑，双颊与下颌覆有细密浅褐色短毛，露出粉米色皮肤；嘴唇较薄，微抿，露出两颗略尖的犬齿。双眼为深褐色虹膜，瞳孔在金光照射下泛出一圈金辉，眼神锐利而沉静，非戏曲式夸张瞪眼。眉形粗黑、上扬，似两道短戟。

发型：头顶束高发髻，以紫金冠固定，鬓角与脑后留出一缕缕凌乱的浅褐色短毛，与冠下垂落的两条暗红色冠带（飘带）共同形成动势；不留长发披肩。

头冠：凤翅紫金冠 — 紫金色（带暗紫泛金的金属反光，非纯黄金），冠身正面为束发束顶造型，左右两侧各起一片向上向后舒展的凤羽形冠翅，凤翅边缘錾刻细密羽脊纹与点状鎏金；冠正中嵌一颗暗红色火焰纹宝珠，冠下两条暗朱红长飘带垂至肩胛，随动作飘动。

服装：上身穿锁子黄金甲（黄金色锁子纹链甲），甲片为细密六角锁子纹（Suozi 纹），覆盖前胸、肩头、上臂；甲表面有战斗划痕、烟尘擦痕与暗色血污，呈"battle-worn"质感；肩头缀两片半月形护肩甲，边缘鎏金錾兽面纹。腰系深褐色皮制宽腰带，悬挂虎皮护腿（点翠色 + 焦褐色斑纹）至膝。下身着深靛色绑腿裤，足蹬藕丝步云履（云头翘尖、履面绣浅金祥云纹）。整体配色：紫金 + 深金 + 暗朱红 + 焦褐 + 深靛，无任何高饱和原色。

身材：身高与正常成年人相当，体格精瘦却肌肉紧绷，肩宽腰窄，四肢比例略长于人类，灵长类肌腱感明显；尾巴细长（约与小腿同长），自然下垂或随动作甩出。

气质：野性 + 神性并存。静止时如绷紧的弓弦，重心微沉、膝盖微屈，永远处于随时跃起的预备态；眼神有「刚睁开却已通晓天地」的沉静。**不要**戏曲式抓耳挠腮的滑稽感，**不要**Q 版萌系比例。

标志性道具：右手或身侧持金箍棒（如意金箍棒）— 黑铁色长棍，两端各有一段约一掌宽的鎏金箍环，箍环上錾刻云雷纹；棍长可视镜头需要在 1.5 米至 4 米之间变化（本片默认全长约 2 米，肩扛或斜持）。

标志性动作：① 肩扛金箍棒、单脚踏石、半侧身回望的「立棍势」；② 蹲踞如猿、双手按地、尾巴翘起的预跃姿态；③ 单手将金箍棒横举过头、棍身泛金光的亮相姿态。

材质 / 渲染口径：写实 CG 质感，对标《黑神话：悟空》游戏 CG，毛发逐根可辨，金属反光含微擦痕，皮革有油脂光泽，整体偏暗调；**禁用** 卡通线条、cel-shading、二次元大眼、低多边形。
```

## 视觉锚点（hex 上下文，与 style_guide.md 一致）

- 凤翅紫金冠 + 锁子黄金甲 + 金箍棒主体：装饰金双层 `#6B4226` 紫金底 + `#C9A96E` 鎏金高光。
- 冠带 + 腰带血污 + 披风内里（若有）：血暮赤 `#9B2D20`。
- 浅褐色短毛（基色）：山岩灰褐 `#5C4A3A`；逆光部分泛金棕 `#8A6A3A`，禁止纯金色全身毛。
- 虎皮护腿斑纹：焦褐 + 点翠色（参考 `#5C4A3A` 与冷蓝点缀）。
- 暖橘金 `#F2A65A` **仅作环境光 / 逆光 rim light 使用**，不作金属表面色。

## 金箍棒规格

- 默认全长：2 米（肩扛或斜持）。允许镜头视角变化在 1.5–4 米之间，但本片 5 镜统一 2 米。
- 发光：默认反射环境暖光，无自身外发光辉。**唯一例外**：Shot 03 climax（约 t=4–6 s）允许棒身钎柄处微弱内发光（暖橘金 `#F2A65A` 仅边缘 rim，覆盖面积 ≤5%）。
