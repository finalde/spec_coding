# 发布元数据 — 《悟空觉醒》

主投平台：YouTube Shorts。文案以中文为主，跨平台复用规则见末段附录。

## 标题（YouTube Shorts）

- 硬规则：≤ 60 英文字符（约 30 中文字），不放 hashtag，不放 logo / 频道名。
- 软规则：1–2 关键词早出现（"悟空" / "觉醒" / "封神" 等），用悬念 / 反差 / 数字开头；不蹭《黑神话：悟空》IP 标签（已经由视觉风格自证）。

**候选标题（4 选 1，发布时手动挑）：**

1. `石卵裂开的那一秒，悟空醒了` — 悬念 + 时间锚点；29 中文字含标点，符合规则。
2. `40 秒看完悟空觉醒全过程` — 数字 + 利益承诺；13 中文字，余裕大可加副标题。
3. `为什么石猴一睁眼，整个三界都在抖` — 反差 + 悬念；20 中文字。
4. `齐天大圣 · 觉醒` — 极简 + 仪式感；7 中文字（适合配同名卡片字幕）。

> 默认推荐 #1（钩子最强 + 时间锚点提升完播预期）。频道调性偏文艺则选 #4。

## 简介（YouTube Shorts description）

- 硬规则：≤ 5000 字符（官方上限）；实操控制在 150–250 中文字。
- 结构（强制）：
  1. 第 1 句 = 钩子 / 一句话场景概述（默认显示在 feed 折叠前）。
  2. 第 2–3 句 = 简短背景（西游记典故 / 黑神话美学锚点）。
  3. 空行。
  4. hashtag 段（3–5 个，见下）。
  5. 创作工具说明一行（提升 AI-content 透明度）。

**候选简介（粘贴前微调）：**

```
石卵裂开的那一秒，齐天大圣孙悟空自天地间觉醒。
本片以一个 40 秒的电影级回环，重述《西游记》开篇——从破壳到立姿，从混沌到神性。视觉锚点对标《黑神话：悟空》的厚重写实，无对白，纯视觉叙事。

#Shorts #孙悟空 #中国神话 #AI视频 #西游记

由 Seedream + Kling + Seedance 制作。
```

字数复核：去 hashtag + 工具说明后正文约 80 中文字（含标点）；含 hashtag + 工具说明后总长约 110 中文字 — 在 150–250 字目标内可继续扩写背景到 200 字左右；当前精简版已能落地。

## Hashtag 规则

- 数量：3–5 个，全部放在描述区前两行；标题不放 hashtag。
- 顺序（前 3 个会自动作为可点击链接显示在标题上方，选最强的三个排前面）：
  1. `#Shorts`（强制，触发 Shorts shelf）。
  2. `#孙悟空`（主 IP 词）。
  3. `#中国神话`（类目词）。
  4. `#AI视频`（透明度 + AI-content 池）— 可选第 4 个。
  5. `#西游记`（品类长尾）— 可选第 5 个。
- 红线：标题 + 描述 hashtag 总数 ≤ 15（超过 → YouTube 视为 0 hashtag）。
- **不蹭** `#黑神话悟空` / `#BlackMythWukong` — 视觉风格已自证，避免被算法分到二创/同人池，也降低首发账号被 AI-mass-content 分类器误判风险。

## 封面建议

- **YouTube Shorts feed 流量主要看 Shot 01 自动抽帧** → Shot 01 t≈2 s 金光迸发峰值帧必须本身就是合格封面：
  - 主体居中偏上 2/3，单一焦点（裂开的石卵 + 金光）。
  - 高对比度（金光 `#F2A65A` 主调 + 背景 `#2E5C6E` 深岩青）。
  - 无文字 / logo / 多人 / 现代元素。
  - 渲染后须将 t≈2 s 帧导出抽帧测试是否合格；不合格则重新生成 Shot 01。
- **自定义封面**仅用于频道页 / 搜索 / 主页推荐 — v1 不强制，但建议生成 1 张：
  - 尺寸 1080 × 1920 px (9:16), ≤ 2 MB, JPG。
  - 主视觉：悟空特写 / 凤翅紫金冠 + 金箍棒 + 金色光晕。
  - 文字 ≤ 4 中文字（"悟空觉醒" 即可），字号占画面短边 1/8 以上，高对比描边。
  - 不放小字（移动端不可读）。
  - v1 暂不为自定义封面单独写 Seedream prompt — 用 `characters/ref_images/main_seedream.png` 立绘截一张特写即可。
