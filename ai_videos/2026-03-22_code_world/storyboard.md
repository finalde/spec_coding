# 凌晨三点，代码把我拉进去了 | Pulled Into the Code at 3AM

## Description 简介
**中文**: 凌晨三点，他还在和bug死磕。屏幕突然闪烁，一只手从代码中伸出，把他拉进了另一个世界——由代码构成的赛博空间。当你凝视深渊时，深渊也在凝视你。
**English**: At 3 AM, a lone programmer battles a stubborn bug. The screen flickers — and a hand reaches out from the code, pulling him into a world made of pure syntax. When you stare into the abyss of code, sometimes it stares back.

## Visual Bible 视觉圣经

**Style**: Cinematic photorealistic with cyberpunk-fantasy crossover. Grounded realism in the "real world" scenes, transitioning to stylized digital-neon aesthetic for the "code world."
**Aspect Ratio**: 9:16 (vertical, mobile-first)
**Color Palette**: Real world — desaturated blue-grey with warm amber from a single desk lamp. Code world — electric cyan, neon green (#00FF00 terminal green), deep indigo, hot magenta highlights.
**Lighting**: Real world — single warm practical desk lamp, cold blue monitor glow on face, deep shadows. Code world — omnidirectional neon glow, volumetric light rays through floating code.
**Mood/Atmosphere**: Real world — exhaustion, isolation, quiet desperation. Transition — shock and surreal horror. Code world — awe, vertigo, overwhelming digital sublime.

### Characters
- **The Programmer (主角)**: East Asian male, mid-20s, lean build, messy black hair, dark circles under tired eyes, wearing a wrinkled oversized grey hoodie and black-framed glasses slightly askew. Unshaven stubble. Expression shifts from exhausted focus to wide-eyed shock to stunned wonder.

### Environment
- **Primary Location — The Room**: Small cluttered apartment desk setup. Single 27-inch monitor glowing with green-on-black terminal code. Energy drink cans, crumpled papers, a mechanical keyboard with RGB backlighting. Curtains drawn, pitch dark outside the window. Desk lamp with warm tungsten bulb is the only other light source.
- **Secondary Location — The Code World**: An infinite three-dimensional space made of floating lines of source code (green and cyan glowing monospace text). Code streams flow like rivers in all directions. The "ground" is a translucent grid of circuit-board traces pulsing with light. Massive structures made of nested curly braces and function blocks tower like buildings in the distance. The sky is a deep indigo void with scrolling commit logs like aurora.

### Style Anchors
cinematic still, 9:16 vertical composition, shallow depth of field, Arri Alexa look, film grain, dramatic chiaroscuro lighting

## Enhanced Narrative 润色后的叙事

**原始描述**: 一个程序员凌晨3点还在改bug，突然电脑屏幕闪了一下，屏幕里伸出一只手，把他拉进了代码的世界。

**润色版本**:

凌晨三点十七分。整个公寓只剩下机械键盘的敲击声和显示器的嗡鸣。程序员已经连续工作了十二个小时，眼睛布满血丝，第四罐能量饮料已经见底。屏幕上的终端密密麻麻——同一个bug，改了又改，始终无法通过。

他揉了揉眼睛，摘下眼镜放在键盘旁。就在他重新戴上眼镜的瞬间——屏幕猛地闪烁了一下。不是普通的闪烁，而是整个画面像液体一样扭曲、涟漪般荡漾开来。

他愣住了，还没来得及反应，一只由发光代码字符构成的手从屏幕中伸出，指尖滴落着零和一的数据流。那只手精准地抓住了他的手腕——冰冷的触感像电流一样蔓延全身。

下一秒，他的身体被猛地拉向屏幕。穿越的瞬间如同坠入深水——压迫感、失重感、视觉的完全重置。当他睁开眼睛，发现自己漂浮在一个无限延展的空间中：脚下是发光的电路板网格，四周是如河流般奔涌的代码流，远处巨大的花括号和函数块如摩天大楼般矗立。

他低头看自己的双手——指尖正在像素化，缓缓融入这个由纯粹代码构成的世界。

**改动说明**: 增加了时间细节（凌晨三点十七分）和环境氛围（键盘声、嗡鸣声），强化了疲惫感的视觉描写（血丝、能量饮料），将"屏幕闪了一下"升级为液态扭曲的超现实意象，补充了触感和穿越时的身体感受，增加了结尾的像素化细节作为视觉钩子和悬念延伸。

## Production Notes 制作备注
- **Total Duration**: ~28s
- **Total Shots**: 9
- **Suggested Music Mood**: Start with minimal dark ambient / drone (Shots 1-2). Build tension with distorted electronic elements (Shots 3-5). Massive impact and silence on the transition (Shot 6). Resolve into grand, awe-inspiring synthwave with a melancholic undertone (Shots 7-9). Final note should be bittersweet — not triumphant, not tragic.
- **Key Consistency Notes**:
  - The programmer's appearance must stay identical across all shots: messy black hair, black-framed glasses slightly askew, wrinkled oversized grey hoodie, mid-20s East Asian male. Check every generated image against this description.
  - The code color must be consistent: green (#00FF00 terminal green) and cyan as primary code colors. Do not let it shift to other colors.
  - The real-world lighting (Shots 1-4) is strictly warm amber desk lamp + cold blue monitor. No other light sources until the supernatural green/cyan glow begins in Shot 3.
  - The code world (Shots 7-9) uses indigo as the void/sky color, with neon green, cyan, and magenta accents. Maintain this across all code-world shots.
  - The pixelation effect on the programmer's body should be subtle in Shot 7 (just edges), and only prominent on the hands in Shot 9. Do not over-apply.
  - All prompts begin with the same Style Anchors to enforce visual consistency. Do not modify or omit these when generating images.
  - For video generation (Kling/Seedance/Sora), use Shot pairs (e.g., Shot 3 as start frame, Shot 4 as end frame) to generate smooth transitions for the most critical moments: the screen distortion, the hand emergence, and the world transition.
