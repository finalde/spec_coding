## Methodology & Limitations

**Data collection method:** yt-dlp v2026.02.21 search queries (ytsearch) targeting recent trending content.

**Limitations:**
1. YouTube's `/feed/trending` page now requires authentication — unauthenticated access returns the home page nudge instead of trending content.
2. View counts reflect totals to date, not just today's views — videos with high overall counts may not be newly trending today.
3. No YouTube Data API key was used. A YouTube Data API v3 key would allow access to `videos.list` with `chart=mostPopular` for region-specific trending data.
4. Shorts trending requires authentication — only publicly searchable short-form content was found.

**To get accurate trending data in a production setting, use:**
- YouTube Data API v3: `GET https://www.googleapis.com/youtube/v3/videos?chart=mostPopular&regionCode=US&videoCategoryId=0&maxResults=50&part=snippet,statistics`
