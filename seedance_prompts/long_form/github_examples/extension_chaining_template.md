# 真實模板：超長視頻分段拼接（>15s）+ 延伸接續

來源：GitHub [woodfantasy/Seedance2.0-ShotDesign-Skills](https://github.com/woodfantasy/Seedance2.0-ShotDesign-Skills) — `references/scenarios.md` §十「超长视频分段拼接模板」

兩種接續方式：**硬切拼接**（各段獨立生成，靠重複的風格行保持一致，剪輯軟體裡接）；**延伸接續**（用 `@视频1` 把上一段延長，模型續寫）。

## 第 1 段（主體生成）
```
[15秒][完整提示词]，
最后2秒：画面稳定过渡至可延续的交接帧，
角色动作收束至静止/可持续状态，光照稳定。
```

## 第 2 段起（延伸接續）
```
将@视频1延长[X]秒，
保持一致的角色外貌/服装/光照条件/色调风格，
[续接动作描述]，[运镜承接]，
全程延续上段的大气效果和色调。
```

## 交接幀三原則（跨段不跳戲的核心）

1. **前段末尾畫面必須穩定**——禁止在運鏡中途中斷。
2. **後段開頭聲明一致性**——明確重申角色 / 光照 / 氛圍。
3. **運鏡連續**——前段 Dolly In 結束 → 後段自然承接同方向運動。

> 原理：用 `@视频1` 這種參考把每段獨立生成但顯式鏈接起來，繞過 15 秒單次生成上限。
