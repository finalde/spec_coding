# 《焚寿录》

> 仙侠 / 男频 / 重生 / 寄生系统 / 60 集长篇连载

## 简介

化神剑修裴长砚，渡劫之夜被五个生死兄弟联手刺穿胸口；睁眼时，他成了澹江洲落雁渊渊底的七岁穷童。胸前浮出二十四格青铜罗盘——每升一阶吞他一年寿元、削他一段记忆。他用前世剩下的修真知识，水边划下新名「裴知秋」，决心在罗盘耗尽前手刃六个旧人。可这一夜，渊口苇草丛里站着的剪影，分明是早就死了的师父——师父怎么会在这里？

> *"我以寿命换刀，以记忆换路；他们以我换天下。"* — 焚寿盘第一格泛黑日，裴知秋
