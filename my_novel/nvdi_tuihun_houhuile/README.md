# 女帝退婚后悔了

> 古风 · 装废逆袭 · 女帝重逢悔婚 · 男频爽文

## 概要

陈国公府三公子陈凡，五年前被女帝赐婚，同期开始装废。京都人尽皆知他是不学无术、沉迷玩乐的纨绔三公子，世家圈众人嗤之以鼻。无人知道，这副废柴皮囊之下，是一个蓄势五年的隐藏强者。

女帝一道退婚诏书送上门来——在旁人眼里是奇耻大辱，在陈凡眼里，却是他等了整整五年才等来的一步好棋。接旨谢恩、装足恭顺，待宣旨太监与老父离去，他缓缓起身，废柴的慵懒一寸寸褪尽，桃花眼底冷金锐光初现。这盘棋，才刚刚开始；而退他婚的那个人，该后悔了。
