# 修订需求 — 《被家族除名，看我武神归来之武神觉醒》

- **task_type**: `ai_video`
- **sub_type**: `novel`（多集，layout 走 `episodes/epNN/`）
- **task_name**: `wushen_juexing`
- **中文标题**: 被家族除名，看我武神归来之武神觉醒
- **题材**: 古风玄幻 / 修真 · 男频爽剧（逆袭·打脸·系统流）
- **载体**: 竖屏短剧 9:16，单集多 shot，每 shot 3–15s

## 目标（Goal）

打造一部以"王府长公子丹田破碎被除名、觉醒武神躯逆袭"为核心爽点的竖屏古风玄幻短剧。
本轮先走完整 agent_team 六阶段流程，并实际产出**第一集 EP1**（含世界观/角色锚定 + EP1 剧本 + 分镜 + 每镜 Kling/Seedance 双提示词 + publish 元数据）。

## 背景与核心设定（Context）

- **主角**: 某王府**长公子**（嫡长子），曾是天才，因**丹田破碎**沦为废物。
- **冲突源**: 被家族嫌弃除名，**弟弟**是主要打压者；父亲（王爷）高高在上。
- **金手指**: **系统**觉醒，提示主角不再低头。
- **逆袭转折**: 主角**主动脱离王府**（自请除名 / 断绝关系，强调"不低头"的骨气爽点），
  随后觉醒 **武神躯** + **王体资质**。
- **资质体系（待 Stage 2/3 锁定）**: 含 **王体**（顶级）、**天阶、玄阶、低阶** 等分层；
  武神躯为特殊体质金手指。
- **师承**: 后**拜入剑修门下**，开启修行逆袭线。

## 期望产出（Desired Outcome）

1. 完整六阶段 spec 工件，落在 `specs/ai_video/wushen_juexing/`。
2. EP1 实际成片素材，落在 `ai_videos/wushen_juexing/`：
