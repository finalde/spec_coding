测资时**上品 / 玄品 / 王品 / 帝品**都要有**相应、明显**的灵石特效，不能像 EP5 里下品那样"只是亮一下就完事"。要点：**不同颜色的光芒升起、不同颜色代表不同等级**（升起的色光柱 + 灵气形态，档越高越盛、光柱越冲）。

## 锁定色阶（§world 1.5① 已落地·游戏 CG 能量光效·呼应 follow-up 024 的 3D CG 风）
| 档 | 色阶特效 |
|---|---|
| 废品 | 灰败死寂、无光 |
| 下品 | **黯白**微光、明灭不定（最低档·只微微一亮） |
| 中品 | **青绿**柔光自石心晕开 |
| 上品 | **湛蓝**明亮光华 + 灵气环石 + 淡蓝光晕笼台 |
| 玄品 | **紫**光盛放 + 紫芒灵气成束螺旋环柱 + 紫晕数尺 |
| 王品 | **橙金**灵气漩涡 + 石身震鸣 + 橙金光柱拔地 |
| 帝品 | **金**光暴涨 + 金色冲天光柱 + 异象虚影漩涡 |

色阶＝资质档视觉读数：灰→黯白→青绿→湛蓝→紫→橙金→金。**上品起每档必有明显「升起的色光 + 灵气/光柱」奇观**，只有下品才是微微一亮。藏锋照旧：色光皆灵石法器读数，人物周身无金光无瞳光；帝品冲天金柱是灵石异象、非人物体放。EP6 落地：上品蓝(S4)/玄品紫(S6)/帝品金(S8·S10 残缺＝金柱缺块裂口)。EP5 若有旧档测资镜按此色阶待重渲。每测资 shot 的 `光线/色调` 写锁定色名（零 hex）、`渲染样式` 写色光升起 + 灵气形态。
