# BDD 场景 — EP1《武神觉醒》第一集行为验收

红果/番茄 男频爽剧 EP1「能不能成」的 feature-level 行为。每个 Scenario 是审片人能逐条走查、对照 `episodes/ep01/` 工件 + 成片节奏即可判定 PASS/FAIL 的具体例子。计时锚点参照 spec FR-7 / dossier story-structure §3.1：6s 钩 / ~30s 系统觉醒 / ~120s 资质觉醒 / 末镜宗门暗线钩；总时长 ∈ [85,100]s（约 136s · follow-up 020 dialogue-rich）。

---

## Feature: 前 3 秒冷开场冲突抓人（红果前 10 秒铁律）

为留住竖屏刷流量的男频观众，EP1 必须用「当众废丹/除族」冲突开场，禁止任何世界观旁白前置。

  Background:
    Given EP1 是 80 集系统流逆袭剧的第一集
    And 观众从信息流竖屏刷到本集，注意力窗口 < 3 秒

  Scenario: 冷开场即冲突，无世界观铺垫
    Given 审片人从 S01 第 0 秒开始播放
    When 看到前 3 秒画面
    Then 画面呈现「裴知秋当众被废丹田 / 被宣布除族」的冲突动作（如丹田碎裂暗红裂纹 + 裴霆宣判口型）
    And 此刻没有任何「这是XX大陆，资质分五档…」式世界观旁白或字幕
    And shotlist 第一镜被标注为 hook 镜
    例: S01 字幕「裴知秋，丹田已碎，逐出裴王府」+ 围观哗然反应，0–3s 完成

  Scenario: ~6s 出二段羞辱（退婚/打压叠加）
    Given 已过冷开场
    When 播放到约第 6 秒
    Then 出现退婚或裴昭当众羞辱的二段冲突，把「至暗」探底
    And 该羞辱镜后紧跟围观者/裴昭的反应镜头
    例: ~6s 裴昭折扇一收冷笑「废人也配姓裴？」+ 婚约一方收回玉佩

  Scenario: 禁世界观旁白先于冲突（反例必须 FAIL）
    Given 审片人检查 S01–S03 的字幕/台词归属
    When 任一镜在首个冲突动作之前插入解释性世界观旁白
    Then 该集判 FAIL，须返工把冲突前置

## Feature: 第 1 分钟首爽点（系统觉醒 / 起身反转）

3 分钟/集需要双爽点；第一波爽点必须在第 1 分钟内落地，给「忍辱→反转」的第一口气。

  Scenario: ~30s 武神觉醒系统绑定（金手指亮相）
    Given 裴知秋处于跌入谷底的至暗时刻
    When 播放到约第 30 秒
    Then 出现「武神觉醒系统」鎏金符文光 + 面板/任务提示 motif
    And 系统光的 motif 为鎏金符文光，与觉醒暖金区分但同属暖色家族
    例: ~30s 眼前浮现鎏金面板「检测到宿主，武神觉醒系统绑定中…」

  Scenario: 系统提示驱动「不向王爷低头」的起身反转
    Given 系统已绑定
    When 裴知秋面对父亲裴霆
    Then 他在系统提示下当众不低头、自请脱离裴王府（行动）
    And 紧接裴霆/裴昭/围观者的震惊反应镜头（反应）
    例: 裴知秋「这王府的姓，我不要了」→ 切裴霆瞳孔骤缩、裴昭折扇停滞

  Scenario: 第一分钟内必须出现首爽点（节奏验收）
    Given 审片人计时第 0–60 秒
    When 统计已落地的爽点（系统绑定 + 起身反转）
    Then 至少一个完整「行动→反应」爽点在 60s 前闭环
    And 若 60s 内无爽点，判节奏 FAIL（fast-cut 松垮风险）

## Feature: ~2 分大爽点（王体资质 + 武神躯觉醒二段打脸）

EP1 的大爽点是双段叠加打脸，先王体资质（资质天花板）后武神躯（体质降维），逐层加压。

  Scenario: ~120s 王体资质觉醒（第一段打脸）
    Given 全场仍视裴知秋为废人
    When 播放到约第 120 秒
    Then 觉醒异象显示「王体资质」——千年一遇·镇国之资的品阶虚影/光柱
    And 切裴昭等天阶天骄的震惊反应镜头（「天阶之上？」）
    例: ~120s 暖金光柱冲天 + 字幕「王体资质·千年一遇」，裴昭后退半步

  Scenario: 武神躯异象再压一层（第二段打脸）
    Given 王体资质刚被全场认知为人间巅峰
    When 紧接下一拍
    Then 武神躯异象覆盖——体表暖金裂纹亮起，呈现「废即资格」的反转
    And 出现「资质决定上限的高度，体质改写上限本身」的设定落点（台词或字幕）
    And 反应镜头从「惊」升级到「不可置信/恐惧」
    例: 暖金裂纹自残血暗红丹田处蔓延全身，裴霆失声「这是…体质异象？」

  Scenario: 二段打脸顺序不可颠倒（验收方向性）
    Given 审片人核对觉醒段镜序
    When 武神躯异象出现在王体资质之前
    Then 判 FAIL——叠加打脸的「加压」逻辑被破坏，须按王体→武神躯排序

## Feature: 末镜 cliffhanger（剑修宗门暗线）

每集必钩；EP1 末镜埋凌虚子/剑修宗门暗线，把观众推向 EP2。

  Scenario: 末镜出现剑修宗门暗线
    Given 大爽点已落地
    When 播放到 EP1 最后一镜
    Then 出现凌虚子（月白广袖剑修）或剑修宗门视角的暗线镜头（远观/出场前兆）
    And 该镜在 shotlist 中被标注为 cliffhanger 镜
    And 镜尾留悬念而非闭合（如剑仙一句「此子…武神躯」后转身/留白）
    例: 末镜 远山薄雾中凌虚子按剑而立，淡声「武神躯…百年未见」，画面收黑

  Scenario: cliffhanger 钩住下一集而非自我了结
    Given 审片人看完末镜
    When 判断末镜是否制造「想看 EP2」的驱动
    Then 末镜留有未解钩子（宗门是谁/会否收徒），不交代结局
    And 若末镜把冲突全部收束（无钩），判 FAIL

## Feature: 每个反派打脸含「行动→反应」反应镜头

男频爽点的落点靠反应镜头兑现；每次主角放招/亮身份后必须切打压者的反应。

  Scenario Outline: 打压者反应镜头成对出现
    Given 裴知秋做出 <主角行动>
    When 该镜播放完
    Then 紧接 <打压者> 的反应镜头（表情/肢体震惊）
    And 反应镜头与行动镜头在 shotlist 中相邻成对

    Examples:
      | 主角行动           | 打压者     |
      | 当众自请脱离王府   | 裴霆       |
      | 王体资质光柱冲天   | 裴昭       |
      | 武神躯裂纹覆体     | 裴霆/围观众 |

  Scenario: 反应镜头缺失即判不达标
    Given 审片人逐对核查行动镜
    When 任一关键打脸行动镜之后没有打压者反应镜头
    Then 该爽点判 FAIL（爽感落点丢失），须补反应镜

## Feature: 每集强钩子密度（前 72 小时高热）

EP1 是分账经济杠杆，钩子密度直接决定追剧留存。

  Scenario: EP1 含多层钩子
    Given 审片人统计 EP1 钩子
    When 盘点系统觉醒钩、双段打脸钩、宗门暗线末钩
    Then 至少存在 1 个开场冲突钩 + 1 个中段爽点 + 1 个末镜 cliffhanger
    And publish.md 标注前 72 小时高热运营提示与封面建议镜

## Feature: 角色同框可辨识（四主色不撞）

四主角主色互不撞色，保证任意同框镜头一眼可辨。

  Scenario Outline: 主角色与 bible 锁定一致
    Given 任一点名 <角色> 的 shot prompt
    When 核对其服装主色
    Then 主色为 <主色>（自然色名，零 hex）
    And 与该角色 character bible 一句话锁定 byte-identical 重贴

    Examples:
      | 角色     | 主色       |
      | 裴知秋   | 玄黑描金   |
      | 裴昭     | 朱红织金   |
      | 裴霆     | 玄青蟒纹   |
      | 凌虚子   | 月白淡青   |

  Scenario: 同框镜四色不撞
    Given 一个裴知秋、裴昭、裴霆同框的群戏镜
    When 审片人辨认三人
    Then 玄黑、朱红、玄青三主色清晰可分，无两人撞色致混淆
    例: 大殿宣判镜 中央玄黑裴知秋 / 侧朱红裴昭 / 高位玄青裴霆

## Feature: 觉醒特效 motif 出现（暖金裂纹 + 鎏金系统光）

觉醒视觉母题必须按 style_guide motif 词典落地，且系统光与觉醒光可区分。

  Scenario: 觉醒态暖金裂纹 + 残血暗红
    Given 武神躯觉醒镜
    When 审片人看觉醒特效
    Then 体表呈暖金裂纹，自残血暗红丹田处蔓延（冷环境暖人物对比）
    And 裂纹强度取 style_guide「中档」（EP1 既定，open question 已定档）

  Scenario: 系统光 motif 与觉醒光区分
    Given 系统绑定镜（~30s）与资质觉醒镜（~120s）
    When 对比两处暖色 motif
    Then 系统为鎏金符文光（面板/符文），觉醒为暖金裂纹+光柱，二者视觉可区分
    And 冷调环境（淡灰青冷底 + 四周压暗）作为统一底色衬托暖色爽点

  Scenario: motif 缺失或越界（反例）
    Given 觉醒/系统镜的 prompt
    When prompt 未含对应 motif，或出现 hex 色码
    Then 判 FAIL（NFR-2 零 hex + motif 一致性）
