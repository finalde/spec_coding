## A. 台词规则集（逐行套用 D1–D6）

| 代号 | 准则 | 本剧落地判据 | 严重度 |
|---|---|---|---|
| D1 | 通俗易懂 | 现代白话，红果男频观众一遍听懂。**禁未触发的文言/玄学格言**（「玄机暗藏因果自衡」类无 stake 空话）。世界观术语（王体资质/武神躯/丹田/境界名）首次出现须由画面或上一句带出语境，不得作为旁白凭空抛。 | fail → **warning** + 提 rewrite |
| D2 | 信息量/钩 | 每行须推进剧情 / 抬高 stake / 揭示角色 / 埋钩之一。可删而不断链 = 装饰行。 | fail → **warning** + rewrite |
| D3 | 节奏 | punchy 行 ≤15 字；仅宣言/仪式句（自请脱离、誓言）可放长，单气泡至多两分句。 | 超长 → **warning** + 拆/压缩 |
| D4 | 角色声口 | 措辞/语气匹配 spec.md 锁定站位（见 §B）。 | 违背 → **blocker** |
| D5 | 反转密度 | EP1 fast-cut：每镜至少一行落转折/揭示/威胁；群像独白镜全场≥1 揭示，不得为 0。 | 整镜零信息变化 → **warning**（钩镜为 0 → 见 S1 blocker） |
| D6 | 名场面短句 | cliffhanger/钩 payoff 的招牌句 ≤7 字理想（如「逐我者，仰我」）。 | 招牌句冗长 → **warning** + 压缩 |

逐行说话人语气须匹配以下站位（spec.md §角色命名 + §视觉锁定），并与 `characters/cN_*/` 一句话锁定不冲突：

## C. 镜头/结构规则集（S1–S5 + 红果铁律）

| 代号 | 准则 | 本剧 EP1 判据 | 严重度 |
|---|---|---|---|
| S1 | 钩落（hook landing） | 被 shotlist 标为黄金钩(S01冷开场)/反转/cliffhanger 的镜，必须在**声明秒数内**于 镜头+动作+台词 三者可见落地。锚点：6s 内当众废丹、~30s 系统绑定、~120s 王体+武神躯异象、末镜宗门暗线。 | 命名但未落地 → **blocker** |
| S2 | 情节链（不可删步骤） | 每镜须是 EP1 plot chain（dossier §3.1 的 15 节拍 → 28–32 镜）不可跳的一步。可删而不断链 = 装饰镜。 | 装饰镜 → **warning**（提合并/绑定剧情转点） |
| S3 | 角色一致/anachronism | 仅可引用本集已建立的角色/事实。**后续 arc 名字早现**（凌虚子门派名号、~40集身世真相、未登场红颜/兄弟线人物名）= 时序穿帮。 | forward-ref → **blocker** |
| S4 | 视觉信息密度 | 相邻镜须各给一个视觉判然不同的 beat，不得「仅换机位、无剧情增量」近重复。 | 近重复无 plot delta → **warning**（合并/差异化） |
| S5 | 台词模板去重 | 同集内两角色不得用近同句式表不同威胁/转折（如裴昭与裴霆都说「今日便是你的劫数」）。 | 模板撞句 → **warning**（差异化语速/喻体） |

**红果前10秒铁律（move #9 末段强约束，叠加在 S1 上）：**

- **禁世界观旁白先于冲突**：S01 必须直接是冲突画面（废丹/除族当面宣读），任何资质轴/境界轴/势力背景旁白若排在首个冲突镜之前 = **blocker**。术语随冲突带出，不前置科普。
- **第1分钟首爽点**：~30s 系统绑定 + 起身气场反转须在第1分钟内可见落地，否则首爽点缺位 = **blocker**。
- **3分钟末强 cliffhanger**：末镜（凌虚子暗线/宗门注视）须为强钩并显式落到 镜头+动作+台词，弱钩或缺失 = **blocker**。

## D. 重复模板 & 时序穿帮（汇总）

- **D-dup（撞句）**：两角色同句模板 → **warning**（同 S5）。本集高风险点：裴昭讥讽句 vs 裴霆宣判句须不同声线/喻体。
- **D-anachronism（早现）**：本集禁出现的 forward-ref 名词清单（任一出现即 **blocker**）：凌虚子门派/剑仙名号未到末镜前不得提；「武神躯」「王体资质」可出现（EP1 即觉醒），但**身世大反转/灭门真相/红颜兄弟线人名**（~40集才揭）严禁在 EP1 台词或旁白透出。

## F. Worked example — 假设弱镜 S01 的 patch

> 假设 S01 草稿如下（**故意写弱**，演示三类典型失败）：
> ```
> 01集01镜｜台词/字幕:
>   旁白: "天地有五重资质，黄玄地天之上更有千年一遇之王体，此乃镇国之资也……"
>   裴知秋 (隐忍): "玄机暗藏，因果自衡，丹田碎而道心不灭。"
> ```

```
shot01:
  D-flags: [D1, D2]
  S-flags: [S1]
  patches:
    台词/字幕 line 旁白 "天地有五重资质…此乃镇国之资也" —
      fails 红果前10秒铁律（世界观旁白前置于冲突）+ D2:
      DELETE 整行（资质科普禁前置；术语后续随觉醒镜带出）

    台词/字幕 line 裴知秋 "玄机暗藏，因果自衡，丹田碎而道心不灭" —
      fails D1（未触发玄学格言，观众听不懂、无 stake）+ D4（一开场即说教，
      不匹配「前段隐忍」声口）:
      rewrite → 裴知秋 (吐血跪地，气音): "丹田……碎了？"（≤6字，内心 OS 体，隐忍）

    动作/镜头 0-6s —
      fails S1（黄金钩「当众废丹」未在画面落地，仅靠旁白交代）:
      重写为 丹田碎裂金光炸开特写 + 裴知秋喷血跪地 + 裴昭一脚踩肩，
      字幕硬落「废了，他丹田碎了」（冲突画面先行，6s 内落钩）
```

> 修补后 S01 同时满足：D1（白话）、D4（隐忍声口）、S1（6s 内冲突落地）、红果禁旁白前置铁律。parent 逐条 Edit 应用后 re-emit，re-run 至 0 blocker + ≤2 warning。
