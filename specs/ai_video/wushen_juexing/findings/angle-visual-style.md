# 视觉风格 + 角色设计 调研（古风玄幻男频爽剧《武神觉醒》）

## 1. 本角度覆盖范围

为《被家族除名，看我武神归来之武神觉醒》建立可直接被 stage 4 继承的两套视觉资产：
（a）**古风玄幻/修真竖屏短剧的成熟视觉语言**——镜头语言、光影、色彩基调、特效 motif（灵气/剑气/觉醒系统光）；
（b）**四位主要角色的中文锁定视觉描述符**（主角王府长公子、嚣张少主弟弟、威严王爷父亲、出尘剑修师父），具体到两位画师读完会画出同一个人；并给出 style_guide.md 的关键词与自然色名调色建议。
**全文不含任何 hex 色码**，配色一律自然中文色名。

## 2. 关键发现（含引用）

### 2.1 玄幻爽剧的视觉爆款公式
- 大陆玄幻网剧/短剧的爆红公式被业内总结为「**玄幻 + 武打 + 特效**」，且面向男频时叠加「兄弟情/家族恩怨」叙事；这一组合「比宫廷斗争剧更受欢迎」，男性观众愿意为视觉刺激付费（[ETtoday 星光云：解密大陆网剧爆红公式](https://star.ettoday.net/news/1093020)）。→ 对本剧含义：每集必须有一个**特效爽点**（觉醒/剑气/打脸），视觉强度优先于场景写实。
- 男频爆款的快节奏要求镜头在前几秒立刻给冲突，慢镜与浅景深只留给「觉醒/亮剑」的高光帧（[知乎：红果女观众爱看的男频爆款](https://zhuanlan.zhihu.com/p/1952068957491142742)）。

### 2.2 电影质感视觉语言（可直接套用到分镜）
权威影视评论把「电影质感」拆为六要素：**娴熟的镜头运动、精准的场面调度、独特美学的构图、富层次的景深、雕琢人物的光线（正光/侧光/逆光/顶光）、张力十足的剪辑**（[中国日报：电影质感融入电视剧](https://cn.chinadaily.com.cn/a/202404/19/WS6621ccc8a3109f7860dda071.html)）。→ 竖屏 9:16 落地：浅景深特写打人物脸部，**逆光描边**塑造剑修/觉醒高光，顶光压暗营造威严王座感。

### 2.3 神话古风调色方法（自然色名，无 hex）
专业调色教程给出可复用的「神话古风片」配方（[澎湃：调出绝美神话古风片](https://m.thepaper.cn/newsDetail_forward_7381975)）：
- 整体先压成**淡灰青**底，正片叠底柔化氛围；
- 背景减暗黄、去黄去洋红、**加青色**，使背景冷而通透；
- 衣物披帛**加洋红与青**使色彩浓郁；末段回**加暖黄**让画面整体浓郁不寡淡；
- **降低明暗对比**提亮暗部，再用曲线**四周压暗**凸显人物、增加空间感；
- 人物周围轻微提亮 + **淡黄色光束（叠加）**赋予「神化意境」；肤色提亮至**白皙通透**。
→ 这正是「冷调环境 + 暖调人物光」的修真常用对比，本剧采用：**环境灵气青白偏冷，主角觉醒时叠暖金光束**。

### 2.4 特效 motif 词典
综合玄幻剧特效惯例（[ETtoday 同上]；[网易：国产特效现状](http://ent.163.com/special/guochantexiao/)）与修真视觉惯例：

| motif | 自然色名 | 视觉表现 |
|---|---|---|
| 环境灵气 | 灵气青白 / 月白 | 飘浮粒子、薄雾、地面流光 |
| 剑气 | 霜白偏冷青 | 锐利刀光弧线、空气扭曲残影 |
| 觉醒/武神躯 | 暖金 + 残血暗红 | 体表金色裂纹脉络、金光柱自天而降 |
| 系统/金手指提示光 | 鎏金 | 角色身侧金色符文光圈、瞳孔泛金 |
| 反派压制 | 阴煞紫黑 | 暗紫能量、压顶黑云 |
| 师父出尘 | 月白 + 淡青 | 衣袂流光、脚不沾尘的薄雾托底 |

## 3. 对 spec 的含义（四角色锁定描述符 + style_guide 关键词，全自然色名无 hex）

> 描述符写法面向 Seedream 立绘画师：固定**性别年龄体型/面貌/发型发色/服装主色/标志道具/气质 + 一句话锁定（≤30字）**。同一描述符须在每个点名该角色的分镜 prompt 中逐字重贴。

**C1 主角·裴玄（王府长公子，废柴→武神）**
男，约 18–20 岁，初期清瘦后期挺拔。剑眉星目、鼻梁高挺、唇线偏薄；面貌随剧情由苍白颓气转为眼神锐利。**长发束高马尾，墨黑发**（觉醒后发梢挑染暖金光）。服装主色**玄黑描金**长衫，初期布料素旧、后期暗纹华贵；觉醒态体表浮**暖金裂纹**。标志道具：**断裂家纹玉佩**（被除名的信物）。气质：颓→锋锐，含怒不发。
- 锁定句：**玄黑描金长衫、墨发高马尾、眸由颓转锋的清瘦少年。**

**C2 弟弟·裴昭（嚣张少主，打压者）**
男，约 16–18 岁，体型偏养尊处优的圆润感。面貌精致带骄气、眉尾上挑、惯性冷笑。**短束发，墨黑发缀金冠**。服装主色**朱红织金**锦袍、披帛华丽。标志道具：**鎏金折扇**（指人时用）。气质：嚣张、轻蔑、外强中干。
- 锁定句：**朱红织金锦袍、金冠折扇、眉尾上挑的骄纵少主。**

**C3 父亲·裴霆（威严王爷，家主）**
男，约 45–50 岁，魁梧伟岸。面貌方正、浓眉鹰目、蓄修剪整齐的短须；不怒自威。**束发戴玉冠，墨黑发染霜白鬓**。服装主色**玄青蟒纹**王袍、玉带。标志道具：**家主令印/座下虎纹扶手**。气质：威严、城府、护短与冷酷并存。
- 锁定句：**玄青蟒纹王袍、霜白鬓玉冠、鹰目不怒自威的家主。**

**C4 师父·凌虚子（出尘剑修/剑仙）**
男，年龄难辨（外貌约 30 岁，气质苍古），清逸修长。面貌淡漠出尘、丹凤眼、肤色白皙通透。**长发半束半披，墨黑发**（发丝泛淡青流光）。服装主色**月白叠淡青**广袖道袍，无华饰。标志道具：**素鞘长剑（剑气霜白偏冷青）**、足下常带薄雾。气质：清冷、超然、藏锋。
- 锁定句：**月白广袖道袍、素鞘长剑、足踏薄雾的清冷剑仙。**

**style_guide.md 关键词建议**
- 画风关键词：电影质感、竖屏 9:16、浅景深特写、逆光描边、顶光威压、东方修真、工笔质感肤色白皙通透。
- 环境调色：**淡灰青底 + 灵气青白冷调**，四周压暗凸显人物。
- 人物光：主角/师父高光帧叠**暖金光束**（冷环境暖人物对比）。
- 特效 motif 锁定：灵气青白、剑气霜白偏冷青、觉醒暖金+残血暗红裂纹、系统鎏金符文光、反派阴煞紫黑。
- 角色主色锁定表：主角玄黑描金 / 弟弟朱红织金 / 王爷玄青蟒纹 / 师父月白淡青——四色互不撞色，便于同框辨识。
