# Angle：红果/番茄 竖屏短剧平台打法（hongguo-fanqie-conventions）

## 1. 本角度覆盖什么

回答《武神觉醒》作为 **红果/番茄 免费短剧（IAA）** 上的男频古风玄幻爽剧，应当遵循的**平台运营与内容打法**：

- (a) 红果/番茄 短剧生态与变现（免费 + 广告 / IAA）模式；
- (b) 男频爽剧开头**黄金前几秒** hook 公式；
- (c) 单集时长 / 集数体量 / 每集钩子与（红果语境下的）卡点位置；
- (d) 标题 / 简介 / 标签 爆款写法；
- (e) 对本剧 EP1 与 `publish.md` 的具体建议。

核心前提：**红果是字节旗下"免费看 + 广告分成（IAA）"平台，不是小程序付费充值剧**。所以本剧没有"第 N 集付费墙"，"卡点"含义是**完播/留存卡点 + 广告插入承受点**，而非充值点。这一区别贯穿全文。

## 2. 关键发现（含引用）

### 2.1 生态与变现：免费 + IAA，而非付费充值

- 红果脱胎于番茄小说的"**免费阅读、广告变现**"模式，由张超将其平移到短剧：全部剧集免费，平台靠**信息流广告 / 激励视频广告**赚钱，再与版权方分成；同时叠加会员、电商带货、IP 授权等混合变现。([流媒体网](https://lmtw.com/mzw/content/detail/id/245832)，[澎湃·湃客](https://www.thepaper.cn/newsDetail_forward_30814493)，[腾讯新闻](https://news.qq.com/rain/a/20250523A09XE100))
- 用户可**看广告 / 做任务赚金币**，金币再用于解锁（少量）付费内容；主路径仍是免费观看。([腾讯新闻](https://news.qq.com/rain/a/20250523A09XE100))
- 体量：截至 2025-04 红果累计上线 **27,715 部**，月新增 **1000+** 部，题材覆盖古代权谋、都市职场、玄幻修真等；2024-11 月活冲到 **1.4 亿**、12 月 **1.58 亿**。([腾讯新闻](https://news.qq.com/rain/a/20250523A09XE100)，[钛媒体](https://www.tmtpost.com/7808795.html))
- 分账：面向版权方/承制方、编剧、演员三类，保底 + 分成；版权方约 **20% 起跳、头部可谈到 30%**。红果首创 **"72 小时高热 + 18 个月长尾"** 模型——**首月分账前 72 小时占 60% 权重**，之后最长 18 个月持续结算。([流媒体网](https://lmtw.com/mzw/content/detail/id/245832))
  - **创作含义**：前 72 小时的**完播率与追剧率决定全剧收益**，所以 EP1→EP3 的钩子密度是经济杠杆，必须最重。
- 拉新：通过抖音投流切片 + 自定义"推广关键词"（4-5 字），拉一个新用户约 **9 元**。([知乎](https://zhuanlan.zhihu.com/p/716369683))

### 2.2 男频在红果的窗口

- 2025 红果出**男频激励政策**，男性用户占比已升至 **48%**（女 52%），性别壁垒被打破。([网易](https://www.163.com/dy/article/KKHJ9O120550S2L1.html))
- 但头部仍以女频为主：2025 年破 10 亿播放的 61 部里**女频占 92%**；男频两部进 20 亿俱乐部的都是**权谋/历史、走事业线而非后宫**。([澎湃](https://m.thepaper.cn/newsDetail_forward_32575296)，[知乎年鉴](https://zhuanlan.zhihu.com/p/1997236213539692779))
- **系统题材**是男频公认爆点之一，曾出现观看量破 10 亿的系统流作品。([钛媒体](https://www.tmtpost.com/7582778.html))
  - **含义**：本剧"丹田破碎被除名→觉醒武神躯系统逆袭"踩中**系统流 + 事业线逆袭**双爆点，方向正确；但要避开纯"打脸爽"同质化，加事业线/世界观厚度。

### 2.3 开头黄金 N 秒 hook 公式（行业可量化节点）

业内共识的注意力卡点：**前 3 秒 / 前 10 秒 / 第 1 分钟 / 第 3 分钟**。([腾讯新闻](https://news.qq.com/rain/a/20240311A0A31C00)，[人人都是产品经理](https://www.woshipm.com/share/6191708.html))

| 时间点 | 必须发生 | 男频玄幻具体化（本剧） |
|---|---|---|
| 0–3 s | 钩住：刺激画面 / 反常情境 | 当众被废丹田 / 除族令当面宣读，无前情铺垫 |
| ~6 s | 进入冲突 | 昔日未婚妻/兄弟当场羞辱、退婚 |
| ~10 s | 抛出悬念 | 主角眼神变化 / 武神残魂之声响起（系统降临前兆） |
| 第 1 分钟 | 给足第一波"爽" | 系统/武神躯觉醒，主角第一次反制（小爽点） |
| 第 3 分钟 | 强 cliffhanger | 实力初显被目击 / 更大威胁登场，逼观众点下一集 |

口诀：**不拖沓、直接高能，第一分钟必须给爽感**。([流媒体网](https://lmtw.com/mzw/content/detail/id/241269))

### 2.4 单集时长 / 集数 / 钩子节奏

- 抖快/红果系竖屏单集多为 **2–3 分钟**；微短剧定义为单集 ≤10 分钟。([百度百科](https://baike.baidu.com/item/%E5%BE%AE%E7%9F%AD%E5%89%A7/23450704))
- 体量：一部短剧典型 **~100 集**（红果常见 70–100+）。([21财经](https://m.21jingji.com/article/20240325/herald/4a95c578c62ade60c69bb3cded49767a.html))
- **付费墙 vs 红果**：付费小程序剧第一卡点在第 9–10 集，后续 20/30 集再卡。([21财经](https://m.21jingji.com/article/20240325/herald/4a95c578c62ade60c69bb3cded49767a.html)) **红果免费无此墙**，但应**保留同位置的强 cliffhanger**——把"付费卡点"转译为"追剧留存卡点 + 广告插入承受点"：每集结尾强钩子拉完播，第 8–10 集、20 集、30 集设**章节级大反转**维持长尾追剧。

### 2.5 标题 / 简介 / 标签 写法

- 标题要**前置爽点 + 身份反差 + 悬念词**：现有片名《被家族除名，看我武神归来之武神觉醒》已含"被除名"(惨)＋"武神归来/觉醒"(爽)＋逆袭弧，符合套路。可考虑更口语化前缀（如"全族跪求…"式钩子句）。([cyzone](https://m.cyzone.cn/article/815999.html))
- 推广关键词：授权平台可申请 **4–5 字自定义关键词**用于抖音投流拉新。([知乎](https://zhuanlan.zhihu.com/p/716369683))
- 简介：1–2 句把"被废→觉醒→打脸/逆袭"弧亮出，制造身份反转预期。

## 3. 对 spec 的启示（EP1 hook + 每集 cliffhanger + publish 建议）

1. **EP1 前 10 秒铁律**：0–3 s 当众废丹/除族，6 s 退婚羞辱，10 s 武神残魂觉醒前兆。**禁止任何世界观旁白铺垫**先于冲突。
2. **EP1 必须在第 1 分钟出第一个系统/觉醒小爽点**，3 分钟末甩强 cliffhanger（实力被目击 / 强敌登场）。
3. **每集 25–35 镜、3 分钟、fast-cut**与本项目设定吻合平台节奏；每集结尾必留钩子。
4. **章节级大反转**放在第 8–10 集、20 集、30 集（对齐行业卡点心智，红果语境下即追剧/留存峰）。
5. **前 3 集（72 小时高热窗口）**做最高钩子密度——直接决定全剧分账权重。
6. 走**事业线逆袭 + 系统流**，弱化纯打脸重复，加世界观/势力厚度以避同质化清退风险（2025 红果在精品化、清退低俗套路化内容）。([澎湃](https://m.thepaper.cn/newsDetail_forward_32575296)，[36氪](https://36kr.com/p/3611356355375875))
7. **`publish.md` 元数据建议字段**：
   - `平台`: 红果短剧 / 番茄；`变现模式`: IAA 免费+广告分成（**非付费充值**）。
   - `题材标签`: 男频 / 古风玄幻 / 系统流 / 逆袭打脸 / 武神。
   - `单集时长`: ~3 分钟；`总集数`: 80；`竖屏`: 9:16。
   - `标题`: 主标题 + 口语化钩子副标题；`推广关键词`: 预留 4–5 字（投流拉新）。
   - `简介`: 1–2 句"被废→觉醒→逆袭"弧。
   - `运营提示`: 前 72 小时高热窗口 / 前 3 集钩子最密 / 每集结尾 cliffhanger。

## 4. 待解问题 / 未研究

- **红果具体广告插入位**（是否每集间插激励视频、是否影响完播口径）未取得官方一手数据——上文为行业综述推断。
- 红果**男频激励政策**的具体分账系数 / 题材白名单未取得原文细则。
- 标题字数上限、标签体系、封面规格等**平台后台投稿规范**未找到官方文档（仅二手运营文）。
- 2025 内容清退红线（低俗/违规）对"废丹当众羞辱"等桥段的**审核尺度**未核实，建议出片前查红果创作者规范。
- 系统题材"破 10 亿"作品具体片名/数据因 tmtpost 抓取超时未二次核实。
- confidence 标 **medium**：变现/生态/hook 节点有多源印证；男频系统流的细粒度运营数据多为推断。
