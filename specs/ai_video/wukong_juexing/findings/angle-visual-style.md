## 1. What this angle covers

锁定《悟空觉醒》的视觉语言：调色盘、镜头语言、光线状态词汇、运镜特征、转场偏好。本角度产物即为 stage 4 的 `style_guide.md` 蓝本，逐字继承。锚点已由 stage 2 拍板：**写实"厚重神话感" / Black Myth: Wukong 同源调性**，而非戏曲、水墨或赛博风；环境为 **暮色山巅 + 星空**，两段微弧（碎裂石蛋特写 → 山巅广角揭示）。

3. **Fox Render Farm — Black Myth: Wukong CG 同人作幕后解析** — <https://www.foxrenderfarm.com/news/behind-the-scenes-analysis-of-the-cg-fan-work-black-myth-wukong/> ：明确写出"冷色背景 + 暖色逆光"的对比策略、"血红暮色"开场、聚焦光分离主体（如悟空颤抖之手）的写法——直接给出"暮色 + 星空 + 主体逆光"该怎么打。

## 3. Locked style table (中文 — 逐字继承到 `style_guide.md`)

### 3.1 调色盘（palette）

| 角色 | 命名 | Hex | 用途 |
|---|---|---|---|
| 主调 1 | 暖橘金（落日金辉） | `#F2A65A` | 悟空逆光勾边、碎石蛋金光、金箍棒高光 |
| 主调 2 | 血暮赤（血红暮色） | `#9B2D20` | 地平线 1° 处的暮色残光、披风内侧 |
| 阴影 | 深岩青（冷青阴影） | `#2E5C6E` | 山岩阴面、悟空甲胄阴影、整体冷背景 |
| 高光天 | 紫罗兰夜霞（暮转夜过渡） | `#3A2A55` | 星空与暮色之间的过渡带 |
| 极光点 | 星辰白 | `#E8E6D9` | 星点、瞳孔金光与白光的爆破核 |
| 中性 | 山岩灰褐 | `#5C4A3A` | 岩面、皮革、金属基底 |
| 装饰金 | 凤翅紫金冠的紫金 | `#6B4226` 与 `#C9A96E` 双层 | 头冠、金箍棒主体 |

**对比策略**：参考 Black Myth 同人解析与 magic-hour 通则，全片采用 **冷青背景 + 暖橘逆光** 的 teal-and-orange 框架；冷色饱和度压制（≤ 35），暖色饱和度抬升（55–75），让人物在低光环境中被金光"切"出轮廓。绝不使用纯黑（`#000`），最深处用深岩青压暗到 `#1A2A30`。

### 3.2 镜头语言（lens & framing）

| 项 | 取值 |
|---|---|
| 画幅 | 9:16 vertical（硬约束） |
| 等效焦距偏好 | 大特写 35mm 等效、广角揭示 24mm 等效、中景 50mm 等效；不使用长焦压缩 |
| 景深 | **浅景深为默认**——主体清晰，背景虚化（f/2.0–2.8 等效）。只有 shot02 山巅广角揭示时改深景深 |
| 耀光 | **横向条形耀光（anamorphic-style flare）** 模拟史诗感，仅在金光爆发瞬间触发 |
| 构图 | 主体置于 9:16 中央上 1/3（手机观看习惯），头顶留 8–12% 安全区给字幕兼容 |
| 视差 | 前景元素（碎石碎屑 / 飞絮）始终存在，制造"层叠厚重感" |

### 3.3 光线状态词汇（lighting state vocabulary）

供 Kling / Seedance prompt 直接拼装，**逐字使用**以下词条：

- `暮色魔幻时刻顶光`：太阳已沉于地平线下 1°，仅余暖橘金天光，无直射阳光
- `星空环境光`：上半天球深紫罗兰至深岩青渐变，星点分布密集但不喧宾
- `金色边缘逆光`：主体后方 30°–45° 高度的暖橘金强光，勾出毛发与甲胄边缘
- `点状装饰光`：散落于地面 / 岩缝的微小暖光点（参考《妖猫传》"灯具即美术"），用于打破暗部
- `体积光丁达尔束`：金光爆发瞬间穿透扬尘 / 雾气，呈可见光锥
- `冷蓝补光`：从画面侧上方注入低强度冷蓝，避免暗部死黑

**禁用**：正午顶光、纯白补光、霓虹彩色光、舞台追光。

### 3.4 色调 / 调色（color grading）

- LUT 取向：`teal-and-orange` 但向"国画文人画"偏移——冷端不滑向工业青，而压向 **深岩青 `#2E5C6E`**；暖端不滑向霓虹橙，而压向 **落日金 `#F2A65A`**
- 对比度：高（暗部细节保留，但黑位压到 `#1A2A30`，白位升至 `#E8E6D9`）
- 饱和度：选择性——冷部低饱和（≤ 35），暖部高饱和（55–75）
- 颗粒：`轻 35mm 胶片颗粒` 全片叠加，避免数字 AI 生成的"塑料感"

### 3.5 运镜（motion characteristics）

| 类型 | 用途 |
|---|---|
| `极缓推近 dolly-in`（每秒 ≤ 2% 画幅） | 碎石蛋特写，制造紧张前压感 |
| `慢速广角揭示 reveal pan`（约 4–5 秒横扫 30°） | shot02 山巅广角揭示 |
| `轨道环绕 arc`（180° 半弧，≤ 6 秒） | 悟空持棒站立全身展示 |
| `手持轻微浮动 handheld micro-sway`（±1° 抖动） | 悟空怒吼 / 蓄力时的中景，加强"在场感" |
| `垂直升降 crane-up`（向上 1.5–2 m） | shot05 收尾，与开场推近呼应形成回环 |

**禁用**：快速 whip-pan、360° 旋转、夸张鱼眼变形、第一人称 POV。

### 3.6 转场偏好（transitions）

- 默认：**hard cut**（硬切），符合 Shorts 节奏
- 允许：**金光闪白过渡**（white flash 0.3–0.5 秒）——仅用于 shot 01 → 02 的"破壳—揭示"枢纽
- 允许：**匹配剪辑 match cut**——shot 04 → 05 的金箍棒姿态匹配
- 禁用：交叉溶解（dissolve）、推拉转场（push transition）、任何 3D 旋转转场

## 4. Implications for the spec（5 镜头映射）

| Shot | 用到的核心元素 |
|---|---|
| **Shot 01 — 碎石蛋特写（hook，≤ 2.5 s）** | 暮色魔幻时刻顶光 + 极缓推近 + 体积光丁达尔束（金光从裂缝穿出）；调色暖橘金 `#F2A65A` 主导，背景压暗到深岩青 `#2E5C6E` |
| **Shot 02 — 山巅广角揭示** | 慢速广角揭示 reveal pan + 24mm 广角 + 深景深；血暮赤 `#9B2D20` 地平线 + 紫罗兰夜霞 `#3A2A55` 上半天球 + 星辰白点；金光闪白衔接自 shot 01 |
| **Shot 03 — 悟空中景蓄力 / 怒吼** | 50mm 中景 + 手持轻微浮动 + 金色边缘逆光从背后 30° 注入；冷蓝补光从侧上方避免死黑；横向条形耀光在嘶吼瞬间触发 |
| **Shot 04 — 持金箍棒全身亮相** | 轨道环绕 arc 180° + 浅景深；凤翅紫金冠紫金 `#C9A96E` + 金箍棒主体 `#C9A96E` 高光；点状装饰光（地面碎石散金）打破暗部 |
| **Shot 05 — 回环收尾（loop-back 到碎石特写）** | 垂直升降 crane-up + 金光闪白匹配剪辑回到 shot 01 构图；金辉淡出至深岩青冷尾，再凝定 0.5 s 留给重播触发 |

## 5. Open questions surfaced

1. **金箍棒发光强度** — 是设定为"自身有金光"还是"反射环境暖光"？前者风格更接近《大圣归来》，后者更接近 Black Myth。建议取后者以维持"厚重写实"。
2. **悟空毛发颜色** — `#5C4A3A` 山岩灰褐 vs 偏金的 `#8A6A3A`？两者影响逆光勾边的可见度。建议 stage 4 在 character bible 里和此处保持一致选项。
3. **星空密度** — 真实星空（极密、点小）还是戏剧化星空（疏、点亮）？AI 生成对前者控制力较弱，建议默认戏剧化星空 + 银河淡带。
4. **"金光闪白"过渡** — Kling / Seedance 对 0.3–0.5 秒白闪的可控性需 stage 5 通过样片验证；若不可控，退化为硬切。
