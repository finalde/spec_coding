## 3. 锁定中文描述符（粘贴块 — 阶段 4 字节级继承）

```
【孙悟空 · 觉醒态 · 锁定描述符 v1】
身份：齐天大圣孙悟空，刚自石卵中破壳而出的觉醒瞬间，气质介于野性灵猴与天界战神之间。

面貌：猴脸，颧骨高耸，眉骨突出，鼻梁短而上挑，双颊与下颌覆有细密浅褐色短毛，露出粉米色皮肤；嘴唇较薄，微抿，露出两颗略尖的犬齿。双眼为深褐色虹膜，瞳孔在金光照射下泛出一圈金辉，眼神锐利而沉静，非戏曲式夸张瞪眼。眉形粗黑、上扬，似两道短戟。

发型：头顶束高发髻，以紫金冠固定，鬓角与脑后留出一缕缕凌乱的浅褐色短毛，与冠下垂落的两条暗红色冠带（飘带）共同形成动势；不留长发披肩。

头冠：凤翅紫金冠 — 紫金色（带暗紫泛金的金属反光，非纯黄金），冠身正面为束发束顶造型，左右两侧各起一片向上向后舒展的凤羽形冠翅，凤翅边缘錾刻细密羽脊纹与点状鎏金；冠正中嵌一颗暗红色火焰纹宝珠，冠下两条暗朱红长飘带垂至肩胛，随动作飘动。

服装：上身穿锁子黄金甲（黄金色锁子纹链甲），甲片为细密六角锁子纹（Suozi 纹），覆盖前胸、肩头、上臂；甲表面有战斗划痕、烟尘擦痕与暗色血污，呈"battle-worn"质感；肩头缀两片半月形护肩甲，边缘鎏金錾兽面纹。腰系深褐色皮制宽腰带，悬挂虎皮护腿（点翠色 + 焦褐色斑纹）至膝。下身着深靛色绑腿裤，足蹬藕丝步云履（云头翘尖、履面绣浅金祥云纹）。整体配色：紫金 + 深金 + 暗朱红 + 焦褐 + 深靛，无任何高饱和原色。

身材：身高与正常成年人相当，体格精瘦却肌肉紧绷，肩宽腰窄，四肢比例略长于人类，灵长类肌腱感明显；尾巴细长（约与小腿同长），自然下垂或随动作甩出。

气质：野性 + 神性并存。静止时如绷紧的弓弦，重心微沉、膝盖微屈，永远处于随时跃起的预备态；眼神有「刚睁开却已通晓天地」的沉静。**不要**戏曲式抓耳挠腮的滑稽感，**不要**Q 版萌系比例。

标志性道具：右手或身侧持金箍棒（如意金箍棒）— 黑铁色长棍，两端各有一段约一掌宽的鎏金箍环，箍环上錾刻云雷纹；棍长可视镜头需要在 1.5 米至 4 米之间变化（本片默认全长约 2 米，肩扛或斜持）。

标志性动作：① 肩扛金箍棒、单脚踏石、半侧身回望的「立棍势」；② 蹲踞如猿、双手按地、尾巴翘起的预跃姿态；③ 单手将金箍棒横举过头、棍身泛金光的亮相姿态。

材质 / 渲染口径：写实 CG 质感，对标《黑神话：悟空》游戏 CG，毛发逐根可辨，金属反光含微擦痕，皮革有油脂光泽，整体偏暗调；**禁用** 卡通线条、cel-shading、二次元大眼、低多边形。
```

> 阶段 4 / 阶段 6 使用约定：上述「【孙悟空 · 觉醒态 · 锁定描述符 v1】」整段必须**字节级**粘贴入 `characters/main.md` 与每个引用孙悟空的 shot prompt 中（可放在「角色」字段下）。任何对该块的措辞修改都构成 v 升级（v2、v3 …），并需通过 follow-up 流程触发下游重生成。

## 4. Seedream 立绘 prompt 结构大纲 + 一例神话/武侠工作样

**参考来源（Seedream / 即梦 4.0 提示词手册）：**
- 苏米客《即梦4.0提示词手册快速上手版》：https://www.xmsumi.com/detail/1550
- 知乎《Seedream 4.0 图像创作模型正式发布》：https://zhuanlan.zhihu.com/p/1948805126555932049
- 火山引擎社区博客《豆包 Seedream 4.0 正式发布》：https://www.cnblogs.com/volcengine-developer/articles/19086394
- Seedream 官方页：https://seed.bytedance.com/en/seedream4_0

**官方建议的 prompt 写法（综合上述来源）：**
> "使用连贯自然语言描述图像内容（主体 + 动作 + 环境），用短语描述视觉美学（风格、色彩、光线、构图），可在 prompt 内直接写明长宽比，或让模型按场景自动推断。"

**本项目立绘 prompt 推荐结构（中文字段顺序，期望长度 250–400 字）：**

1. **主体定义**（约 20 字）— 一句话说明"谁 + 什么状态 + 什么视角"，例：「全身立绘：孙悟空，刚自石卵破壳而出的觉醒瞬间，3/4 侧身正面视角」。
2. **面貌细节**（约 60 字）— 五官、毛色、皮肤、眼神、表情。直接节选 § 3 描述符的"面貌"段。
3. **发型 + 头冠**（约 50 字）— 节选"发型"+"头冠"段。
4. **服装**（约 80 字）— 节选"服装"段，按头部 → 肩 → 上身 → 腰 → 下身 → 足的顺序。
5. **身材 + 姿态**（约 40 字）— 节选"身材"+"气质"+ 默认立绘姿态（双脚分开与肩同宽，重心微沉，右手肩扛金箍棒）。
6. **道具**（约 30 字）— 节选"标志性道具"。
7. **画面控制**（约 50 字）— 构图：居中全身像，头顶留 5%、脚下留 5% 余白；背景：纯净浅灰色到深灰色渐变背景（#E0E0E0 → #707070），无任何环境元素；光线：顶部柔光 + 左前方 45° 主光，呈电影级三点布光；色调：紫金 + 暗朱红 + 焦褐主调，整体偏暗。
8. **风格 / 质感**（约 40 字）— 「写实 CG 渲染，质感对标《黑神话：悟空》游戏 CG；毛发逐根可辨，金属含微擦痕；电影级镜头，胶片颗粒感；禁用卡通、二次元、cel-shading、低多边形。」
9. **比例 + 输出**（约 15 字）— 「比例 9:16 竖构图，分辨率 2K」。
10. **负向提示**（如平台支持，约 30 字）— 「avoid: 卡通、二次元、Q 版、cel-shading、戏曲妆、低多边形、扭曲手指、模糊面部」。

**一例工作样（同档位神话/武侠角色 — 关公 / 赵云式武将立绘，仅作结构演示，非本项目用）：**

```
全身立绘：三国名将赵云，少年银甲将军定身亮相，3/4 侧身正面视角。
面貌：少年面相，剑眉星目，鼻梁挺直，薄唇微抿，皮肤白皙，眉宇间有英气。
发型 + 头冠：高束发髻，戴银鎏金束发冠，冠身錾雷纹，冠下两条墨绿色长飘带垂至肩。
服装：上身银鳞鱼鳞甲，胸前圆护心镜镶蓝玉，肩头双狮吞肩兽护甲；腰系深红宽腰带，腰侧悬黑漆剑鞘；下身墨绿战裙，足蹬黑色高筒战靴。整体配色：银 + 墨绿 + 暗朱 + 玄黑。
身材 + 姿态：身高七尺余，肩宽腰细，四肢修长。双脚分开与肩同宽，左手按剑柄，右手提丈八蛇矛斜指地面。
道具：丈八蛇矛 — 暗银枪杆，红缨在矛尖。
画面控制：居中全身像，头顶/脚下各留 5% 余白；背景纯净浅灰到深灰渐变（#E0E0E0 → #707070）；顶部柔光 + 左前方 45° 主光；色调银 + 暗朱主调。
风格 / 质感：写实 CG 渲染，对标《三国志14》开场 CG；金属含微擦痕，皮革有油脂光泽；电影级镜头，胶片颗粒感；禁用卡通、二次元、Q 版。
比例 9:16 竖构图，分辨率 2K。
avoid：卡通、二次元、Q 版、cel-shading、京剧脸谱、低多边形、扭曲手指。
```

阶段 4 应将本结构应用于 § 3 锁定描述符，产出 `characters/ref_images/main_seedream.md` 的最终立绘 prompt。

## 5. 对规格的隐含要求（Implications）

1. **`characters/main.md` 必须包含 § 3 整段锁定描述符**，不得删减或重写；可在其上方加项目语境段（角色身份、出场场次），但描述符块本身字节锁定。
2. **`characters/ref_images/main_seedream.md` 应严格按 § 4 的 10 段结构生成**，字数控制在 250–400 中文字。
3. **每个 `prompts/shotNN_kling.md` / `shotNN_seedance.md` 必须在「角色」字段下粘贴 § 3 整段描述符**（这是 image-first lock 契约的核心；阶段 6 验证器须按字符串相等校验）。
4. **5 个镜头共享同一张 Seedream 立绘**作为 image-to-video 的参考帧；阶段 6 用户先生成立绘，再分别用 Kling 与 Seedance 跑五镜。
5. **风格守门**：style_guide.md 必须明确禁用清单（卡通、二次元、Q 版、戏曲红黄妆、cel-shading、低多边形），并明确 reference IP 为《黑神话：悟空》游戏 CG，**不**为 86 版西游记。

## 6. 浮现的开放问题

- **Q1**: 描述符默认凤翅紫金冠为「金属反光带紫泛金」，但 R5 BWIKI 图标偏纯紫；阶段 4 是否需在立绘 prompt 中再加一句「冠身整体偏紫调，金辉为高光局部」以避免歧义？建议加。
- **Q2**: 是否需要为「破壳瞬间」准备**第二张** Seedream 立绘（覆有蛋壳碎屑、毛发湿润状）作为 Shot 01-02 的专用 ref？v1 不强制；建议留作 follow-up。
- **Q3**: 金箍棒在不同镜头长度不同（默认 2 米，但有「可大可小」的设定）。锁定描述符已声明「本片默认全长约 2 米」；阶段 4 须在每个 shot prompt 显式重述棍长，避免 Kling/Seedance 自由发挥。
- **Q4**: 锁子黄金甲的「黄金」与凤翅紫金冠的「紫金」在画面里需明显区分（黄金更亮、紫金偏暗），但中文表述上易混淆；建议在 style_guide.md 加一行 hex 锚点：黄金 ≈ #C9A14A，紫金 ≈ #6B4A78 + 局部金辉 #C9A14A。
