**Q2.3 — 系统类型**
- A: **任务型 + 双修机缘 + 丹药机缘 三轨混合**（用户自定义，不在 4 选项中）
- 用户原话："击杀内心邪恶之人，还有需要关键机缘和道具，机缘比如和女主双修，道具比如是药王谷女次主的丹药，帮助快速提升修为回复实力。要引入一套修为等级体系，用来衡量所有任务包括反派的实力"
- 拆解：
  - **任务模块**：系统派发"诛伪君子 / 救凡人 / 揭露阴谋"类任务，奖励为修为提升 / 神识强化 / 魔功秘籍。

  - **丹药机缘**：药王谷女医师所炼丹药为另一关键阶级跃迁辅助。
  - **修为等级体系**：用来衡量所有任务（包括反派实力）的客观标尺——明确写入 stage 4 spec 的 `world.md`。

**Q2.4 — 系统强度**

### Category 4 — Visual style（视觉风格）

**Q4.1 — 主美学方向**
- A: **传统仙侠 · 仙霖颜色 + 白衣隐逆**（用户选 B，不是默认推荐 A "暗黑仙侠"）
- 含义：主框架走传统仙侠（白衣山门、剑气漂浮、青绿色仙气），不走暗黑/血色路线。
- **但**与 Q4.2 主色调"黑金沉郁"产生有意张力 — 详见下条整合。

**Q4.2 — 主调色方案**
- A: **黑金沉郁 · 黑 + 护黄金 + 深青**（用户选 B，不是默认推荐 A "黑红血色"）
- hex 锁定（stage 4 写入 `style_guide.md`）：
  - 主色：黑（#0a0a0a）
  - 辅色：护黄金 / 深沉金（#a8842c）
  - 点缀：深青 / 墨青（#1a3038）
  - 高光（仅用于雷劫 / 系统提示 / 高潮）：白银 / 米白（#f5f5f0）

**Q4.1 ↔ Q4.2 整合（pipeline 分析）**：
- 传统仙侠"白衣山门"作为**正派伪君子**的视觉标识（明面光鲜、衣冠楚楚），让画面表层看似"传统仙气"。
- 黑金沉郁作为**男主魔尊**与全剧高潮场景的色调主基（雷劫云海、宗门暗室、魔功展开等）。
- 两套色彩在每集中都做明暗反差：白衣伪君子 → 表面仙气 → 真相揭露后过载黑金沉郁 → 男主黑色长袍现身。
- 这不是色彩冲突，而是 **表里反差**——视觉语言精确呼应剧情核心反讽。

**镜头时长 / 单集镜头数**（pipeline 默认）：
