## 1. 本角度覆盖范围

本研究为《魔尊归来》的 **9 个核心锁定角色** 给出可直接灌入 stage 4 spec 的命名套件：
- **男主双形态**（魔尊前世 + 乞丐转生）的中文姓名 / 称号候选；
- **3 位女主**（正派圣女 / 酒楼老板娘 / 药王谷女医师）的命名候选；
- **5 位伪君子宗主**（淫魔派 / 炼器贩人派 / 争位派 / 灭门夺宝派 / 暗通魔界派）的宗派 + 宗主命名候选；
- **每个核心角色的中文锁定描述符样板**（面貌 / 发型 / 服装 / 身材 / 气质 / 标志动作 / 配色 hex），用于跨集 byte-identical 复制；
- **Seedream 立绘 prompt 模板**（9:16 / 黑金沉郁 / 风格关键词 / 光线 / 负向）；
- **男主每阶魔功招式命名建议**（练气 → 渡劫后顶层共 7 阶标志魔功）。

不覆盖：宗派内部架构（弟子 / 长老 / 阵法）、剧情台词、单集分镜——分别归属 plot-and-arc、worldbuilding、shot-language 三个角度。

## 2. 关键发现

### 2.1 仙侠网文男主命名模式（适用于男主双形态）

主流仙侠 / 修真男主名遵循 "**网文姓 + 道意单字 / 双字**" 套路。常见姓氏池：萧、叶、方、陈、楚、沈、苏、李、秦、林、陆、唐、许；常见名字用字：东、南、北，易、逸、毅，宣、轩、风、枫、渊、运、云、尘、辰、阳、寒、墨、玄、凡（[起点中文网 — 修仙小说主角的名字大全男生](https://m.qidian.com/ask/tenqbosbp)）。CSDN 整理的 100 位玄幻男主名（[CSDN 100 位玄幻男主名](https://blog.csdn.net/lsylovejava/article/details/148979418)）中验证了 **韩立 / 王林 / 叶凡 / 张小凡 / 秦羽 / 陈平安 / 李逍遥** 等代表案例。

**魔尊前世名** 的取名要"威压感 + 古意"，参考 *苍兰诀* 的"东方青苍"（复姓 + 冷色单字 + 古字尾，[东方青苍 百度百科](https://baike.baidu.com/item/%E4%B8%9C%E6%96%B9%E9%9D%92%E8%8B%8D/57291136)）与 *长月烬明* 的"澹台烬 / 冥夜"（复姓单字 + 暗色意象，[澹台烬 百度百科](https://baike.baidu.com/item/%E6%BE%B9%E5%8F%B0%E7%83%AC/59195722)）。共同结构：**1-2 字姓 + 单字主名 + 偶带尊号"魔尊 X X"**。

**乞丐转生名** 应"贱名长寿"——网文常用质朴小户姓 + 反差暗指——参考 *琉璃* 禹司凤幼年化名 ([禹司凤 百度百科](https://baike.baidu.com/item/%E7%A6%B9%E5%8F%B8%E5%87%A4/53460625)) 与 *凡人修仙传* 韩立（凡人姓"韩"，名一字"立"，[韩立 百度百科](https://baike.baidu.com/item/%E9%9F%A9%E7%AB%8B/2508547)）。

### 2.2 反派宗主命名模式

仙侠反派宗主常用 "**宗派意象色字 + 宗主称号 + 道号**" 三段：
- 宗派色字：苍 / 玄 / 紫 / 赤 / 墨 / 玉 / 寒 / 焚（[起点 — 玄幻小说中常用的宗门名字](https://m.qidian.com/ask/tenjdycminj)）。
- 道号：真人 / 真君 / 道君 / 长老 / 上人。
- 表里反差：仙侠剧 *琉璃* 的离泽宫主柏麟（[柏麟 百度百科](https://baike.baidu.com/item/%E6%9F%8F%E9%BA%9F)）以"清雅道号 + 内里执痴"为反派范式；*长月烬明* 的庞宜之、谛冕（[长月烬明 维基](https://zh.wikipedia.org/zh-hans/%E9%95%B7%E6%9C%88%E7%87%BC%E6%98%8E)）则以"姓 + 单字阴柔字"突出"伪正派"气场——这是本剧 5 大宗主直接可复用的范式。

### 2.3 女主命名模式

仙侠女主双轨：
- **正派圣女**：清雅花木字 / 玉石字 / 单字 + 双字花名。参考 *苍兰诀* 小兰花 / 木槿（[小兰花 百度百科](https://baike.baidu.com/item/%E5%B0%8F%E5%85%B0%E8%8A%B1)）、*琉璃* 褚璇玑（[褚璇玑 百度百科](https://baike.baidu.com/item/%E8%A4%9A%E7%92%87%E7%8E%91/53419290)）、*长月烬明* 黎苏苏 / 桑酒（[桑酒 百度百科](https://baike.baidu.com/item/%E6%A1%91%E9%85%92/60657301)）。
- **凡间烟火型**（酒楼老板娘）：常用通俗姓 + "娘 / 阿"前缀。参考短剧女主修真赛道盘点（[澎湃 — 短剧女主修真赛道](https://m.thepaper.cn/newsDetail_forward_30820092)）。
- **药 / 医系女医**：药 / 兰 / 蕊 / 苓 / 璎 等草药字。

### 2.4 修为体系与魔功招式命名套路

主流九阶 + 渡劫后高阶（[起点修仙等级表](https://m.qidian.com/ask/qzfhvosamxa) / [元婴期 百度百科](https://baike.baidu.com/item/%E5%85%83%E5%A9%B4%E6%9C%9F/10612997)）。魔功命名验证案例：**梵圣真魔功 / 涅槃圣体（凡人修仙传）**（[梵圣真魔功 百度百科](https://baike.baidu.com/item/%E6%A2%B5%E5%9C%A3%E7%9C%9F%E9%AD%94%E5%8A%9F/20271639)）、**业火 / 琉璃火（苍兰诀）**、**罗喉计都魔识（琉璃）**。命名结构 = "**意象动词字 (噬 / 焚 / 灭 / 摄 / 蚀) + 自然 / 神话名词 (九幽 / 血雨 / 星辰 / 魔龙 / 雷劫) + 决 / 诀 / 印 / 掌 / 阵 / 杀**"（[资料 — 武侠玄幻招式名称大杂烩](https://zhuanlan.zhihu.com/p/663943295)）。

### 2.5 Seedream 立绘 prompt 模板

Seedream 5.0 / 3.0 官方文档（[Seedream 3.0 提示词指南 火山方舟](https://www.volcengine.com/docs/82379/1795150) / [Seedream Studio 5.0](https://seedreamv5.com/)）确认：原生 4K 输出、支持 Chinese Style / 国漫风格 / 古风插画 / Character design 关键词、9:16 比例。

### 3.2 中文锁定描述符样板（写入 `characters/{role}.md`，跨所有镜头 byte-identical 复制）

**统一字段格式**（10 字段，去掉任意一项即视为不一致）：

```
1. 性别 / 年龄 / 体型：[男 / 三十出头 / 高瘦劲健，肩宽腰窄，身高约 1.85m]
2. 面貌：[剑眉，丹凤眼眼尾微挑，鼻梁高挺，薄唇略压]
3. 瞳色：[赤红 (#8b1a1a) — 战斗时进化为金赤双瞳]
4. 发型 / 发色：[漆黑长发束半髻，余发披至腰际，鬓边一缕银]
5. 服装 / 主色：[黑金高领大袍，金色魔纹绣兽吞肩，黑底 (#0a0a0a) + 护黄金 (#a8842c) 绣纹]
6. 标志道具：[黑金手势护符，无具体本命法宝]
7. 标志动作：[左手负后，右手二指轻屈，魔气如墨自指尖溢出]
8. 气质：[冷漠睥睨众生，唇角微抿，无情无怒即是最大威压]
9. 配色 hex：[主 #0a0a0a / 辅 #a8842c / 点缀 #1a3038 / 高光 #f5f5f0]
10. 一句话锁定 (不可改一字)：[黑发披肩，黑金长袍，赤红瞳孔，左手负后，魔气如墨。]
```

每个角色都按本格式产出 `characters/{role}.md`。**第 10 字段"一句话锁定"是跨集 byte-identical 复制的核心**——任一 Kling/Seedance prompt 中提到该角色，必须 byte-identical 嵌入此句。

### 3.4 Seedream 立绘 prompt 模板（9:16 标准版）

按 Seedream 官方提示词指南 ([火山方舟 Seedream 3.0](https://www.volcengine.com/docs/82379/1795150)) 的"主体 + 细节 + 风格 + 参数"四段式：

```
[主体 + 锁定句子] [面貌 5 字段] [服装 / 配色 hex] [姿态] [构图 + 镜头]
[环境 + 光线] [整体风格关键词] [负向词]
```

**通用模板（替换 {role_name} / {locked_descriptor} 即可）**：

```
{role_name}，{locked_descriptor}。
半身立绘，正面构图，目视镜头，三庭五眼东方面孔。
背景：黑金沉郁色调（#0a0a0a 主 / #a8842c 辅 / #1a3038 点缀），淡雾沉香，微星河流光。
光源：左上 45 度冷月光（#f5f5f0 高光），右下淡金辅光（#a8842c）。
风格：传统中国仙侠插画 + 国漫风格 + Character design + 高细节服化道，水墨写意 +
工笔精绣，类比《苍兰诀》东方青苍 / 《长月烬明》冥夜 的造型质感。
画幅：9:16 竖屏，4K 原生分辨率。
负向：不要现代服饰，不要西方面孔，不要文字水印，不要多余手指，不要血腥镜头，
不要低饱和褪色，不要赛博朋克元素，不要暖色橙红主调。
```

## 4. 留给后续阶段的开放问题

1. **男主"沧冥"vs"玄无极"** —— stage 4 spec 阶段可让用户在剧本朗读时复检发音 / 网感记忆点；推荐沧冥（与黑金沉郁色调发声更同构），但玄无极在抖音搜索热度更高（候选 2 留作 v2 follow-up）。
2. **宗派"正道盟"统称** —— 是否叫"五道盟"/"中州五宗"/"九霄正道联"等，留给 worldbuilding 角度共同决定。
3. **男主前世魔殿名** —— "无极魔殿" / "沧冥魔域" / "九幽魔宫"——与 worldbuilding 角度协同确定。
4. **乞丐转生地点的酒楼名** —— 老板娘所在酒楼候选："醉仙楼"/"红袖招"/"小阁灯"等，待 plot-and-arc 角度的城镇命名定型后回填。
5. **每阶魔功招式的修为对应数值化** —— 是否需要"修为值 / 战斗力 / 法力上限"数表，留给 system-design 角度（任务型系统）。
6. **宗主弟子线 / 长老线命名** —— 9 角色锁定外的次要正派角色（如圣女师姐、宗主嫡子）是否需要 v1 锁定，stage 5 validation 阶段决定。
7. **乞丐期男主 → 觉醒期男主** 中间形态是否需要第三份 Seedream 立绘（例如金丹刚成时"半魔半凡"形态）—— stage 4 spec 决定是否独立锁定。
