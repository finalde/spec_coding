## 1. 本角度覆盖范围

本角度回答的是："以《魔尊归来》既定的'传统仙侠主框架（白衣山门、剑气漂浮、青绿仙气）+ 黑金沉郁主色调（反讽张力）'美学方向出片，应当如何把视觉语言锁定为可写进 Kling / Seedance prompt 的关键词"。覆盖五块：

1. **调色锁定**——为四类主场景（魔尊本相、白衣伪君子、乞丐凡间、雷劫与系统提示）各自固定 hex 与命名，使 60 集均一适用。
2. **镜头语言**——景别 / 运动 / 镜头时长惯例，契合 9:16 竖屏 + 8–10 秒 / 镜头 + 10 镜头 / 集的硬约束。
3. **光影状态词典**——把"魔气""仙气""雷劫""神识""丹炉光"等仙侠特定光影翻译成 prompt 可识别的关键词。
4. **转场词典**——从仙侠剧 / 古风 MV 沉淀的常用过场，给到双管线（Kling / Seedance）能稳定重现的写法。
5. **9:16 构图惯例**——单主体 / 双主体对峙 / 群像登场的三种竖屏构图模板。

明确**不**覆盖的：单个角色立绘（划归 character-design 角度）、单集分镜（划归 narrative-pacing / shotlist 角度）、平台发布元数据（划归 publishing 角度）。

## 2. 关键发现（镜头语言 / 调色 / 光影 with URL 引用）

### 2.1 黑金沉郁 = 已被市场验证的"魔尊型男主"色谱

《长月烬明》澹台烬"魔神黑甲"是 2023 年仙侠 + AI 视频圈最常被复刻的"黑金沉郁"模板：黑底加入蓝、金、红等金属色织入，形成"五彩斑斓的黑"的层次感，避免纯黑勿盖死光的塌掉感。这一组态可以直接对应本剧"魔尊原形"。

- [《长月烬明》：一幅重谱仙侠奇幻的国风史诗 | 澎湃新闻](https://www.thepaper.cn/newsDetail_forward_22637066)

借鉴元素（来自 `research/xianxia_storylines/chang_yue_jin_ming.md` 第 377–388 行的"AI 视频可借鉴的视觉元素"）：
- **黑甲长发魔神**：金属反光 + 黑金兽吞肩 + 龙鳞胸甲 + 血红披风内衬，雪夜山顶 / 高阶大殿俯视构图。
- **末世废墟**：破败仙门 + 红色血月 + 灰白尘土 + 仙骨残骸（可作第一卷"被镇压"开场）。
- **同悲道殉道**：黑色岩石阶梯 + 红色裂缝 + 金色神光 + 灰飞烟灭粒子（可作第六卷"反杀大本营"高潮）。

### 2.2 白衣伪君子 = 反向利用《琉璃》《沉香如屑》"白衣帝君"模板

`research/xianxia_storylines/chenxiang_ru_xie.md` 明确指出 2022–2023 仙侠"白衣帝君"路线（《沉香如屑》）是"红衣魔尊"路线（《长月烬明》）的对极。本剧反讽设定恰好把白衣端在伪君子一方——剧迷视觉记忆里"白衣 = 仙气 = 正道"的条件反射可以被剧情翻转利用，反差越强越爽。

借鉴元素：
- 《沉香如屑》"忘川送魂"——白银身影立于桥头不语（`chenxiang_ru_xie.md` L455）。
- 《琉璃》"摘面具一镜"——仅露下颌的白衣男子缓缓抬手，银面具滑落入掌（`liu_li.md` L419-421）→ 可作"伪君子真面目揭穿"的标准模板。
- 《琉璃》"剐肉刑名场面"——白衣浸血 + 仰拍长跪（`liu_li.md` L395-397）→ 可作魔尊被五大宗主联手镇压的开场。

### 2.3 9:16 竖屏构图 = 几何中心 + 大景深 + 3-5 秒钩子

短视频专业写作普遍约定竖屏微短剧的镜头规律：

- 单主体几何中心 / "井"字构图，把人脸或眉眼锁在画面上 1/3。
- 大景深拉伸纵深感，弱化背景细节、把情绪压在表情 / 衣着。
- 单镜头被压在 3–8 秒区间快速切换，"动接动"维持紧张感；开场 3–5 秒必须建立钩子。

参考链接：
- [干货｜新媒体时代竖屏微短剧的剧本特点及拍摄技巧（豆瓣小组）](https://www.douban.com/group/topic/310200172/)
- [短剧制作的比例是 9:16 还是 3:4？拍摄短剧应该注意什么问题（榜中榜）](https://m.maigoo.com/goomai/738742.html)

### 2.4 Kling / Seedance prompt 关键词 = 中文运镜词 + 风格锚点 + 色温对位

Kling 官方说明 6 类一线运镜（推 / 拉 / 摇 / 移 / 跟 / 升降），并且**对中文 prompt 的语义识别比 Runway / Sora 更精准**。这意味着我们可以在中文 prompt 内直接写"低角度仰拍 + 慢动作风扬披风 + 银枪反光"这种成段动作复合句，不需要翻译成英文动词。

参考链接：

- [Kling 3.0 Prompt 完整指南：掌握 AI 视频生成（中文版）](https://seavidgen.com/zh/blog/kling-3-0-prompt-complete-guide)

### 2.5 prompt 可写 vs 后期才有 ——必须严格分桌

经多剧视觉拆解（`yu_feng_xing.md`、`liu_li.md`、`chang_yue_jin_ming.md`），稳定可由 Kling / Seedance 单镜头一次出片的元素与必须靠后期叠加 / 调色才能呈现的元素需要先分清楚，否则 prompt 会被塞进过多"它做不到"的氛围词，反而让画面失焦。

| 类别 | 可写进 prompt（Kling / Seedance 一镜出） | 需要后期 |
|---|---|---|
| 服化造型 | 黑金长袍、白银兽吞肩甲、长发飞扬、银面具、乞丐褴褛粗麻 | — |
| 单点光源 / 主光位 | 月光逆光、烛火侧光、雷劫顶光、丹炉腹光 | — |
| 简单粒子 | 桃花瓣旋落、雪粒、金色符纹飞旋、灰烬粒子 | — |
| 单一基调色 | 黑金 / 白银 / 深青 / 米白 整场色温 | — |
| 简单运镜 | 推 / 拉 / 摇 / 跟 / 仰拍 / 俯拍 / 360° 环绕 | — |
| **跨镜头 LUT 一致性** | — | DaVinci 二级调色统一 |
| **复合特效叠加** | — | After Effects（如黑红业火 → 金紫琉璃火转化） |
| **多层符箓字 + 镜头同步** | — | AE 做字幕层 + 抖动 |
| **音频驱动闪频** | — | 剪映 / Premiere 节奏点切 |
| **角色一致性细节微调** | 靠 Seedream 立绘 + 锁定描述符固定，但仍有飘移 | 后期遮罩修脸 / 局部重画 |

|---|---|---|---|
| 玄夜黑 | `#0a0a0a` | 魔尊本相、宗门暗室、雷劫云海、虚空背景 | 全剧主调 60% |
| 护黄金 | `#a8842c` | 魔气描金、金线刺绣、丹炉腹光、系统任务 UI 边框 | 高频强调 25% |

| 仙气青绿（辅） | `#5a7a6c` | 山门白日 / 凡间药王谷 / 莲花楼式江湖外景 | 偶尔过场，避免纯绿 |
| 血纹红（点缀） | `#5a0a14` | 魔气红瞳、契约红绳、伪君子隐私揭露的暗红血迹 | 极低频，单镜限制 |
| 凡间灰褐 | `#4a3d2c` | 第二卷乞丐期土墙 / 烟火街市 | 第二卷主调，全剧约 8 集 |

调色规则：
- **每集 ≥ 70% 镜头使用主三色（玄夜黑 / 护黄金 / 墨青）。**
- **银白高光只在"雷劫劈下""系统任务弹窗""白衣伪君子摘面具"三类节点出现。** 出现在普通对话场景就稀释了反差爆点。
- **第二卷（乞丐期）允许凡间灰褐 + 仙气青绿过半占比**，制造"已死"的伪平淡假象，第三卷觉醒一刻立刻切回黑金。

### 3.2 镜头语言关键词字典（直接 copy-paste 进 prompt）

| 类别 | 中文 prompt 关键词 | 适用情境 | 默认时长 |
|---|---|---|---|
| 景别 · 大全景 | `大全景 / 远景俯瞰 / 山门远景 / 云海开阔` | 卷开场、转场环境镜 | 6–8 s |
| 景别 · 全景 | `全景中距 / 长袍立人 / 群像并立` | 五大宗主合议、魔尊登场 | 8 s |
| 景别 · 中景 | `中景半身 / 双人对峙 / 师门论道` | 主要对白镜头 | 8–10 s |
| 景别 · 近景 | `近景特写 / 眉眼锐利 / 红瞳隐现 / 唇角冷笑` | 情绪爆点 | 5–8 s |
| 景别 · 特写 | `特写微距 / 指尖魔气 / 剑刃寒光 / 丹炉裂纹` | 道具 / 特效空镜 | 3–5 s |
| 运动 · 推 | `镜头缓推 / 慢推近主角眉眼` | 心理逼近、揭穿真相前一拍 | 5–8 s |
| 运动 · 拉 | `镜头慢拉 / 抽离远去 / 拉至大全景` | 高潮收束、卷尾留白 | 6–10 s |
| 运动 · 跟 | `跟随长袍 / 跟拍奔跑 / 后景跟入` | 行动戏 | 5–8 s |
| 运动 · 环绕 | `360° 环绕 / 摄像机围绕主体顺时针` | 真身觉醒、神识展开 | 8–10 s |
| 运动 · 升降 | `镜头从地面缓升至俯瞰 / 由俯下沉至平视` | 卷转折、阵法启动 | 6–8 s |
| 运动 · 仰拍 | `低角度仰拍 / 自下而上仰望黑袍` | 反派 / 反英雄出场（《yu_feng_xing.md` 战阵镜头同写法） | 5–8 s |
| 运动 · 俯拍 | `高角度俯拍 / 俯视跪地之人` | 镇压、被审判 | 5–8 s |
| 速度 | `慢动作 / slow motion / 慢镜风扬披风` | 高情绪节点 | +2 s |
| 速度 | `定格瞬间 / 时间凝滞 / 凝住一拍` | 揭穿、闪回引子 | 1–2 s |

镜头时长惯例（与单集 90 s ÷ 10 镜头吻合）：

- 默认 **每镜 8–9 秒**。
- 开场钩子镜头压到 **3–5 秒**，先用一个特写或冲击镜定调（参考竖屏短剧"3–5 秒立钩"规律，源：[豆瓣短剧创作技巧](https://www.douban.com/group/topic/310200172/)）。
- 高潮镜头允许放到 **12–15 秒**（仍在 ai_video.md 规则 6 上限内），用慢动作 + 360° 环绕拖住情绪。
- 转场空镜压到 **3–5 秒**，承担节奏调味，不抢戏。

### 3.4 转场词典

| 转场名 | 中文 prompt 写法 | 接续方式 | 用例 |
|---|---|---|---|
| 闪白 | `画面被银白雷光瞬时充满、全屏过曝 0.3 秒` | 雷劫 → 下一阶修为 | 每次境界突破 |
| 闪黑 | `画面骤然全黑、余烬粒子缓飘` | 出招完 → 后果镜 | 魔功施放后果 |
| 烟雾过场 | `黑色魔气盘绕逐渐遮蔽全屏、金线最后熄灭` | 场景 A → 场景 B | 跨地点切换 |
| 桃瓣 / 灰烬过场 | `桃花瓣（凡间）/ 灰烬粒子（魔境）旋落充满画面` | 时间跳跃 | 卷间 / 集间 |
| 镜面碎裂 | `银白镜面碎裂飞散、碎片中映出新场景` | 闪回引入 | 前世今生切换 |
| 长袍掠过 | `镜头被一道黑袍 / 白袍掠过 0.5 秒` | 跟拍跨场 | 同一卷内紧凑切换 |
| 雷劫切 | `银白雷电自上劈下、画面震荡 0.5 秒、再开新场景` | 高强度场景间 | 反派覆灭、卷高潮 |
| 字幕浮现 | `黑底浮出护黄金竖排小篆字、缓淡入 1 秒、淡出 1 秒` | 章节标题 / 卷名 | 卷开场 |

每集**用且仅用 1–2 种转场为主**，其余靠"动接动"硬切（contiguous cut）；切忌每镜一转场，会让 90 秒看起来比 90 秒还碎。

### 3.5 9:16 竖屏构图惯例

| 构图模板 | 写进 prompt 的描述 | 用例 |
|---|---|---|
| 单主体几何中心 | `9:16 竖屏 / 主体居中 / 顶 1/3 留天 / 底 1/3 留地 / 浅景深虚化背景` | 角色单人立绘式镜头（魔尊登场、白衣摘面具） |
| 双主体对峙 | `9:16 竖屏 / 上下分屏式构图 / 上方主体高位 / 下方主体跪地 / 中央留 1/3 空气` | 镇压 / 揭穿 / 双修前的对视 |
| 群像登场 | `9:16 竖屏 / 五人一字排开 / 顶光均匀照亮 / 长阶背景纵深` | 五大宗主联盟登场 |
| 移动跟拍 | `9:16 竖屏 / 跟随主体奔跑 / 主体始终位于画面右 1/3 / 左 2/3 留环境流动` | 乞丐期街市奔逃 |
| 反英雄低角度 | `9:16 竖屏 / 极低角度仰拍 / 主体顶天而立 / 长袍下摆充满画面下半` | 魔尊真身归位 |
| 神识俯视 | `9:16 竖屏 / 极高角度俯拍 / 主体浮于深青虚空中央 / 周围星点旋转` | 闭关 / 内视 / 双修共修 |

构图三大铁律：

1. **眉眼必须落在画面顶 1/3 区间**，竖屏抖音播放器把底 1/3 给评论 / 互动栏挡住，主体放低就被吃。
2. **每镜至少有一个"留白带"**（顶天 / 底地 / 单侧空气），密构图竖屏看着窒息。
3. **群像永远横排**，竖屏不适合纵深排列五人；五大宗主"长阶向远延伸"那种构图在 9:16 会糊成一团。

## 4. 开放问题

留给 stage 4 spec 与 stage 6 execution 决定，研究阶段无法独立锁定：

1. **"魔功招式"具象 prompt 化对照表**——interview 已给出"九幽噬魂掌 / 魔气化龙 / 血色雷 / 魔影分身 / 虚空蚀心 / 血雨九重 / 星辰魔阵"等 7 阶招式（`qa.md` Q3.2 注释），但每个招式的镜头编排（特写 / 全景 / 慢动作 / 升格倍率）需要 stage 4 在 character-design 角度成果出来后联动锁定。
2. **白衣伪君子五宗的差异化色彩与材质**——五大宗主需要在"统一白衣"前提下区分（剑修银线 / 丹修朱砂线 / 阵修金线 / 符修青线 / 兽修黑线 之类），具体绑定 stage 3 character-design 的产出。
3. **凡间副女主酒楼**的色温——"烟火气暖橙"与全剧"黑金沉郁"的边界处理：是设第二卷为暖色专属，还是全程在远景嵌入冷色暗调（暗示魔尊永远未真正属于凡间）？建议后者，但需 stage 4 决断。
4. **Seedance vs Kling 同 prompt 的 LUT 飘移度**——同一镜头双管线产出可能在饱和度 / 色温有可见差异；是否需要在 stage 6 引入"prompt 内强制写 hex"还是"放任后期统一调色"，需要 stage 5 validation 团队给出 production 级建议。
5. **YouTube Shorts 的色域适配**——抖音偏 sRGB 高对比，YouTube 默认 Rec.709 略平，黑金在 YouTube 上易发灰。是否需要在 publish 阶段产出两套调色版本，留给 stage 4 publishing 角度决定。
6. **第六卷"反杀大本营"的色彩反差极大值**——是用最饱和的金（接近 `#d4a136`）做"魔尊归位"的终极爆点，还是反向把金压到比主调更暗（暗示"复仇即虚无"）？这是与 narrative 角度的"开放结局 vs 闭合结局"绑定的视觉抉择。
